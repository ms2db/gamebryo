Generating Tool Plug-in Scripts

Each invocation of the Tool Plug-in Batch Tool application can require as many as three scripts (although by default, none are required). How these scripts are generated depends on the exact type of script.

Processing and export scripts

You can create or edit processing and export scripts in one of three ways:

  • Using the script dialog in the Gamebryo 3ds Max Plug-in or the Gamebryo Maya Plug-in
  • In the Gamebryo Tool Plug-in Testbed application, using the script dialog
  • Using a standard text editor (for example, Notepad)

Import Scripts

You create import scripts in a standard text editor, following a specific format. Import scripts must contain a reference to a single plug-in, which must be an import plug-in. In addition, all options must be specified explicitly. The following is a sample import script:

Script ("Default_MYFORMAT_Import","Import")

{

Plugin ("MYFORMAT_ImportPlugin","MYFORMAT 
 Importer","1.0","IMPORT")

{

Parameter ("UseTrueDoubleSided","TRUE");

Parameter ("UseInternalTextures","TRUE");

Parameter ("AddDefaultLights","TRUE");

}

}
