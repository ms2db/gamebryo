Understanding How the Tool Plug-in Batch Tool Works

Conceptually, the Gamebryo Tool Plug-in Batch Tool works in several basic steps:

  1. Locates an import plug-in to load the specified import file, by:
    1. (Default) Finding the first import plug-in that is capable of loading the file (keyed off of the filename extension) and using its default import options.

      - OR -

    2. (Script specified) Using the command-line-specified import plug-in script to define the desired import plug-in and import options.
  2. Uses the selected import plug-in and options to load the specified import file.
  3. If a processing script name is specified, loads and applies that processing script to the previously-loaded Gamebryo data.
  4. Locates an export plug-in to load the specified export file, by:
    1. (Default) Finding the first export plug-in that is capable of saving the file (keyed off of the filename extension) and using its default export options.

      - OR -

    2. (Script specified) Using the command-line-specified export plug-in script to define the desired export plug-in and export options.
  5. Loads the selected export script and saves the processed Gamebryo data to the specified destination filename/format.
