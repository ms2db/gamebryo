Tool Plug-in Batch Tool Command-line Usage

The command-line syntax of the Tool Plug-in Batch Tool is as follows:

ToolPluginBatch.exe [-i ImportScriptName] [-p ProcessScriptName] [-r RendererName] [-e ExportScriptName] ImportFileName ExportFileName

-i ImportScriptName

(Optional) The name (and path) of the import script, which must be a Gamebryo Tool Plug-in script file. If specified, this script must include only one plugin, which must be an import plug-in capable of reading files of the type specified as ImportFileName. If this script is not specified, a default import plug-in for the given file extension is located and used.

-p ProcessScriptName

(Optional) The name (and path) of the processing script, which must be a Gamebryo Tool Plug-in script file. If specified, this script will be loaded and applied to the data imported from the import file. If no script is specified, the data will be passed through this stage unchanged.

-e ExportScriptName

(Optional) The name (and path) of the export script, which must be a Gamebryo Tool Plug-in script file. If specified, this script must include one export plug-in as the last in its pipeline, and that plug-in must export files of the extension specified by ExportFileName. This script will be applied following the processing stage to save the data to the file specified by ExportFileName.

-r RendererName

(Optional) The name of the renderer to use when processing assets. By default, it will use the generic renderer. Valid options are generic, dx9, d3d10, d3d11, xenon, and ps3.

ImportFileName

The name of the file to be imported and processed

ExportFileName

The name of the file to be written at the end of the process
