Moving Entities Between Layers in World Builder

You might find that an entity or group of entities is contained within a layer that is not appropriate for your team's workflow. If this is the case, you can move the entity (or group) to a different layer to better suit your team's needs.

World Builder: Moving a layer

Note

You cannot move entities to an unloaded layer. Refer to Loading and Unloading Layers for more information.

To get started

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. In World Builder, select the entities that you want to move to a different layer.
  5. If the Layer Editor is not already open, open it (View > Layer Editor).
  6. In the Layer Editor, make sure the target layer and the layer containing the entities is writable by right-clicking them and selecting Revision Control > Add / Edit from the context menu.

To move all selected entities to a different layer

  1. Right-click the target layer and select Add Selection to Layer from the context menu.
  2. The entities that you selected are now assigned to the target layer.

To move entities from the current layer to a different layer

  1. Select the entities that you wish to move from this layer into the new layer.
  2. Drag these entities onto the desired layer.
  3. The entities that you selected are now assigned to the target layer.
