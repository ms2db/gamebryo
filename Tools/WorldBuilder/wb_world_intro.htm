Working with World Builder - Introduction

World building involves taking existing models and presets and instantiating them as entities. In this context, these instantiations represent the initial conditions for the targeted game. Working with World Builder not only necessitates an understanding of how the tool works, but also, how it works in relation to both modeling and game content.
