World Builder - Menus

World Builder

  • Export Block - Exports the current block.
  • Export Block As... - Exports the current block to an alternate location.
  • Test Content - Launches the application specified by the "[User] > World Builder > Test Content App Path" setting.
  • Sync World - Manually triggers a synchronize between World Builder and any changed EMT files your block uses.
  • Terrain
    • Create New Terrain - Creates a new terrain entity.
    • View Terrain Properties - Opens the properties for the current terrain.
    • Snap Selection To Terrain - Snaps the currently selected entities to the terrain.
  • Viewport Layout
    • Viewport Layout 1
    • Viewport Layout 2
    • Viewport Layout 3
    • Viewport Layout 4
    • Maximize Active Viewport
  • Export to Autodesk FBX - Exports the current content to Autodesk's FBX format for import into other digital content creation applications like 3ds Max or Maya. See Entity Export to DCC for more details.
  • Create Camera From Viewport - Creates a camera based on the current perspective. This only works for perspective cameras.
  • Randomizer - Launches the Randomizer.
  • Scene Query Tool - Opens a data visualization dialog for the current scene.
