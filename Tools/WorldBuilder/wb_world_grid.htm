Working with the Grid

World Builder provides a grid that can be toggled on and off. The grid is a guide to give perspective on the distance, scale, and orientation of entities relative to each other. Through snap and precision setting options, the grid can additionally help facilitate aligning entities so that they fit seemlessly together.

Snap Settings

Snap settings force specific increments in value settings when performing translation, rotation, and scale. Using snap settings effectively is important for ensuring entities fit together seemlessly. Each of the three operations (translation, rotation, and scale) has its own snap settings. Related to snap settings are precision settings. Precision settings are used to help translations and scales stay at even numbers always. In a way, precision is very similar to snap in that causes entities to move in set increments. The advantage of precision is that it does not operate on the translation or scale relative to the starting translation or scale, but rather makes its final value round off to an even increment.

Translation Snap

Translation snap can be enabled or disabled by the above-left icon. Additionally the snap values can be selected via the drop down combo box to the right of the icon. The values represent the distance between each translation increment when Translation Snap is enabled.

Rotation Snap

Rotation snap can be enabled or disabled by the above left icon. Additionally, the snap values can be selected via the drop down combo box to the right of the icon. The values represent the degrees between each rotation increment when snap is enabled.

Scale Snap

 Scale Snap can be enabled or disabled by the above left icon. Additionally, the snap values can be selected via the drop down combo box to the right of the icon. The value represents the distance between each scale increment when snap is enabled.

Precision Settings

Translation and scale values can be automatically rounded to user-configurable increments. To enable, disable, or configure these operations, open the Settings dialog. Precision settings are in the Document.WorldBuilder.Precision category.

When precision snapping is enabled, only the final translation and scale values (as appropriate) are rounded off; intermediate results are computed at full precision.

To change the appearance of the Grid

  1. Open the Settings dialog by choosing Utilities > Settings from the main menu.
  2. From the Settings dialog, drill down into User.WorldBuilder.Grid settings. The following settings will be made available.
  3. From the these settings, the appearance of the grid can be modified. The "Grid Extents" specifies how far the grid will be seen from the origin of the world. The color of the grid lines can be controlled by the "Grid Major Line Color" and "Grid Minor Line Color" properties. Finally, the spacing can be controlled by the "Major Line Spacing" and "Minor Line Spacing" properties.
