Overriding Mix-in Properties

Entities in World Builder are instances of the entity models constructed in Entity Modeling Tool. As such, it is possible for them to inherit values from their parent models. This inheritance is beneficial as it lets users tweak values for a large number of entities at once without having to reopen all existing files. However, you may also want to make a particular instance unique. Some properties are obviously unique like position and rotation.

Overriding Mix-in Properties

When you want to make a property value unique, you will need to override the default value from the parent entity model.

  1. Select the entity you wish to edit.
  2. Open the Entity Editor panel in World Builder.
  3. Begin to edit the value. When you press "Enter" to accept the field, the override checkbox will be set beside the value field.

Reverting an Overridden Property

Once you've overridden a value, you may also find that you may need for the value to return to its default value.

  1. Select the entity you wish to edit.
  2. Open the Entity Editor panel in World Builder.
  3. Uncheck the override checkbox next to the property. The value in the editor should revert to the default value stored in the parent entity model.
