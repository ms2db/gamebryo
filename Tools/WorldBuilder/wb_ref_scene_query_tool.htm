World Builder - Scene Query Tool

The scene query tool allows the user to trigger queries on different types of objects in order to display important information about those objects. It currently provides information about a terrain's points, materials and meta data. However this can be extended to provide feedback on any number of objects.

Launching the interface

The scene query tool can be accessed through the World Builder menu.

Figure 1: Launching the scene query tool

Scene Query Dialog

The scene query dialog provides a number of queries that can be activated or deactivated using the associated check box. Each query belong to a category that is displayed as a tab in the dialog.

Terrain Geometry queries

Under the Terrain Geometry tab, you will find three queries that allow you to retrieve information about a specific geometrical point on the terrain. All those queries will perform different types of ray collisions to retrieve collision point, collision normal, the height and slope of the point as well as the leaf and sector the point belongs to.

  • 3D Mouse Ray Query: When activated, this casts a 3D ray from the mouse pointer location following the camera's direction. Users can modify the Level of detail the ray should be targeting. This updates every time the mouse moves.
  • 2D Mouse Ray Query: When activated, this first casts a 3D ray from the mouse pointer and then uses the resulting collision point to cast a second vertical 2D ray. As for the 3D query, users can modify the level of detail and the query is updated on mouse move.
  • Manual Ray Query: When activated, this casts a ray using the given origin, direction and length for the ray. This is updated anytime inputs are changed.

Figure 2: The available terrain geometry queries

Terrain Materials queries

Under the Terrain Materials tab, you will find two queries that allow you to retrieve information about materials and meta data applied to a specific point. Each query will perform a 3D ray collision anytime the mouse is clicked and if they are activated.

  • Terrain Material Query: When activated, this query will retrieve all the available materials at a given point and display them in a list view. It will show to he user the opacity of the material at that point as well as all the meta-data associated with that material.
  • Blended Meta-Data Query: When activated, This displays to the user all the meta-data available at a given location. This is retrieved by performing a blend operation. Any meta data key that is referenced with two or more data types will be circled in red to let the user know about the error.
