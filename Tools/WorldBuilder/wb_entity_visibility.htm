Managing Entity Visibility

When working with a large number of entities, it is often useful to only display a subset. This can be done to increase performance or to reduce the number of items to consider. There are two categories of entity visibility control: per-entity or per-layer.

Per-Entity Visibility

Users can adjust the visibility of a single entity by pressing the "Hide Selected Entity" button in the  Entity Editor panel. If the entity has a graphical representation, it will no longer render in the viewports, although it will be selectable through the Entity View panel. Note that this visibility toggle does not affect visibility in the game itself. In order to affect game visibility, please adjust the "IsVisible" property on the Renderable object. This will also cause the object to no longer render, but has the advantage of also not rendering in the game.

Per-Layer Visibility

It is often valuable to only show the entities for a subset of the layers in the block. In the Layer Editor, it is possible to set the visibility of all entities in a given layer. This value takes precedence over any per-entity settings. As with the "Hide Selected Entity" button, adjusting layer visibility does not affect visibility in the game itself.
