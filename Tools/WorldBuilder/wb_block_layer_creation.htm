Creating a Layer in World Builder

While you work in World Builder, you might find that you have a large number of entities in your level (.block). To make the job of managing these entities easier, you can divide them into smaller groupings that are known as layers (.layer). You can create an empty layer, or you can create one from a selection in World Builder.

World Builder: Layer option buttons

Note

Layer creation is disabled when the selected layer is currently unloaded. Refer to Loading and Unloading Layers for more information.

To get started

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. In the Layer Editor, locate the desired parent layer you wish to edit. This could be the master block file or any other existing layer file.
  6. Select the layer from the list. If this layer is read-only, make it writable by right-clicking and selecting Revision Control > Add / Edit from the context menu. This will make the file writable and open it for edit in your revision control system.

To create an empty layer

  1. Click the Create New Empty Layer button.
  2. In the Set Layer Name dialog that displays, type the name of the layer. Only valid filename characters are allowed; 512 character maximum. If the layer name matches one that already exists in the block or on disk, you will be prompted to choose a different name. When you are done, click OK.

    The new layer is now active in the Layer Editor. Any subsequent entities that you create will be placed in this layer file.

  3. Save your project to write the layer to disk.

To create a layer from a selection

  1. In World Builder, select the level item(s) from which you want to create a layer.
  2. Click the Create New Layer From Selected Items button. If any of the entities exist in a layer that is read-only, you will be prompted to open those layers for edit. If this is not the desired action, you can cancel the entire operation.
  3. In the Set Layer Name dialog that displays, type the name of the layer. Only valid filename characters are allowed; 512 character maximum. If the layer name matches one that already exists in the block or on disk, you will be prompted to choose a different name. When you are done, click OK.

    The new layer is now active in the Layer Editor. Any subsequent entities that you create will be placed in this layer file.

  4. Save your project to write the layer to disk.
