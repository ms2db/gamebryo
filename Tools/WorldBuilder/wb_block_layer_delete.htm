Deleting a Layer in World Builder

While you are working with layers, you might discover an orphaned layer or one whose content is no longer relevant. When this occurs, you can remove the layer from the block file.

World Builder: Deleting a layer

Note

An unloaded layer cannot be deleted. Refer to Loading and Unloading Layers for more information.

To delete a layer

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. Select the layer and delete it by clicking the Remove Selected Layers button (trash can) or by right-clicking the layer and selecting Remove Layer from the context menu.
  6. The layer and all of the entities that it contains are removed from the scene. If that layer had any child layers, they are also removed from the scene. The .layer file remains on disk if you need to re-import it later.

Note

Deleting a layer does not actually delete the layer's file. Thus the deleted layer can be imported later or used by another block file. This also means that you will not be able to create a layer of the same name without first deleting the layer file manually (there is no way to delete the layer file within Toolbench).
