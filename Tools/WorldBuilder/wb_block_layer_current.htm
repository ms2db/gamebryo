Making a Layer Current in World Builder

When you make a layer current (active) in World Builder, all entities that you place in the world thereafter will be children of that layer. Making a layer the current or active layer is a good practice to perform before you add entities so that you do not have to spend time later organizing entities.

World Builder: Making a layer current

Note

An unloaded layer cannot be chosen as the current layer. Refer to Loading and Unloading Layers for more information.

To make a layer current

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. Select the layer from the list. If this layer is read-only, make it writable by right-clicking and selecting Revision Control > Add / Edit from the context menu. This will make the file writable and open it for edit in your revision control system.
  6. Right-click the layer and select Set as Current from the context menu.
  7. The layer becomes the current layer (note the icon).
