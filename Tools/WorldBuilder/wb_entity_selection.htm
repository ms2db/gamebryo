Selecting Entities

There are several ways to select an entity for manipulation in World Builder.

Viewport Click Selection

Users may select an entity at any time in the rendered viewport via left-mouse click. Left-clicking on any selected entity will de-select the entity. Clicking on any other entity will de-select the current selection and select the new entity instead. In order to append a selection to the list, press the "Control" key when selecting. De-selecting an entity with "Control" pressed will only de-select that particular entity and will not clear the rest of the selection. De-selecting without the "Control" key pressed will clear the selection.

Viewport Box Selection

Users may select multiple entities at once using the box selection mechanism. Pressing the "Alt" key while holding down the left mouse button will enable box selection. A 2D box will be drawn from the initial mouse click to the current mouse position. Releasing the button will cause all entities whose bounds intersect the box to be selected.

Box selection has the following modifier keys:

  • Shift - If held, the selection becomes additive; the selection will merge with any other selected entities that were previously selected instead of clearing the selection.
  • Ctrl - The selection toggles; if an entity within the box is not selected, it will be selected. Otherwise it will be deselected.
  • Ctrl + Shift - The selection becomes subtractive; if an entity within the box is already selected, it will be deselected.

Note

Note that the entity bounds are often larger than the entities themselves, occasionally causing the box selection to select entities that aren't visually within the box. When this occurs, use "Control" click to de-select the undesired entities.

Entity Locking

When working with a large number of entities, it is often useful to prevent certain entities from being selected in the viewport. This is especially true for large objects like terrain or skydomes. There are two categories of entity locking control: per-entity or per-layer.

Per-Entity Locking

Users can adjust the selectability of a single entity by pressing the "Lock Selected Entity" button in the  Entity Editor panel. If the entity has a graphical representation, it will no longer be selectable in the viewports, although it will be selectable through the Entity View panel.

Per-Layer Locking

It is often valuable to only allow selection for a subset of the layers in the block. In the Layer Editor, it is possible to set the selectability of all entities in a given layer. This value takes precedence over any per-entity settings.

Selection Sets

If a user wishes to retain a selection for future use, named selection sets provide this functionality. Please see the description of the selection set editor for more details.

Entity View

The entity view provides several mechanisms for selecting entities. Simply selecting an entity in the view will select it in World Builder. "Control"-selection appends to the selection. "Shift" allows drag selection to occur as the mouse is dragged over the list. Note that this view is also useful for organizing entities by folder and by parent model, two common ways to assist in selection. Please see Entity View for more information.
