Manipulating Entity Transformations

Entities may be transformed in the world using various transformation gizmos. For more information refer to the transform toolbar documentation.

Transformation Coordinate Systems

All of the Transformation Modes work relative to the active coordinate system. The active coordinate system can be selected from the transform toolbar.

World Space

This is the coordinate system of the block file. When multiple entities are selected and the coordinate system is set to world space, transformations occur in the block file's coordinate system. For example, if three entities are selected and rotated simultaneously, each entity will rotate about the center of the selection.

Local Space

This is the coordinate system of the entity itself. When multiple entities are selected and the coordinate system is set to local space, transformations occur in the space local to each entity. For example, if three entities are selected and each entity is tilted in a particular direction, rotating about the z-axis will rotate each entity about its own z-axis.
