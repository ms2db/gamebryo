Editing a Layer in World Builder

If a layer is not editable (read-only), a red icon () appears next to its name in the Layer Editor. Within World Builder, you can make the layer file writable to continue doing what you need to do to the layer.

World Builder: Editing a layer

To make a layer editable

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. RightRight-click the read-only layer and select Revision Control > Add / Edit from the context menu. If you are using a revision control system like Perforce or Subversion, the file will also be checked out.
  6. The file is now writable (note the icon).
