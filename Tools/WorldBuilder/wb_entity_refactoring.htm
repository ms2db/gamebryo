Refactoring an Entity

During the course of your game's development, you'll often want to change the entities that you've already instanced into the world. Depending on the nature of the change you wish to make, you can make the change directly in World Builder. For many changes, however, you'll need to edit the parent entity model in the Entity Modeling Tool and then resynchronize the block file with the entity modeling project.

Changing an Entity's Parent Model

The easiest change to make in World Builder is to change the Entity instance from one Entity Model parent to a different parent. Where possible, any overridden values will be preserved. If the property name does not exist in the new model, it will not be preserved.

  1. Select the entity instance(s) you wish to convert.
  2. Open the Palette View panel in World Builder.
  3. Select the new entity model that you wish to convert the selected entity instance into.
  4. Right-click on the entity model to bring up the context menu. Select "Transmogrify".
  5. The entity instance should now be of the new model type.

Note

Additional refactoring tools may come available in future releases of the World Builder and Entity Modeling tool sets.
