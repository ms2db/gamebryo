Creating an entity

There are several ways to instance entities into your block or layer in World Builder. The most important first step is to author your parent EntityModel or Preset in Entity Modeling Tool.

Creating from Palette View - Drag and Drop

The simplest method for adding an entity to the world is to simply drag and drop from the Palette View.

  1. Open the Palette View panel in World Builder
  2. Find the model you wish to instance into the world.
  3. Select the model and drag onto the render view. When you release the mouse button, an entity will be created underneath the mouse cursor. You can subsequently move the entity using the translation modes.

Note

The entity will be created in the currently open entity folder and on the current layer. Please see Entity View Reference or Making a layer current for more details.

Creating from Palette View - Creation Mode

If you are going to be creating many instances of the same entity model at the same time, it is easier to use the creation mode.

  1. Open the Palette View panel in World Builder
  2. Select the model you wish to instance into the world.
  3. Click on the render view.
  4. Adjust the camera position if desired.
  5. While holding down the space bar, left-click in the render view. A new entity will be created under the mouse cursor each time you click.
  6. To rotate the new entity, move the mouse while still holding the left mouse button. The new entity is finalized when you release the left mouse button.

Note

Rather than holding down the space bar, you can also press the "Creation Mode" toolbar button () to toggle Creation Mode on and off.

Note

When using Creation Mode, the new entities take their rotations from the previously created entity. If you need to place several entities with each facing the same direction, you can rotate the first entity in this fashion. Subsequent placements will start with the same rotation. Leaving and re-entering Creation Mode will reset the starting rotation to the default for the model.

Creating from Existing Entities - Cloning

If you would like to duplicate entities that already exist in the game world, it is easier to use cloning.

  1. Select the entities you wish to duplicate.
  2. Activate the translation, rotation, or scale gizmos.
  3. While holding down the shift key, transform the selected entities using the gizmo. You will instead transform newly created entities, leaving the original selection unchanged.

Creating from an asset file from Windows Explorer - Drag and Drop

  1. Open the Windows Explorer
  2. Find the asset file.
  3. Select the file and drag onto the render view.
  4. Create entity dialog will prompt you to choose the model-property to instantiante.
  5. Choose model and property.
  6. (Optionally) Check "Use as default settings for this asset type".
  7. (Optionally) Check "Don't show this dialog when default setting exist.".
  8. An entity will be created where you dropped the asset file. You can subsequently move the entity using the translation modes.

Note

The entity will be created in the currently open entity folder and on the current layer. Please see Entity View Reference or Making a layer current for more details.

Note

The asset file needs to be in your Asset Web Root directory. Please see Configuring the Asset Runtime Service.
