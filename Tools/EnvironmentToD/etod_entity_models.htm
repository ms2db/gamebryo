Environment and Time Of Day Entity Models

There are three different models available in the standard library to use with the editor.

  • TimeOfDay model is the object controlling the animation. This must be added to the scene in order to use the editor.
  • TimeOfDayEditable model is an abstract model that should be mixed in with any models that should be animated by the editor.
  • Environment model is currently the only model within the standard library that can be instantiated and mixes in the TimeOfDayEditable model.

The TimeOfDay and the Environment model do not currently have graphical proxies.
