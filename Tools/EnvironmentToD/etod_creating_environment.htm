Creating an Environment Entity

Creating a time of day editable environment is done in two steps. First of all a TimeOfDay model must be added to the scene if it is not already present. TimeOfDay models are global and only one can be added per block. To create a TimeOfDay entity, simply select the model from the StandardModelLibrary in the Palette View and drag it on the scene. This entity has no graphical data or proxy.

Users can then add an Environment model. When dragging the Environment model from the StandardModelLibrary onto their scene, a window will open to choose where to save the time of day animation. The user can choose to use an existing file, to create a new file or not to use the time of day editor and therefore select no file. Using an existing file will attempt to load the data.

The 'Ok' button will only become enabled if the selected file path is correct or the 'No file' option is selected.
