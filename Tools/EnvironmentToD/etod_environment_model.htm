The Environment Model

Fig. 1: Environment properties

The environment model holds a number of properties to allow for user customization. Not all properties are animatable using the time of day editor.

Attribute Type Description
EnvironmentSunEntity EntityRef The directional light the environment uses. If nothing is set, a default light will be used. This is not animatable.
EnvironmentAutoFogColor Bool Controls whether the fog color is automatically calculated according to the horizon color or not. This is not animatable.
EnvironmentUseSunAngles Bool Defines whether the environment should modify the associated sun entity using the EnvironmentSunAzimuthAngle and the EnvironmentSunElevationAngle properties
EnvironmentFogColor Color The color of the fog when EnvironmentAutoFogColor is set to false.
EnvironmentSunAzimuthAngle Float The angle of the sun's direction around the z axis.
EnvironmentSunElevationAngle Float The angle of the sun's direction around the x axis.
EnvironmentSunSize Float The size of the sun visible in the sky.
EnvironmentSunIntensity Float The intensity of the sunlight. Controls the white effect on the horizon.
EnvironmentPhaseConstant Float Controls the blurring effect around the sun.
EnvironmentRayleighConstant Float Controls the color at the horizon.
EnvironmentMieConstant Float Controls the scattering of light around the sun.
EnvironementHDRExposure Controls the brightness of the sun.
EnvironmentRedWavelength Float The length of the red color wave. Modifying this will change the colors of the sky in accordance with the other wavelengths.
EnvironmentGreenWavelength Float The length of the green color wave. Modifying this will change the colors of the sky in accordance with the other wavelengths.
EnvironmentBlueWavelength Float The length of the blue color wave. Modifying this will change the colors of the sky in accordance with the other wavelengths.
EnvironmentDomeRadius Float The apparent radius of the skydome.
EnvironmentDomeDetail Float The level of detail used to display the sky. Increasing this will greatly affect performances.
EnvironmentDomeDetailAxisBias Float Modifies the level of detail displayed on the dome vertically.
EnvironmentDomeDetailHorizonmtalBias Float Modifies the level of detail displayed on the dome at the horizon.

By default, environment properties are set to match Earth's characteristics.
