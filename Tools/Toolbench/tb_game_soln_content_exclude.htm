Excluding Content from your Game Solution

If you no longer need specific files in your game solution, but may need them at a later date, you can exclude them. When you exclude a file from a game solution, it does not get deleted.

To exclude content from a game solution

  1. In the Game Solution view, right-click the item that you want to exclude.
  2. From the context menu, click Exclude.

The file is removed from your game solution.

Tip

To restore excluded content, choose Add and Existing Content from the game solution context menu.
