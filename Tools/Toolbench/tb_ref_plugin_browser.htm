Toolbench - Plug-in Browser

The Plug-in Browser displays the list of currently loaded plug-ins and allows you to load other plug-ins.

To see the Plug-in Browser, open the View menu and select Plugin Browser.

For each plug-in currently registered with Toolbench, the Plug-in Browser displays the following information:

  • The name of the plug-in.
  • Whether it is currently loaded.
  • The name of the library file which contains the plug-in.
  • The version of the plug-in.
  • The organization which created the plug-in.

To load a currently unloaded plug-in, double-click on its entry in the list. The plug-in and any plug-ins it requires will load. Explicitly loading a plug-in in this way is not usually necessary as most plug-ins load automatically in response to opening content or interacting with menus and toolbars.
