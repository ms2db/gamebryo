Editing Revision Controlled Content

Toolbench does not automatically check-out files when you edit them in Toolbench. You need to edit files and then either save them or manually check them out in Toolbench. When you save or check-out edited files, the files are, first, saved to the local file system. This sequence ensures that the file data is always preserved.

If the revision control check-out operation fails, you must perform a manual check-out to ensure that the edited files are included in the changes that are committed to the revision control server.

To edit revision controlled content

  1. Edit files in Toolbench.
  2. Check the edited files out from the revision control system by doing one of the following:
    • From the Toolbench menu bar, click File > Save or File > Save All.
    • From the context menu, select Revision Control > Add/Edit.

      A manual check-out via right-clicking is especially useful if you want to inform other revision control users that you intend to modify the selected content.

  3. When you are ready to do so, use your revision control client to check in your changes. See Checking in Content to Revision Control.
