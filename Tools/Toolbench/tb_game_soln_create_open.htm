Creating/Opening Game Solutions

When using Toolbench to manage the content in your game, you must first create a game solution. The game solution is the container that provides easy access to a working set of game content. If you create a new game solution or open an existing one, while a game solution is already open, the solution that is currently open will close because only one game solution can be open at a time.

Note

A game solution must be open before creating or opening entity model projects and block files.

To create a new game solution

  1. Do one of the following:
    • From the Toolbench menu bar, click File > New
    • From the toolbar, click the New Content icon.
    • Press Ctrl+N

    The New Content dialog box opens.

  2. In the Content Categories group box, select Game Solution.
  3. In the Content Types group box, select Game Solution.
  4. In the Name text box, type a name that best represents the game which you are working on.
  5. In the Folder text box, type in the full path where you want the game solution file to be saved. (You can click the Browse button to browse through the folders on your computer or network to select a file.) The folder must be empty.
  6. Click OK to launch the Game Solution Wizard, which will guide the rest of the game solution setup process.
  7. Confirm the root of the asset web to be used by the new game solution and click Next. All referenced content must reside in or below this folder. The wizard will default to an existing asset web if one can be located, else will use the path of the new game solution.
  8. Choose content libraries to copy into the asset web and include in the new game solution, then click Next. If no libraries are listed, be sure that the environment variable EMERGENT_PATH has been specified and points to the Gamebryo installation. These libraries can seed a game solution with useful initial content. If you later decide that you wish to import additional content, please see Importing Content Libraries into your Game Solution.
  9. Choose whether to add a default entity modeling project and world block file to the new game solution. Generating these files will allow jumping straight to content creation. Click Finish to complete the wizard.

To open an existing game solution

  1. Do one of the following:
    • From the Toolbench menu bar, click File > Open
    • From the toolbar, click the Open Content icon.
    • Press Ctrl+O

    The Open Content dialog box opens.

  2. Navigate to the location on your computer or network where the game solution resides.
  3. Select the .gsl file and click Open.

The game solution opens in Toolbench.

Tip

To open a game solution or other file that was recently opened, click File > Recent Files > filename.
