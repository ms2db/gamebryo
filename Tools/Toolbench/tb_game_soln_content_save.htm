Saving Content within your Game Solution

It is a good practice to save edited content on a regular basis. You can save individual documents or all open documents at the same.

To save content

  • From the toolbar, click one of the following icons:
    • Save Content to save the file or document that has the current focus for editing.
    • Save All Content to save all open files and documents that have been directly modified.

The open file or files are saved.
