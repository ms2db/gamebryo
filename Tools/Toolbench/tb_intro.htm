Toolbench: Introduction

Toolbench is a completely plug-in driven application devoted to hosting the wide variety of tools needed to build a game. These "sub-tools" include:

  • Entity Modeling Tool
  • World Builder
  • Script Debugger

Developers can extend Toolbench with game-specific modifications to these tools and add entirely new capabilities, building on the core facilities of the Toolbench platform.
