Rolling Back Revision Controlled Content

The revision control system can be used to rollback to a previous version of the asset web. Local files can be rolled back using the revision control client.

Follow these steps roll back to a previous revision of the asset web in revision control:

Perforce

  1. In Toolbench, close the game solution (File -> Close All).
  2. Use P4V Back Out Submitted Changelist feature (right-click a changelist and select Rollback...) or follow steps for Backing Out Submitted Changelists.
  3. In Toolbench, reopen the game solution (File -> Recent Files -> [games solution].gsl).

Subversion

  1. In Toolbench, close the game solution (File -> Close All).
  2. Follow steps for Fetching Older Repository Snapshots.
  3. In Toolbench, reopen the game solution (File -> Recent Files -> [games solution].gsl).
