Renaming Game Solutions

When you first create a game solution, you must name it. You can, however, rename a game solution at anytime for any reason.

To rename a game solution

  1. In the Game Solution view, do one of the following:
    • Select the game solution node and press F2.
    • Right-click the game solution node and select Rename from the context menu.
  2. Type a new name and press Enter.

The game solution is now renamed.
