Using Text Boxes

Several of the windows in Toolbench contain fields or text boxes in which you can type property values or specific information about an item in your project. The name of the text box or the name of the column in which the text box appears identifies the type of information the field represents, for example, name, type, and Mix-ins.

You may notice that many text boxes have an icon or check box beside them. These visual elements represent one of the following:

Visual Element Description
The property value is either inherited from the parent or has been overridden and modified.
Additional properties exist for the current property. Click the Expand icon to display the additional properties. Click the icon a second time to hide the properties.
Additional editing tools exist for the current property values. Click the Expand icon to display the additional editing tools. Click the icon a second time to hide them.
