Adding Content to your Game Solution

After creating your game solution, you need to start populating it with content.

A typical game solution will include one or more entity modeling projects for the different parts or concepts in your game. These projects are then populated with the entity models defining your game. World block documents can also be created to layout entities within a 3D world.

To add new content to your game solution

  1. Right-click the game solution node in the Game Solution view.
  2. From the context menu, click Add > New Content.

    The New Content dialog box opens.

  3. From the Content Categories list, select Game Content to add specific game content to your game solution, including project and block document files.
  4. In the Content Types group box, select the type of file you want to add to your game solution.
  5. In the Name text box, type a name that best represents the file.
  6. In the Folder text box, type in the full path where you want the file to be saved. (You can click the Browse button to browse through the folders on your computer or network to select a file.)
  7. Click OK.

The new file is created and added to the game solution.
