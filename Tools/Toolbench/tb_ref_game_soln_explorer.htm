Toolbench - Game Solution View

The Game Solution view is where you manage and organize all the different items within your game. A game solution is a virtual container that holds all of the content within your game, including entity modeling projects and world block files. The Game Solution view is the main window in Toolbench because this is where you add, move, rename, and delete items from your game. You can access specific commands by right-clicking any of the items in the Game Solution view.

Note

Only one game solution can be displayed in the Game Solution view at a time.
Interface Element Description
Game Solution view Displays a list of items within the currently loaded game solution. The following items are displayed within the Game Solution view:
  • Entity Modeling Projects— group of entity models and other project items that serve as the blueprints for game entities.
  • World Block Documents— set of entity models and presets laid out within a three-dimensional environment that represent a certain area of your world space at time=0.
