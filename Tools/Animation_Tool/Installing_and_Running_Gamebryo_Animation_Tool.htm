Installing and Running the Gamebryo Animation Tool

The Gamebryo Animation Tool is installed when you run the main Gamebryo installer. The installer creates a shortcut on the Windows Start Menu. During the install, you have the option to install sample files. We recommend that you install these samples because they provide examples of the correct way to use the Animation Tool. By default, these files install to the following location:

%EMERGENT_PATH%\Samples\Models\AnimationToolExamples

The .NET component is required for the Animation Tool to run. In Visual Studio .NET 2005 (VC80), the Animation Tool is built on the .NET Framework 2.0. In Visual Studio .NET 2008 (VC90), the Animation Tool is built on the .NET Framework 3.5. You can download the .NET Framework from the Microsoft web site as follows:

D3D10 / D3D11 Support

By default, the Animation Tool uses the DX9 renderer. You can choose to use the D3D10 or D3D11 renderer instead. To use the D3D10 renderer, specify "-D3D10" on the command-line when you launch the tool. To use the D3D11 renderer, specify "-D3D11" on the command-line when you launch the tool. If you want to change the default behavior, add this command line option to the tool's Windows Start Menu shortcut.

KFM File Association

Upon running the Gamebryo Animation Tool for the first time, all files with a ".kfm" extension are associated with the Gamebryo Animation Tool. Thus, double-clicking KFM files in the future will automatically open them in the Gamebryo Animation Tool.

If the "-D3D10" or "-D3D11" command line switch was used to launch the Gamebryo Animation Tool, then it will also be used to launch the Gamebryo Animation Tool when KFM files are double-clicked in the Windows Explorer.

For example, if the Gamebryo Animation Tool was last launched using the "-D3D10" command line option, double-clicking on a KFM file would automatically launch the Gamebryo Animation Tool with D3D10 renderer. In the same way, if the Gamebryo Animation Tool was previously launched without the "-D3D10" or "-D3D11" command line switches (which defaults to the DX9 renderer), then double-clicking on a KFM file will launch the Gamebryo Animation Tool using the DX9 renderer.
