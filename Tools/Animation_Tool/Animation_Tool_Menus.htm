Animation Tool Menus

The menus in the Animation Tool are described below:

File menu Description
New Create a new Animation Tool project.
Open Open an existing Animation Tool project from a KFM file.
Recent Files Select from a set of recently opened KFM files.
Save Save the current Animation Tool project to a KFM file. The save operation requires that you have set up the model and model root.
Save As Save the current Animation Tool project with a new filename. The save operation requires that you have set up the model and model root.
Exit Exit the application.
Edit menu Description
Docking Management
  • Save Dock Configuration—Save the current layout of the windows in the Animation Tool.
  • Reset Dock Configuration—Reset the saved layout of windows to the default. The changes will take place the next time the Animation Tool is loaded.
Message Box Management
  • Manage Message Boxes—Advance feature to turn on or off individual "Do Not Show Again" message boxes.
  • Reset Message Boxes—Reset all of the "Do Not Show Again" message boxes to be shown.
Scene Options Open the Scene Options dialog box. This Scene Options dialog box allows the user to change the animation playback rate, modify the up axis, choose the Image Subfolder, and invert the pan mode of the Render View.
Transition Table Settings Open the Transition Table Settings dialog box. This dialog box allows the user to set up background color and text color preferences for the cells in the Transitions View
View menu Description
Project View Toggle the visibility of the Project View
Edit View Toggle the visibility of the Edit View
Render View Toggle the visibility of the Render View
Transitions View Toggle the visibility of the Transitions View
Output View Toggle the visibility of the Output View
Interactive Preview Activates and sets focus to Interactive Preview
Help menu Description
Contents Opens the Animation Tool help to the "Contents" section.
Index Opens the Animation Tool help to the "Index" section.
Search Opens the Animation Tool help to the "Search" section.
About Animation Tool Opens a dialog box with product version information
