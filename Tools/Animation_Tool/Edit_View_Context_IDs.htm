Edit View: Sequence IDs and Group IDs

The actions that you can perform in the Edit View change depending on the type of object that is active in the Animation Tool. When you select Sequence ID Overview or Sequence Group ID Overview in the Project View, the Edit View displays a list box with the IDs in the left column and the names of the sequences (or sequence groups) on the right. You can modify IDs by double-clicking in the ID field and changing the value. When you press ENTER or TAB, the input focus moves to the next ID field. This is a convenient way of rapidly changing sequences and sequence groups IDs.
