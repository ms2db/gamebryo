Changing sequence IDs in the Animation Tool

You can use the Animation Tool to manually set sequence IDs. Sequence IDs are unique identifiers for sequences. They are used at runtime to interact with the sequence. These IDs are automatically generated when sequences are added to the project. However, some applications may need a particular animation to have a specific sequence ID.

You may also need to manually change Sequence IDs when a new KFM file is created for a character, yet your game or application already has logic based on particular ID values. For example, the Lenguins on Ice sample requires that IDs higher than a particular value represent a jump. It also requires that the baby and momma characters have some matching IDs. When a new KFM was required for the baby Lenguin character, all of the Sequence IDs were re-mapped to match those of the previous KFM. In general, Sequence ID re-mapping may be required when a new KFM file is exported from a DCC tool or a new animation tool project is started for an existing application.

To change sequence IDs

  1. Launch the Animation Tool and load the following file:

    %EMERGENT_PATH%\Samples\Models\AnimationToolExamples\Tutorials\Tutorial4.kfm

  2. Select Sequence ID Overview in the Project View.

    The Edit View should now change to appear like the image below.

  3. Double-click the ID field for walk. You should enter into the edit mode for that field. Type in 30 and press the Enter key. Note that the entry for walk is now 30. Also note that the entry for bind_pose is also open for editing. To cancel an edit, press the Escape key.

You have now changed a sequence ID.
