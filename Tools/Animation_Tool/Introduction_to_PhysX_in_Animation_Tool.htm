Introduction to PhysX in Animation Tool

The PhysX-enabled version of Animation Tool supports physics preview in the tool as animations are playing. This allows you to preview blends and transitions between animated control and physical control, and allows you to see how secondary physical objects, such as a cape, move while the character is animated.

This guide includes the following pages:

  • We give an overview of how to manipulate and interact with physical sequences in Animation Tool.
  • The Physics View is a control panel for physics in Animation Tool. It allows users to specify which sequences are physically controlled and to define the parameters of the underlying physics simulation.
  • The Physical Forces page describes the force interface and how it works when an animation is playing.
  • A tips page provides many useful recommendation for achieving better performance and a smoother work-flow. It includes known issues and things to avoid.

Samples

The directory %EMERGENT_PATH%\Samples\Models\AnimationToolExamples\Slim in the Gamebryo installation contains an example of a physical model and Animation Tool settings. You can also look at the Slim character model from the DodgeBall sample, which differs in its use of accumulation.

Requirements

Support for physics preview requires the PhysX system installed on your machine, and only works with models that were exported with PhysX properties associated with them. Some parts of the physics interface will be visible but disabled if you do not have PhysX installed.

The version that ships in the Gamebryo installers has PhysX enabled. Support for PhysX is optional when the Animation Tool is rebuilt from source code, and build configurations exist that do not use PhysX. Note that a non-PhysX build will have a different executable name so that studios can use both versions simultaneously if necessary.

If your studio works with a different physics system, there is a good chance you can use the PhysX integration with Animation Tool as a template for integrating your preferred physics system.
