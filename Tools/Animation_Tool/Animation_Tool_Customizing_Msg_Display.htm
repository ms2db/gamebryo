Showing or Hiding Dialog Boxes in the Animation Tool

You can choose to show or hide Animation Tool dialog boxes. You can also re-enable the display of dialog boxes that might be hidden after you checked the "Do Not Show Again" option on a dialog box.

To display or hide dialog boxes

  1. In the Animation Tool, click Edit > Message Box Management > Manage Message Boxes.
  2. In the Message Boxes Registry Edit dialog box, uncheck dialog boxes that you want to suppress and check message that you want to display.

You can reset dialog box display to the default (out-of-box) settings at any time by selecting Edit > Message Box Management > Reset Message Boxes.

The handles in the Message Boxes Registry Edit dialog correspond to RTF files that are used by the Animation Tool for dialog generation. You can find the RTF files in the following location:

%EMERGENT_PATH%\Source\Tools\AnimationTool\AnimationToolUI\Resources\MessageBoxes
