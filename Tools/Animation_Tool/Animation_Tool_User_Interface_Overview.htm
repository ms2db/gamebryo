Animation Tool User Interface Overview

The Animation Tool user interface is divided into seven principal parts.

Gamebryo Animation Tool user interface

  1. Menus and Toolbars—These are the toolbars and menus for the application.
  2. Project View—This view displays all of the data concerning sequences and sequence groups in the project.
  3. Render View—This view displays the currently selected item in the project or transition view as well as providing user interface for interacting with the display and playback.
  4. Edit View—This view allows editing the currently selected item in the project or transition view.
  5. Transitions View—This view displays all of the transitions between sequences in the project.
  6. Output View—This view displays a log of internal messages and warnings.
  7. Physics View—This view enables you to specify forces that you want to exert on objects while they are simulating
