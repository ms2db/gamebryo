Project View

You can use the Project View in the Animation Tool to visualize a hierarchy of all of the data in the active project. When you click an item in the tree, the Edit View updates with all settings and options that are available for the obejct. You can also right-click items for quick access to operations that you can perform on the item. This view enables you to work efficiently with various animation components and concepts.

To display this view, select View > Project View or click the Project View button on the main toolbar.

Project View

The Project View tree comprises the following items:

  • Character Model—Select this item to edit properties of the character model in the Edit View. Editable properties include the NIF file that is to be used and the root node for the character model.
  • Sequences—This item serves as a root level folder to house all of the sequences in the project. In addition, it may contain a user-customizable hierarchy of folders that contains the various sequences. When you select an individual sequence item, you can preview it in the Render View. You can view and edit various properties in the Edit View.
  • Sequence Groups—This item serves as a root level folder to house all of the sequence groups in the project. In addition, it may contain a user-customizable hierarchy of folders that contains the various sequence groups. When you select an individual sequence group, it becomes active in the Render View. You can view and edit various properties in the Edit View.
  • Sequence ID Overview—When you select this item, the Edit View refreshes to provide a comprehensive overview of all of the sequences in the project along with their respective sequence IDs. You can view and edit various properties in the Edit View.
  • Group ID Overview—When you select this item, the Edit View refreshes to provide a a comprehensive overview of all of the sequence groups in the project along with their respective group IDs. You can view and edit various properties in the Edit View.
  • Transitions Preview—When you select this item, you activate the currently selected transition in the Transitions View. You can view the transition in the Render View while you edit its properties in the Edit View. You can only select individual transitions from the Transitions View itself.
  • Interactive Preview—When you select this item, you enter a preview mode. You can can set up keystroke bindings in the Edit View, and you can interactively preview transitions between sequences using these bindings in the Render View.
