Manage Message Boxes

The "Message Boxes Registry Edit" dialog box is an option for advanced Animation Tool users who wish to modify the "Do Not Show Again" registry entries for individual dialog boxes within the tool. The dialog displays handles to the various dialogs used throughout the system. Checked boxes indicate which dialogs will be presented to the user while un-checked boxes indicate those dialogs that will be suppressed.

For users interested in using this feature, the handles in the "Message Boxes Registry Edit" dialog correspond to RTF files that are used by the Animation Tool for dialog generation. You can find the RTF files in the following location:

%EMERGENT_PATH%\Source\Tools\AnimationTool\AnimationToolUI\Resources\MessageBoxes
