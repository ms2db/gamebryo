Animation Tool Overview

The Gamebryo Animation Tool augments the art pipeline by providing you with a feature-rich tool that helps you preview, edit, and organize game character animations. With the Animation Tool, you can do the following:

  • Preview individual animation sequences
  • Preview and edit animation sequence transitions
  • Interactively drive character animations to mimic game behavior
  • Mix and match multiple animations
  • Preview animations on different models
  • Generate programmer information for driving the animations in your application
  • Preview additive animation blending
