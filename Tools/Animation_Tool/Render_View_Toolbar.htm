Render View toolbar

Use this field... To...
Orbit mode Enter orbit mode
Pan mode Enter pan mode
Scene Height mode Enter scene height mode
Decrease BoneLOD Decrease the Bone level of detail
Increase BoneLOD Increase the Bone level of detail
Wireframe Toggle the visualization between wireframe and normal mode
Scene Lights Toggle scene lights ON/OFF
Scene Background Toggle the background scene ON/OFF
Text display Toggle text display ON/OFF
Collision proxies Toggle collision proxy visibility ON/OFF
Accumulation trail Toggle the visibility of the accumulation trail ON/OFF
Toggle node visualization Toggles a graphical representation of the nodes in the scene which shows their position and orientation.
Toggle hierarchy visualization Toggles a graphical representation of the hierarchical relationships between nodes. Parent nodes are connected to their children by a line, while nodes which affect skinned characters are shown as a sphere and a pyramid pointing in the direction of the bone.
Toggle bounds visualization Toggles a graphical representation of the bounding spheres of mesh objects in the scene.
Pixel/vertex shaders Reload and reset all pixel and vertex shaders in the scene
Reset character model position Reset the position of the character model to the origin
Reset orbit point Reset the orbit point of the Orbit mode to the center of the character model
Scene options Open the Scene Options dialog box
Zoom to screen extents Zoom the camera to the extents of the scene
Camera follows character Toggle the camera to follow or not follow the character
Previous camera Switches the viewport to the previous camera in the scene. This will look through different cameras that are part of the scene, including animated cameras. However, only the default camera can be moved using the camera controls, so those functions will be locked until you return to that camera.
Next camera Switches the viewport to the next camera in the scene. This will look through different cameras that are part of the scene, including animated cameras. However, only the default camera can be moved using the camera controls, so those functions will be locked until you return to that camera.
Default camera Returns the viewport to the default camera which can be moved by the user using the camera controls.
