Physics View

You can use the Physics View in the Animation Tool to specify forces that you want to exert on objects while they are simulating. The forces are saved and re-applied every time the simulation restarts. This enables you to apply a force and then change the view to see from various angles how the object behaves under the force.

To display this view, select View > Physics View or click the Physics View button on the main toolbar.

While the Physics View is visible, you can float it, add it to a tab group, or dock it within the Animation Tool interface. Values that you specify in the Physics View are saved with the settings file for the current project and are available when the project is reloaded.

Physics View in the Animation Tool

Use this field... To...
Physical Sequences View and/or select sequences that are currently under physics control. When one of these sequences is animating, physics-derived values are written to the sequence's pose buffer. This pose buffer data is blended with other active sequences before the final result is propagated to the scene graph.
Add/Remove Open a dialog box in which you can add or remove physical sequences
Apply Forces Control force exertion while the animation is playing. This toggle button is part of the group that controls the Render View's response to mouse input (like the Orbit, Pan and Ground height buttons). When this button is selected, mouse input is interpreted as applying a force.
Clear Clear force from memory.
Strength Control the strength of the force.
Ground Plane Friction Define the amount of friction on the ground plane.
Note: There is a physical ground plane in the same location as the rendered ground plane. The friction and bounce values interact with the friction and bounce of the model that is being simulated - the default is for the collision to have the average of the two values. Hence, doubling the bounce or friction of the ground may not double the apparent bounce or friction.
Ground Plane Bounce Define the amount of bounce on the ground plane.
Debug Rendering To visualize the collision shapes and joints of the physics model as it animates.
Scale Scale the debug geometry as it is rendered.
Gravity Specify the force and direction of gravity.
Timestep Set the step size used by the underlying physics simulation. For stable simulation, the value should be kept within a tight range; between 0.05 down and 0.005 works best. Use smaller steps for faster or heavier objects.
Save As Defaults Saves the current values so that they're used as the defaults when Animation Tool opens a new project.
Restore Defaults Reset defaults to the current values.
