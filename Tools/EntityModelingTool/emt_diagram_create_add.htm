Creating Entity Model Diagrams

You can create any number of diagrams within your project to help you visualize important subsections or strategic concepts within your project. For example, let's say you are creating a dungeon crawl game. In this case, you could create diagrams to help visualize the following:

  • The fundamental elements that determine how entities are displayed and moved through the world.
  • Offensive and defensive capabilities in the combat system and how they are assigned to monsters.
  • The collection of abilities that can be used in spell effects or assigned to magical items.

When creating diagrams, you can explicitly determine which models will be included, their location within the diagram, as well as any relationships that may exist between models. After the diagrams are created, you can then go on to add, exclude, rearrange, and link items within the diagram.

To create a diagram

  1. Open a game solution.
  2. In the Game Solution view, double-click a project node to open it for editing.
  3. In the EMT Project Explorer, select Add > New Diagram from the menu.

    The new diagram is created within the open project.

  4. Specify a name for the diagram.

    You can now begin adding and linking models and presets within the diagram.
