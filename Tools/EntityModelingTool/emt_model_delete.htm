Deleting Models

If you no longer need a model file, and never plan to use it again, you can delete it. The Delete command removes the model from your project and deletes the model file stored on disk.

Note

If you delete a model by mistake, you can undo it.

To delete a model file

  1. In the EMT Project Explorer, select the model that you want to delete.
  2. From the context menu, click Delete, or simply press the Delete key.

    Toolbench will provide you with a list of models that are mixing in the selected model and will ask you to confirm the deletion.

  3. Click Yes.

The model is removed from your project.
