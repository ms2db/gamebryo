Renaming Presets

When you first create a preset, it is given a default name, such as Preset1. You can, however, rename a preset at anytime for any reason.

To rename a preset

  1. In the EMT Project Explorer, do one of the following:
    • Select the preset item and press F2.
    • Right-click the preset item and select Rename from the context menu.
  2. Type a new name and press Enter.

    The preset is now renamed.

Note

You can also rename a preset in the Entity Model Editor.
