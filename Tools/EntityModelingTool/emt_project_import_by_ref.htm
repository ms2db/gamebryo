Importing a Project by Reference into Another Project

To manage your game content effectively, you will most likely break it down into different projects, where each project contains a unique set of models, presets, and diagrams. There may be cases, however, when you want to use specific models or presets in more than one project. To avoid duplicating the models in each project, EMT allows you to import a project by reference, giving you access to all the items within that project.

To import a project by reference into another project

  1. In the Game Solution view, expand the project node where you want to import a project by reference. (If you can't expand a project node, you need to open it for editing first. To load a project, simply double-click the node in the Game Solution view.)
  2. Right-click the References folder and select Add Reference from the menu.

    The Open dialog box opens.

  3. Navigate to the folder where the project resides, select it, and then click Open.

    The project is added to the current project's References folder in the Game Solution view. You can now use the items within the referenced project as if they were in the current project.

  4. Repeat steps 2-3 to import another project by reference.
