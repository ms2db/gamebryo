Selecting/Multi-Selecting Entity Models and Presets

When you select a model or preset in the Mix-in or Diagram Editor, the model's properties and behaviors are automatically displayed in the Entity Model/Preset Editor, making it easy for you to edit the properties and behaviors of a model or preset.

To select an entity model or preset within the Mix-in/Diagram Editor

  1. In the EMT Project Explorer, do one of the following:
    • Double-click a model or preset to load it into the Mix-in Editor.
    • Double-click a diagram to load it into the Diagram Editor.
  2. To select a model or preset in the both editors, simply click the box that represents it.

    The box turns blue and the corresponding properties and behaviors are displayed in the Entity Model Editor.

Note

You can select multiple models and/or presets using Ctrl+click and Shift+click. You can also use the rectangle selection tool by dragging the selection around the boxes directly in the editor. When multiple models are selected, the properties and behaviors are not displayed in the Entity Model Editor. However, multiple presets can be selected and edited at once.
