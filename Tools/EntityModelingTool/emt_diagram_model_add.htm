Adding Entity Models to a Diagram

The Diagram Editor is very flexible, making it very easy for you to add and remove items from a diagram. You can add any number of models and presets to a diagram to help design important subsections and strategic concepts within your game. Within each diagram, you can experiment with different mix-in possibilities, and then exclude those models and presets that you no longer need.

To add models and presets to a diagram

  1. In the EMT Project Explorer, double-click a diagram to load it into the Diagram Editor.
  2. Do one of the following:
    • From the EMT Project Explorer, drag a model or preset into the Diagram Editor.
    • In the Diagram Editor, right-click and select New Entity Model or New Preset from the menu.

    A box that represents the model or preset is created in the Diagram Editor.

  3. Continue to add other models and presets into the Editor using either method.

You are now ready to experiment with different relationships between the various models and presets within the diagram.
