Deleting Presets

If you no longer need a preset file, and never plan to use it again, you can delete it. The Delete command removes the preset from your project and deletes the preset file stored on disk.

To delete a preset file

  1. In the EMT Project Explorer, select the preset that you want to delete.
  2. From the context menu, click Delete, or simply press the Delete key.

    The preset is removed from your project.
