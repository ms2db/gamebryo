Entity Model Projects

An entity model project is a group of entity models, presets, and diagrams that make up a specific part or concept in your game; for example, a level, zone, or special mission. Projects allow you to break up your game into manageable parts and can be as simple or as complex as you need. The items within each project can also be organized into folders in whichever way you want.

Entity model project contents

Each project within your game solution is considered as a separate item, but you can "import" information from one project to another using references. When you add references to a project, all items within the referenced project become available in the current project. This is very useful when you want entity models in different projects to use the same properties and/or follow the same behaviors.
