Saving Content within your Projects

It is a good practice to save the content within your project on a regular basis. You can save individual documents or all documents at the same time.

To save the content in your project

  • From the toolbar, click one of the following icons:
    • Save Content to save the file or document that is currently focused for edit.
    • Save All Content to save all open files and documents that are flagged as dirty.

The file or files are saved to disk.
