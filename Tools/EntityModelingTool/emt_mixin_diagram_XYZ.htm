Using the Mix-in and Diagram Editors

The Mix-in and Diagram Editors show the graphical representations of the relationships between the entity models and presets within a project. They both function in a similar manner, although the Diagram Editor is more flexible with layout whereas the Mix-in Editor has the advantage of automatic generation.

To work efficiently in the Mix-in and Diagram Editors, you need to understand how to perform the following tasks:

  • Select/Multi-select Entity Models and Presets
  • Move Entity Models and Presets within the Diagram Editor
  • Navigate in the Mix-in/Diagram Editor
  • Edit Property Values within the Mix-in/Diagram Editor
  • Change the Stacking Order of Models and Presets
  • Convert Models into Presets
  • Remove Models and Presets from the Diagram Editor
  • Delete Models and Presets from the Diagram Editor
