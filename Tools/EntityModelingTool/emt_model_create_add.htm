Creating/Adding Entity Models to a Project

Before creating a series of models in EMT, it is a good idea to spend some time up front to decompose your game into distinct entities. You will need to decide on the level of detail that is required, which things to include and which to ignore, as well as the best way to structure the complexity of your system. This work will help you decide how best to represent the entities in EMT. After this is all decided, you can then break down your game into manageable projects. Within each project, you can create the entity models following the strategy that was developed when you decomposed your game.

To create a new entity model within a project

  1. Open a game solution.
  2. In the Game Solution view, double-click an EMT project node to open it for editing.
  3. Right-click in the Contents sub-pane of EMT Project Explorer and select Add > New Model from the menu.

    The new entity model is created within the open EMT project.

  4. Specify a name for the new entity model.
