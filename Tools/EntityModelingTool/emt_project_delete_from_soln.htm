Deleting Entity Model Projects from a Game Solution

If you no longer need a project, and never plan to use it again, you can delete it from the game solution. Unlike the Exclude command, the Delete command removes the project file from your game solution and then deletes the file stored on disk.

To delete a project from a game solution

  1. In the Game Solution view, right-click the project that you want to delete.
  2. From the context menu, click Delete.

    Toolbench prompts you to confirm the deletion.

  3. Click Yes.

The project is removed from the solution and deleted from disk.
