Deleting Models and Presets from the Diagram Editor

If you no longer need a model or preset file, and never plan to use it again, you can delete it from within the Diagram Editor. When you delete a box from the Diagram Editor, the model/preset is deleted from the project and the corresponding file stored on disk is also deleted.

To delete a model or preset file from the Diagram Editor

  1. In the Diagram Editor, right-click the box of the model or preset that you want to delete.
  2. From the context menu, click Delete.

    The model/preset is removed from the editor and the project.
