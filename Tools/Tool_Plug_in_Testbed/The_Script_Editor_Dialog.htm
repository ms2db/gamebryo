Tool Plug-in Testbed Script Editor Interface Options

The script editor dialog is a powerful tool in customizing the art pipeline. It allows you to generate and edit scripts for each plug-in.

Script Editor

Use this field... To...
Current Script Select the script from a list of known scripts that match the script type. The selected item from this list defines what the current script is for the rest of the dialog. Each template script indicates the source file that it was loaded from in parentheses. If a script has an asterisk (*) character after the parentheses, it means that the script has been altered in the current dialog session.
New Create an entirely new script with no plug-ins. This capability is useful when you are starting from scratch and generating a script.
Save Save the currently selected script to disk. At this point, the script becomes a template script and can be used by others.
Clone Create a new script based on an existing script. A dialog box opens. Select the script that you want to clone and specify the name of the script that you want to create.
Rename Open a dialog box in which you can specify a new name for an existing script.
Script Type Toggle through types of scripts that this dialog will display.
Known Plug-ins List Choose from categorized lists of known plug-ins.
Plug-in List For Current Script Review the plug-ins used by the current script. The order that the plug-ins are listed is the order in which they are run. It is possible to have multiple copies of a plug-in in the list as well as different parameters for each copy.
Add Plug-in Add a new instance of the currently highlighted plug-in in the known plug-ins list to the current script. The script will use the default parameters of the plug-in.
Remove Plug-in Remove the plug-in selected in the "Plug-in List For Current Script" from the current script.
See Description View a description of the currently selected plug-in in the "Known Plug-ins List".
Edit Plug-in Edit the currently selected plug-in in the "Plug-in List For Current Script" list.
Move Up Move the currently selected plug-in in the "Plug-in List For Current Script" list up one position in the execution order.
Move Down Move the currently selected plug-in in the "Plug-in List For Current Script" list down one position in the execution order.
