Removing breakpoints in the Script Debugger

You can use the Breakpoints View to view all of the breakpoints that have been added by clicking in the code editor margin.

You can remove breakpoints in several ways:

  • In the code editor margin, click an existing breakpoint icon to toggle the breakpoint OFF.
  • In the Breakpoints View, right-click and remove one or all breakpoints.
  • In the Breakpoints View, select (highlight) a breakpoint and press the DELETE key.
