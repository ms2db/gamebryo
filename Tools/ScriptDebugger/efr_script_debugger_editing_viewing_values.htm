Editing values in the Script Debugger

In the locals view, the auto entity view and the watch entity view you can edit variable values. You can enter editing mode by simply double clicking on variable value or by choosing "edit value" in the context menu. Integer values can be entered as decimal values or as hexadecimal values.

Viewing values in the Script Debugger

In the locals view, the auto entity view, the watch entity view and the watches view you can choose to view the integer values as decimal or as hexadecimal. To change the display bring the context menu up and check "hexadecimal display" option. Note that when you activate/deactivate hexadecimal display in one view it will propagate in other views as well.
