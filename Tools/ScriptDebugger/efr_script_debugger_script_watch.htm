Adding script watches in the Script Debugger

You can use the Watches View to track the value of arbitrary script expressions. To add a watch, right-click within the Watches View and select Add Watch from the context menu. The expression that you enter will be evaluated each time script execution breaks. This is useful for tracking global variables, examining specific elements of complex types, or continuously monitoring the return value of functions.

Script Debugger: Adding script watches

You can modify or delete watches by right-clicking within the Watches View.
