NiThumbnailExtractor: Introduction

NiThumbnailExtractor is a Microsoft Windows COM DLL that generates thumbnails of NIF and KFM files for Explorer. In order to generate thumbnails, the following restrictions must be true:

  • If a NIF file is selected, a thumbnail must be embedded in the NIF file. This can be done by using the ThumbnailGeneratorDLL tool plug-in at export time.
  • If a KFM file is selected, a thumbnail must be embedded in the NIF file referenced by the KFM file.

If no thumbnail is embedded in the file, a Missing Thumbnail Data image is displayed instead.

The DCC plug-ins (3ds max, Maya, XSI) all have several tool plug-in pipeline scripts to generate thumbnails for you. If you aren't seeing thumbnails, make sure that the script that you are exporting with includes the ThumbnailGeneratorDLL tool plug-in.

Installation

This tool is installed using the installer shipped with the product by default or it can be installed as an Administrator manually using a command-line like:

regsvr32 %EMERGENT_PATH%\sdk\Win32\DLL\NiThumbnailExtractor<GamebryoVersion><CompilerVersion>S.dll

The following registry settings are created/modified when registering this DLL with Windows:

  • HKEY_CLASSES_ROOT\.nif\Shellex\{BB2E617C-0920-11d1-9A0B-00C04FC2D6C1}\{2735371A-C925-4a6c-BF1A-A49313CDB27B}
  • HKEY_CLASSES_ROOT\.kfm\Shellex\{BB2E617C-0920-11d1-9A0B-00C04FC2D6C1}\{2735371A-C925-4a6c-BF1A-A49313CDB27B}
  • HKEY_CLASSES_ROOT\CLSID\{2735371A-C925-4a6c-BF1A-A49313CDB27B}\

Vista 64

If you are using a 64-bit version of Vista, you will need to run the following app from the command-line in Administrator mode:

%windir%\syswow64\regsvr32 %EMERGENT_PATH%\sdk\Win32\DLL\NiThumbnailExtractor<GamebryoVersion><CompilerVersion>S.dll

Additionally, the DLL is a 32-bit DLL. It will not display in the default Vista explorer shell. It will show up in a 32-bit version of the explorer shell. It will also show up in Toolbench and other 32-bit applications. You can start a 32-bit explorer shell using the following steps:

  1. Click on the Start menu
  2. Type in the following into the search: %windir%\SysWOW64\explorer.exe /separate, /root, c:
  3. You should now have a 32-bit explorer shell pointing at your C drive. Thumbnails will work in this view.
