EEDeploy: Introduction

You can use the EEDeploy tool (EEDeploy.exe) to deploy a sample or game application (separate out the shippable data from source assets, data for other platforms, and so on).

The EEDeploy tool is driven by an XML script (typically named ProjectName.deployscript), which specifies the rules for deploying a particular sample.

Multiple platforms or build-configurations (debug vs release, for example) are supported by specifying key=value on the command line, and using conditional expressions in the script. See the examples in EEDeploy Script File Format.

You can find this application in the following location (which is part of the system PATH):

%EMERGENT_PATH%\SDK\Win32\DLL\EEDeploy.exe
