World Builder Terrain Artist Guide

The World Builder Terrain tools within Toolbench is used for creating and editing terrain entities. With these tools the artist can create, sculpt and paint terrains.

The topics below cover the possible operations with these tools.

  • Terrain User Interface
    Introduces all the components of the terrain user interface.
  • Creating a New Terrain
    Describes how to make a new terrain within Toolbench.
  • Terrain Properties
    Describes the terrain properties.
  • Painting on the Terrain
    Describes how to paint materials and sculpt the terrain.
  • Editing and Using Materials
    Describes the management of materials.
  • Editing and Using Multiple Sectors
    Describes how to use multiple sectors in the terrain.
  • Saving and Loading the Terrain
    Describes how to load and save a terrain from Toolbench.
  • Importing and Exporting Heightmaps
    Details the operation of importing and exporting the heightmaps for the terrain.
