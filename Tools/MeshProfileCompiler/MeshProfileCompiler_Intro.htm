MeshProfileCompiler: Introduction

MeshProfileCompiler (MeshProfileCompiler.exe) is a tool intended to help profile authors validate profiles that they create. The MeshProfileCompiler tool takes an input mesh profile directory, parses all the profiles, and dumps out a log file. This log file not only displays warnings and errors, but also gives a verbose English description of the profile. Because profiles may be fairly complex and have renderer-specific data, they can be difficult to follow. The verbose English summary helps eliminate some of that complexity.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\MeshProfileCompiler\Win32
