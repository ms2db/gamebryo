MeshProfileCompiler Command-line Usage

To compile the mesh profiles and output a log file, do the following.

Launch: %EMERGENT_PATH%\Source\Tools\DeveloperTools\MeshProfileCompiler\Win32\MeshProfileCompiler.exe

Optional arguments to control the inputs and outputs are specified as follows:

  • -in Path to profiles
  • -out Output log file name

If no -in argument is specified, by default the tool will simply parse the profiles from the default profile directory:

%EMERGENT_PATH%\Media\ToolPluginsData\MeshProfiles.

The -out argument must be specified. The output log file location defaults to the root of the active drive letter.
