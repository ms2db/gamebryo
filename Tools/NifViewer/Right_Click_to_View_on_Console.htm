Right-click to View on Console

The Gamebryo installer adds console-specific entries to Windows context menus. When you right-click .NIF files, you can choose to view the selected .NIF file in any of the following consoles:

  • PS3
  • PS3 (PhysX)
  • Xbox 360
  • Xbox 360 (PhysX)

These options are not available for non-console versions of Gamebryo.

Xbox 360 Details

  • The NifViewer application will be launched on the default Xbox 360 development kit.
  • All shaders located in %EMERGENT_PATH%\sdk\Xbox360\Shaders\Data will be copied over to the target Xbox 360 development kit along with the NIF.
  • Only textures inside the NIF will be displayed. External textures will not be copied over to the target Xbox 360 development kit.

PS3 Details

  • The host PC must be connected to a target in the SN Systems Target Manager.
  • The .NIF file must reside on the same drive as the Gamebryo installation.
  • Right-click to View on PS3 automatically sets the SN Systems Target Manager file serving host and root to the NifViewer installation folder so that it can access the data files that it needs.
  • Right-click to View on PS3 automatically sets the NifViewer's -shaderpath command-line argument, by default, to:

    %EMERGENT_PATH%\Media\Shaders\Data\PS3
