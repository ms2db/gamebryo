NifViewer Menus

The options in the NifViewer are accessible through an on-screen menu. You can activate/deactivate the menu by pressing the START button. You can change the current menu selection by using the LUP, LDOWN, LLEFT, and LRIGHT buttons. Execute a menu command by pressing the RDOWN button.

File Menu Description
Screen Shot PS3 Only. Creates a screenshot image in %EMERGENT_PATH%\SDK\Win32\DLL\ToolPlugins\ConsoleViewers\PS3
Exit Exits the viewer.
View Menu Description
Camera Enables you to select the camera that is being used to view the scene. All the cameras in the scene being viewed will be listed in this sub-menu as well as an additional camera created by the NifViewer. The camera created by the NifViewer has the label "NifViewer" and appears as the last entry in the camera list.
Textures Display the texture browser. You can cycle through the textures in the scene and explore their sizes, bit depths, names, and mipmap settings. The texture browser is exclusive and will hide the menu. Pressing start or selecting the exit option from the browser will exit the texture browser.
Wireframe Toggle wireframe view.
Reset Resets the current camera to its initial position and orientation.
Stats Only available in NiProfile builds. Toggles a box that displays performance information.
FPS Only available in non-NiProfile builds. Toggles a framerate display.
Sorter Provides the following options:
  • None—No sorter is used to render the scene. If there are transparent objects in the scene, they will probably look incorrect from certain viewing angles when this option is chosen. This is the default option for a scene that has no alpha-blended objects.
  • Alpha—An NiAlphaAccumulator is used as the sorter. NiAlphaAccumulator defers the rendering of alpha-blended objects until after all opaque objects have been rendered, and then renders the transparent objects from back to front. This is the default option for a scene that contains any alpha-blended objects.
Animation Menu Description
Freeze Toggles animations ON/OFF.
Reset Time Resets the time being fed to the animation system to zero (0), even if the scene is frozen.
Loop Toggles whether or not the viewer resets the time that it is feeding to the animation system to 0 when the end time for the animations in the scene is reached. Looping is enabled by default in scenes that do not contain any automatically-looping animation controllers.
Speed Provides the following options:
  • Slower—Decreases the animation speed.
  • Faster—Increases the animation speed.
  • Reset—Resets the animation speed to 100%.
Gamepad Menu Description
Tumble Toggles the gamepad interface for navigating through the scene between TUMBLE and TURRET mode.
Invert Camera Invert the up and down rotation on the left thumbstick for TURRET navigation.
Rotate (0) Enables you to adjust the sensitivity of using the gamepad sticks to rotate the camera (TURRET mode) or the scene (TUMBLE mode). The number in parentheses indicates the current sensitivity.
  • Slower—Decreases rotation speed.
  • Faster—Increases rotation speed.
Translate (0) Enables you to adjust the sensitivity of using the gamepad to translate the camera through the scene. The number in parentheses indicates the current sensitivity
  • Slower—Decreases translation speed.
  • Faster—Increases translation speed.
Lock Up Axis (*) Lock the up axis for TURRET mode navigation. The default is the Z axis.
