NiMaterialNodeXMLLibraryParser: Introduction

NiMaterialNodeXMLLibraryParser (NiMaterialNodeXMLLibraryParser.exe) is a command-line application that converts an input XML file into a C++ source and header file pair that can be compiled and included by an application.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\NiMaterialNodeXMLLibraryParser\Win32\<VC80 or VC90>
