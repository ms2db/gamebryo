Creating ASCII Fonts in NiFontCreator

To launch the NiFontCreator application, double-click NiFontCreator.exe in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\NiFontCreator\Win32\<VC80 or VC90>

In the application window, choose the Select Font button. In the Font dialog box, select an ASCII font and the desired font settings. Next click OK.

The selected settings will be reflected in the grayed-out elements of the main NiFontCreator dialog box.

Next, select the AntiAlias checkbox, if desired. If AntiAlias is not checked, the pixels that define the characters in the font image will be fully opaque. If AntiAlias is checked, the edge pixels of the characters will be partially transparent.

Finally, click the Save NFF button and specify a location for the resulting NFF file that will be created.

The default filename for the NFF file includes the name of the font, letters to indicate the options (B for bold, I for italic, U for underlined, S for strikethrough, A for antialiased), and digits to indicate the point size for the font. For example, Verdana_A_14.nff is a NFF file representing a 14-point Verdana font, with anit-aliasing.

There is no need to create or select a Font configuration file for ASCII character sets, although it is required for Unicode character sets.

The tool will generate 95 glyphs (Ascii codes 32-126) to support ASCII character sets in a NFF file.
