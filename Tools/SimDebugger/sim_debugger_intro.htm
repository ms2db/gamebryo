Simulation Debugger: Introduction

The Simulation Debugger is a remote debugging tool designed to help developers grasp both an overall picture of the entities in a running game and internal state details for specific entities.

By connecting to a game and receiving updates of entity lifecycles and property changes, the Simulation Debugger builds up a simplified database mirroring that of the actual entity system. The tool then creates a visualization of this data. When used in conjunction with the Script Debugger, developers can identify, isolate, and fix a wide range of game issues without digging into code.

Note

Simulation Debugger is not a standalone tool! Simulation Debugger functionality is accessed by using Toolbench to create or open a .simdebug file.
