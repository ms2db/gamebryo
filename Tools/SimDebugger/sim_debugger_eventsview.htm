Simulation Debugger Events View

The Events View provides a list of all recorded events directly related to the entity selected via the World View, including:

  • Entity creation
  • Periodic position and facing updates
  • Real-time property changes
  • Entity destruction

These events are used to drive the World View visualization.

Because property changes are reported immediately, with timestamp, analyzing the record of events can help identity cause and effect relationships in the entity system and pinpoint bugs.
