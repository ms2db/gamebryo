Gamebryo Asset Viewer Animation View

The Animation View displays information about the animation playback of the current scene. Select View > Animation Controls to display the extended animation toolbar.

Use this field... To...
Time slider Set the animation start time. Drag the slider handle to change the animation start time. Or, if you want to watch the entire animation, allow the slider handle to move as the animation progresses. The end time of the slider is set to the time of the last animation key found in the scene. You can manually set the start and end times by clicking the Animation Options button.
First Frame Set the time slider to the animation's first frame.
Previous Frame Set the time slider to the animation's previous frame.
Play Toggle the animation ON/OFF.
Stop Stop animation playback and reset the time slider to the animation's first frame.
Next Frame Set the time slider to the animation's next frame.
Last Frame Set the time slider to the animation's last frame.
Animation Options Open the Animation Options dialog box.
Playback Mode Set the animation playback mode. If you choose Loop, the animation will restart playback when the animation reaches the last frame. If you choose Continuous, the animation will never reset the time to the first frame and will keep playing until you stop the animation.
