Asset Viewer Animation Options Dialog Box

You can use the Animation Options dialog box to view detailed information about time controllers and animation keys in the current scene. You can also use this dialog box to adjust the time slider start and end times as well as the playback tick rate. You can adjust the playback rate anywhere from 0% to 400% by dragging the slider. To open the Animation Options dialog box, click Options > Animation Options.

Animation Options dialog box

The settings in this dialog box are not saved between Gamebryo Asset Viewer program executions. The next time the Gamebryo Asset Viewer is opened, these settings will have been set appropriately for the new scene.
