Asset Viewer Menus

The menus in the Asset Viewer are described below:

File Menu Description
New Creates a completely new scene graph, removing any currently loaded NIF or KFM files.
Open File Opens a NIF or KFM file, replacing any currently loaded files.
Add File Adds a NIF or KFM file to the current scene, retaining any currently loaded files.
Remove File Removes a NIF or KFM file from the current scene.
Set Image Subfolder Sets the subfolder to use when loading external images. This should be set before opening or adding the file with external images.
Recent Files Recently opened or added files will be listed here. Clicking on an entry opens (not adds) the file in the Asset Viewer.
Exit Quits the Gamebryo Asset Viewer application.
View Menu Description
Toolbars Sub-menu for turning docking toolbars on and off.
Standard Toggles the Standard Toolbar on and off.
Camera Navigation Toggles the Camera Navigation Toolbar on and off
Mini-Animation Toggles the Mini-Animation Toolbar on and off.
Docking Views Toggles the Docking Views Toolbar on and off.
Status Bar Toggles the Status Bar on and off.
Scene Graph Tree Toggles the Scene Graph Tree View on and off.
Scene Graph List Toggles the Scene Graph List View on and off.
Animation Controls Toggles the Animation View on and off.
Performance Statistics Toggles the Statistics View on and off.
Collision Lab Toggles the Collision Lab on and off.
Message Window Toggles the Message Window on and off.
Options Menu Description
Scene Graph List Options Opens the Scene Graph List Options dialog box.
Animation Options Opens the Animation Options dialog box.
Preferences Opens the Preferences dialog box.
Cameras Menu Description
Cameras All the cameras in the scene are listed in this menu. Select a camera to make it the current camera
Help Menu Description
Contents Displays the Asset Viewer help file.
About Gamebryo Asset Viewer Displays the About Asset Viewer dialog box.
