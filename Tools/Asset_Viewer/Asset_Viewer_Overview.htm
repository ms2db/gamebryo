Asset Viewer Overview

The Asset Viewer displays and analyzes the performance of NIF files in the Gamebryo engine. A rendered view of the scene is shown, as is a hierarchical view of the underlying scene graph that provides detailed information about any object in the scene.

Using the Asset Viewer to view NIF files that were exported for non-Windows platforms may result in error textures or improperly or incompletely rendered meshes. Some data formats that are supported by other renderers are not supported by the Direct3D renderers. In this case, the mesh in question will fail to render, but can still be inspected in the scene graph view. View the output log of Asset Viewer debug builds for detailed information about rendering failures.
