Asset Viewer Property Sheets

Property Sheets in the Asset Viewer show important information about any Gamebryo object. The following screen shot is from the property sheet of an NiNode. Each property sheet is divided into multiple panels that contain specific kinds of information like "Local Transforms," "Children," or "Extra Data." Property sheet information is updated in real time to match what is visible in the scene.

Right-clicking in a property sheet brings up a pop-up menu allowing you to expand or collapse all of the panels or manipulate individual panels.

Left-clicking in a property sheet and dragging will scroll the property sheet window.

Property Sheets often have links to other property sheets. For instance, clicking on the "node_animation" parent button will open up the property sheet for that object. Double-clicking on the "CyberMonkey_Rig:body1Shape" item in "Children" will open that object's property page.

In this case the object is a NiMesh object. It opens up a property page to display a variety of mesh related data. Including mesh modifiers and data streams.

You may also double click the mesh modifier to display information about that given modifier. In this case double clicking on "NiSkinningModifier" will open the NiMeshModifier property page displaying all kinds of useful information customized to the specific mesh modifier.

The "DataStream Refs" list displays all the current NiDataStreamElements contained inside the mesh. You can double click on the data stream to get a property page displaying the data stored in the data stream. It is good to note that the data is refreshed per frame, if an animation is playing you may notice a performance issue.

You may also click on links to property pages for NiProperties. One example is the NiTexturingProperty property page, which lists basic information about the NiTexturingProperty and the associated NiTexture object(s). It also lists the source texture target platform, in case the NiSourceTexture is platform-specific (Xbox360, Playstation3, or DX9) as opposed to "generic". In addition, information about texture transforms and bump map specific information is included.
