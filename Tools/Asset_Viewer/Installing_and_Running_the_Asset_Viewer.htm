Installing and Running the Gamebryo Asset Viewer

The Gamebryo Asset Viewer is installed when you run the main Gamebryo installer. The installer creates a shortcut on the Windows Start Menu.

The Asset Viewer requires Windows XP Service Pack 2 or later and DirectX 9.0c or later to run. The installer will inform you if you do not have these prerequisites.

To uninstall the Asset Viewer, use the Add/Remove Programs applet in Windows Control Panel.

D3D10 / D3D11 Support

By default, the Asset Viewer uses the DX9 renderer. You can choose to use the D3D10 or D3D11 renderer instead. To use the D3D10 renderer, specify "-D3D10" on the command-line when you launch the tool. To use the D3D11 renderer, specify "-D3D11" on the command-line when you launch the tool. If you want to change the default behavior, add this command line option to the tool's Windows Start Menu shortcut.

NIF File Association

Upon running the Gamebryo Asset Viewer for the first time, all files with a ".NIF" extension are associated with the Gamebryo Asset Viewer. Thus, double-clicking NIF files in the future will automatically open them in the Gamebryo Asset Viewer.

If the "-D3D10" or "-D3D11" command line switch was used to launch the Gamebryo Asset Viewer, then it will also be used to launch the Gamebryo Asset Viewer when NIF files are double-clicked in the Windows Explorer.

For example, if the Gamebryo Asset Viewer was last launched using the "-D3D10" command line option, double-clicking on a NIF file would automatically launch the Gamebryo Asset Viewer with D3D10 renderer. In the same way, if the Gamebryo Asset Viewer was previously launched without the "-D3D10" or "-D3D11" command line switches (which defaults to the DX9 renderer), then double-clicking on a NIF file will launch the Gamebryo Asset Viewer using the DX9 renderer.
