Asset Viewer Preferences Dialog Box

You can set Asset Viewer preferences in the Preferences dialog box. To open the Preferences dialog box, click Options > Preferences.

Note

All preferences are stored in the Windows registry for each user.

Asset Viewer Preferences dialog box

Use this field... To...
General Preferences
Use default lights Add lights for the entire scene when a NIF or KFM file that does not contain any lights is added to Asset Viewer.
Prompt when discarding open files Specify whether or not the Asset Viewer will prompt you if you are about to discard currently open files. This prompt will appear if you choose File > New or File > Open File when other files are already open.
Note: You can also turn off this prompt in the message box that displays the prompt.
Highlight selected objects in Render View Display a green rectangular reticule in the Render View around objects that are selected in the Scene Graph Tree View or Scene Graph List View.
Display alternate bounding volumes (ABVs) Display a blue wireframe representation of all alternate bounding volumes (ABVs) in the scene.
Background Color Specify the background color that appears in the Render View. Click the Choose Color button to select a color.
Keyboard Preferences
Translation Speed Specify the camera translation speed for each axis when you use the keyboard controls. These numbers specify the percentage of the scene radius to move each second. For example, 50 will translate the camera one-half the radius of the scene in one second.
Pitch Speed Specify the speed in degrees per second at which the camera tilts up and down when you use the keyboard controls.
Yaw Speed Specify the speed in degrees per second at which the camera pans right and left when you use the keyboard controls.
Up AxisSpecify the up axis for the scene. By default Maya has an up axis of Y and 3ds max has an up axis of Z.
Camera control mode Specify the type of camera control to simulate while in object selection mode. Camera controls are simulated for both 3ds max and Maya.
