Asset Viewer Scene Graph Tree View

The Scene Graph Tree View displays the current scene graph in a hierarchical format. It is designed to provide a way to navigate the scene graph in an easy-to-read fashion.

Asset Viewer: Scene Graph Tree View

Right-clicking on a tree item will bring up a popup menu. This menu will allow you to expand or collapse the entire sub-tree or view properties on the currently selected item. You can also Show/Hide objects by using this menu which will set their AppCulled flag.

Double-clicking on a leaf item or pressing "Alt+Enter" when a tree item is selected will bring up its property sheet. This property sheet will display useful information about the object itself. See Asset Viewer Property Sheets for more information about object properties.
