Source License Issues

Source License Compilation Options

Source-license clients will find that the .vcproj files shipped with the source components of Gamebryo have debug, release, and shipping build configurations for static and dynamic libraries. Building a configuration will place libraries into a configuration-dependent subdirectory of %EMERGENT_PATH%\sdk\Win32\Lib\VC80 or %EMERGENT_PATH%\sdk\Win32\Lib\VC90 . See Gamebryo Build Configurations for more details. All dynamic link libraries are located in sdk/Win32/DLL. Gamebryo DLLs are named uniquely by their library name, Gamebryo version, compiler, and build configuration. Debug DLLs end in 'D'; Release DLLs in 'R'; Shipping DLLs in 'S'.

Bink Video Library Source

Compiling the NiVideo library for the Bink Video requires the Bink SDK.

The NiBinkVideo library is included as a part of the Gamebryo source and evaluation distributions, although, as mentioned earlier, this fact does not grant Gamebryo users a license to the Bink SDK. That license must be obtained directly from RAD Game Tools at http://www.radgametools.com/.

Note that the NiBinkVideo library is not built in the standard build configurations with the rest of the UtilityLibs projects. The rationale for this exclusion is to avoid having source builds fail due to an absence of the Bink SDK. Contact sales@radgametools.com) to request an evaluation SDK.

NVIDIA PhysX™ Library Source

Compiling the Gamebryo-PhysX integration libraries and tools requires the PhysX SDK. The Gamebryo-PhysX integration is available to all Gamebryo developers, although this fact does not grant Gamebryo users a license to the PhysX SDK. That license must be obtained directly from NVIDIA.

The Gamebryo %EMERGENT_PATH%\SDK\Wins32\DLL directory contains the PhysXLoader.dll library. It is present to simplify the environment variables needed to run the Gamebryo-PhysX demos.

Note

The presence of PhysX DLLs in the Gamebryo SDK folder does not in any way grant a PhysX license to Gamebryo customers or end-users. The PhysX SDK must be requested from NVIDIA. Contact NVIDIA to request an evaluation version of the PhysX SDK.

To run the Gamebryo-PhysX samples and develop with Gamebryo, you must install the PhysX 2.8.3 System Software and PhysX Core. More specifically, the latest version of Gamebryo LightSpeed has been developed and tested using:

  • PhysX SDK Version 2.8.3.21 on Win32.
  • PhysX SDK Version 2.8.3.34 on Xbox 360.
  • PhysX SDK Version 2.8.3 (version B1) on Playstation 3.

The System Software may safely be of a later version. For development we also recommend the PhysX samples. Art tools require specific installers for the art package available from NVIDIA.

The build files for Gamebryo-PhysX are separate from the base Gamebryo files to enable all non-PhysX Gamebryo components to build without the need for the PhysX SDK.
