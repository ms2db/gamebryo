Tools Directory Orientation

The Gamebryo codebase contains a large amount of code, and it can be difficult at first to understand how the engine is organized on disk. Before the directory structure of the engine can be explored, it is important to understand how Gamebryo is organized functionally.

Gamebryo's functionality is broken down into a number of subsystems, of which the primary subsystems are Foundation, CoreRuntime, and GameFramework.

Many high-level directories in the engine, such as Build, Samples, and Source, are further broken down by subsystem.

Note

World Builder is the premier world building tool in Gamebryo LightSpeed. Because it is actually constructed from a series of Toolbench plug-ins, it is missing from the list above.

AnimationTool

The Gamebryo Animation Tool augments the art pipeline by providing you with a feature-rich tool that helps you preview, edit, and organize game character animations. With the Animation Tool, you can do the following:

  • Preview individual animation sequences
  • Preview and edit animation sequence transitions
  • Interactively drive character animations to mimic game behavior
  • Mix and match multiple animations
  • Preview animations on different models
  • Generate programmer information for driving the animations in your application

ArtPlugins

This folder contains code for the following exporter plugins:

Directory Contents
MaxPlugin Source to the 3ds max exporter plugin
MayaPlugin Source to the Maya exporter plugin
XSIPlugin Source to the XSI exporter plugin

AssetController

In addition to running alongside Toolbench, the Asset Controller also has several useful features that are accessible by using it as a command-line tool. This usage is handy for scripting functionality into your build system and for some simple testing to troubleshoot problems with locating assets at run-time.

AssetViewer

The Asset Viewer displays and analyzes the performance of NIF files in the Gamebryo engine. A rendered view of the scene is shown, as is a hierarchical view of the underlying scene graph that provides detailed information about any object in the scene.

ChannelServer

The ChannelServer is a simple application that can be run from the command line to provide a NetService compatible Channel Manager that other applications can connect to. When multiple processes need to communicate using the NetService, they do so by connecting to a common Channel Manager Service. In the event that none of the applications actually embed this service themselves, you can use this dedicated ChannelServer application to provide the common connection point.

DDEHeaderGen

This application is used as part of the build processing to generate header files containing unique system id's for items like messages, classes, log modules, and system services. The input is an XML document that enumerates the necessary Id's. The output is a C/C++ header file that is used to define the Id's in the code.

Libs

This folder contains source for the following libraries:

Directory Contents
etkRCS Library that provides base support for Toolbench revision control integration
etkRCSP4 Library that provides support for Toolbench integration with Perforce
etkRCSSVN Library that provides support for Toolbench integration with Subversion

Remote Viewer

The Remote Viewer is a tool that can be launched from Maya to render the particle systems and scene in real-time using the Gamebryo renderer on a console or the PC.

Toolbench

Toolbench is an application platform written from the ground up to be an enterprise-level hosting platform for any and all tools associated with the development of video games. Toolbench is driven purely through the use of plug-ins where plug-ins define their extensibility in a data-driven manner.

ToolPlugins

This folder contains the source code for Gamebryo's tool plug-ins that work with the tool plug-in framework. These plugins can be called from plug-in scripts that can be run as part of the export process or in a stand-alone application.

UpdateArtistTools

This folder contains a script that updates the Gamebryo tools, SDK, and supplemental files from a location on the network.
