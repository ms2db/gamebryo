Gamebryo LightSpeed System Requirements

To install and use Gamebryo LightSpeed, your system should meet or exceed the following specifications. Please note that these requirements will allow the use of prebuilt samples and tools. Users who wish to compile samples, libraries, or their own applications should read Building Gamebryo LightSpeed for more information about required SDK setup for each platform.

  • Hardware
    • RAM—512MB minimum; 2GB recommended
    • Disk space—20GB minimum; 40GB recommended
    • CPU—2GHz minimum; Multi-core or multiple processors at 2GHz or above recommended
    • Graphics card—Shader Model 2.0 (SM2.0) minimum; SM 3.0 with 512MB and above recommended
  • Software (* = included and optionally installed during LightSpeed install)
    • Microsoft .NET Framework 3.5 SP1 *
    • Microsoft Visual C++ 2005 SP1 Redistributable (with ATL Security Update) *
    • Microsoft Visual C++ 2008 SP1 Redistributable (with ATL Security Update) *
    • Microsoft DirectX Runtime Libraries (February 2010) *
    • NVIDIA PhysX System Software 9.10.0129 (for samples requiring PhysX. GPU acceleration of PhysX requires an appropriate NVIDIA GeForce graphics card with drivers later than v177.81) *
    • Autodesk 3ds Max or Maya or SoftImage XSI (for artists)

Version support for the digital content creation tools is outlined in the following:
