Samples Directory Orientation

The Samples directory contains game and graphics demos, as well as sample libraries and tutorials.

Samples Directory

Directory Contents
GameDemos Significant samples that show how Gamebryo can be used to build a full game.
GameFrameworkTechDemos Tech demos that demonstrate specific aspects of the GameFramework subsystem such as using System Services and Entities.
GraphicsTechDemos Tech demos that demonstrate specific aspects of the CoreRuntime subsystem.
Models Sample art assets that show off how artists can create specific effects in Gamebryo.
OnlineTechDemos Tech demos that demonstrate specific aspects of the Online subsystem.
SampleLibs Sample libraries that demonstrate how customers can write plug-ins to Gamebryo art tools.
Tutorials Series of step-by-step sample applications that introduce Gamebryo Core Runtime functionality to new customers.

GameDemos

These sample games can serve as excellent starting points for larger customer applications and games. These are samples that use all parts of Gamebryo in an actual game-like environment. They are intended to demonstrate how various systems in Gamebryo work together, rather than how a specific system can be used. Full source and artist files are included with these samples.

Directory Description
DodgeBall Demonstrates the use of physics rag-doll and blending in a light-hearted game environment. Also shows Gamebryo shadows.
Lenguins on Ice Demonstrates high end rendering effects and large scale physical interactions
MangledMetal Demonstrates how the GameFramework subsystem can be used to build a game.
Metal Wars Demonstrates a rich, "third person shooter"-like game environment and shows many advanced features of Gamebryo.
MOUT Military Operations in Urbanized Terrain is a first-person perspective application that demonstrates the use of Gamebryo for simulation and training.

GameFrameworkTechDemos

The GameFrameworkTechDemos directory contains a number of samples that demonstrate how to use the Game Framework subsystem.

Directory Description
HelloWorld Demonstrates the basic concepts and components of the Gamebryo Game Framework.
BaseGame Demonstrates how to create a small game starting from the HelloWorld sample.
CustomTypeExample1 Demonstrates how to pass a user-defined custom data type to and from built-in APIs in a scripting language using SWIG generated bindings.
CustomTypeExample2 Demonstrates how to extend the custom data type bound to scripting in CustomTypeExample1 and use it as an Entity property type.

GraphicsTechDemos

The GraphicsTechDemos directory contains a number of samples that show off different areas of CoreRuntime functionality, often in isolation. For more information on how this directory is organized, see Graphics Tech Demos Directory Orientation page.

Models

The Models directory contains a number of art assets that can be used with Gamebryo. They demonstrate various techniques artists can use when creating their own assets.

OnlineTechDemos

The OnlineTechDemos directory contains a number of samples that demonstrate how to use the Game Framework subsystem.

Directory Description
efdNetLibEnet Provides reliable UDP networking communication on Windows.
efdNetLibPS3UDPP2P Provides network communication on Playstation 3.
efdNetLibXbox360XRNM Provides network communication on Xbox 360 using the XRNM library.
MatchmakingSample Demonstrates basic matchmaking functionality on the console platforms.
OnlineSession Contains the XSessionSample which demonstrates the management of an Xbox Live session.
PeerGame This is an extension of BaseGame to add multiplayer support.

SampleLibs

The SampleLibs folder contains examples of how developers can write plug-ins to some of Gamebryo's tools.

Directory Description
SamplePluginsDLL Demonstrates how to use NiPluginToolkit to create your own custom tool plug-ins, which can then be scripted and inserted into the art pipeline.

Tutorials

The Tutorial tracks provide step-by-step introductions to using Gamebryo. There are two programming tracks in this directory, each in its own subdirectory

Directory Description
Graphics Illustrates the use of a simple screen texture.
PhysX Covers the creation and display of a rendered texture.
