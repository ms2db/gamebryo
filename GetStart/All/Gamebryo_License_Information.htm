Gamebryo LightSpeed License Information

Binary/Partner/Academic/Evaluation Customers

Binary, partner, academic, and evaluation customers will be prompted for a valid license code during the product install. Binary installers include all of the Gamebryo LightSpeed libraries and tools, but they do not include source code.

The installer sets the license code in the NDLDEVLOCK environment variable. The license code is also set for all licensed platforms in the following file:

%EMERGENT_PATH%\SDK\<platform>\Include\NiLicense.h

If you need to change your license code, you can manually modify the NDLDEVLOCK environment variable and all NiLicense.h files. Alternatively, you can use Emergent's license update utility, which is installed by default in the following location:

%EMERGENT_PATH%\update_license.exe

This command-line utility automatically updates the environment variable and all of the NiLicense.h files to contain your new license code. Usage takes the following form:

%EMERGENT_PATH%\update_license.exe 12345678ABCDEFGH %EMERGENT_PATH%\sdk

where 12345678ABCDEFGH is your new license code.

All applications that are built from a LightSpeed binary installation must contain the following source code to correctly link:

#include <NiLicense.h>
NiEmbedGamebryoLicenseCode;

For academic, partner, and evaluation licenses, statically linked executables will have a small watermark in the lower right corner of the screen. Executables with a watermark will continue to run past the expiration of the license code. Binary licenses do not display a watermark.

For binary LightSpeed installations, for all license types, dynamically linked executables (such as tools or exporter plug-ins) do not have a watermark, but they will no longer run when the license code has expired. The Gamebryo tools will expire on the date provided to you by the sales team (sales@gamebryo.com).

Source Customers

Source customers are not required to embed a license code in NiLicense.h. A valid license in the NDLDEVLOCK environment variable is also not required. The above code for embedding a LightSpeed license code will compile and execute safely, but it is not necessary.
