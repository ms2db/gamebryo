Documentation Overview

The documentation that ships with Gamebryo LightSpeed is broken into the following major sections:

  • The Getting Started section contains basic orientation information such as product software requirements, how things are laid out on disk, and where to look for additional information.
  • The Release Notes include platform-, tool-, subsystem-, and library-specific changes and known issues.
  • The Conversion section contains information for customers who are upgrading from a previous product version.
  • The Learning Gamebryo section introduces the product's principal features and includes guided tutorials and demos to get you familiarized with LightSpeed.
  • The Artist section contains information about using exporters for digital content creation tools.
  • The Programmer section includes information for application developers.
  • The Tools section contains information about the artist, designer, and developer tools that are supplied with LightSpeed.
  • The Quick Reference section contains additional high level info on LightSpeed configuration options, models, and system services.
  • The Reference section contains API-level class reference documentation.

When you install LightSpeed, a shortcut to the documentation is created on the Windows Start Menu. We recommend that you view the documentation in Microsoft Internet Explorer 7.x or 8.x or Mozilla Firefox 3.x.

Alternately, you can open the documentation directly from the file system by double-clicking the following file:

%EMERGENT_PATH%\Documentation\HTML\documentation.htm

Note

Documentation is tested in the recommended browsers on various versions of Microsoft Windows products. Please contact us if you notice issues with other browser and/or operating system configurations.
