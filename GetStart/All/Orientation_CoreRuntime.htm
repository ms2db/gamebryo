CoreRuntime Directory Orientation

CoreRuntime Directory

Directory Contents
AppFrameworks Source code for the NiApplication and NiSample application framework libraries.
EngineLibs Source code for all of the standard runtime Gamebryo libraries.
PhysXLibs Source code for all of the core Gamebryo-PhysX integration libraries.
ShaderLibs Source code for all of Gamebryo shader libraries.
ToolLibs Source code for all Gamebryo tool libraries.
UtilityLibs Source code for all of the supplemental runtime Gamebryo libraries.

AppFrameworks

This folder contains the source code for the NiApplication and NiSample application frameworks.

Directory Contents
NiApplication Basic cross-platform application layer.
NiSample More functional cross-platform application layer.

EngineLibs

The EngineLibs folder contains the source code for the runtime libraries in the CoreRuntime subsystem. These libraries handle the more fundamental functionality for this subsystem, such as scene graph management, animation, and rendering.

Directory Contents
ecrD3D11Renderer Rendering library for Win32 that uses the D3D11 interface
NiAnimation Animation library
NiCollision Collision detection and picking
NiD3D10Renderer Rendering library for Win32 that uses the D3D10 interface
NiDX9Renderer Rendering library for Win32 that uses the D3D9 interface
NiEnvironment Environment library, including sky and time of day systems
NiFloodgate Cross-platform stream processing library
NiMain Main scene graph manipulation library
NiMesh Library for managing representation of geometry data
NiParticle Particle simulation library
NiPortal Portal system library
NiPS3Renderer Rendering library for SP3
NiSystem Thin cross-platform library for accessing platform-specific services
NiTerrain Terrain library
NiXenonRenderer Rendering library for Xbox360

PhysXLibs

The PhysXLibs folder contains the source code for the PhysX integration libraries.

Directory Contents
NiPhysX Main PhysX integration library
NiPhysXFluid Integration library for PhysX fluids
NiPhysXParticle Integration library for PhysX particles
NiTerrainPhysX Library to integrate PhysX with NiTerrain

ShaderLibs

The ShaderLibs directory contains libraries that enable Gamebryo's use of shaders. These libraries contain NiShaderLibrary objects and NiShaderParser objects that allow the use of NSF and FX based shaders.

Directory Contents
ecrD3D11BinaryShaderLib Implementation of NSB shaders for D3D11.
ecrD3D11EffectShaderLib NiShaderLibrary for loading FX shaders for D3D11.
NiBinaryShaderLib Implementation of NSB shaders for DX9 and Xbox360.
NiCgShaderLib Library that enables Cg shader programs for DX9.
NiCommonMaterialLibrary Library providing a number of common materials to applications and tools.
NiD3D10BinaryShaderLib Implementation of NSB shaders for D3D10.
NiD3D10EffectShaderLib NiShaderLibrary for loading FX shaders for D3D10.
NiD3DXEffectShaderLib NiShaderLibrary for loading FX shaders for DX9 and Xbox360.
NiFXLShaderLib NiShaderLibrary for loading FX shaders for Xbox360 more efficiently.
NiMaterialNodeXMLLibraryReader Library for parsing Material Node XML files.
NiPS3BinaryShaderLib Implementation of NSB shaders for PS3.
NSBShaderLib NiShaderLibrary for loading NSB files
NSFParserLib NiShaderParser for reading NSF files

ToolLibs

The ToolLibs directory contains source code for libraries that can be used for:

  • Processing during content generation and export from plug-ins
  • Optimization
  • Profiling
  • Conversion

These libraries are not optimized for runtime performance, so they should generally be used in performance-critical applications.

Directory Contents
NiAnimationCompression Library for compressing animation data
NiControllerExtractor Library to extract animation data into KF files
NiDevImageQuantizer Library for creating palettized textures
NiFBXConverter Library for exporting to FBX files.
NiHDRReader Library for reading Radiance HDR textures
NiMaterialToolkit Helper library for loading common material and shader files
NiMeshProfileProcessor Library for managing mesh profiles in the tool plug-in pipeline
NiMeshProfileXMLParser Library for parsing mesh profile XML files
NiMeshTools Library of static helper functions for managing meshes in the tools
NiNBTGenerator Library for generating normal/binormal/tangent frames
NiOpenEXRReader Library for reading OpenEXR textures
NiOptimization Library of scene graph optimization functions
NiPhysXSharedData Library of classes for holding shared PhysX-related data for use with NiPluginToolkit
NiPhysXTools Library of helper functions for managing PhysX content in the tools
NiPluginToolkit Library for implementing the plug-in pipeline
NiPNGReader Library for reading PNG textures
NiProgramLauncher Library for launching applications on local PC or console dev kits.
NiPS3ProgramLauncher Library that implements NiProgramLauncher on PS3.
NiStandardSharedData Library of classes for holding shared data for use with NiPluginToolkit
NiStripify Library for optimizing geometry data
NiViewMath Library for manipulating the camera
NiXbox360ImageTiler Library for tiling images for use on the Xbox360.
NiXbox360ProgramLauncher Library that implements NiProgramLauncher on Xbox360.

UtilityLibs

The UtilityLibs folder contains libraries that provide supplemental features, such as audio, video, font, and input handling. These libraries are not a required part of the engine.

Directory Contents
ecrD3D11RendererSetup Library that interacts with NiSettingsDialog to handle construction and customization of an ecrD3D11Renderer.
NiAudio Library for integrating Miles audio
NiCursor Library for displaying a cursor
NiD3D10RendererSetup Library that interacts with NiSettingsDialog to handle construction and customization of an NiD3D10Renderer.
NiDX9RendererSetup Library that interacts with NiSettingsDialog to handle construction and customization of an NiDX9Renderer.
NiFont Library for creating fonts and displaying text on screen
NiInput Cross-platform library for handling input
NiMetricsOutput Library that implements modules for outputting metrics data
NiSceneGraphUpdateService Scene graph synchronization library used by Rapid Particle Iteration.
NiSettingsDialog Library for displaying a dialog at startup to allow users to adjust renderer-creation settings on Win32
NiUserInterface Library for displaying and interacting with UI elements in screen
NiVideo Library for integrating Bink video
NiVisualTracker Library for displaying performance graphics on screen
