Gamebryo PhysX Integration Overview

Gamebryo is integrated with the PhysX package from Nvidia, and the integration tools and code ship with the core Gamebryo distribution. Here we provide links to related documentation.

  • Demos of Gamebryo-PhysX include MangledMetal, other Game Demos, and technical samples.
  • Source License Issues discusses the details for licensing third-party code for use with Gamebryo, including PhysX.
  • NVIDIA PhysX Resources provides a set of links for obtaining the PhysX SDK and obtaining PhysX-specific support.
  • The Introduction to Gamebryo PhysX Max Support and Introduction to Gamebryo PhysX Maya Support pages are the starting point for artists seeking to create PhysX content.
  • The Introduction to Gamebryo PhysX Programming is the starting point for programmers working with PhysX Core Runtime.
  • The PhysX System Service Intro is the starting point for programmers working with PhysX in the Game Framework.
