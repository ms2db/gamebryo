Information for Artists and Designers

For artists and designers, the suggested reading list is:

  1. 3ds max Plug-in, Maya Plug-in, or XSI Plug-in documentation: Artists who will be creating content for use with Gamebryo will want to read these sections. They include samples and details on using the major digital content creation tools to produce scenes that maximize Gamebryo efficiency.
  2. Entity Modeling Tool: Designers who are creating and editing game objects should learn about this powerful tool.
  3. World Builder: Designers or artists who are responsible for world or level layout, editing, and tuning should refer to this documentation.
  4. Asset Viewer: Artists who want to use Gamebryo's asset previewing tool should consult this documentation.
  5. Animation Tool: Artists who are interested in using Gamebryo's Animation Tool to set up character animation sequences should consult this documentation.
  6. Game Demos: Examples of both basic and advanced Gamebryo techniques:
    • MangledMetal: A Game Framework-based demo that takes place in a world of giant steam-powered mechanical beings called Warjacks, from Privateer Press's table-top role-playing game, Warmachine™. You control the actions of a mechanical being and are able to interact with the world via physical actions (using NVIDIA PhysX™) and combat (moderated by Lua scripts). It is a full-featured, robust application that demonstrates many of the features and best practices of the Emergent Game Framework.
    • Hello World: A tutorial that demonstrates some of the key features of LightSpeed's entity modeling and runtime simulation systems.
    • Base Game: A tutorial that extends the Hello World tutorial. This tutorial highlights the use of PhysX triggers and Lua behavior scripts to add game play dynamics quickly to a basic application.
    • MetalWars: MetalWars demonstrates an array of advanced shader and animation blending techniques. Full source and artist files are included with the sample, which can serve as an excellent starting point for larger customer titles.
    • Lenguins on Ice (requires PhysX to run): This sample demonstrates PhysX by smashing little birds around with giant ice cubes. The demonstration is fully playable and also demonstrates several interesting Gamebryo shader techniques.
    • DodgeBall (requires PhysX to run): Demonstrates the use of physics rag-doll and blending in a light-hearted game environment. Also shows Gamebryo shadows.
