Online Overview

Online takes Gamebryo LightSpeed into the realm of multi-platform, multiplayer online games. It builds upon previous releases to simplify the creation of games with an online component.

The Networking system extends LightSpeed's messaging system to support multiple processes on separate machines. Each platform has specific network libraries, called NetLibs for short, that plug into the NetworkService to provide custom support for platform-specific networking features from a single cross-platform API.

  • PC Networking
  • Playstation 3 Networking
  • Xbox 360 Networking

The Entity Replication system simplifies the task of synchronizing entity data between multiple processes. Using a publish-subscribe model, you can ensure that the correct data is sent to every process that needs the data.

Samples and Tutorials

This release includes several examples of online technologies for use as starting points for learning about these features. These include:

  • Peer Game Demo and Tutorial—intended to be a bare bones example of a working peer-to-peer game. It demonstrates functionality from the BaseGame demo, reimplemented to support multiple players, including support for rapid iteration. The accompanying tutorial walks you through all of the code changes required to convert Base Game into the online-enabled Peer Game.
  • Multiplayer Rapid Iteration Tutorial—demonstrates how you can rapidly iterate on multiplayer games, just as you can on standalone games. This tutorial walks you through some examples of rapidly iterating on Peer Game with multiple clients connected.
  • Matchmaking Sample—We provide NetLib implementations for all console platforms that include basic integration with the matchmaking services offered by that platform. Each of the console platforms provides slightly different matchmaking functionality. Platform-specific samples demonstrate the integration of LightSpeed networking with the matchmaking functionality of that platform.
