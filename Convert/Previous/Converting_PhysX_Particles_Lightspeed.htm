Converting PhysX Particle to Gamebryo LightSpeed 3.0

The APIs for PhysX-based particle and fluid systems have changed in Gamebryo LightSpeed 3.0 to adapt to changes in Gamebryo particle systems. In general, particle systems created in DCC tools and loaded from file will not require any conversion. However, changes must be made to convert particle systems created in code.

The following classes have changed:

  • NiPhysXPSMeshParticleSystem: The Create function has a new set of arguments that control new particle system features.
  • NiPhysXPSParticleSystem: The Create function has a new set of arguments that control new particle system features.
  • NiPhysXPSFluidEmitter: The constructor has a reduced set of arguments to reflect movement of some code from emitters to the particle systems themselves.
  • NiPhysXPSFluidSystem: The Create function has a new set of arguments that control new particle system features.
  • NiPhysXPSMeshFluidSystem: The Create function has a new set of arguments that control new particle system features.

Note that classes from NiParticle that are used to construct PhysX-based systems might also have changed. See the NiParticle Conversion documentation for details.
