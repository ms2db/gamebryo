Texture Conversion Custom Attributes

These custom attributes have been designed to assist developers in setting up textures for export out of 3ds max. There are a variety of options for texture conversion, including downsampling the image and reformatting the way the pixels are laid out. These attributes can be assigned by using the Gamebryo toolbar icons discussed in the Gamebryo MaxScript Toolbar Introduction.

Texture Export Parameters

Selected Platform

This option lets the artist toggle the pixel quality settings on a per-platform basis.

Use Current Platform for All

This option forces the pixel quality settings to be the same on all platforms. The selection is limited to the quality options that apply to all platforms. Platform-specific options cannot be selected when this is set.

Export Mode

Use Global Settings defers the export mode to the setting on the export dialog. The remaining three settings will place the current texture map in the NIF file, as a separate external NIF file, force Gamebryo to load the original image, or use original images loaded directly into the renderer. These options are described in more detail in the Exporting Textures documentation.

Filtering

This sub-panel allows you to set the way in which mip-mapping and filtering occurs when rendering this texture. This value will override the Pyramidal\ Summed Area\ None values in the Bitmap rollout.

To achieve the same result as Pyramidal, set the Filter Mode to TriLerp. To achieve the same result as Summed Area, set the Filter Mode to BiLerp. To achieve the same result as None set the Filter Mode to Nearest. Max Anisotropy has no effect when filtering is not set to Anisotropic. For more information on precisely what this means, go to the Texture Pixel Interpolation and Filtering section.

Final Pixel Quality

Image Default uses whatever pixel layout the image has on disk. Use Global Settings uses the options set on the export dialog. Other options are dependent on the final export platform. For instance, 16 bit textures on Win32 may have a slightly different pixel layout than 16 bit textures on other platforms.

Note

It is important to remember this is only a preferred format. Not all formats are supported on all platforms and during the export process the destination renderer will determine the final destination format.

Final Texture Size

This option essentially chooses the appropriate level in the mip-map pyramid and makes it the uppermost mip-map level. This can result in a blurry texture, but can substantially help in reduction of memory usage.
