Helper Objects and 3ds Max

While Gamebryo does not directly export out the parameters of many of the standard Helper objects in Max, it will by default export out an NiNode with the Helper object's position, rotation, and scale. This node will have the same name as the Helper object, but no children. This is often useful for adding attachment points or frames of reference in your scene graph.

The Gamebryo Optimization plug-in, part of the "Default Process Script" will automatically remove all of these Helper NiNodes unless the "NiOptimizeKeep" tag is specified in the User Defined Properties of the Helper object. Alternatively, the NiOptimization toolbar utility can be used to set the tag.
