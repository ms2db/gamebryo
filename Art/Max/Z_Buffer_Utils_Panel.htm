Z-Buffer Utils Panel

This panel allows quick configuration of the z-buffer properties of objects in the scene graph. This property determines whether or not the object reads and/or writes to the z-buffer.

Z-Buffer Utils panel

Read From Z-Buffer: This option toggles whether or not the object reads from the z-buffer.

Write To Z-Buffer: This option toggles whether or not the object writes to the z-buffer.

Apply To Selection: This button applies the current state of Read From Z-Buffer and Write To Z-Buffer to the currently selected objects in the scene.

Select Matching Objects: This button selects the objects in the scene graph matching the current state of the Read From Z-Buffer and Write To Z-Buffer check boxes.

The utility operates by adding or removing text from the user-defined properties of the object. The user defined properties text buffer for an object is manipulated to contain one of the following four strings:

String Description
zMode00 Z-buffer Test and Write off
zMode01 Z-buffer Test off, Write on
zMode10 Z-buffer Test on, Write off
zMode11 Z-buffer Test and Write on (default)
