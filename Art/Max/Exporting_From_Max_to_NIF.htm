Exporting From 3ds Max to NIF

You can use the Gamebryo 3ds Max PLug-in to export your Max art assets to Gamebryo's NIF format.

To export a Max art asset to NIF

  1. Make sure that the Gamebryo Maxscript Toolbar is loaded and available in 3ds Max.

    See Gamebryo MaxScript Toolbar Introduction.

  2. In 3ds Max, create a teapot (or any other object). You should now have something similar to the image below.
  3. Click the Export to Gamebryo button (see image below).
  4. In the Export File dialog box, type the name of the file that you want to save and then click Save. (If you've exported previously, the application will remember the last file name used. In this case, we chose "Teapot.nif".)
  5. In the Export Scene to NIF File dialog box, click OK to export with the default settings.

    See Gamebryo 3ds Max Export Options.

After the file is exported, you can view it with the Gamebryo Asset Viewer. If you need to tweak the art asset in Max, you can do so. When you export to NIF again, the settings that you chose in the first export will be remembered. If you save the Max file after an export, it will save your last export settings in the file itself.
