Particle Color and Opacity Map

The "Particle Color" and "Particle Opacity" fields should only be used for materials that are applied to a particle system that uses Standard (or sprite-based) particles. Both fields will only accept a "Particle Age" material and together allow particles to change color and opacity as they age. For "Particle Opacity" the r,g,b values in the "Particle Age" material will be averaged together to yield the required alpha value.

If you are using Instanced Geometry (or mesh-based) particles, the material applied to the source particle geometry will be used for the particles. You can animate any value on that material, and each particle will be updated with its age at runtime, achieving the same effect as the "Particle Color" and "Particle Opacity" slots for sprite-based particles.
