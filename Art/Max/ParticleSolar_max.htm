ParticleSolar.max

The ParticleSolar.max sample uses a Max Super Spray particle system to create a glowing effect. The particles do not have any velocity, and the effect is accomplished through the particle's 'Grow For' parameter and rotation settings.

The material applied to the particle system uses a radial monochrome texture. It gets its coloring through vertex colors. The vertex colors change over the lifetime of each particle according to the Particle Age map in the material's Particle Color map slot. The vertex colors are set to E for emissive and alpha blending is set to Additive. Because additive alpha blending is used, the Particle Age map uses black as the color at times when particles should fade out.

The scene also contains hidden geometry that can be used for picking or collision detection.
