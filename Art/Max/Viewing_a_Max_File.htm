Previewing a 3ds Max File

You can use the Gamebryo 3ds Max Plug-in to preview your Max assets. This enables you to quickly determine how the asset will look in Gamebryo's default viewer or a target platform viewer.

To preview a Max file

  1. Make sure that the Gamebryo Maxscript Toolbar is loaded and available in 3ds Max.

    See Gamebryo MaxScript Toolbar Introduction.

  2. In 3ds Max, create a teapot (or any other object). You should now have something similar to the image below.
  3. Click the Quick Preview toolbar button indicated in the picture below.

    The scene is converted into the Gamebryo format. It then loads in a separate window using the default viewer or a platform-specific viewer that you might have configured previously.

  4. You can manipulate the asset within the viewer by using several different keyboard and mouse commands. The left mouse click or UP, DOWN, LEFT, and RIGHT keyboard buttons move the camera forward, backward, left, and right. If you press and hold down the CTRL key while you click the left mouse button or any of the previously mentioned keyboard keys, you can orbit the camera around the origin of the world.
  5. Optionally, you can change the viewer's background color (Rendering > Environment). In the Environment and Effects dialog, select the color that you want to use for the background.
  6. Optionally, you can change which viewer is launched when you click the Gamebryo Default Viewer button. This button will launch the viewer that is selected in the Gamebryo Max Plug-in Viewer Selection panel. Simply select the viewer that you want to use as the default. The Gamebryo Max Plug-in will use this setting across sessions.
