ProjectedAndAttenuatedLights.max

The ProjectedAndAttenuatedLights.max sample is an example of a texture projected from a light and a light with attenuation. Both are animated. The projected spotgray.tga texture is loaded through the light channel on the Lights modifier panel, under the "Directional Parameters, Projector Map" section in Max.

This sample shows how to use a projected light map with normal dynamic lighting. It consists of three objects (Box01, Light Projector Marker, and Point Light Marker) two lights (Omni01 and Projected Light Map), and two paths (Circle01 and Circle02).

Box01 is the geometry upon which all the lights are cast. Light Projector Marker and Point Light Marker mark the position of, respectively, the "Projected Light Map" light and the Omni01 light. As its name suggests, the "Projected Light Map" light is used to create a project light map. The Omni01 is a simple dynamic light. The "Projected Light Map" light uses an inclusion list so that it will affect only Box01.

Finally, "Projected Light Map" is attached via a path controller to Circle01. Circle02 path animates Omni01. When run, this sample demonstrates the simultaneous effect of both the project light and the dynamic point light.
