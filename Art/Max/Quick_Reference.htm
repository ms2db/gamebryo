Keyframe Manager Note Tracks

Track Description
start Starting key in animation note track
end Ending key in animation note track
-name mysequencename Name your sequence
-loop Set Biped character animations to loop
-at [all | none | {xyz}] Accumulate translations, all, none, or any combination of x,y, and/or z
-ar [all | none | x | y | z ] Accumulate rotations along: all axes, no axis, or one—either the x, y, or z axis
-smooth Make accumulated rotations smooth.
morph: mymorphtargetname Indicates instant, e.g. "knees together", at which to morph between animations
myblendtargetname Indicates instant, e.g. "knees together", at which to blend between animations
-GlobalRatio [1-100] The default animation compression ratio is for the actor or layer root containing this key
-GlobalCompressFloats [true | false] Indicates whether to compress floats in animation keys to unsigned shorts.
-GlobalDontCompress [true | false] Indicates that the animations for this actor or layer root should not be compressed at all.
-SeqRatio [1-100] The animation compression ratio for all child objects during the sequence defined by the same key
-SeqCompressFloats [true | false] Whether to compress the animation key floats to unsigned shorts for the sequence defined by the same key
-SeqDontCompress [true | false] Indicates if all animations in the current sequence should not be compressed at all
-ObjRatio [1-100] The animation compression ratio for the object for all sequences. This key can be found on objects that are not actor or layer roots
-ObjCompressFloats [true | false] Whether to compress the floats in animation keys for this object. This key can be found on objects that are not actor or layer roots
-ObjDontCompress [true | false] Indicates if the object's animation keys should not be compressed at all. This key can be found on objects that are not actor or layer roots
-[sequence name]Ratio [1-100] Indicates that for a particular sequence, the object that owns this key should use this compression ratio. This key can be found on objects that are not actor or layer roots
-[sequence]CompressFloats [true | false] Indicates that for a particular sequence, the object that owns this key should not compress floats in its animation keys. This key can be found on objects that are not actor or layer roots
-[sequence]DontCompress [true | false] Indicates that for a particular sequence, the object that owns this key should not be compressed at all. This key can be found on objects that are not actor or layer roots
-[sequence]Override [true | false] Indicates that when the object compression settings are changed, that the sequence settings for this object should not be cleared because the per-object per-sequence data was intentionally set

Transparency Considerations

Issue: Object has completely opaque or completely transparent regions only

What you should use: Alpha testing

Issue: Object has regions that are semi-transparent

What you should do: Alpha blending

Issue: Object has some semi-transparent or opaque regions, and some completely transparent

What you should do: Alpha testing with alpha blending
