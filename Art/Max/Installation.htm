3ds Max Plug-in Installation

When you install Gamebryo, the installer identifies the version of Max on the target host. By default, the installer will install the plug-in for any supported version of Max that it finds. So, if you have more than one version of Max installed, you should verify that the target Max paths are correct if you want to install the plug-in for each Max version. If you do not want to install the plug-in for a specific version of Max, deselect that version during the install.

After you install the 3ds Max Plug-in, you can set up the Gamebryo MaxScript toolbar.

Gamebryo MaxScript toolbar

This toolbar provides quick access to a variety of Gamebryo utilities. For more information about installing and using this toolbar, see Gamebryo MaxScript Toolbar Introduction.

If you plan to create PhysX content for use with Gamebryo, you must also install the PhysX Max plug-in. This plug-in can be found in the following location:

%EMERGENT_PATH%\Installers\NVIDIA_PhysX_Plugin_for_3ds_Max_1.0.6.exe

Build Considerations on Windows Vista

To build the plug-in on Windows Vista, you need to be logged on as a user with administrator-level permissions. However, even with these permissions, it is possible that you might have to modify the security settings on the Max folder to allow user access. You will have to make these modifications if you see a large number of "Access denied" errors during the post-build step of the plug-in itself. These errors will occur if the post-build scripts cannot copy Gamebryo scripts and plug-ins into the installation folder.

To modify the security settings, open Windows Explorer, right-click the top-level folder of each Max installation, and select (click) Properties. On the Security tab, click the Edit button. In the Group or user names window, click the Users entry and then click the Modify checkbox under the Allow column. Click OK. Click OK again to apply this change.
