Billboard Utilities Panel

This panel allows quick configuration of the billboard settings of a set of objects.

Billboard Utilities panel

Setting Description
None Object is not billboarded in any way
Standard Object follows camera, min difference between up and Y
Up Object follows camera, keeps up coordinates up in world
Rigid Object appears locked in camera screen space
Center Object points directly at camera, min. difference between up and Y
RigidCenter Object points directly at camera, appears locked in camera screen space
Apply To Selection Applies the current state of the Billboard Modes radio buttons to the currently selected objects in the scene.
Select Matching Objects Selects the objects in the scene graph matching the current state of the Billboard Modes radio buttons.
