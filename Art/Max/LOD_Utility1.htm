LOD Utility

This MAXScript utility was designed to replace the 3ds Max LOD utility for Gamebryo users. The settings for this utility are much more intuitive and match Gamebryo implementation.

LOD Utility

Field Description
Refresh LOD Sets Updates the entire LOD utility plug-in with the contents of the 3ds max scene. This step is required if you rename objects or destroy objects.
Recenter Objects when they are added to the set Tells the plug-in whether or not to move all objects in a LOD set to a common origin. The objects are moved according to their pivot.
Set Name Dropdown list that shows all of the known LOD sets in the current Max scene graph. If you delete or manipulate the User Property buffer of an item in the set, please press the "Refresh LOD Sets" button.
LOD List Displays all of the LOD levels for this particular LOD Set. The list is sorted by smallest to largest distance from the camera. The value in parentheses is the distance from the camera in world units that this object first becomes visible. The object disappears from view when the next object in the list becomes visible.
Show All Shows all the items in the LOD list. These items are all made visible.
Show Active Only Shows only the selected LOD in the LOD list. This item is made visible, but all others in the LOD list are hidden.
Match Active Viewport Shows the LOD that should be visible in the currently active viewport. Perspective and camera views will produce accurate results. The object is updated on a regular basis, but rapid movement will not produce valid switches in LOD until the utility has a chance to update. Orthographic views like Top, Bottom, Left, Right, Front, and Back do not work with the LOD preview since they have no real perspective on the object.
Add Selection To LODs Takes the currently selected objects and adds them to the current LOD set.
Remove Selection From Set Removes the current LOD selected in the LOD list from the LOD list.
Make Sel. Child of LOD Attaches the currently selected objects as a child of the current LOD level selected in the LOD list.
LOD Name This name is the name of the currently selected LOD level in the LOD list.
Start Distance This value is the distance at which the current LOD becomes visible.
Use Distance From Viewport This button sets the start distance on the current LOD level to match the object's distance from the current viewport camera.
