Supported Materials

Supported Material (Shader) Type Description
Gamebryo Shader This Shader was designed specifically to take advantage of all of the features in Gamebryo. This Shader type is the preferred one for most Gamebryo artists.
Blinn Ambient, diffuse, specularity, self-illumination, and opacity are all supported. The specular highlights panel is not supported.
Phong Ambient, diffuse, specularity, self-illumination, and opacity are all supported. The specular highlights panel is not supported.
Oren-Nayar-Blinn Ambient, diffuse, specularity, self-illumination, and opacity are all supported. Advanced diffuse and the specular highlights panel are not supported.

Gamebryo supports wire and 2-sided modes on all materials. The Face Map and faceted options are not supported under the Shader Basic Parameters rollout.

2-Sided Materials

Since Gamebryo does not create two-sided triangles, the Gamebryo 3ds max Plug-in must perform a trick on all objects that use Max's "2-Sided" feature.

The fastest way to draw a double-sided face involves turning off back face culling for that object. This will allow the object to appear double sided. However, the object will only be lit from the side that has the forward facing polygons. The other side will duplicate the lighting on the opposite side. If this is a problem, create two unique but identical objects with opposite facing normals and merge them into one geometry object.

DirectX Shader Material

The Gamebryo Max Plug-in supports DirectX Shader materials, but significant restrictions apply to the types of FX Shaders that can be converted. For more information on this topic, read DirectX Shader in Max.
