Tutorial: Applying Pixel and Vertex Shaders in 3ds Max

Series: Max Introduction

Audience: Artists new to Gamebryo

The Gamebryo Shader allows developers to assign customized pixel and vertex shaders to objects on the PC, PS3, and Xbox 360. Each shader requires a certain level of graphics hardware to function properly. In the case of this tutorial, make sure that your video card is capable of doing 1.1 pixel shaders and 1.1 vertex shaders.

1. Follow the directions in the Gamebryo Shader tutorial to apply a Gamebryo Shader to an object. In this case, we'll re-use the teapot created in the Gamebryo Shader tutorial.

2. Open the Max Material Browser. Go to the maps rollout in the Gamebryo Shader that you added to the teapot in step 1. Disable every map except for the Base map by un-checking the box next to the map slot.

3. Let's preview the current teapot to see where we're starting from. If you haven't already installed the Gamebryo toolbars, follow the Gamebryo Max Plug-in Toolbar installation instructions. You should install the Asset Viewer if you haven't already. If you do install the Asset Viewer, save the Max file, restart Max, and then load the file. Click on the "NIF Viewer" icon displayed below.

4. You should now see the teapot with the brick texture in the Asset Viewer.

5. Go back into the Max Material Browser. Scroll until the "Gamebryo Shader" rollout is visible. Towards the bottom of this rollout, there is a sub-panel titled "Custom Shader". This is where pixel and vertex shader functionality can be accessed. The sub-panel should match the one below.

6. Click in the "Selected Shader" drop-down list. Select "ToonShadingWithOutline". This shader uses pixel and vertex shader 1.1 technology to do a simple toon shading effect with a simple colored outline. Once you have selected the shader, your "Custom Shader" panel should look like the one below. If you are interested, click on the "More Info" button for additional details about the shader. Click on the "Apply" button in the "Custom Shader" sub-panel.

7. Scroll down to the bottom of your Material Browser panel. A set of custom attributes has been added for the pixel and vertex shader that you just applied. These options control various tweakable parameters of the shader, including images and colors.

8. Click on the "NIF Viewer" icon that brought up the Asset Viewer in step 3. You should now see a teapot with the brick texture applied and a toon shading effect. If you zoom in, you will notice that there is also a very thin black outline around the teapot.

You have now finished the Pixel and Vertex Shader tutorial. For more information on the Pixel and Vertex Shader system, please refer to the following documentation.
