Decalmap.max

This 3ds Max sample adds a crack to the wall by using the decal.tga texture map in the Decal Map slot of the Gamebryo Shader. The first explicit UV channel is reused because this decal should only tile once across the entire polygon. By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\decalmap.max
