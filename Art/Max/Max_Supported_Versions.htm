Max Plug-in: Supported Versions

Each release of Gamebryo is guaranteed to support, at a minimum, the three versions of 3ds Max that are available at the time the release ships. This Gamebryo release includes support for the following 32-bit versions of 3ds Max:

  • 3ds Max 2008
  • 3ds Max 2009
  • 3ds Max 2010

If you plan to create PhysX content for use with Gamebryo, you must also install the PhysX Max plug-in. Gamebryo 3.2 is intended for use with a pre-release PhysX Max plug-in, found in the installation media at PhysX\NVIDIA_PhysX_Plug-in_3dsMax_x86_WithAPEX_2.40.0421.1156.msi. Note the license agreement found in the same directory.

The PhysX Max plug-in version 2.40.0421.1156 does not expose all of the data needed for Gamebryo to export PhysX content. An additional plugin, named fp_basics, is included and must be installed in the Max plugins directory. See the readme in the installer PhysX directory.

The most recent Max plug-in does not support cloth authoring, but it does support loading and exporting cloth created in previous Max PhysX plug-in versions. We recommend installing the previous plug-in for an older version of Max, say the NVIDIA_PhysX_Plugin_for_Max_1.0.6.exe for Max 2008. See the page PhysX Cloth with Max for more information.

Note

Gamebryo is only compatible with version 1.0.6 of the Max PhysX Plug-in. See NVIDIA PhysX™ Resources for additional information about PhysX.
