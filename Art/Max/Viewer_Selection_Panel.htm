Viewer Selection Panel

Viewer Selection panel

Use this field... To...
Direct3D11 Use the NiSceneViewer plug-in with a D3D11 renderer.
Direct3D10 Use the NiSceneViewer plug-in with a D3D10 renderer.
Direct3D9 Use the NiSceneViewer plug-in with a D3D9 renderer.
Xbox 360 Use the Xbox 360 on-target viewer plug-in.

There is an option to set the name of the development kit that you want to use. If the target is Default Xbox360, then the viewer will attempt to use your default Xbox 360 debug station. The Xbox 360 Viewer Plugin (found in the developer tools) must be installed in order for this option to become available.

The Xbox 360 NifViewer will use %EMERGENT_PATH%\Media\Shaders\Data as the custom shader source directory. The EMERGENT_PATH environment variable is set during installation.
Note: You must have licensed and installed the Xbox 360 version of the product to access this viewer.
PS3 Use the Playstation 3 on-target viewer plug-in.

There is an option to chose the target development kit to use, but in order to choose a target, it must have already been added to SN Systems' Target Manager. The PS3 Viewer Plugin (found in the developer tools) must be installed in order for this option to become available.

The PS3 NifViewer will use %EMERGENT_PATH%\Media\Shaders\Data\PS3 as the custom shader source directory. The EMERGENT_PATH environment variable is set during installation.
Note: You must have licensed and installed the PS3 version of the product to access this viewer.
PhysX Viewer Use the PhysX NIF Viewer.
Note: You must have installed the PhysX extensions of the product to access this viewer.
Asset Viewer Use the NIF Viewer plug-in. This plug-in launches the Gamebryo Asset Viewer tool.
KFM Preview Use the KFM Viewer plug-in. This plug-in launches the application that is associated with KFM files, which, by default, is the Gamebryo Asset Viewer tool.
Custom Specify a custom viewer.

The button immediately below this option indicates the current Custom View (if any). When you click the Custom View button, the Script Management dialog opens. Get programmer support to help implement your game executable to run from this file so you can instantly export art into your game easily!
View Export the current file and launch the selected viewer.
Optimize for the viewer Toggle the optimization option for the selected viewer.
X and Y size spinners Set the preview window's size. In full screen hardware mode, you will need to know which resolutions your hardware card will support (for example, 640x480, 800x600, and so on) to appropriately set the X and Y fields.
Windowed Specify whether the preview will be full-screen or windowed.
