Envmap.max

This 3ds Max sample illustrates a "reflective" environment map (envmap03.bmp) that is placed on some moving objects.

The sample demonstrates how to set up an environment map. All of the objects in the scene (Sphere01, Cone01, Teapot01, Sphere02, Cone02 and Teapot02) are shaded with the same material (Base). The Base material has the arcade.bmp texture as a diffuse map. It uses no other textures. An "Environment Map" light is set up to cast the environment map only onto Sphere01, Cone01, and Teapot01 by using an inclusion list and lighting/environment maps. The environment map is added on top of the base texture so that both effects operate simultaneously. The Omni02 light is included to light all the objects from the right. Multiple shapes (the Cone, Sphere and Teapot) are included in the scene to demonstrate how an environment map will appear on different geometries.

By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\envmap.max
