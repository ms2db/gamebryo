Gloss.max

This 3ds Max sample uses the Metal-gloss.bmp image to modulate the intensity of an environment map on a rusting ball.

Gloss mapping can add some visual interest to an application. This sample consists of a sphere (Sphere01) and a texture effect (Env Light) that is being used to create an environment map. This combination causes the environment map to appear in the "shiny" area of the base texture, but the environment map disappears in the "flat" regions.

By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\gloss.max
