Introduction to Gamebryo-PhysX Max Support

Gamebryo's support for PhysX content authoring in 3ds Max includes the ability to create rigid-bodies, rag-dolls, physics-based particle systems, and fluid systems. These can be exported as part of a Gamebryo NIF file ready for import into a Gamebryo-based application.

Gamebryo's support for PhysX content in 3ds Max is built upon the PhysX plug-in, found in the PhysX directory on the Gamebryo install disk. Before working with Gamebryo's PhysX support we strongly recommend you read about and familiarize yourself with the PhysX 3ds Max plug-in, using the documents and examples provided with the plug-in. This documentation does not repeat information discussed in the PhysX plug-in documentation.

The following topics are covered by this artist guide:

  • Exporting PhysX content for Gamebryo.
  • Rigid-body extras provided by Gamebryo.
  • PhysX Constraints
  • Creating Rag-dolls for use with Gamebryo.
  • PhysX-based Particle Systems set-up and export.
  • Fluids authoring.
  • Cloth authoring.

Several samples of PhysX content authored with Max are provided in the %EMERGENT_PATH%\Samples\Models\Max Artist Examples\PhysX directory of your Gamebryo install.

See the Known Issues and Limitations page for a description of Gamebryo-PhysX 3ds Max issues and suggested work-arounds.
