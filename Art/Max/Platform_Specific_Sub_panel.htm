Platform Specific Sub-panel

Platform: The Gamebryo 3ds max Plug-in enables you to select which target platform for which you wish to export. This selection can alter the behavior of scripts with respect specific optimizations with images and data. In some cases, such as skinning, it is necessary to choose this option to get correctly set up hardware skinning for a given platform.

The "generic" platform, which is the default setting, indicates that NiPixelData-based source textures and source cubemaps will be exported, so that image conversion can be done at load time, if necessary. Thus, the "generic" setting works across all platforms, although it requires more memory and time at load time for on-the-fly conversions. Platform-specific options can save memory and time at load time, but will not necessarily display in the PC-specific Gamebryo tools, nor on other platforms. To specify platform-specific textures for export from 3ds max, you may select any of the non-generic platform types with internal textures (i.e., textures are included in the main NIF). Specifically, these platform-specific internal texture choices include: Xbox360, Playstation3, D3D10, D3D11, and DX9. See Texture Streaming Options for Internal Textures for more details.

Image File Subdirectory: When exporting external texture NIFs, it is possible to select a subdirectory to save all texture NIFs.

Default Content Quality: When exporting with external texture NIFs or internal textures, it is possible to select the default content quality for the textures. This selection will enable the user to down sample larger texture formats into smaller texture formats. It is important to remember this is only a preferred format. Not all formats are supported on all platforms and during the export process the destination renderer will determine the final destination format.

Override Per-Texture Settings: When using the default content quality options, options set in the Gamebryo Shader for texture format type will not be overridden by default. Therefore, if the user wishes to override these settings, checking this box is required.
