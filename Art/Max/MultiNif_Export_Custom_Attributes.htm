MultiNifExport Parameters

These custom attributes, available through the Gamebryo Toolbar, allow the artist to specify multiple files for export out of a single source Max file.

Use Global Settings—This option forces this instance of the custom attributes to match the settings in the Multi-Nif exporter plug-in.

Use Object Name—This option automatically generates the filename based off of the object name.

Delete After Export—This option removes the object from the scene graph once it is exported.

Repair Emitter w/o Particles—This option repairs emitters in the event that the particle system gets broken out.

Translate Type—This option determines what the translation is set to on the object when it gets its own scene graph.

Rotate Type—This option determines what the rotation is set to on the object when it gets its own scene graph.

Scale Type—This option determines what the scale is set to on the object when it gets its own scene graph.

Note

These custom attributes require the Multi-Nif Exporter plug-in to function correctly. Please see the documentation on the Multi-Nif Exporter for more details.
