TwoPass

<TwoPass.NSF>

Requirements

  • NSF Shader Library
  • Implementation VS11:
    • Vertex Shader 1.1
  • Implementation VS11_PS11:
    • Vertex Shader 1.1 and Pixel Shader 1.1

Description

TwoPass is a programmer sample to illustrate how multiple passes can be placed in a single shader.

Art Requirements

  • Base map (Using first UV set)

Custom Attributes

  • TestMap—This map is alpha-blended over the base map of the model in the second pass. (Using first UV set)
  • SelectedColor—The color used to modulate the base map.

Global Variables

There are no global variables for this shader.

Object Variables

There are no object variables for this shader.
