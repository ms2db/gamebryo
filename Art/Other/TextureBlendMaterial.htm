TextureBlendMaterial

<NiCommonMaterial>

Requirements

  • NiCommonMaterial Library
  • Implementation PerPixel:
    • Pixel Shader 2.0
  • Implementation PerVertex:
    • Vertex Shader 2.0

Description

This material blends four textures together using a fifth texture as a control map. It is derived from the standard material, so it supports texture transforms, lighting, projected textures, etc.

Art Requirements

No additional art requirements beyond the Custom Attributes.

Custom Attributes

  • BlendMap0—RGB values from this texture are multiplied by ControlMap.R and then added to the total.
  • BlendMap1—RGB values from this texture are multiplied by ControlMap.G and then added to the total.
  • BlendMap2—RGB values from this texture are multiplied by ControlMap.B and then added to the total.
  • BlendMap3—RGB values from this texture are multiplied by ControlMap.A and then added to the total.
  • ControlMap—RGBA values from this texture control the color influence of each of the blend maps.
