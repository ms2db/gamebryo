Halo

Programmatic shader from ShaderLibrary.nl*

Requirements

  • ShaderLibrary.nl9 or ShaderLibrary.nlx
  • Vertex Shader 1.1
  • Vertex Shader 1.1 and Pixel Shader 1.1 on Xenon

Description

Halo is a programmatic shader that is stored in the ShaderLibrary library, which is a part of the ShaderSample sample. It draws a halo effect around an object. It also has a mode where the halo acts like an outline around a toon shaded object.

Art Requirements

No additional art requirements beyond the Custom Attributes.

Custom Attributes

  • EnableToon—This toggles the Toon effect on and off.

Global Variables

There are no global variables in this shader.

Object Variables

There are no object variables in this shader.

Sample Art Files

BigShipHalo.max, located with the ShaderSample source

BigShipToon.max, located with the ShaderSample source

Programmer Notes

The m_kColor, m_kScaling, m_fPulse, and m_fEdgeSize variables in the HaloShader class all control the halo parameters.
