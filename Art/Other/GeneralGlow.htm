GeneralGlow

<GeneralDiffuseSpecular.fx>

Requirements

  • FX Shader Library
  • Vertex Shader 1.1 and Pixel Shader 2.0

Description

The techniques in the GeneralDiffuseSpecular.fx file are used for most of the unprocessed scene in the MetalWars demo. Generally speaking, all of them implement some variation of a lighting model that is based on two HDR image cube maps, and two point light sources. The HDR cube maps (one for specular and one for diffuse) are artist generated and applied to almost everything in the demo scene. The point lights are used in the demo for the nearest fire and for the light from the gun firing. The application code is responsible for assigning the closest fire light.

This particular shader is much simpler than the GeneralDiffuseSpecular in that it does not use any cube maps or dynamic lights for lighting at all—only a glow map.

Art Requirements

  • Base map
  • Glow map

Custom Attributes

  • NormalMap—The normal map texture in tangent space. The glossiness of the specular component should be stored in the alpha channel of this texture..
  • LightingMap_cube—The cubic environment map used for diffuse lighting.
  • ReflectionMap_cube—The cubic environment map used for specular lighting.
  • bumpiness—Scalar value used to bias the normal map.
  • GlobalDimmer—Multiplier for the diffuse lighting cube map.
  • GlowMultiplier—Multiplier for the glow map.

Global Variables

  • g_bModifyAlpha—Boolean to determine whether to modify alpha for x-ray effect.
  • g_fCharacterDist—Distance from camera to the character. Used in the x-ray effect.
  • g_fDiffLightMultiplier—Global multiplier for diffuse cube map lighting.
  • g_fSpecLightMultiplier—Global multiplier for specular cube map lighting.
  • g_fPointLightMultiplier—Global multiplier for object lighting.

Object Variables

  • LightPos0—Position of point light 0.
  • LightDiff0—Diffuse color of point light 0.
  • LightAtten0—Attenuation constants of point light 0.
  • LightPos1—Position of point light 1.
  • LightDiff1—Diffuse color of point light 1.
  • LightAtten1—Attenuation constants of point light 1.

Programmer Notes

These shaders are primarily intended for use in the MetalWars demo. Not all applications will be able to effectively use them.
