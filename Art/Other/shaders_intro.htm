Gamebryo Shader Reference

The shaders that ship with Gamebryo are divided into two categories. Simple shaders are examples that, in most cases, would not be used in a production environment without customization. Game shaders are more advanced and can be used in a production environment with little or no customization.

Gamebryo also provides the Gamebryo Standard Material (NiStandardMaterial). This flexible dynamic shader system enables you to create shaders without having to code them by hand. Gamebryo provides GUIs for this system for each of the DCC tools. For more information about these GUIs, see the following:

  • Gamebryo Shader and 3ds Max Materials
  • Introduction to the NiMultiShader (Maya)
  • Specifying materials in the XSI Plug-in
