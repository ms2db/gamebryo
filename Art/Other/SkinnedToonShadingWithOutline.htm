SkinnedToonShadingWithOutline

<SkinnedToonShadingWithOutline.NSF>

Requirements

  • NSF Shader Library
  • Vertex Shader 1.1 and Pixel Shader 1.1

Description

Implement toon shading on a skinned character using a simple luminance transfer and the outlining technique used in the Outlining shader. It is essentially a skinned version of the ToonShadingWithOutline shader. Supports 20 bones per partition.

Art Requirements

  • Skinned model
  • Base map (Using first UV set)
  • Two lights of any kind

Custom Attributes

  • outlineThickness—Used to scale the thickness of the outline.
  • outlineColor—The outline color.
  • ToonRamp—The 1D texture ramp that maps luminance to a color. This works with toon shading when only one or two colors are used in the ramp.

Global Variables

There are no global variables for this shader.

Object Variables

  • Light1—A generic light whose diffuse color and direction are used to light the model.
  • Light2—A generic light whose diffuse color and direction are used to light the model.
