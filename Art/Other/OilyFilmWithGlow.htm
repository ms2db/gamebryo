OilyFilmWithGlow

<OilyFilmWithGlow.NSF>

Requirements

  • NSF Shader Library
  • Vertex Shader 1.1 and Pixel Shader 1.4

Description

OilyFilmWithGlow is the same as OilyFilm except that it uses the glow map instead of the warp effect map.

Art Requirements

  • Base map (Using first UV set)
  • Glow map
  • Two lights of any kind

Custom Attributes

  • Exponent—This is the specular exponent for the effect.
  • filmRamp—The 1D ramp texture for the thin film interference effect. The texture should be approximately the interference pattern for thin film interactions where increasing U is increasing thickness of the film.

Global Variables

There are no global variables in this shader.

Object Variables

  • Light1—A generic light whose diffuse color and direction are used to light the model.
  • Light2—A generic light whose diffuse color and direction are used to light the model.

Sample Art Files
