TextureBlendUsingVertexColors

<TextureBlendUsingVertexColors.NSF>

Requirements

  • NSF Shader Library
  • Vertex Shader 1.1 and Pixel Shader 1.1

Description

This shader allows three independent textures to be blended together using the red, green, and blue vertex color channels as alpha modifiers. Further control can be gained by using the BlendMapMultiplier. A dark map is then applied to the final color. Note that this shader supports up to 8 Point light sources.

Art Requirements

  • Vertex color arrays in the geometry
  • Eight lights of any kind

Custom Attributes

  • BlendMapRed—Texture map modulated by the red vertex color channel. (Using first UV set)
  • BlendMapGreen—Texture map modulated by the green vertex color channel. (Using second UV set)
  • BlendMapBlue—Texture map modulated by the blue vertex color channel. (Using third UV set)
  • DarkMap—Dark map used to modulate all three blend maps. (Using fourth UV set)
  • BlendMapMultiplier—Scales each individual channel of the vertex colors. Use 0.0 to turn off a channel.
