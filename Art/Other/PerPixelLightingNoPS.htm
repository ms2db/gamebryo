PerPixelLightingNoPS

<PerPixelLighting.NSF>

Requirements

  • FX Shader Library
  • Vertex Shader 1.1

Description

This shader applies a base map and performs per-pixel lighting using a normal map and a single point light. This shader differs from the PerPixelLightingPS shader in that it uses the fixed-function pixel pipeline to perform the texture lookups.

Art Requirements

  • Base map
  • Point light
  • Normal, binormal, and tangent arrays (automatically calculated during export)

Custom Attributes

  • normalMap—The tangent-space normal map for this shader.

Global Variables

  • AmbientIntensity—Global scalar for ambient light intensity.

Object Variables

  • MSLightPos—Model-space light position.
