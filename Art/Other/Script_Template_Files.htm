Script Template Files

Script Templates are customized asset processing pipelines that have been saved to disk in the ".script" format. These files can be shared amongst modelers and artists, allowing an art director to define a standard set of pipelines. All characters can share the template script for characters. All levels can share the template script for levels. When you change a template script, all asset files that use that script can be updated by exporting again. If you find that you need to add a plug-in to all character export scripts, all you need to do is alter the template. If all of your characters use the template script, then all exports of any character from that point on includes the new plug-in.
