ChromeSkinned

<Chrome.fx>

Requirements

  • FX Shader Library
  • Implementation ChromeSkinned:
    • Vertex Shader 1.1 and Pixel Shader 2.0
  • Implementation ThreePassChromeSkinned:
    • Vertex Shader 1.1 and Pixel Shader 1.1

Description

This shader produces a chrome effect using a base map, diffuse cube map, specular cube map, and normal map (that contains gloss in alpha channel). There are multiple implementations of this particular effect, including skinned and static versions of the geometry.

The ChromeSkinned implementations use 29 bones per partition.

Art Requirements

  • Skinned model
  • Base map
  • Normal, binormal, and tangent arrays (automatically calculated during export)

Custom Attributes

  • NormalMap - The normal map texture in tangent space. The glossiness of the specular component should be stored in the alpha channel of this texture.
  • LightingMap_cube - The cubic environment map used for diffuse lighting.
  • ReflectionMap_cube - The cubic environment map used for specular lighting.
  • bumpiness - Scalar value used to bias the normal map.
