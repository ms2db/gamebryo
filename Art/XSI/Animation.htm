Working with animations in the XSI Plug-in

Linear animation of position, rotation and scale is supported with the following limitations:

  • Non-linear animations (e.g. cubic) are converted to linear before exporting
  • Only uniform scale animation is supported, non-uniform scaling animation results in a warning on export
  • Animations are assumed to be local
  • Only clamped animations are supported

Animation Mixer

Clips defined in the XSI Animation Mixer are exported in a manner compatible with the Gamebryo Controller Extractor plug-in. Clips are converted and exported end-to-end in the Gamebryo NIF file. The start and end frame of each clip is suitably tagged with NiTextKeyExtraData so the Gamebryo Controller Extractor plug-in can detect and create animation keyframe files (KF and KFM files) with multiple sequences as part of the export process.

Transitions, mixings, and weights defined in the Animation Mixer are not exported. Use the Gamebryo Animation Tool to define transitions between the sequences stored in KFM files created by the Controller Extractor.

Mesh Morphing

Mesh morphing is supported with the following features and limitations:

  • Merging meshes using weights is supported
  • Animated weights are supported
  • Morph animations must use linear interpolation
