Understanding how geometry is exported in the XSI Plug-in

The exporter only supports polygons and triangle-based geometry. You can triangulate more complex geometries prior to exporting and modify the Crosswalk export options to govern how complex geometry (such as NURBS) is decomposed into simpler planar geometry.

The following object properties are supported by the exporter:

  • Position, rotation, and scale transformations
  • Pivot
  • Neutral pose
  • Visibility
  • Node names are preserved on export

Note

Only the default XSI axis setting is supported.
