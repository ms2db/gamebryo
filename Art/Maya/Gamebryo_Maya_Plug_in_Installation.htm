Maya Plug-in Installation

When you install Gamebryo, the installer identifies the version of Maya on the target host. By default, the installer will install the plug-in for any supported version of Maya that it finds. If you have more than one version of Maya installed, you should verify that the target Maya paths are correct. If you do not want to install the plug-in for a specific version of Maya, deselect that version during the install.

After you install the plug-in, you use the Maya Plug-in Manager to load it in Maya.

To load the Maya Plug-in

  1. Launch Maya.
  2. From the menu bar, choose Window > Settings/Preferences > Plug-in Manager.
  3. In the Maya Plug-in Manager dialog, ensure that the Loaded and Auto load checkboxes are checked beside the GamebryoMayaPlugin.mll plug-in.
  4. Click Close to close the Maya Plug-in Manager dialog.
  5. From the Maya menu bar, choose File > Save Preferences or restart Maya.

If you plan to create PhysX content for use with Gamebryo, you must also install the PhysX Maya plug-in. Gamebryo 3.2 is intended for use with a pre-release PhysX Maya plug-in, found in the installation media at PhysX\NVIDIA_PhysX_Plug-in_Maya_x86_2.01.0529.1836.msi. Note the license agreement found in the same directory.

After the plug-in is installed and loaded, a Gamebryo Tools Shelf is added to the Maya interface. You can access a variety of Gamebryo utilities through this shelf.

Build Considerations on Windows Vista

To build the plug-in on Windows Vista, you need to be logged on as a user with administrator-level permissions. However, even with these permissions, it is possible that you might have to modify the security settings on the Maya folder to allow user access. You will have to make these modifications if you see a large number of "Access denied" errors during the post-build step of the plug-in itself. These errors will occur if the post-build scripts cannot copy Gamebryo scripts and plug-ins into the installation folder.

To modify the security settings, open Windows Explorer, right-click the top-level folder of each Maya installation, and select (click) Properties. On the Security tab, click the Edit button. In the Group or user names window, click the Users entry and then click the Modify checkbox under the Allow column. Click OK. Click OK again to apply this change.
