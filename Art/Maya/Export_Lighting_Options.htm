Export Lighting Options

The Gamebryo Maya Plug-in supports inclusion lists for lights using the "Relationship Editor" in Maya.

Lighting Options

Within Gamebryo, during rendering, there is some overhead associated with the setup of lighting. Because Maya doesn't really maintain a single scene root, lighting is applied to each object at the top of the scene individually. You can minimize this overhead by selecting (checking) the Optimize Scene Root Lights option. This applies a light to the scene root that affects every object at the root of the Maya scene graph.
