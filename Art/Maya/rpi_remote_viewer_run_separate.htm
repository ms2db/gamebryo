Running the Gamebryo Remote Viewer on a Separate Workstation

The Remote Viewer can be launched manually with a command line argument specifying the IP address (and optionally the port number) of the workstation that is running Maya. For example:

RemoteViewer.exe -editorHost 192.168.1.151 -editorPort 13215

This command line option can be used to run the PC Remote Viewer on a separate workstation than the one that is running Maya.
