Gamebryo Remote Viewer Status Messages

When the Remote Viewer starts it will immediately attempt to establish a network connection back to the workstation that is running Maya. This can be seen in the viewer’s Output Log panel; a message such as this will be displayed:

Trying to connect to 192.168.1.151

Once a connection has been established the message will change to:

Connected to Editor @ <IP Address : Port>

The Remote Viewer on a console attempts to connect back to the host PC using the primary network adapter returned by Windows. Connection may fail if this address is not accessible from the console, as is the case with some VPN and router set-ups. See Knowledge Base Article 894564 for information on switching the PC's primary adapter to one that the console can access. Alternatively, the remote viewer could be launched manually with the required IP address.
