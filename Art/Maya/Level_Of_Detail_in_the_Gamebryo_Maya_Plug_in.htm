Level Of Detail in the Gamebryo Maya Plug-in

Create Centered LOD Group button

The Gamebryo Maya Plug-in supports "Level of Detail" groups found within Maya. As shown below, in SwordLOD.mb, multiple objects are grouped together and, depending on the distance to the camera, one of the group members is shown. This approach enables you to create simplified versions of a model which are used when the camera isn't close enough to see the detail found in a complex model. Within Maya, you can set the threshold transition values between models. Those threshold values are used by the Gamebryo Maya Plug-in, when creating a corresponding NiLODNode.

SwordLOD.mb example

Gamebryo supports two different types of LOD. Range LOD (NiRangeLODData) uses the distance to the camera to determine the visible LOD level. Screen LOD (NiScreenLODData) uses the size of the objects bounding volume on screen to determine which LOD level to use.
