Creating Environment Maps

Environment maps are part of a larger concept in Gamebryo called projected textures. To create an Environment Map, you will be creating a Dynamic Texture Light and setting its parameters appropriately to generate an environment map. The position and orientation of the light will determine the position and orientation of the environment map.

Environment map example

To create an environment map

  1. Click the "Create Gamebryo Dynamic Texture Light" on the Gamebryo shelf in Maya.
  2. Select the spot light that was just created and view it with the Attribute Editor.
  3. In the Attribute Editor, select its shape tab.
  4. Link the shape's color to a file texture. This texture will be your environment map. To create an environmental cube map, use Maya's Environmental Cube Map material instead of a file texture. You will then need to apply file textures to cube attributes: front, back, left, right, top, and bottom. Important: The cubic environment map will get its orientation from the light, not the attached 3D texture transform. To visualize things easiest, move the 3D Texture Transform to be a child of the dynamic texture light and adjust the light to adjust your cube map.
  5. Select the Dynamic Texture Light and click its transform tab in the Attribute Editor.
  6. The Gamebryo section custom Gamebryo attributes.
  7. Set the Texture Type to "Environment Map".
  8. Set the Coordinate Type to "Sphere Map". If you are creating a cubic environment map, set the Coordinate Type to "Specular Cube Map".
  9. Customize the other projected texture light attributes.
  10. Select Light centric linking from the \Windows\Relationship Editors\Light Linking\Light-Centric menu. All objects affected by this light will have the environment map in Gamebryo.
