NiForce Node Reference

Base Parameters

NiForce Base Parameters user interface

Attribute Keyable Description
Type No Possible values for this attribute are:
  • GravityField - Modifies the velocity of each particle to simulate the effects of directional gravity.
  • AirField - Updates the particle velocity to simulate the effects of air movements like wind, fans, or wake.
  • TurbulenceField - Randomly adjust the velocity of a particle.
  • VortexField - Particles are accelerated around the directional axis by the Magnitude (velocity).
  • DragField - The drag force is applied to the particle velocity vector unless Use Direction (on the Drag Field Parameters) is true.
  • RadialField - Updates particle velocity to simulate the effects of point gravity.
Magnitude Yes The strength of the field.
Attenuation Yes Controls how the field is affected by distance to the field object. As the Attenuation value increases, the force object has less effect on particles that are father away from it. Zero attenuation means the effect of the force will remain constant over distance.

As the Attenuation increases, the drop off of the force effect increases very rapidly (resulting in a decreasing exponential curve).
Use Max Distance No Determines whether or not to use a maximum distance from the field object after which the force has no effect.
Max Distance Yes The maximum distance from the field object at which the force has any effect. This distance is only used if Use Max Distance is true.

Air Field Parameters

NiForce Air Field Parameters user interface

Examples of Air Field Parameters for common effects:

  • Wind
    • Magnitude = wind speed
    • Inherit Velocity = 0.0
    • Inherit Rotation = false
    • Enable Spread = false
  • Fan
    • Magnitude = air speed
    • Inherit Velocity = 0.5
    • Inherit Rotation = true
    • Enable Spread = true
  • Wake
    • Magnitude = 0.0
    • Inherit Velocity = 1.0
    • Inherit Rotation = true
    • Enable Spread = false
Attribute Keyable Description
Air Friction Yes How quickly particles will accelerate to the magnitude of the air field.
Inherited Velocity Yes Controls the amount of the air field's velocity that will be added to the magnitude and direction. This value is between 0.0 and 1.0.

Air Field Velocity = (particle system to field object transform – previous transform) * Inherited Velocity / (time since last update)
Inherit Rotation No Whether or not changes in the rotation of the field object will be reflected in the final direction of the air field.
Enable Spread No Whether or not the air flow is cone-shaped relative to the direction of the air field.
Spread Yes The angle of the air field cone if Enable Spread is true.

NiForce Drag Field Parameters user interface

Attribute Keyable Description
Use Direction No Whether or not the drag force applies only in the direction specified by direction of the arrow. See Maya documentation for an excellent description of this attribute.

Radial Field Parameters

NiForce Radial Field Parameters user interface

Attribute Keyable Description
Radial Factor No Controls how the radial field's effect diminishes with attenuation. This value must be between 0.0 and 1.0.

A value of 1.0 causes the effect of the radial field to drop quickly to zero when a particle approaches the max distance from the field.

A value of 0.0 causes the effect of the radial field to gradually approach (but never reach) zero as a particle approaches the max distance from the field.
