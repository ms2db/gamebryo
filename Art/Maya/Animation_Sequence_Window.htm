Animation Sequence Editor

Animation Sequences are a powerful tool for creating compelling interactive media. By utilizing a higher-level unified concept of a character, Gamebryo can help you organize your art to make it easy to use and create. Character sequences allow animation accumulation that simplifies the creation of walk and run cycles.

In one export, all of the sequences for a character will be saved. The Gamebryo Maya Plug-in supports exporting of Layered Sequences as well as multiple characters per Maya file. For convenience, the sequences are ordered by starting frame. The exporter uses character sequences to find the characters in the scene and differentiate between a character and any other object in the scene.

  • The attribute Ni_CharacterRoot is used to tag an object as a Character Root.
  • The attribute Ni_Sequence_Animation_Accumulation is used to tag an object as a Accumulation Root.
  • There is not a specific attribute for layer roots. All dag nodes with sequences are by default layer roots.
  • Sequences are stored in the compound multi attribute Ni_Sequence with children Ni_Sequence_Valid, Ni_Sequence_Name, Ni_Sequence_Start_Time, Ni_Sequence_End_Time, Ni_Sequence_Looping, Ni_Sequence_Rot, Ni_Sequence_Tran_X, Ni_Sequence_Tran_Y, and Ni_Sequence_Tran_Z. (The sequences section will be grayed out if there are no Character Roots in the scene.)
  • Morph Tags are stored in the compound multi attribute Ni_MorphTag with children Ni_MorphTag_Valid, Ni_MorphTag_Name and Ni_MorphTag_Time. (The Morph Tags section will be grayed out if there are no Sequences for the current Layer Root.)
  • Text Keys are stored in the compound multi attribute Ni_TextKey with children Ni_TextKey_Valid, Ni_TextKey_Name and Ni_TextKey_Time. (The Text Keys section will be grayed out if there are no Sequences for the current Layer Root.)

Animation Sequence Editor

Use this field... To...
Character Roots
Character Roots drop-down Select character roots that have already been added to the scene.
Reload Refresh the Sequence Editor. This field is useful if, while the Sequence Editor is open, an event occurs that changes an object that the Sequence Editor depends on. For instance, opening a new scene will require the Sequence Editor to be reloaded if it was open while the new scene is loaded.
Add Selected Add new objects as Character Roots. Works with one object because ideally there should be only one Character Root per scene.
Delete Delete the Character Root information, including all of its Layer Roots, Sequences, Morph Tags, and Text Keys. Does not delete the transforms from the Maya scene; it will only delete the character info.
Select Select the currently highlighted Character Root in the Maya scene.
Accumulation Root
Accumulation Root textbox Review the name of the current Accumulation Root. If your scene somehow finds itself in an invalid state with two Accumulation Roots, this box will say "Resolve Duplicates!" Re-adding the correct Accumulation Root will fix this problem. If your scene does have two Accumulation Roots, only one will get exported.
Add Selected Add new objects as the Accumulation Root. The Accumulation Root will have to be a descendant of the Character Root.
Select Select the current Accumulation Root in the Maya scene.
Layer Root
Layer Root drop-down Select Layer Roots that have already been added to the scene.
Add Selected Add new objects as Layer Roots. Works only with one object selected.
Delete Delete the Layer Root info and all of its Sequences, Morph Tags, and Text Keys. Will not delete the transforms from the Maya scene; only deletes their associated character info.
Select Select the currently highlighted Layer Root in the Maya scene.
Sequences
Sequences drop-down Select sequences that have already been added to the scene.
Add Add new sequences
Rename Rename the selected sequence.
Delete Delete the selected sequence. This also deletes the sequence's Morph Tags and Text Keys.
Start Time Enter the sequence start time.
End Time Enter the sequence end time.
Make Time Range Fit Update the play back range of the Time Slider to reflect the start and stop times of the sequence.
Looping Turn looping ON/OFF for the selected sequence.
Accumulate Rotation Turns ON/OFF rotation accumulation.
Accumulate Translate Turns ON/OFF translation accumulation.
Morph Tags
Morph Tags drop-down Select Morph Tags that have already been added to the selected sequence.
Add Add new Morph Tags.
Rename Rename selected Morph Tags.
Delete Delete selected Morph Tags.
Frame To associate Morph Tags with a specific frame.
Jump To Time Set the current Maya time to the given frame.
Text Keys
Text Keys drop-down Select Text Keys that have already been added to the selected sequence.
Add Add new text keys.
Rename Rename selected text keys.
Delete Delete selected text keys.
Frame To associate text keys with a specific frame.
Jump To Time Set the current Maya time to the given frame.
