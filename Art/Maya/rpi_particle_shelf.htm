Gamebryo Particle Shelf

The Gamebryo Particle shelf contains all the buttons necessary for creating and modifying Gamebryo particle systems. These buttons either create custom Maya nodes or associate existing nodes with each other to create effects. These associations can be viewed and deleted in the Maya Hypergraph.

Warning

Creating Gamebryo particle system node associations in the Hypergraph is not recommended since the Gamebryo nodes use custom (non-standard) attributes to associate with each other.
Shelf Icon Description
Remote Viewer Options

This button opens the Remote Viewer Options dialog that can be used to configure how the scene is rendered in the viewer.
Start Remote Viewer

Launch the realtime particle iteration viewer on the local workstation or a console development kit.
Refresh Remote Viewer

Update non-particle system changes to the scene displayed on the Remote Viewer. For example, if a material has been changed, use this button to update the scene displayed in the Remote Viewer. This button exports the entire scene and sends it to the viewer. The Gamebryo export progress dialog will be displayed. Depending on the scene’s size (or complexity) the progress dialog will be displayed anywhere from less than a second to a couple of minutes. Refer to Automatic versus Manual Updates for additional details on when to push this button.
Note: Pressing the F5 key will also refresh the scene in the Remote Viewer

Resume

Pause

 Resume/Pause Remote Viewer Updates

This is a toggle button that indicates whether the Gamebryo plug-in is actively monitoring particle system nodes. When the Resume button is displayed, the Gamebryo plug-in is not currently monitoring particle system nodes.

These buttons can be used to temporarily disable monitoring while the artist makes changes that do not require viewing in real-time (reducing processing overhead and improving Maya performance).

In addition, these buttons can be used to reset monitoring system and the scene displayed in the Remote Viewer if a unrecoverable error occurs. For example, if the viewer stops responding to changes to particle systems or the Refresh button.
Note: Opening a scene or creating a new scene in Maya will cause Remote Viewer Updates to automatically pause. To re-enable remote viewer updates, press the Resume button.
Create Sprite Particle System with Emitter

Create a basic particle system consisting of NiParticleSystem and NiEmitter nodes that emits sprites.
Create Mesh Particle System with Emitter

Create a basic particle system consisting of NiParticleSystem and NiEmitter nodes that emits meshes instead of sprites. Select the mesh to be used as the particle and then click this button to create the particle system.
Create Particle System with Surface Emitter

Select the mesh that will emit particles and press this button to create a basic particle system consisting of NiParticleSystem and NiEmitter nodes.
Create Particle System with Curve Emitter

Select a curve that will emit particles and then press this button to create a basic particle system consisting of NiParticleSystem and NiEmitter nodes. Only curves with uniform knot spacing are supported for curve emitters.
Create Emitter and Attach to Selected Particle System

Create a new NiEmitter object. If a particle system is not selected, the emitter will be added to the scene, but it will not be associated with a particle system and will not be exported.
Create Surface Emitter and Attach to Selected Particle System

Create a new NiEmitter object associated with a mesh and attach it to an existing NiParticleSystem.
Create Curve Emitter and Attach to Selected Particle System

Create a new NiEmitter object associated with a curve and attach it to an existing NiParticleSystem. If a particle system is not selected, the curve emitter will be added to the scene, but it will not be associated with a particle system and will not be exported. Only curves with uniform knot spacing are supported for curve emitters.
Create Collider and Attach to Selected Particle System

This button creates a new NiCollider node and associates it with an existing particle system.
Create Force and Attach to Selected Particle System

Creates a NiForce node and associates it with an existing particle system.
Create Living Spawner and Attach to Selected Particle System

Creates a NiLivingSpawnerParticleSystem node and associates it with an existing particle system. The existing particle system’s particles will emit the spawned particles during their lifetime (as defined by the living spawner node).
Create Death Spawner and Attach to Selected Particle System

Creates a NiDeathSpawnerParticleSystem node and associates it with an existing particle system. The existing particle system’s particles will emit the spawned particles upon death. This can occur when the particle’s lifespan is reached or it comes in contact with a collider node.
Create Gamebryo Collision Spawner and Attach to Selected Particle System

Creates NiColliderSpawnerParticleSystem node and associates it with the selected particle system and collider nodes.
Attach Mesh as Mesh Particle: Select one or more meshes and then select particle system

Use this button to assign an existing mesh (to be emitted by the particle system) to an existing particle system. If multiple meshes are attached to a particle system, the mesh to be used for a individual particle will be selected using a uniform random distribution.

Attaching a mesh to an existing sprite particle system will convert it into a mesh particle system.
Attach Emitter to Particle System: Select emitter and then select particle system

Link an existing emitter to an existing particle system. This button can be used to configure multiple particle systems to emit from a single emitter. If the particles are emitted at the same time, it may be necessary to adjust the bounding volume of one of the particle systems to ensure one is always on top (closer to the camera).
Attach Mesh to be Used by Surface Emitter: Select one or more meshes and then select particle emitter

This button associates one or more meshes (that are to emit particles) with an existing surface emitter on a particle system.
Attach Curve as Curve Emitter: Select curve and then select particle emitter

This button attaches a curve that will emit particles to an existing particle system.
Attach Collider: Select one or more colliders and then select particle system

Associates existing colliders with an existing particle system.
Attach Force: Select one or more forces and then select particle system

Associates existing force with an existing particle system.
Attach Particle System as Living Spawner: Select one particle system as the spawner and last select another particle system as the host

Associates existing living spawner with an existing particle system.
Attach Particle System as Death Spawner: Select one particle system as the spawner and last select another particle system as the host

Associates existing death spawner with an existing particle system.
Attach Particle System as Collision Spawner: Select one particle system as the spawner, then select a collider or colliders and last the particle system as the host

Associates existing collision spawner with an existing particle system.
Convert Existing Maya Particle Systems to Gamebryo Particle Systems

This button converts any Maya particle system nodes in the scene into equivalent Gamebryo particle system nodes. This conversion removes the original Maya particle system nodes from the scene. Given the possible complexity of some Maya particle systems, some converted systems will likely require additional tweaking by the artist.
