Export Animation Options

Use this field... To...
Export Animation Range Control export of the entire scene while limiting the animation to a select range. Only key frame data within the supplied range will be exported. This option is only used by the D3D viewer.
Loop over the Animation Range Set up the viewer to loop along the animation range. For example, a scene may extend from frame 0–1000, but by setting the animation range slider, you can cause the viewer to loop over frames 200–250. This option is only used by the Direct3D viewer.
Maximum Frame Rate Control the frame rate of animations in the view window. Artists will typically want to see how animations look at different frame rates to accurately judge how they will look when implemented in the game. This option is only used by the Direct3D viewer.
Minimum Bone Influence Specify the minimum percentage of influence that bones in a vertex have. Bones that influence a vertex by an amount that is less than this percentage value are ignored.
Sample All Rotation Force the exporter to create a rotation keyframe for every frame of the exported time range. Sampling is recommended for complex scenes.
Sample Rotation As Quaternions Export rotation as quaternions instead of Euler angles. This is recommend since it prevents flipping from IK and takes up less memory and is more efficient than Euler sampling.
Sample All Translation Force the exporter to create a translation keyframe for every frame of the exported time range. This is recommend for complex animation because Gamebryo does not support weighted tangents for keyframes and does not support tangents on a per axis basis.
