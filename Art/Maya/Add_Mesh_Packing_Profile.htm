Add Mesh Packing Profile

Every piece of geometry has a mesh packing profile associated with it. The Packer tool plug-in uses this profile to create platform-specific geometry streams that graphics cards use. Depending on the situation, it can be necessary for a specific mesh to override the scene default packing profile. In this case, the mesh uses its own profile. To override the scene default packing profile, click the Add Mesh Packing Profile button and choose the desired profile from the Mesh Packing Profile dropdown.

Add Mesh Packing Profile button

When you click the Reload Profiles button, the plug-in reparses the %EMERGENT_PATH%\Media\ToolPluginsData\MeshProfiles directory for new and modified mesh profiles. The drop-down list values also refresh.
