Viewing Maya Plug-in Version Information

You can view version information for the Maya Plug-in and the NiMultiShader plug-in. You can also review the version of Gamebryo with which each plug-in was created.

To view version information

  1. In the Gamebryo Tools Shelf in Maya, click the Export Options button.

    The Export All Options dialog displays.

  2. View version information within the File Type Specification Options group box.
