Spawn On Collide

Both Maya and Gamebryo support particle spawning when a particle hits a collider. By using "Particle Collision Events", you can specify the new particles to spawn.

Spawn on collide example

Collision Event Attributes

Particle Collision Events

Use this field... To...
Type Specify the event type. Currently only the "Emit" type is supported.
Random # Particles Specify that a random number of particles will be created when a collision occurs.
Num Particles Set the number of particles that will be spawned when a collision occurs.
Spread Set the randomness that will be applied to the direction of spawned particles. Values range from 0.0 (the direction of particle) and the 1.0 (any direction).
Target Particle Define the new particle that will be spawned when a collision occurs.
Note: It is important to remember that you need to specify the "shape" of the target particle.
Original Particle Dies Specify that the original particle should die after the collision.
