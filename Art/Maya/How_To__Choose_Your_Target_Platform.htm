Choosing Your Target Platform

The "generic" platform, which is the default platform setting, indicates that NiPixelData-based source textures and source cubemaps will be exported, so that image conversion can be done at load time, if necessary. Thus, the "generic" setting works across all platforms, although it requires more memory and time at load time for on-the-fly conversions.

Platform-specific options can save memory and time at load time, but will not necessarily display in the PC-specific Gamebryo tools, nor on other platforms. To specify platform-specific textures for export from Maya, you may select any of the non-generic platform types with internal textures (i.e., textures are included in the main NIF). Specifically, these platform-specific internal texture choices include: Xbox360, Playstation3, D3D9, D3D10, and D3D11.

The Mesh Profile System has platform specific options that can control the format of the NiMesh objects. If the "generic" option is selected the assumption is that you wish to have a D3D9 version of your asset. If a profile specifies a format for the D3D9 version that is not compatible with other platforms it will not be able to be viewed on that platform even if the "generic" option is selected. If a true "generic" asset is required it is recommended that you do not finalize the asset. Once an asset is finalized it can not be finalized again.

Export-Display Options

To choose your target platform

  1. In Maya, select "Export All" from the "File" menu. Alternately one could click on the Export Options Button on the Gamebryo Tool Shelf.
  2. In the "Export All" dialog, click the "Options" button.
  3. In the "Export All Options" dialog, set the file type to NIF.
  4. Under the "Export-Display Options" section, select the viewer you wish to use.
  5. In the upper left corner of the window there is a menu option labeled "Edit". Click on the item labeled "Save Settings", and close the Export All Options window.

Note

Viewing platform-specific content (i.e., non-generic) on a different platform may yield unexpected results. For example, if a character is exported specifically for the Xbox 360, it will be formatted such that it is incompatible with other platforms. Using the Gamebryo Scene Viewer on a PC might not be able to show the scene without "error replacement textures".
