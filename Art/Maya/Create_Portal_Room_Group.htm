Create Portal Room Group

Click the Create Portal Room button to create a room group and name it appropriately. During the export, the portal conversion code detects a room group by locating "_NDLRG" in the name.

Create Portal Room Group button

Depending on what is selected during the export, you will see three different types of behavior.

  • Nothing selected—an empty group named "RoomGroup_NDLRG" will be created in Maya
  • One transform selected—the "_NDLRG" tag will be appended to the end of the selected object's name
  • Multiple transforms selected—the selected objects will be grouped together under a new group named "RoomGroup_NDLRG"
