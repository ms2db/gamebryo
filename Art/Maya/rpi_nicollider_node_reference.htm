Base Parameters

NiCollider Base Parameters user interface

Attribute Keyable Description
Type No Possible values for this attribute are:
  • Planar - The collision shape will be a plane with a size defined by Width and Height attributes.
  • Spherical - The collision shape will be a sphere whose size is specified by the Radius attribute.
Bounce No The amount of bounce that particles will have upon collision. The collider bounce is multiplied by the particle system bounce to equate the total bounce when a collision occurs. The larger the total bounce the faster a particle will reflect from the collider.

Total Bounce = Particle System Bounce * Collider Bounce
