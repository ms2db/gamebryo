Introduction to Gamebryo-PhysX Maya Support

Gamebryo's support for PhysX content authoring in Maya includes the ability to create rigid-bodies, rag-dolls, physics-based particle systems, and fluid systems. These can be exported as part of a Gamebryo NIF file ready for import into a Gamebryo-based application.

Gamebryo's support for PhysX content in Maya is built upon the PhysX plug-in, found in the PhysX directory of the Gamebryo install disk.

Before working with Gamebryo's PhysX support, we recommend that you read about and familiarize yourself with the PhysX Maya plug-in, using the documents and examples provided with the plug-in. This documentation does not repeat information discussed in the plug-in's documents.
