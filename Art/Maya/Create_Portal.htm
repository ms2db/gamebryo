Create Portal

Click the Create Portal button to create a two-way portal between two portal rooms. The portal is created by creating a default plane with its normal pointing into the starting room and adding the appropriate "Extra Attributes". During the export, the portal conversion code detects a portal by locating "_NDLPR" in the name.

Create Portal button

There are a few things you need to remember when you create portals between rooms.

  • You MUST select two portal rooms for this to work correctly.
  • The first room you select will be the starting room and the second will be the destination room.
  • The portal's normal must point INTO the starting room.
  • The portal MUST be positioned coplanar with a portal wall. If they are not coplanar you will see a flash as you move through the portal.
  • The name of the destination room will be found as an extra attribute on the created portal named "NiData_Portal". This extra attribute is a string which contains <NDLADJOINER "pTorus1_NDLRM">. The name of the destination portal room is found in the quotations. This extra attribute is automatically created and set for you.
  • To make this a one-way portal, you need to delete the "Extra Attribute" named "NiData_TwoWay". This attribute will be found on the created portal.
