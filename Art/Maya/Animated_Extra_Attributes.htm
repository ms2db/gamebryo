Animated Extra Attributes

Animated "Extra Attributes" are a feature of Gamebryo that allows for custom data to be embedded in Gamebryo objects, and that data to be animated in Maya. There are three different types of attributes you can animate: Float, "Colors", and Double3. Colors are a special case added to Maya to support animated RGBA values. Creating "Extra Attributes" is complex, so the necessary MEL script is provided to support adding animated attributes to the selected object.

Float Attribute

addAttr -ln NiData_YOURNAME -at "float";

Double Attribute

addAttr -ln NiData_YOURNAME2 -nc 3 -at double3;

addAttr -ln NiData_YOURNAME2X -at double -p NiData_YOURNAME2;

addAttr -ln NiData_YOURNAME2Y -at double -p NiData_YOURNAME2;

addAttr -ln NiData_YOURNAME2Z -at double -p NiData_YOURNAME2;

Color Attribute

addAttr -ln NiData_YOURNAME3 -nc 3 -at float3 -uac;

addAttr -ln NiData_YOURNAME3R -at "float" -p NiData_YOURNAME3;

addAttr -ln NiData_YOURNAME3G -at "float" -p NiData_YOURNAME3;

addAttr -ln NiData_YOURNAME3B -at "float" -p NiData_YOURNAME3;

addAttr -ln NiData_YOURNAME3_Alpha -at "float";
