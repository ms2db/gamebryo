Viewer Scripts

The export process happens in two stages. First, in the Script Selection dialog, you select the script that you want to use to finalize the scene. Then, a viewer script is run to view the finalized scene. Selecting the Scene Viewer in the Export All Options dialog box chooses the specific viewer script to run.

The Gamebryo Maya plug-in uses default viewer scripts to configure the viewer for the scene. However, you may want to customize the individual viewer scripts. You can find the scripts in the Script Selection dialog, and you can customize them in the Script Editor. For example, you may create all of your scenes without any lights, but you still want to see some default lights when you are viewing the scene. By creating a simple process plug-in that adds default lights to the scene graph and adding it to your custom viewer script, all your exported files would not have lights but default lights would appear when viewing with on the target viewer.
