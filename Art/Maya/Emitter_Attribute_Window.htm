Emitter Attribute Window

When you click the Create Emitter and Particles button to create a particle system, Gamebryo-specific extra attributes are added to the emitter. These attributes help control the emitter as it animates. For backward compatibility, there are two ways to control when a particle system emits particles. You can specify the begin key and start frames, or you can directly animate the "Rate".

Surface Emission
Location Specify the location on a surface where particles will be emitted. This value is stored in the Ni_SurfaceEmitLocation attribute. There is an inherent overhead associated with choosing a location of a created particle. The following options are listed from least expensive to most expensive:
  • Vertices—Particles will be emitted by randomly selecting a vertex and emitting from that location.
  • Edge Center—Particles will be emitter by randomly selecting a triangle then a random edge of that triangle. The particle will be emitted from the center of that edge.
  • Edge Surface—Particles will be emitted by randomly selecting a triangle and then a random edge of that triangle. The particle will be emitted from a random location on that edge.
  • Face Center—Particles will be emitted by randomly selecting a triangle and then emitting from the center of that triangle.
  • Face Surface—Particles will be emitted by randomly selecting a triangle and then the particle will be emitted from a random spot on that triangle.
Direction Specify the particle direction. This value is stored in the Ni_SurfaceEmitDirection attribute. Options include:
  • Use Normals—The particles direction will be based off the normal of the vertex or the triangle it was created from.
  • Use Random—The particles direction will be determined randomly.
  • Use Direction—The particles direction will be specified using the Emit Axis.
Emit X Specify the directional vector used when a particle is created on the surface of an object. This value is stored in the Ni_EmitterSurfaceEmitAxisX attribute.
Emit Y Specify the directional vector used when a particle is created on the surface of an object. This value is stored in the Ni_EmitterSurfaceEmitAxisY attribute.
Emit Z Specify the directional vector used when a particle is created on the surface of an object. This value is stored in the Ni_EmitterSurfaceEmitAxisZ attribute.
Animation
Begin Keyframe Specify the frame when to begin the emitters emission rate sequence.
End Keyframe Specify the frame when to end the emitters emission rate sequence.
Emitter Start Frame Specify the frame when the emitter becomes active.
Note: You can still use this field and the Emitter Stop Frame field, but their functionality has been superseded by animating the "Rate" of the emitter.
Emitter Stop Frame Specify the frame when the emitter becomes inactive. See note above.
Emitter Looping Type Specify if the emitter will be a looping animation or if there will be no looping. This value entered here is stored in the Ni_EmitterLoopingType attribute.
Reset on Loop Cause the particle system to kill all particles and start over when the emitter animation resets. For example, if you wanted to create the smoke from a fire, you would NOT want to set Ni_ResetOnLoop. This value is stored in the Ni_EmitterResetOnLoop attribute.
