Morphing Skinned Objects

You can use the Gamebryo Maya Plug-in to morph skinned objects. For example, you might choose to do use this kind of morphing to create facial animations for skinned characters. However, this type of morphing comes with a significant performance cost, so we recommend that you minimize the number of morphed vertices. You might create the character's face or head as a separate object, allowing only that portion to be morphed.

Within Maya there are a number of ways to morph and skin, but for consistent export, it is important to follow the steps that follow. If you don't follow these steps, it will probably look something like spaghetti, as the vertices will be in a different order between the morph targets.

To skin morphed objects

  1. Create your base shape.
  2. Texture the base shape.
  3. Triangulate the base shape.
  4. Duplicate the base shape for each target.
  5. Create the blend shape.
  6. Skin the base shape to the bones.
  7. Animate the bones and blend shape weights.
