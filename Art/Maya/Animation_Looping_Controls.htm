Animation Looping Controls

By default, the Gamebryo Maya exporter uses the playback settings to determine the style of looping. By using the Animation Looping Controls, you can control how animations behave. Click the Looping Controls icon in the Attribute Editor to create the necessary Extra Attributes.

Looping Controls icon

Select one of the following values from the Looping Type drop-down:

  • Loop Animation—plays continuously, starting over immediately after they finish
  • Reflect loop—plays continuously from the beginning to end and then reverses to play from end back to the beginning
  • No Looping—plays from the start to the end frame just once

Click Propagate to Children if you want to propagate the selected looping type to all children. This value can be overridden by looping controls found lower in the scene graph.
