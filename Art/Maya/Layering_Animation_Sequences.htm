Layering Animation Sequences

With the Sequence Editor, you can layer sequences on top of one another. This is made possible with Gamebryo's Actor Manager.

In the DestroyerBot.mb example sequences are found on two different parts of the character. The node "DestroyerBot" is the root of the character, and all of the basic character sequences are found there: Start Walk, Walk, End Walk. Turn Left, Turn Right, and Idle. The "Cockpit" node also contains a sequence. The "Scan" sequence is merely a slow movement of the robot head, scanning left and right. The Actor Manager allows the "Scan" sequence to be layered with the "Walk" sequence to create an animation where the robot walks and scans side to side.

Note

Layer sequences cannot accumulate animations.
