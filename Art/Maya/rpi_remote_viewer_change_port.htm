Changing the Gamebryo Remote Viewer Network Port Number

By default, the Remote Viewer uses TCP port number 13215 for all network communication. This default port number can be changed by creating (or modifying) the Remote Viewer configuration file. The file can be found here:

%EMERGENT_PATH%\Media\ToolPluginsData\RemoteViewerNetwork.ini

Edit (or create) the RemoteViewerNetwork.ini file and set the ListenPort to the desired port number. For example:

[ChannelManager]
ListenPort=2222
ListenIP=127.0.0.1
