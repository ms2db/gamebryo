Viewing and exporting from Maya

A number of buttons on the Gamebryo Tools Shelf enable you to view and export objects in a variety of ways.

Button Name Description
Export options Opens the Export Options dialog box.
Export Exports your scene using the settings that are defined in the Export All Options dialog box.
Export Selected Exports the selected objects using the settings that are defined in the Export All Options dialog box.
Gamebryo Maya Plug-in View Scene Displays the scene in the viewer of your choice; uses the settings that are defined in the Export All Options dialog box.
View Selected Displays the selected objects in the viewer of your choice; uses the settings that are defined in the Export All Options dialog box.
Recursive Export From the Selected Directory Opens the Browse for Folder dialog. When you have selected a directory, Maya will load each Maya file (*.mb) in that directory and "Export All" to a NIF file of the same name. The Recursive Export will continue to all sub-directories, exporting as it goes. For more information, see the Recursive export section below.
Recursive View From the Selected Directory Opens the Browse for Folder dialog. When you have selected a directory, Maya will load each Maya file (*.mb) in that directory and view that scene. The Recursive View will continue to all sub-directories. For more information, see the Recursive view section below.

Recursive export

After selecting the folder to search, the Gamebryo Recursive Export dialog will contain the files in a list box.

Recursive Export dialog box

Use this field... To...
Silent Export Disable scene viewing. When this option is selected (checked), it also suppresses the Script Selection dialog. The last script used to export this scene will be used during export. If no previous script information is available the "Default Export" script will be used.
Force Default Export Export the scene using the "Default Export" script or "Default Character Export" if the scene contains a character. Force Default Export is only available during silent export.
Export Next Scene Export a file by selecting the file and clicking the "Export Next Scene" button. After exporting this scene, the list box will automatically update to the next scene.
Export All Remaining Export all of the remaining scenes in the list box. This automated export is only valid when "Export Silent" is selected (checked).

Recursive view

After selecting the folder to search, the Gamebryo Recursive View dialog will contain the files in a list box. You can choose to view by selecting the file and clicking the "View Next Scene" button. After exporting this scene, the list box will automatically update to the next scene.
