Key Frame Reduction in Maya

The key frame reduction window is designed to automate the time consuming task of reducing character animation data. When characters are animated using IK controllers, you must "bake" the animation to convert it to key frame animations. During the baking process, an enormous number of unnecessary keys are created. Key frame reduction is necessary to control the size of animations by removing unnecessary animation keys. Unfortunately, key frame reduction can introduce compounding numerical error when the start or end keys of character sequences are reduced. You might see this where a character will slowly sink into the ground the farther the character runs or perhaps a slight rotation is applied every time the character moves. To solve this problem, the key frame reduction window will avoid removing keys at the beginning and end of a sequence. The first object selected in Maya will be used to determine what character sequences you will use. Internally the key frame reduction is done using Maya's "Simplify" command.

Simplify Parameters
Simplify Method Choose between the classic simplify routine used by Maya or a newer routine which is specialized for the dense data associated with baking or motion capture data. The Classic options uses Maya's simplify command, The Dense Data uses the filterCurve command. The Time Tolerance and Value Tolerance parameters are used by simplify.
Time Tolerance Specify the x-axis tolerance (defaults to 0.05 seconds) for time-input animCurves. For more information, refer to your Maya product documentation.
Value Tolerance Specify the value tolerance (defaults to 0.01). For more information, refer to your Maya product documentation.

Character Root and Character Sequences
Character Root View the name of the character. The character is found by searching up from the first selected object when you clicked the Key Frame Reduction button.
Character Sequences Select the sequence that you want to reduce.
Jump To Selected Sequence Set the time slider to the beginning and end of the currently selected sequence
Reduction Method
Reduce Selected Sequence Reduce the currently selected sequence
Reduce All Sequences Reduce all sequences of this character.
Note: Only the keys contained in sequences will be reduced.
Reduce Time Slider Range Reduce all keys within the range that is specified by the time slider regardless of whether or not the keys are in a sequence. The keys at the beginning and end of every sequence will be preserved.
