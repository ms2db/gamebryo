NiParticleSystem*Spawner Node Reference

The spawner node is used to associate particle systems whose particles spawn particles (Input) with particle systems being spawned (Output). This node is used for Living, Death, and Collision Spawners.

Connections

NiParticleSystem*Spawner Connections Parameters user interface

Attribute Keyable Description
Input No A convenient way to view and navigate to the particle system that is spawning particles.
Output No A convenient way to view and navigate to the particle system that represents the spawned particles.

Base Parameters

NiParticleSystem*Spawner Base Parameters user interface

Attribute Keyable Description
Percentage Spawned No A ratio determining the likelihood of a particular particle being spawned. Must be between 0.0 and 1.0.
Min Number to Spawn No The minimum number of particles to spawn for any given original particle.

This value may be the same as the Max Number to Spawn, in which case the same number of particles will always be spawned. If these values differ, the number of particles actually spawned will be randomly selected from this range.

For example, if a living spawner has a Living Spawn rate of 1.0, a given particle will spawn secondary particles once every second, but with Min and Max set to 10, it will spawn 10 particles instead of just one (all 10 at once, as opposed to evenly spaced out over a second).
Max Number to Spawn No The maximum number of particles to spawn for any given original particle.

This value may be the same as the Min Number to Spawn, in which case the same number of particles will always be spawned. If these values differ, the number of particles actually spawned will be randomly selected from this range.
Speed Factor No This attribute controls how much the speed of the spawned particle will vary from that of the original particle.

For example, if the original particle is traveling at 5.0 and Speed Factor is 1.5, then the spawned particles will be traveling at 6.5. Likewise, if Speed Factor is -1.5, spawned particles will be traveling at 3.5.
Speed Factor Variance No Adds randomness to the spawned particle’s speed. A spawned particle’s speed will be calculated as:

Original Particle Speed + [Speed Factor +/- Speed Factor Variance]
Direction Chaos No Determines how much the direction of the spawned particle will vary from that of the original particle.
Life Span No The life span (in seconds) of spawned particles.
Life Span Variance No Determines the amount by which the life span for the spawned particle will vary. The life span will be in the range of [Life Span +/- Life Span Variance]
