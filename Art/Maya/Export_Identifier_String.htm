Export Identifier String

An identifier string is added to each NIF file generated by the Gamebryo Maya Plug-in. This identifier will aid in art production by allowing artists to determine the origin of a particular NIF file. The identifier can be found in a NiStringExtraData object associated with the root node of the NIF file. It is visible in the Gamebryo Asset Viewer or using the Scene Graph Printer.

The information in the identifier string includes:

  • Maya File Name and Path (D:\MyGameArt\MainCharacter.mb)
  • Gamebryo Version
  • Gamebryo Maya Plug-in Version
  • NiMultiShader Version
  • Maya Product Name
  • Target Platform
