Command Line Export

Maya can be run from the command line and passed an argument that specifies a MEL script to execute. To do this, provide the argument -command followed by MEL script statements surrounded in quotes. Using this functionality, a batch export system can be constructed.

Note

The Gamebryo Maya Plug-in is not compatible with Maya's batch mode, specified by passing -batch or calling mayabatch.exe. Emergent recommends running Maya in GUI mode (just use maya.exe) for command line processing.

Although any arbitrary script can be executed when Maya starts from the command line, the Gamebryo Maya plug-in provides a helper export script named GamebryoMultiBatchExport.mel. The script takes in a file path that points to a file with a list of files to export. Each line of the file list refers to a Maya source art file and optionally to one or more destinations. Each line can take one of the formats listed below:

  • SourceMayaFilePath
  • SourceMayaFilePath;DestNifFilePath
  • SourceMayaFilePath;Platform=DestNifFilePath;Platform=DestNifFilePath;...

In the case where only a source Maya file is specified, the export script will create a file in the same directory as the source file with the .nif extension. Appending a destination path separated by a semicolon will allow you to export to a chosen path. The third option is discussed in more detail in the platform-specific section below.

The export options and export scripts that are stored within each Maya file will be used for each file. In the case where no export settings are present in the file, the most recently used values will be used.

Platform specific export

The Gamebryo batch export Mel Script has the ability to export to multiple platforms at once for each source Maya file. To specify a platform for export, you put the string 'platform=' before the destination path where the valid values for platform are any of the following:

  • DX9
  • PS3
  • D3D10
  • D3D11
  • Xenon

Multiple platforms can be specified per Maya file as long as they are delimited by a semicolon.

Command line example

Below is a example of how to call the script from the command line.

maya.exe -command "GamebryoMultiBatchExport(\"C:/data/filelist.txt\")"

Below is an example of the contents of a file list text file. Note that not all files need to be exported to the same platforms.

C:\Source\ArtModels\IKRig.mb;DX9=c:\Game\DX9\IKRig.Nif;Xenon=c:\Game\360\IKRig.Nif;PS3=c:\Game\PS3\IKRig.Nif
C:\Source\ArtModels\Joints.mb;DX9=c:\Game\DX9\Joints.Nif;Xenon=c:\Game\360\Joints.Nif;PS3=c:\Game\PS3\Joints.Nif
C:\Source\ArtModels\Animation.mb;c:\MayaTest\Animation.Nif
...
