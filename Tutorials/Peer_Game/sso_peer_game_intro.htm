Peer Game: Introduction

Peer Game builds upon Base Game functionality to create a simple multiplayer simulation. In terms of supported gameplay, it is nearly identical to Base Game, adding only the ability for multiple players to connect, to observe one anothers' avatars moving within the game world, and to (minimally) interact. As its name implies, Peer Game uses a simple peer-to-peer topology.

In this section, we include a tutorial that walks you through the process of extending Base Game into Peer Game. In the process, the various parts of the Peer Game application are described. We also discuss why these pieces are used and how they fit into the Game Framework. We also include a description of the Peer Game demo itself, including an overview of how it works, its controls, and its principal source, script, and configuration files.
