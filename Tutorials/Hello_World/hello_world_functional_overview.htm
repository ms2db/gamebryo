Hello World Demo Functional Overview

The Hello World demo is driven by the Gamebryo Game Framework, a few C++ files that are specific to the application, and some application-specific data, including script files for implementing game logic. The demo contains one level that includes the following:

  • a simple room
  • one computer-controlled character (called AIJack)
  • one player-controlled character (called Avatar)
  • two boxes

When you start the demo, the AIJack (the smaller of the two characters) starts running toward the near left corner of the room. When AIJack reaches the corner, he will stop moving and stay there.

Independent of the AIJack, you can move the Avatar around in the room. You can also pick up a box, move to a different location, and drop off the box at the new location.

The level and the entities that are included in the demo are created in Toolbench. The entity behaviors are written in Lua. The primary purpose of Hello World is to illustrate a simplest possible Game Framework-based application that demonstrates:

  • Programmer features:
    • Setting up a solution for compiling a Game Framework application
    • Creating a custom system service
    • Creating and initializing common Foundation and Game Framework classes and services
    • Asynchronous creating entities and waiting for creation to complete
  • Designer and Scripter features:
    • Creating models using the Entity Modeling tool
    • Creating a level using the World Builder tool
    • Implementing game logic using Lua scripted behaviors
