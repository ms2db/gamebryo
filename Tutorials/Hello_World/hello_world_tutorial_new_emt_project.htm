Hello World: Creating a New Toolbench Game Solution

A game solution is the container that provides easy access to a working set of game content. A game solution is analogous to a Visual Studio solution. In this section of the tutorial, you'll create a new game solution in Toolbench. You'll also add some sample assets to your game solution.

Create a New EMT Project

In Toolbench, you need to create a new EMT project so you can model your application. First, run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench). Within Toolbench, select File > New Content. In the New Content dialog, select Game Solution within the Content Categories list on the left side. Give your new solution a name and set the output folder. For the purposes of this tutorial, we create our new solution in the following location:

C:\Emergent\HelloWorldTutorial\Media

Toolbench: New Content dialog

When you've entered the name of your solution and the output folder, click OK.

Tip

The remainder of this tutorial assumes you've named your solution 'HelloWorld'. If you've given your solution a different name, make note of the name so you can use it later.

Following the Game Solution Wizard

After you define the name of your new solution and the folder to use for it, the Game Solution Wizard launches. This wizard walks you through the remaining steps in your game solution setup.

Organize Game Content

The Organize Game Content dialog is the first dialog in the wizard. If you create your game solution in a folder that doesn't have a pre-existing asset web, the Asset Web Root is relative to your solution's output folder. Click Next to continue.

Game Solution Wizard: Organize Game Content

Copy Standard Libraries

The next dialog in the Game Solution Wizard is the Copy Standard Libraries dialog. Gamebryo LightSpeed comes with a set of standard libraries that you can use as a basis for the models in your game. These libraries include models like a base model for all Entities, lights, cameras, and PhysX models. In this tutorial, we want to import the Standard Model Library but not the PhysX Model Library. Go ahead and check the Standard Model Library checkbox and then click Next.

Game Solution Wizard: Copy Standard Libraries

Create Initial Assets

The Create Initial Assets dialog is next. The wizard will create empty entity model and block files for you. You can bypass this step by unselecting the appropriate checkbox. You can also put the resulting files in different directories and give them different names. For this tutorial, go ahead and leave the checkboxes selected to accept the wizard-generated defaults.

Game Solution Wizard: Create Initial Assets

Click Finish to finalize the creation of your new game solution. In Toolbench, the result should look something like this:

Toolbench: Game solution

Preparing HelloWorld Sample Assets

The HelloWorld demo comes with a set of pre-created mesh and animation assets. To use them during this tutorial, you need to copy them to your project directory. We've provided a script (CopySampleAssets.bat) that automates this process for you. To prepare HelloWorld sample assets for use in this tutorial, run the following, passing the full path to your game solution file on the command line:

CD %EMERGENT_PATH%\Samples\GameFrameworkTechDemos\HelloWorld
CopySampleAssets.bat C:\Emergent\HelloWorldTutorial\Media
