Hello World Demo Usage and Controls

In the Hello World demo, you can control the Avatar's movements as follows:

  • Move forward
  • Turn right/left
  • Pick up a box when close to it
  • Drop the box after the Avatar has picked it up

The following table outlines the keyboard and game pad controls.

Keyboard Game Pad Action
UP_ARROW Left Analog Forward Move the Avatar forward.
LEFT_ARROW, RIGHT_ARROW Left Analog Left, Right Turns the Avatar left/right.
M/m A button Drop the box.
N/n B button Pickup the box.
ESC Start & Select/Back buttons Quit the application.
