Base Game Demo Code Overview

By default, files that are specific to the Base Game demo install to the following locations:

  • Source files: %EMERGENT_PATH%\Samples\GameFrameworkTechDemos\BaseGame
  • Entity behavior scripts: %EMERGENT_PATH%\Media\Samples\GameFramework\BaseGame
  • Configuration files: %EMERGENT_PATH%\Samples\GameFrameworkTechDemos\BaseGame
File Description
Source files
BaseGame.cpp
  • Primary system service for BaseGame.
  • Instantiates required system services, registers them with the ServiceManager.
  • Initializes logging.
  • Verifies pre-requisite system services are running in OnInit.
GameLoadingState.cpp
  • Game State responsible for loading the block file.
GamePlayingState.cpp
  • Game State responsbile for game playing.
  • Creates the Avatar and camera asynchronously.
  • Registers handlers and creates action maps for keyboard controls.
Main.cpp
  • Main entry point for the application.
  • Instantiates the BaseGame instance and calls the Go() method.
Entity behavior scripts
EseAnimated.lua
  • SetAnimationSequence() behavior sets a new animation sequence.
EseBgAIJack.lua
  • Behaviors for the BgAIJack entities.
  • Goal Selection between two goals.
EseBgAvatar.lua
  • Behaviors for the BgAvatar entity.
  • Handlers for user input.
  • OnReset() behavior to reset Avatar position.
EseBgBox.lua
  • Behaviors for the BgBox entities.
  • Remove() behavior to destroy a box.
  • Carry() and Drop() behaviors used when a box is picked up and dropped.
EseBgDoor.lua
  • Behaviors for BgDoor entities.
  • Open and Close animation control.
EseBgNoSign.lua
  • Behavior for NoSign entities.
  • Remove() behavior that make the "do not enter" sign on door way disappear.
EseCollideExecuteLocal.lua
  • OnPhysXTrigger() behavior to signal collision.
EseCollideExecuteRemote.lua
  • OnPhysXTrigger() behavior to signal collision on a remote entity.
EseInventory.lua
  • Behaviors to implement picking up and dropping an entity.
EseSecretRoomTrigger.lua
  • OnOpenSecretRoom() behavior to uncover the secret room by destroying the ceilings.
EseTreasureChest.lua
  • CreateSecretTrigger() behavior plays the animation of opening Treasure Chest and makes the AIJacks disappear.
EseTriggeredAnimation.lua
  • TriggerAnimation() behavior sets a new animation.
point3.lua
  • Point3 helper functions.
