Rapid Iteration: Introduction

Rapid iteration is the concept of editing content while simultaneously previewing and testing it on-target. Each edit or group of edits is automatically transmitted to a preview application or a running instance of a game. The application or game incrementally reloads the content without restarting. This development process avoids lost time while deploying assets and restarting the application for each set of edits.
