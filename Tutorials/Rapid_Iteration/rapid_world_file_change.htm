Rapidly Iterating on a World File

  1. Double-click HelloWorld.block in the Hello World game solution opened in Toolbench to open World Builder.
  2. Find a Box in the world and select it.
  3. In the Entity Editor (right pane) in Toolbench, set the NifAsset property under Mesh to TreasureChest.
  4. Set the Scale property under Placeable to '0.3'.
  5. Click the Export Block button on the World Builder Toolbar. If prompted, overwrite the existing HelloWorld.xblock file under the %EMERGENT_PATH%\Media\Samples\GameFramework\HelloWorld directory.

    In the application window, the box in the main room becomes a Treasure Chest.

  6. To keep the changes, click save to commit them to the source block. This will also export the block.

As the world changes, the Rapid Iteration Service notifies the running game that a new world file has been detected. The running game then attempts to load the updated world.
