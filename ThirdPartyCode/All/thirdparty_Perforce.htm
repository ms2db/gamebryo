Third Party Code - Perforce

Version

Perforce C/C++ API r08.2 http://www.perforce.com/perforce/loadsupp.html#api

Description

Perforce is a commercial source revision control system. This system allows multiple local or remote people to collaborate on a shared set of files. A key feature of revision control is the ability to record incremental changes between files and rollback to previous versions.

Usage

Toolbench can optionally use Perforce to revision control asset web files. The code that interfaces directly with the Perforce C/C++ API is wrapped in a plug-in DLL (see etkRCSP4). Toolbench exposes some revision control functionality through its UI to improve the user experience.

The etkRCSP4 DLLs included with Gamebryo LightSpeed are linked with the Perforce C/C++ SDK's libraries. Neither the Perforce C/C++ API source code nor the Perforce C/C++ SDK libraries are included with the LightSpeed SDK. To rebuild the etkRCSP4 plug-in, you must download the Perforce C/C++ SDK from the Perforce web site (see above) to obtain the necessary headers and libraries.
