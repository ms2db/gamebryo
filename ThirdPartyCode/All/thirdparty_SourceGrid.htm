Description

SourceGrid is a Windows Forms grid control.

Usage

The Gamebryo Animation Tool includes a .NET control called SourceGrid that was originally developed by Davide Icardi. The relevant files are in the Animation Tool directory as:

%EMERGENT_PATH%\Source\Tools\AnimationTool\SourceGrid2\*
