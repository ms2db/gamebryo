Third Party Code - DockPanel Suite

Version

DockPanel Suite 2.2.0.0: http://sourceforge.net/projects/dockpanelsuite.

Description

The docking library for .Net Windows Forms development which mimics Visual Studio .Net developed by Weifen Luo.

Usage

This control is used in Emergent's Animation Tool and Toolbench applications. They are used to add "docking" behavior to some controls (the ability to float or link a tray of controls to other controls in an interface).

Toolbench uses a slightly modified version of the DockPanel source, which can be found under Source/Tools/Toolbench/WeifenLuo.DockPanel. The modifications are to support the UI styles applied in Toolbench.
