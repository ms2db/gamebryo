Third Party Code - ImageButton

Version

No named version given.

Description

This class dispays a Windows Forms button that contains only an image.

Usage

The Animation Tool includes a .NET control originally written by O&O Services GmbH. The Gamebryo files affected include:

  • %EMERGENT_PATH%\Source\Tools\AnimationTool\AnimationToolUI\Controls\ImageButton.cs
  • %EMERGENT_PATH%\Source\Tools\AnimationTool\AnimationToolUI\Controls\ImageButton.resx
