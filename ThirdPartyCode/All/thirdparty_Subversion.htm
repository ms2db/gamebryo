Version

Description

Subversion is an open source revision control system. This system allows multiple local or remote people to collaborate on a shared set of files. A key feature of revision control is the ability to record incremental changes between files and rollback to previous versions.

Usage

Toolbench can optionally use Subversion to revision control asset web files. The code that interfaces directly with the Subversion API is wrapped in a plug-in DLL (see etkRCSSVN). Toolbench exposes some revision control functionality through its UI to improve the user experience.

The Subversion source is not included with the Gamebryo LightSpeed SDK. The etkRCSSVN DLLs included with the SDK are linked with the Subversion libraries. Rebuilding the etkRCSSVN plug-in requires the Subversion SDK to be downloaded from the Subversion web site (see above).
