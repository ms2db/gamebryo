Third Party Code - Atmospheric Scattering Shader Implementation

Version

"Accurate Atmospheric Scattering", Chapter 16 in GPU Gems 2, Editor: Matt Pharr, Addison Wesley, © Copyright 2005.

Description

This code is a shader implementation to approximate atmospheric scattering calculations.

Usage

The Gamebryo atmospheric scattering code in NiEnvironment files:

%EMERGENT_PATH%\Source\CoreRuntime\EngineLibs\NiEnvironment\NiAtmosphereMaterialNodeLibrary\*
