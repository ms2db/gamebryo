Third Party Code - Xerces C++

Version

Xerces-C++ 2.7 : http://xml.apache.org/xerces-c.

Description

Xerces-C++ is a validating XML parser written in a portable subset of C++. It is licensed under the Apache 2.0 license, a copy of which is included with the distribution and can be found at http://www.apache.org/licenses/LICENSE-2.0.txt.

Usage

Xerces is used to convert COLLADA material files to NSF files (%EMERGENT_PATH%\Source\Tools\Collada2NSF\).
