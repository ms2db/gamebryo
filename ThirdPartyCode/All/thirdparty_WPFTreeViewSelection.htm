Third Party Code - WPF TreeView Selection

Version

1.0. http://www.codeproject.com/KB/WPF/TreeView_SelectionWPF.aspx

Description

This code helps manage selection in a WPF TreeView control.

Usage

The WPF TreeView Selection code is integrated with Toolbench to assist with the tree UI elements in the Game Solution, Entity Modeling Tool, and World Builder. It has been modified for Toolbench use.

Location: %EMERGENT_PATH%\Source\Tools\Toolbench\Plugins\Emergent.Toolbench.UI
