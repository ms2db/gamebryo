NiApplicationMetrics

<ApplicationMetrics.h>

Description

This class contains the NiMetrics enumeration and a static list of metric names. The NiMetrics enumerates measurements that are specific to NiApplication and memory. NiApplicationMetrics::ms_acNames is the static list of names for the enumerations.

Supported Enumerations

The NiApplicationMetrics class contains one public enumeration.

NiApplicationMetrics::NiMetric

The NiMetric enumeration identifies the type of values that are being measured. This enumeration can be one of the following:

Enumeration Description
FRAMERATE The number of frames rendered per second.
CULL_TIME The time spent in culling objects.
UPDATE_TIME The time spent updating objects.
RENDER_TIME The time spent rendering objects.
CREATE_SCENE_TIME The time spent creating the scene.
INITIALIZED The time spent initializing the scene.
ACTIVE_MEMORY The active memory.
ACCUMULATED_MEMORY The accumulated memory.
ACTIVE_ALLOCATIONS The active memory allocations.
ACCUMULATED_ALLOCATIONS The accumulated memory allocations.
ACTIVE_TRACKER_OVERHEAD The active tracker overhead.
ACCUMULATED_TRACKER_OVERHEAD The accumulated tracker overhead.
UNUSED_BUT_ALLOCATED_MEMORY The unused but allocated memory.
NUM_METRICES The number of items in this enumeration.
