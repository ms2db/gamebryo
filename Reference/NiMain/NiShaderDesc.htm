NiShaderDesc

: public NiShaderRequirementDesc

<NiShaderDesc.h>

Description

NiShaderDesc provides a description for a Gamebryo Shader.

Constructors

NiShaderDesc()

The default constructor creates an uninitialized NiShaderDesc. It clears the implementation description array and the attribute list.

Supported Member Functions

const unsigned int GetNumberOfImplementations() const

Returns the number of implementations provided by the shader.

const NiShaderRequirementDesc* GetImplementationDescription(unsigned int uiImplementation) const

Retrieve the description for the implementation at index uiImplementation. If no implementation exists there, this function will return NULL.

const unsigned int GetNumberOfAttributes() const

Returns the number of attributes the shader accesses.

const NiShaderAttributeDesc* GetFirstAttribute() const

Returns the first attribute description in the array.

const NiShaderAttributeDesc* GetNextAttribute() const

Returns the next attribute description in the array. GetFirstAttribute and this function provide a means for iteration over attribute descriptions.

const NiShaderAttributeDesc* GetAttribute(const char* pcName) const

Returns the description for the attribute of the given name. If no such attribute exists, this function will return NULL.
