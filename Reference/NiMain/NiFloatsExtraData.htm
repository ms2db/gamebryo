NiFloatsExtraData

: public NiExtraData

<NiFloatsExtraData.h>

Description

Extra data in the form of an array of arbitrary size of floating point values that can be attached to a NiObjectNET.

Constructors

NiFloatsExtraData(const unsigned int uiSize, const float* pfValue)

Creates an extra data object that stores an array of arbitrary size of floating point values.  A copy is made of the input array of floating point values.

Supported Member Functions

void GetArray(unsigned int &uiSize, float* &pfValue) const

void SetArray(const unsigned int uiSize, const float* pfValue)

These functions retrieve and set the extra data as an array of arbitrary size of floating point values.  For the latter, a copy is made of the input array of floating point values, and any existing data in the object is deleted.

float GetValue(const unsigned int uiIndex) const

bool SetValue(const unsigned int uiIndex, float fValue)

These functions retrieve and set an individual entry in the array of floating point extra data.
