NiDynamicEffectStateManager

: public NiMemObject

<NiDynamicEffectStateManager.h>

Description

The NiDynamicEffectStateManager class is used by the renderer to ensure that each NiDynamicEffectState object is unique. Typically customers will not need to directly interact with this class, but it is possible to specify the map size if desired.

Supported Static Member Functions

static inline NiDynamicEffectStateManager* GetEffectManager();

Returns the global NiDynamicEffectStateManager instance.

Supported Member Functions

inline void ResizeMap(NiUInt32 uiPrimeSize);

Resizes the internal map to the size specified by uiPrimeSize, which must be a prime number.
