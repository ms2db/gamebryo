NiCompositeValidator

: public NiRenderClickValidator

<NiCompositeValidator.h>

Description

This class is a render click validator that holds other render click validators. It is an abstract class, because it does not implement the pure virtual function ValidateClick in NiRenderClickValidator. Thus, classes must derive from NiCompositeValidator to use the list of validators stored in this class.

Supported Member Functions

void AppendValidator(NiRenderClickValidator* pkValidator)

void PrependValidator(NiRenderClickValidator* pkValidator)

void RemoveValidator(NiRenderClickValidator* pkValidator)

void RemoveAllValidators()

NiTPointerList<NiRenderClickValidatorPtr>& GetValidators()

const NiTPointerList<NiRenderClickValidatorPtr>& GetValidators() const

These functions provide access to the list of render click validators stored in this validator.
