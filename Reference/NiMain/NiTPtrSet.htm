NiTPtrSet

: public NiTSet

<NiTPtrSet.h>

Description

NiTPtrSet is an extension of NiTSet that is safe for smart pointer use.  NiTPtrSet sets its elements to zero when they are removed so that the ref count for smart pointers will be properly decremented.

Constructors

NiTPtrSet(unsigned int uiInitialSize = 1)

The constructor takes the initial size of the internal storage for the set.

Supported Member Functions

void RemoveAt(unsigned int uiIndex)

This function removes the element at index uiIndex, setting the removed element equal to zero so that smart pointers will decrement their ref count.

void OrderedRemoveAt(unsigned int uiIndex)

This function removes the element at index uiIndex, preserving the order of the set and setting the removed element to zero so that smart pointers will decrement their ref count. WARNING: This function is slow and should be used with caution.

void RemoveAll()

This function removes all the elements from the set, setting each one equal to zero so that smart pointers will decrement their ref count.
