NiGPUProgramDescriptor

: public NiMemObject

<NiGPUProgramDescriptor.h>

Description

This class is the base class for all GPU program descriptors. Descriptors use a bitfield to encode data about how a GPU program should be assembled. This bitfield is 128-bits in size.
