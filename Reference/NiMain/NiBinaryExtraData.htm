NiBinaryExtraData

: public NiExtraData

<NiBinaryExtraData.h>

Description

Extra data in the form of a user-defined "blob" or chunk of data, of user-defined size, that can be attached to an NiObjectNET. The data is stored in a char* and is deleted as such in the destructor. Keep this in mind when assigning other types of data to this extra data object. For example, if it is pointing to an array of classes, the destructor will not be called on those objects when deleted. Also, since the data is streamed as an array of chars, it is not guaranteed to be aligned properly when streamed in.
