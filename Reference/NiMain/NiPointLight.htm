NiPointLight

: public NiLight

<NiPointLight.h>

Description

NiPointLight is an object that represents a point light source in a scene. Point lights have a specific location in space and a distance attenuation function. Point lights project illumination in all directions from their model space origin. They may be positioned anywhere by changing the translation of the light object.

Constructors

NiPointLight()

The default constructor creates a point light with a location at [0.0,0.0,0.0], a constant attenuation factor of 0.0, a linear attenuation factor of 1.0, and a quadratic attenuation factor of 0.0.
