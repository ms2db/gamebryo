NiGPUProgram

: public NiRefObject

<NiGPUProgram.h>

Description

This class represents a program that can be run on a platform's Graphics Processing Unit.

Constructors

NiGPUProgram(ProgramType eType)

Create a program of the specified type.

Supported Public Member Functions

ProgramType GetProgramType()

Returns the program type of this GPU program.

virtual unsigned int GetVariableCount()

Returns the number of uniform variables for this GPU program.

virtual const char* GetVariableName(unsigned int uiIndex)

Returns the name of a variable specified by the given index for a GPU program.

Notes

ProgramType Enum

This enumeration indicates the type of GPU program.

Enumeration Meaning
PROGRAM_VERTEX Program run per vertex
PROGRAM_PIXEL Program run per pixel
PROGRAM_GEOMETRY Program run per geometry primitive
PROGRAM_HULL Pre-tessellation program run per patch
PROGRAM_DOMAIN Post-tessellation program run per tessellated vertex
PROGRAM_COMPUTE Non-rendering program that does general computation on the rendering device
