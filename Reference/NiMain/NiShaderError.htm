NiShaderError

<NiShaderError.h>

Description

NiShaderError is a mechanism to assist in error reporting by the Gamebryo shader system. Refer to the error topics of the various shader systems, such as NiD3DError, for the shader system-specific error codes.

Typedefs

typedef unsigned int NiShaderError

NiShaderError is typedef-ed as a 32-bit unsigned int. The upper 16 bits are reserved for describing the identity of the shader system that produced the error, and the lower 16 bits are reserved for specifying the actual error code.

Enumerations

NiShaderErrorPrefix

NiShaderErrorPrefix is an enumerated type that identifies the shader system that produced an error:

Prefix Meaning
NISHADERERR_GLOBALPREFIX Global error - refers to all NiShader systems.
NISHADERERR_D3DPREFIX D3D shader error - refers to all D3D-based NiShader systems.
NISHADERERROR_DX9PREFIX DX9 shader error - refers to the DX9 NiShader system.
NISHADERERROR_XENONPREFIX Xenonshader error - refers to the Xenon NiShader system

Macros

#define NiDeclareShaderError(name, prefix, value) \
NiShaderError name = (((prefix) << 16) | (value))

Declares an error named name, with a shader system prefix of prefix and an error value of value.

#define NiDeclareGenericShaderError(name, value) \
NiDeclareShaderError(name, NISHADERERR_GLOBALPREFIX, (value))

Declares an error named name, with a shader system prefix of NISHADERERR_GLOBALPREFIX and an error value of value.
