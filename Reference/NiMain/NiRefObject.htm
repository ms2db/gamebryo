NiRefObject

: public NiMemObject

<NiRefObject.h>

Description

NiRefObject provides the reference counting functionality needed for use of Gamebryo smart pointers. The use of smart pointers is described in the object system documentation. NiRefObject stores a reference count and provides functions for manipulating that reference count.

Constructors

NiRefObject()

Sets the reference count to 0.

Static Member Functions

static unsigned int GetTotalObjectCount()

Returns the number of currently allocated objects that are instances of classes derived from NiRefObject. (NiRefObject is not strictly abstract, but it is abstract in practice.)

Supported Member Functions

void IncRefCount()

void DecRefCount()

Increment and decrement an object's reference count. These functions are called automatically when assignments are made to Gamebryo smart pointers, and never need to be called by any code outside of the class which implements smart pointers, NiPointer.

unsigned int GetRefCount() const

Returns an object's reference count.

Supported Protected Member Functions

virtual void DeleteThis()

This method is called in order to free the memory associated with the current object when the reference count drops to zero. This method is virtual in order to facilitate derived classes with custom deletion code.
