NiAlphaSortProcessor

: public NiBackToFrontSortProcessor

<NiAlphaSortProcessor.h>

Description

This class is a render list processor that first renders all opaque objects in the order they were found in, and then sorts all transparent objects in back-to-front order and renders those objects in that order. The sort can occur either by bounding volume center or bounding volume closest point. The no sort hint on NiAlphaProperty objects can be optionally observed. The functionality of this processor is very similar to that of NiAlphaAccumulator.

Constructors

NiAlphaSortProcessor(bool bObserveNoSortHint = true, bool bSortByClosestPoint = false)

The sole constructor for this class takes Boolean variables indicating whether or not to observe the no sort hint on NiAlphaProperty objects and whether to sort by closest point or bounding volume center.

Overridden Virtual Functions

virtual void PreRenderProcessList(const NiVisibleArray* pkInput, NiVisibleArray& kOutput, void* pvExtraData)

Overridden from NiRenderListProcessor. This function first renders all opaque geometry objects in pkInput in the order they appear in that array. Transparent geometry objects then sorted and rendered. The kOutput array will be empty when this function returns.

Supported Member Functions

void SetObserveNoSortHint(bool bObserveNoSortHint)

bool GetObserveNoSortHint() const

These functions set and get whether or not to observe the no sort hint on NiAlphaProperty objects when rendering. If this value is true and a geometry object has an NiAlphaProperty whose GetNoSorter function returns true, it will be treated as if it were opaque and will be rendered immediately rather than sorted back-to-front before being rendered.
