NiWireframeProperty

: public NiProperty

<NiWireframeProperty.h>

Description

An NiWireframeProperty allows applications to switch between drawing geometry as solid or as wireframe outlines.

Constructors

NiWireframeProperty()

This default constructor creates a property with the wireframe enable flag set to false.

static int GetType()

static NiWireframeProperty* GetDefault()

See NiProperty for details on these non-virtual functions.

Supported Member Functions

bool GetWireframe() const

void SetWireframe(bool bWireframe)

Retrieve and set the wireframe enable flag. If this flag is set to true, then the objects in the subtree below will be drawn using wireframe. If it is set to false, then objects will be drawn as solid.
