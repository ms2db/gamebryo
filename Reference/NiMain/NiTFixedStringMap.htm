NiTFixedStringMap

: public NiTFixedStringMapBase

<NiTFixedStringMap.h>

Description

A mapping or hash table between NiFixedString keys and any type of value. This class is useful for mapping NiFixedString-based names to objects. See the documentation for NiTMap.

Constructors

NiTFixedStringMap(unsigned int uiHashSize = 37)

The uiHashSize argument is sent to the base class constructor.
