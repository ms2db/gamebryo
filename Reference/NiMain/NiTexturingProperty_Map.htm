NiTexturingProperty::Map

: public NiMemObject

<NiTexturingProperty.h>

Description

NiTexturingProperty::Map is a nested class of NiTexturingProperty and acts as a combination of a texture, a filter mode, a clamping mode, and an index to a set of texture coordinates. This class does not contain the texture coordinate data itself, but instead provides information about the data. A discussion of the supported filter modes and clamp modes is contained in the NiTexturingProperty documentation and in the Texturing documentation.

Constructors

Map()

Default constructor. Constructs a Map with no texture, a clamp mode of WRAP_S_WRAP_T, a filter mode of FILTER_BILERP, and a texture set index of 0.

Map(NiTexture* pkTexture, unsigned int uiIndex, NiTexturingProperty::ClampMode eClampMode = WRAP_S_WRAP_T, NiTexturingProperty::FilterMode eFilterMode = FILTER_BILERP, NiTextureTransform* pkTextureTransform = NULL)

Constructs a Map with the given image, textures set index, clamp mode, filter mode, and texture transform.

Map(const Map& kObj)

Constructs a new Map based on the Map passed in. Texture data is shared.
