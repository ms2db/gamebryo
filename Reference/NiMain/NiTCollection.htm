NiTCollection

<NiTCollection.h>

Description

NiTCollection is a set of templated utility classes that adhere to the TAlloc interface requirements for NiTArrays and NiTSets.

NiTMallocInterface

This class implements the interface for allocating arrays of primitive or intrinsic types of data.

NiTNewInterface

This class implements the interface for allocating arrays of NiMemObject derived objects.

NiTExternalNewInterface

This class implements the interface for allocating arrays of objects that are not managed by the engine.
