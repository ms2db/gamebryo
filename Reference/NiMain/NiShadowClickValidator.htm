NiShadowClickValidator

: public NiRenderClickValidator

<NiShadowClickValidator.h>

Description

This class is a view render click that holds a reference to the NiShadowGenerator object that contains it, a dirty flag that defines if the click needs to be executed, and a force render flag that forces the click to always be executed. Shadow render click objects are used to render geometry to shadow maps for the shadowing system.

Overridden Virtual Functions

virtual bool ValidateClick(NiRenderClick* pkRenderClick, unsigned int uiFrameID)
This function returns a boolean that defines if the specified render click should be executed for the specified frame.

Supported Member Functions

void SetCamera(NiCamera* pkCamera)

NiCamera* GetCamera() const

These functions set and get the scene NiCamera provided to the shadow manager. This is the camera that is used to render the final scene, not the camera used to render to a shadow map.

void SetCullingProcess(NiCullingProcess* pkCullingProcess)

NiCullingProcess* GetCullingProcess() const

These functions set and get the culling process that will be used on the geometry in the scenes associated with the render click.
