NiClickRenderStep

: public NiRenderStep

<NiClickRenderStep.h>

Description

This class is a render step that holds and renders a list of NiRenderClick objects. It contains a pure virtual function for obtaining a list of render clicks and simply iterates through that list rendering each render click.

Overridden Virtual Functions

virtual bool SetOutputRenderTargetGroup(NiRenderTargetGroup* pkOutputRenderTargetGroup)

Overridden from NiRenderStep. This function sets the render target group on the last render click in the list of render clicks to be pkOutputRenderTargetGroup. If no render clicks exist, this function returns false.

virtual NiRenderTargetGroup* GetOutputRenderTargetGroup()

Overridden from NiRenderStep. This function returns the render target group that is set on the last render click in the list of render clicks. If the render click's render target group is null, this function returns the renderer's default render target group. If no render clicks currently exist, this function returns a null pointer.

virtual unsigned int GetNumObjectsDrawn() const

virtual float GetCullTime() const

virtual float GetRenderTime() const

Overridden from NiRenderStep. These functions return the rendering statistics from the most recent frame. These values are stored as member variables in NiClickRenderStep and are updated appropriately each frame during the RenderClicks function by querying each render click for its values.

virtual void PerformRendering()

Overridden from NiRenderStep. This function first calls GetRenderClicks to get the list of render clicks and then passes that list to the virtual RenderClicks function to render them.

Supported Virtual Functions

virtual const NiTPointerList<NiRenderClickPtr>& GetRenderClicks() = 0

This pure virtual function returns a list of render clicks. The PerformRendering function calls this function and passes the resulting list into the RenderClicks function.

virtual void RenderClicks(const NiTPointerList<NiRenderClickPtr>& kRenderClicks)

This virtual function iterates through the provided list of render clicks, rendering each click. It first calls ShouldPerformRendering on each render click with the current frame ID and then calls Render on that render click if ShouldPerformRendering returns true.
