NiShadowMap

: public NiMemObject

<NiShadowMap.h>

Description

This class is used by the shadowing system and acts as a container used to reference a texture, a shadow render click, a clamping mode, and a filtering mode. This class is used in a similar fashion to the NiTexturingProperty::Map class, except it is only used for storing shadow maps. These objects are to only be created by the NiShadowManager at request.

Supported Static Member Functions

static const NiFixedString& GetMapNameFromType(NiStandardMaterial::LightType eType)

This function returns an NiFixedString of "DirShadow", "PointShadow", or "SpotShadow" based on the light type that is passed provided.

static const bool GetLightTypeFromName(const NiFixedString& kName, unsigned int& uiID)

This function will set the second parameter, uiID, to the correct NiStandardMaterial::LightType based on the name that is passed provided by the first parameter. If the light type can not be found for the name provided uiID will not be set and the function will return false.

Supported Member Functions

TextureType GetTextureType()

This function returns TextureType of the texture used by the shadow map.

const float* GetWorldToShadowMap()

This function returns a float pointer to the 4x4 matrix used transform a world coordinate into shadow map space. This 4x4 matrix comes from the NiCamera used in the NiRenderView of the NiShadowMap's NiRenderClick object.

NiRenderedTexture* GetTexture()

This function returns the NiRenderedTexture used by the shadow map.

NiShadowRenderClick* GetRenderClick()

This function returns the NiShadowRenderClick used to render the shadow map.

void SetClampMode(NiTexturingProperty::ClampMode eClampMode)

NiTexturingProperty::ClampMode GetClampMode() const

Retrieves and sets the clamp mode for the Map. See the Texturing documentation for more information on texture filter modes.

void SetFilterMode(NiTexturingProperty::FilterMode eFilterMode)

NiTexturingProperty::FilterMode GetFilterMode() const

Retrieves and sets the filter mode for the Map. See the Texturing documentation for more information on clamp modes.

unsigned short GetMaxAnisotropy() const

Returns the maximum level of anisotropy for this map.

void SetMaxAnisotropy(unsigned short usMaxAnisotropyValue)

Sets the maximum level of anisotropy for this map. The value isn't used unless the filtering mode is set to anisotropic.

Notes

TextureType

This enumeration determines how this material deals with transformations.

Enumeration Meaning
TT_SINGLE Single shadow map with one render click.
TT_CUBE Cube shadow map with 6 render clicks.
