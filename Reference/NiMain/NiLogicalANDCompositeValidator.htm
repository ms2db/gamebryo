NiLogicalANDCompositeValidator

: public NiCompositeValidator

<NiLogicalANDCompositeValidator.h>

Description

This class is a composite render click validator that returns true only if all of its sub-validators return true.

Overridden Virtual Functions

virtual bool ValidateClick(NiRenderClick* pkRenderClick, unsigned int uiFrameID)

Overridden from NiRenderClickValidator. This function calls ValidateClick on each sub-validator, returning false if any of them return false. In such a case, all subsequent sub-validators are skipped. If all sub-validators return true from their ValidateClick functions, this function returns true.
