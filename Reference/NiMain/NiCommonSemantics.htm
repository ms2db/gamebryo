NiCommonSemantics

<NiCommonSemantics.h>

Description

The NiCommonSemantics class is a container for an enumeration of available semantics to associate with NiMesh data streams. This class supports converting these enumerations between pure integer values and human-readable NiFixedString objects. Not all of these data streams are valid on all platforms.

These functions return the NiFixedString conversion of the associated enumeration type.

static const NiFixedString& GetString(Enumeration e)

Returns the NiFixedString associated with the enumeration e. If there is no mapping, then it returns an NiFixedString containing a NULL string.

static Enumeration GetEnumeration(const NiFixedString& kString)

Returns the enumeration value associated with the string kString. If there is no mapping, then it returns E__Invalid.
