NiTMetricsScopeTimer

: public NiTMetricsTimer

<NiMetricsTimer.h>

Description

This class is a convenience timer class that starts itself and calls the NiMetricsLayer::StartTimer function on itself when instantiated and stops itself and calls the NiMetricsLayer::StopTimer function when it falls out of scope and is destroyed. This class has no member functions other than a constructor and destructor. Because this happens automatically, the NiMetricsLayer::StartTimer and NiMetricsLayer::EndTimer should not be called on objects of this type.

This class takes a templated type T for its timer type. See NiTMetricsTimer for more details. For convenience, NiMetricsScopeClockTimer and NiMetricsScopePCTimer have been #typedef'd to instances of this class that use the system clock and the performance counter, respectively.

Constructors

NiTMetricsScopeTimer(const char* pcName)

This is the only constructor for this class and it creates a scope timer with a particular metric name. Like the NiMetricsTimer constructor notes, pcName is only copied shallowly and must exist for the duration of this timer's existence.
