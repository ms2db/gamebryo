NiBackToFrontSortProcessor

: public NiRenderListProcessor

<NiBackToFrontSortProcessor.h>

Description

This class is a render list processor that sorts geometry objects in back-to-front order by either bounding volume center or bounding volume closest point. The functionality of this processor is very similar to that of NiBackToFrontAccumulator.

Constructors

NiBackToFrontSortProcessor(bool bSortByClosestPoint = false)

The sole constructor for this class takes a Boolean variable indicating whether to sort by bounding volume center or by bounding volume closest point.

Overridden Virtual Functions

virtual void PreRenderProcessList(const NiVisibleArray* pkInput, NiVisibleArray& kOutput, void* pvExtraData)

Overridden from NiRenderListProcessor. This function adds all geometry objects from the pkInput array to the kOutput array, sorting the kOutput array in back-to-front order before returning. No geometry objects are rendered by this function.

Supported Member Functions

void SetSortByClosestPoint(bool bSortByClosestPoint)

bool GetSortByClosestPoint() const

These functions set and get whether or not the geometry objects will be sorted based on their bounding volume centers or the closest point on the bounding volume to the camera.
