NiMaterialResource

: public NiRefObject

<NiMaterialResource.h>

Description

This class represents an input or output of an NiMaterialNode in a shade-tree. In order to properly hook this resource up with the engine at run-time, this resource is defined by how it interacts with an NiShaderConstantMap. See the notes for more details.

Constructors

NiMaterialResource()

The default constructor.

Notes

Source Enum

This enumeration determines how this resource interacts with the NiShaderConstantMap.

Enumeration Meaning
SOURCE_UNKNOWN This resource is set from an unknown source.
SOURCE_OBJECT This resource is set from an NiShaderConstantMap::ObjectMappings mapping.
SOURCE_PREDEFINED This resource is set from an NiShaderConstantMap::PredefinedMappings mapping.
SOURCE_CONSTANT This resource is set from an variable in an NiMaterialNode
SOURCE_ATTRIBUTE This resource is set from an extra data object attached to the geometry object.
SOURCE_GLOBAL This resource is set from the global constant table.
