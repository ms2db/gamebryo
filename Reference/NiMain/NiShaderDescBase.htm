NiShaderDescBase

: public NiRefObject

<NiShaderDescBase.h>

Description

NiShaderDescBase is the base class for all the shader description helper classes. These classes are used primarily by the tools to allow presenting the artist with some helpful names and descriptions.

Constructors

NiShaderDescBase()

The default constructor creates an uninitialized NiShaderDescBase. The name and description members are set to NULL.

Supported Member Functions

virtual const char* GetName() const

Returns the name member of the class.

virtual const char* GetDescription() const

Returns the description member of the class.
