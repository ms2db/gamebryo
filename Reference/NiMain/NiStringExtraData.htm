NiStringExtraData

: public NiExtraData

<NiStringExtraData.h>

Description

Extra data in the form of text that can be attached to an NiObjectNET.

Constructors

NiStringExtraData(const NiFixedString& kString)

Creates an extra data object that stores the text of the input string. A copy is made of the input string.

Supported Member Functions

const NiFixedString& GetValue() const;

void SetValue(const NiFixedString& kString);

These functions retrieve and set the string extra data. For the latter, a copy is made of the input string, and any existing string in the object is deleted. This function also computes how much space is required on disk to store the string should the extra data be streamed to disk.
