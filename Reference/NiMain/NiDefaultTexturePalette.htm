NiDefaultTexturePalette

: public NiTexturePalette

<NiTexturePalette.h>

Description

Hash table container that maps filenames and paths to NiTexture objects. The NiStream objects use instances of this class by default to implement texture sharing.

Constructors

NiDefaultTexturePalette(bool bSearchGlobalTextureList = false)

Default (and sole) constructor. Creates an empty texture palette. Optionally, this texture palette can search the linked list of all known textures in addition to searching its internal hash table of textures.

Supported Member Functions

virtual NiTexture* GetTexture(const char* pcPath, NiTexture* pkSelf) const

Returns the texture (if any) in the palette's hash table that is associated with the given path. To avoid finding the same texture being added during streaming, there is an option to provide a pointer to a texture to skip when searching the global texture list.

virtual void SetTexture(const char* pcPath, NiTexture* pkTex)

Sets the texture associated with the given pathname. Adds the texture to the hash table for later lookups.

virtual bool GetSearchGlobalTextureList() const

Returns whether or not this texture palette will search the global texture list.

virtual void SetSearchGlobalTextureList(bool bSearch)

Sets whether or not this texture palette will search the global texture list.
