NiBMPReader

: public NiImageReader

<NiBMPReader.h>

Description

The NiBMPReader is a subclass of the abstract NiImageReader class that implements the reading of 4, 8 and 24-bit BMP files. Most applications will not use this class directly, but rather will use the NiDevImageConverter, which includes this class as one of its loaders.

Constructors

NiBMPReader()

Default (and sole) constructor. Creates the internal data structures required for loading BMP files.

Supported Member Functions

virtual bool CanReadFile(const char* pcFileExtension) const

Returns true if the given extension is .bmp, and false otherwise. The base class interface uses this virtual to determine whether a generic image loader can load a given file.

virtual NiPixelData* ReadFile(NiFile& kIst, NiPixelData* pkOptDest)

Reads the given file (assuming that it is a BMP file) and returns the data from the file as a flat NiPixelData object. The output format will be NiPixelFormat::RGB24 for 24-bit BMP files and NiPixelFormat::PAL8 for 8-bit and 4-bit BMP files. The function returns NULL if the BMP file appears to be of the wrong format, is not 4-, 8-, or 24-bit, or is compressed. The calling function should not rely on the position of the file pointer following a call to this function.

virtual bool ReadHeader(NiFile& kIst, unsigned int& uiWidth, unsigned int& uiHeight, NiPixelFormat& kFormat, bool& bMipmap, unsigned int& uiFaces)

Returns true if the file appears to be a valid and readable by the NiBMPImageReader and false otherwise. If the function returns true, the uiWidth, uiHeight, kFormat, and bMipmap parameters will contain the width of the image file, the height of the image file, the destination pixel format, and whether or not the file contains mipmap data, respectively. Note that kFormat will be either NiPixelFormat::RGB24 or NiPixelFormat::PAL8. BMP files do not contain mipmap levels, so bMipmap will always be set to false. The face count will always be 1 on BMP files. The calling function should not rely on the position of the file pointer following a call to this function.
