NiMaterialResourceBinding

: public NiRefObject

<NiMaterialResourceBinding.h>

Description

This class represents a directed edge in a shade-tree that connects an output of one NiMaterialNode to an input of another NiMaterialNode.

Constructors

NiMaterialResourceBinding(NiMaterialResource* pkSrc, NiMaterialNode* pkSrcFrag, NiMaterialResource* pkDest, NiMaterialNode* pkDestFrag)

This constructor creates a binding that links the output resource (pkSrc) from the source node (pkSrcFrag) to the input resource (pkDest) in the destination node (pkDestFrag).

Supported Member Functions

NiMaterialResource* GetSourceResource()

Get the output resource that is the source of this binding.

NiMaterialResource* GetDestinationResource()

Get the input resource that is the destination of this binding.

NiMaterialNode* GetSourceNode()

Get the node that is the source of this binding.

NiMaterialNode* GetDestinationNode()

Get the node that is the destination of this binding.

const NiFixedString& GetVariable()

void SetVariable(const NiFixedString& kVar)

Get and set the variable name attached to this binding.
