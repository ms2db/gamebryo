NiSingleShaderMaterialLibrary

: public NiMaterial

<NiSingleShaderMaterialLibrary.h>

Description

NiSingleShaderMaterialLibrary encapsulates the generation of NiSingleShaderMaterials from the shaders contained within the NiShaderFactory.

Supported Member Functions

virtual NiMaterial* GetMaterial(NiRenderer* pkRenderer, const NiFixedString& kName)

Return the material specified by name. This will attempt to call NiSingleShaderMaterial's create function with the name parameter.

virtual NiShaderDesc* GetFirstMaterialDesc()

Get the first material description in this list.

virtual NiShaderDesc* GetNextMaterialDesc()

Get a subsequent material description from this list.

virtual unsigned int GetMaterialCount() const

Get the number of materials in this library.

virtual const char* GetMaterialName(unsigned int uiMaterial)

Get the name of the material at the specified index in this library.
