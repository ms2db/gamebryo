NiTextureCodec

: public NiMemObject

<NiTextureCodec.h>

Description

NiTextureCodec implements expansion of NiPixelData objects with formats of NiPixelData::DXT1, NiPixelData::DXT3, and NiPixelData::DXT5 into NiPixelData objects with a format of NiPixelFormat::RGBA32. Most applications will not use this class directly, but rather will use the NiDevImageConverter, which includes this class as one of its converters. The NiDevImageConverter uses this object internally to assist renderers that do not support compressed texture formats to render textures that are loaded as compressed data.

Constructors

NiTextureCodec()

Default (and sole) constructor. Creates the internal data structures required for expanding compressed pixel data.

Supported Member Functions

NiPixelData* ExpandCompressedData(const NiPixelData& kSrc)

Expands the given NiPixelData object from a compressed format into an RGBA32 format NiPixelData object. The function will convert all mipmap levels in the given source NiPixelData object. Returns NULL if the source object is not compressed.

Notes

Expansion of NiPixelFormat::COMPRESS1 Format NiPixelData objects

Currently, the NiTextureCodec expands all compressed data into RGBA destinations, even though it is possible that a COMPRESS1 object may have no translucent pixels.  Since this expansion code is "fallback" code for systems that cannot support compressed data, this issue is a minor one, and is unlikely to be changed for performance reasons.
