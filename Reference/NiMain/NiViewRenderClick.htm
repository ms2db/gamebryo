NiViewRenderClick

: public NiRenderClick

<NiViewRenderClick.h>

Description

This class is a render click that holds a list of NiRenderView objects, which it uses in its PerformRendering function. It also holds an NiRenderListProcessor that can be used to process all geometry objects before they are rendered.

Overridden Virtual Functions

virtual unsigned int GetNumObjectsDrawn() const

virtual float GetCullTime() const

virtual float GetRenderTime() const

Overridden from NiRenderClick. These functions return the rendering statistics from the most recent frame. These values are stored in member variables in this class and are computed each frame during the PerformRendering function.

virtual void PerformRendering(unsigned int uiFrameID)

Overridden from NiRenderClick. This function iterates through all the render views, querying each for its array of potentially visible geometry, which it then hands off to the stored NiRenderListProcessor for processing. If no processor has been set, a default instance of the NiRenderListProcessor class is used. The processor passes back an output array of geometry that is then rendered. That output array is then passed back to the processor for further processing.

These functions provide access to the list of render views. The GetRenderViewAndPosByName function returns the first render click in the list with the specified name, storing the list iterator to that render click in the supplied variable. GetRenderViewByName and GetRenderViewPosByName both call GetRenderViewAndPosByName under the hood and are provided for convenience purposes.

void SetProcessor(NiRenderListProcessor* pkProcessor, void* pvProcessorData = NULL)

NiRenderListProcessor* GetProcessor() const

void* GetProcessorData() const

These functions set and get the render list processor and a pointer to its custom data. The processor is used in the PerformRendering function. If no processor is set, a default instance of the NiRenderListProcessor class is used.
