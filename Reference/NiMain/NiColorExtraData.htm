NiColorExtraData

: public NiExtraData

<NiColorExtraData.h>

Description

Extra data in the form of a NiColorA (red, green, blue, alpha) that can be attached to an NiObjectNET.

Constructors

NiColorExtraData(const NiColorA kColorA)

NiColorExtraData(const NiColor kColor)

Creates an extra data object that stores the color of the input parameter. If the input parameter is a NiColor instead of a NiColorA, no alpha component is preserved.

Supported Member Functions

const NiColorA GetValue() const

void SetValue(const NiColorA kColorA)

These functions retrieve and set the color extra data as a NiColorA 4-tuple (red, green, blue, alpha).

const NiColor GetValueColor3() const

void SetValue(const NiColor kColor)

These functions retrieve and set the color extra data as a NiColor 3-tuple (red, green, blue).

float GetRed() const

void SetRed(const float fRed)

float GetGreen() const

void SetGreen(const float fGreen)

float GetBlue() const

void SetBlue(const float fBlue)

float GetAlpha() const

void SetAlpha(const float fAlpha)

These functions are component accessor functions. They retrieve and set the float values for the individual components of the color extra data.
