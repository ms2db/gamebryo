NiGeometricUtils

<NiMath.h>

Description

This class contains several useful functions for geometric operations and computation. Functions are declared as public, static members. The class holds no data and effectively functions as a namespace.

Static Member Functions

static float GetTriArea(const NiPoint3& kP0, const NiPoint3& kP1, const NiPoint3& kP2)

Get the area for the triangle specified by the three points.

static NiPoint3 GetTriCenter(const NiPoint3& kP0, const NiPoint3& kP1, const NiPoint3& kP2)

Get the center of gravity also known as the intersection of the medians for the triangle specified by the three points.

static NiPoint3 GetTriNormal(const NiPoint3& kP0, const NiPoint3& kP1, const NiPoint3& kP2)

Get the normal for the triangle assuming counterclockwise winding order. The resulting vector will be normalized.

static float GetDistToSeg(const NiPoint3& kPoint, const NiPoint3& kSegPoint, const NiPoint3& kSegDirection)

static float GetDistSqrToSeg(const NiPoint3& kPoint, const NiPoint3& kSegPoint, const NiPoint3& kSegDirection)

Calculated the distance or distance squared between kPoint and the line segment specified by the origin point, kSegPoint, and the vector kSegDirection.
