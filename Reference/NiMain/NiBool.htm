NiBool

<NiBool.h>

Description

NiBool is just a typedef of unsigned char. This NiBool may later become a full class implementation. However, it is currently just a typedef that is designed to have the same size across multiple platforms. The C++ standard bool may differ in size across platforms, and can cause trouble with Gamebryo's streaming system.

This header file defines a macro called NIBOOL_IS_TRUE which can be used to convert an NiBool value to a regular C++ standard bool. The macro definition is shown below.

#define NIBOOL_IS_TRUE(bBool) ((bBool) != 0)
