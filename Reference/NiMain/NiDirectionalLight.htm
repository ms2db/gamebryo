NiDirectionalLight

: public NiLight

<NiDirectionalLight.h>

Description

NiDirectionalLight is an object that represents a light located at infinity with a uniform parallel light rays traveling in a specific direction. The light will project illumination in its model-space positive X-axis. The directional light may be pointed in any world-space direction by rotating the light.

Constructors

NiDirectionalLight()

Default constructor. Constructs a light with the light direction (in world space) set to [1.0,0.0,0.0].

Supported Member Functions

const NiPoint3& GetWorldDirection() const

Returns the light direction in world space.
