NiTimeSyncController

: public NiShaderTimeController

<NiTimeSyncController.h>

Description

NiTimeSyncController is used by the shaders to keep track of the last time a piece of NiAVObject was updated. This is then fed into a piece of NiFloatExtraData named "NDL_UpdateTime". This value is then used by the shader system to input time as a shader constant.

Constructors

NiTimeSyncController()

The default constructor creates a default NiTimeSyncController.

Supported Member Functions

virtual void Update(float fTime)

Stores fTime in the NiAVObject's "NDL_UpdateTime" extra data.
