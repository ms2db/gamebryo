NiMetricsOutputModule

: public NiRefObject

<NiMetricsOutputModule.h>

Description

This class is an abstract virtual interface for output modules for the metrics system to derive from. If this output module is added to NiMetricsLayer via the NiMetricsLayer::AddModule function, then these functions will be automatically called whenever the corresponding NiMetricsLayer function is called.

Virtual Member Functions

virtual void AddValue(const char* pcName, float fValue) = 0

virtual void StartTimer(NiMetricsTimer& kTimer) = 0

virtual void EndTimer(NiMetricsTimer& kTimer) = 0

virtual void AddImmediateEvent(const char* pcName, float fValue) = 0

virtual void Update() = 0

These functions are called when the corresponding NiMetricsLayer function is called.
