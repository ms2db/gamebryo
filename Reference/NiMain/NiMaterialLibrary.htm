NiMaterialLibrary

: public NiRefObject

<NiMaterialLibrary.h>

Description

This class contains a set of NiMaterials that it can generate as needed by the application. It also contains the descriptions of the materials so that user-interface can be generated by the tools.

Constructors

NiMaterialLibrary(const NiFixedString& kName)

The default constructor, which takes the name of the library.

Supported Static Functions

static bool GetCreateReturnsNullIfMissingFlag()

static void SetCreateReturnsNullIfMissingFlag(bool bRemove)

If this flag is true, then when a new material fails to be created (such as if the shader cannot be found or an appropriately library is not available), that material will return NULL. If this flag is false, then instead an invalid NiSingleShaderMaterial with the correct name will be returned in its stead. This is to allow for streaming in and out of assets (such as in a tool), even if its shaders cannot be loaded.
