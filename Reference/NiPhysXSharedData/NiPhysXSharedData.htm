NiPhysXSharedData

: public NiSharedData

<NiPhysXSharedData.h>

Description

This shared data class contains information necessary for the PhysX extractor plug-in.

Constructors

NiPhysXSharedData()

Default constructor.

Destructors

virtual ~NiPhysXSharedData()

Default destructor.

Supported Member Functions

unsigned int GetPropCount()

Get the number of NiPhysXProp objects stored in the shared data object.

NiPhysXProp* GetPropAt(unsigned int uiIndex)

Obtain a pointer to the prop at the specified index, which must be less than the value returned by GetPropCount and may be NULL.

void AddProp(NiPhysXProp* pkProp)

Add a prop to the stored set.

void RemoveProp(NiPhysXProp* pkProp)

Remove the specified prop from the stored list. The prop will be deleted if nothing is holding a smart pointer to it after deletion.

NiPhysXScene* GetScene()

void SetScene(NiPhysXScene* pkScene)

Get and set the PhysX scene object. This member is no longer used by any Gamebryo plug-ins.

NiUInt32 GetStateCount()

Get the number of states, other than the default, that are stored in the props.

StateInfo& GetStateAt(NiUInt32 uiIndex)

Get the state information structure for a given state index.

void AddState(const StateInfo& kState)

Add information about a stored state. The information is copied.

void RemoveState(const StateInfo& kState)

Remove a given state.

Notes

The NiPhysXSharedData::StateInfo class is declared as follows:

class StateInfo : public NiMemObject
{
public:
    NiFixedString m_kStateName;
    float m_fStateTime;
};
