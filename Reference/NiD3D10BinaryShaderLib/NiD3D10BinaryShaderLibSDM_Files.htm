NiD3D10BinaryShaderLibSDM Files

Description

These files, NiD3D10BinaryShaderLibSDM.cpp and NiD3D10BinaryShaderLibSDM.h, provide the static data management handling for the shader library. There are two functions provided,Init and Shutdown, which are described below.

NiD3D10BinaryShaderLibSDM Functions

void Init()

Does nothing in the current release, but would be used to initialize the shader library.

void Shutdown()

Clean up the shader library.
