Description

ComputeRenderClick is an NiRenderClick that is responsible for invoking compute shaders.

Compute shaders are run without a mesh, and so cannot be invoked from the standard frame rendering system. A ComputeRenderClick enables applications to insert a compute shader (or a set of compute shaders) into the frame at the appropriate point.
