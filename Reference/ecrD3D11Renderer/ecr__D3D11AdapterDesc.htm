Description

D3D11AdapterDesc describes a display adapter on the current system.

D3D11AdapterDesc is a descriptor of the capabilities of a piece of 3D rendering hardware. It describes the supported display devices attached to the adapter (as D3D11OutputDesc objects), as well as the various supported driver types (as D3D11DeviceDesc objects).
