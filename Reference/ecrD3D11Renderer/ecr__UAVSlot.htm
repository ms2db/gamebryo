Description

Represents an unordered access view variable on a D3D11ShaderCore object.

During D3D11Pass::ExposeShaderParameters, any unordered access view variables that a shader requires will have a UAVSlot created for them on the D3D11ShaderCore object. An application can then attach an UnorderdAccessResource to the UAVSlot to provide a specific D3D11 resource to that shader variable.

New UAVSlot objects should be created on a D3D11ShaderCore object using D3D11ShaderCore::AddUAVSlot.
