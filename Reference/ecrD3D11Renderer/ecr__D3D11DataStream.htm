Description

D3D11-specific implementation of NiDataStream.

Data streams will typically be created through the NiDataStream interface itself, which will create an instance of this class using the correct factory. ecr::D3D11DataStream objects encapsulate an ID3D11Buffer, regardless of how the buffer will actually be bound to the device.

Public Methods

Adjusts the size of the stream, specified in bytes.

Contents of stream are undefined afterwards.

D3D11_USAGE GetD3D11Usage() const

Returns the D3D11 usage for the buffer, which describes the read/write behavior the buffer expects.

efd::UInt32 GetD3D11BindFlags() const

Returns the D3D11 bind flags for the buffer, which indicate how the buffer can be used.

efd::UInt32 GetD3D11CPUAccessFlags() const

Returns the D3D11 CPU Access flags, which indicate whether an application can read or write to the buffer from main memory.

ID3D11Buffer* GetBuffer() const

Returns the D3D11 buffer.
