Description

ecr::D3D11SourceTextureData manages the D3D11 resource and resource view representing a static texture loaded from memory or a file.

Textures of all dimensions, including cube maps, can be represented with this class. Textures that require multiple resource views can be managed by separate NiSourceTexture objects that refer to the same D3D11 resource but have different resource views.
