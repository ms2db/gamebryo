Description

D3D11SystemDesc creates and maintains a description of the current system.

D3D11SystemDesc exposes all of the adapters that are installed and configured in the machine. Its main purpose is to serve as a container class for all of the system's Gamebryo adapter descriptors. See the notes below for a hierarchy diagram of the classes that are attached to this system descriptor.
