Description

The WorldFactoryRequest is used to request an asynchronous World (xblock file) load.

These requests are passed to the AssetFactoryManager for processing. The resulting world information is returned to the caller in a WorldFactoryResponse object.

The requester only needs to provide the urn to the xblock file. The AssetFactoryManager is responsible for providing the fully qualified path to the asset.
