Description

Message implementation for preloading an entity.

The EntityPreloadRequest is sent by EntityManager on the category returned by EntityManager::GetEntityPreloadCategory(). All registered preload services must handle this message and perform any desired preloading. Each service must send an EntityPreloadResponse to the category it registered with when complete.

If the entity is not of a type normally managed by a preload service, the service must still send an EntityPreloadResponse.
