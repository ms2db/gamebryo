Description

The FlatModelFactoryRequest is used to request an asynchronous flat model file load.

The resulting FlatModel instance is returned to the caller in an FlatModelFactoryResponse.

The requester only needs to provide the urn. The AssetFactoryManager is responsible for providing the fully qualified path to the asset and setting this path on the request. If using the FlatModelFactory separate from the AssetFactoryManager, the caller is responsible for providing this path.
