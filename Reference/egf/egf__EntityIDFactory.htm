egf::EntityIDFactory

<EntityIDFactory.h>

Description

The standard EntityIDFactory implementation.

Use this static class to generate new EntityIDs

: This class contains ALL static members and methods. For use be an entire process with common data elements.

: This class is intended for use in the main application thread only. It is not reentrant and should not be used from multiple threads. Use in multiple threads is undefined.

Public Static Methods

static bool InitIDFactory(
efd::UInt32 shardID,
efd::UInt32 netID,
efd::UInt64 baseID)

Init the static class members up to the last ID that was generated.

This will generally happen through some sort of initializing phase involving querying the database for the last known ID used.

Parameter Description
shardID The value that should be placed inside the shard bit pattern for generated EntityIDs
netID The value that should be placed inside the net bit pattern for generated EntityIDs
baseID The last known Base ID value. Each new generated EntityID will increment this value by 1.
Return Value bool true if all input values are within valid ranges, false otherwise.

static bool InitIDFactory(EntityID entityIDValue)

Init the static class members up to the last ID that was generated.

This will generally happen through some sort of initializing phase involving querying the database for the last known ID used.

Parameter Description
entityIDValue The full GUID value of the last EntityID.
Return Value bool true if entityIDValue value is within a valid range, false otherwise.

static bool ReinitIDFactory(efd::UInt32 netID)

Re-init the ID Factory with a new NetID.

This should happen when a network connection is established or re-established. The baseID is not reset and will continue to increment from the previous value.

Parameter Description
netID The value that should be placed inside the net bit pattern for generated EntityIDs
Return Value bool true if netID value is within a valid range, false otherwise.

static EntityID GetNextID()

Fetch the next EntityID.

Parameter Description
Return Value EntityID The next EntityID, or zero if the ID range has overflowed.
