Description

The EntityFactoryRequest is used to request an asynchronous Entity load.

These requests are passed to the AssetFactoryManager for processing. The resulting Entity is returned to the caller in an EnityFactoryResponse.

The requester only needs to provide the urn and EntityID. The AssetFactoryManager is responsible for providing the fully qualified path to the asset.

Parameter Description
urn : URN for the model required by the Entity.
responseCategory : Category to send the response.
entityID : The entityID to use for the entity returned by this request.
assetPath fully qualified path to the Flat Model for the new entity.
isMasterEntity : True if this is a master, owned entity. False if constructing a reflected entity owned by another simulator.
isBackground if true, load is attempted in the background thread.
isPreemptive if true, load is done as soon as possible.
isReload indicates this is a reload request.
