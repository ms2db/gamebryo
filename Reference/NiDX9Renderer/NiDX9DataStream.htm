NiDX9DataStream

: public NiDataStream

<NiDx9DataStream.h>

Description

NiDX9DataStream is a DX9-specific implementation of an NiDataStream. Data streams will typically be created through the NiDataStream interface itself which will create an instance of this class using the correct factory.

Constructors

NiDX9DataStream(const NiDataStreamElementSet& kElements, NiUInt32 uiCount, NiUint8 uiAccessMask, Usage eUsage)

This constructor is an override for NiDataStream. It creates a data stream containing a set of elements.

NiDX9DataStream(NiUInt8 uiAccessMask, Usage eUsage)

This constructor is an override for NiDataStream. It creates data stream for streaming.

Supported Member Functions

virtual void Resize(NiUInt32 uiSize)

Reallocates a new data stream of size uiSize.

void SetAccessMask(NiUInt8 uiAccessMask)

Sets the access flag for this stream.

void* MapBuffer(NiUInt8 uiLockType, NiUInt32 uiReadLockCount, NiUInt32 uiWriteLockCount)

Maps the buffer for the data stream and returns a pointer to that memory. Returns NULL on failure or if the underlying buffer does not exist to be mapped.

void UnmapBuffer(NiUint8 uiLockType, NiUint32 uiReadLockCount, NiUint32 uiWriteLockCount)

Unmaps a previously mapped data stream.

NiUint32 GetDX9UsageFlags()

Returns the D3DUSAGE flag for this data stream.

Returns a pointer to the DX9 vertex buffer resource.

LPDIRECT3DINDEXBUFFER9 GetIndexBuffer() const

Returns a pointer to the DX9 index buffer resource.
