NiDX9PixelFormat

: public NiMemObject

<NiDX9PixelFormat.h>

Description

NiDX9PixelFormat is a utility class that adds the ability to match D3DFORMAT enumeration values to Gamebryo pixel formats. This class is used internally in the NiDX9Renderer to implement fast comparisons of pixel formats.

Static Member Functions

static D3DFORMAT DetermineD3DFormat(const NiPixelFormat& kDesc)

Tries to determine the matching standard D3D texture/pixel format enumeration that matches the given NiPixelFormat, or returns D3DFMT_UNKNOWN if there is no match.

static void InitFromD3DFormat(D3DFORMAT eD3DFmt, NiPixelFormat& kFormat)

Initialize an NiPixelFormat object's internal members by specifying the desired D3DFORMAT.

static NiPixelFormat* CreateFromD3DFormat(D3DFORMAT eD3DFmt)

Create a new NiPixelFormat object which is then initialized with InitFromD3DFormat.
