NiD3DShaderProgramCreatorAsm

: public NiD3DShaderProgramCreator

<NiD3DShaderProgramCreatorAsm.h>

Description

NiD3DShaderProgramCreatorAsm is the class that generates NiD3DShaderProgramCreator objects from assembly shader code, stored either in a file or in a buffer in memory.

Constructors

This class has no public constructors.

Supported Member Functions

virtual unsigned int GetShaderCreationFlags() const

virtual void SetShaderCreationFlags(unsigned int uiFlags)

Set and retrieve an unsigned int value that, after getting OR-ed with the universal shader creation flags obtained from the base NiD3DShaderProgramCreator class, will be passed into the Flags parameter of the D3DXAssembleShader* function. The renderer-specific implementations will ensure that only flags that are appropriate for the shader assembly function are used.

Notes

File Extensions

By default, NiD3DShaderProgramCreatorAsm is registered as the NiD3DShaderProgramCreator function to be used with files whose extension is .vsh or .psh. These registrations can be changed using the function NiD3DShaderProgramFactory::RegisterShaderCreator, so that applications can specify their own extensions. Refer to the documentation of NiD3DShaderProgramFactory for more details.
