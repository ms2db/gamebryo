NiDX9SourceTextureData

: public NiDX9TextureData

<NiDX9SourceTextureData.h>

Description

NiDX9SourceTextureData contains the DX representation of a source texture.

Constructors

This class contains no public constructors the static Create function below is used to create NiDX9SourceTextureData objects.

Static Member Functions

NiDX9SourceTextureData* Create(NiSourceTexture* pkTexture, NiDX9Renderer* pkRenderer)

Creates the renderer data for the specified NiSourceTexture using the specified NiDX9Renderer. The DX9 source texture renderer data will be created from either an associated NiDX9PersistentSrcTextureRendererData object or an associated NiPixelData object.

bool SetMipmapSkipLevel(unsigned int uiSkip)

unsigned int GetMipmapSkipLevel()

Get and set the number of mipmap levels to skip with loading the DX representation from the texture.

Supported Member Functions

virtual void Update()

Forces the DX representation of the texture to update itself based on the source texture.

virtual NiPalette* GetPalette()

Returns the NiPalette associated with this texture.
