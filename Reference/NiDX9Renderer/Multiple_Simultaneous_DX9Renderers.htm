Multiple Simultaneous DX9Renderers

Applications should never create multiple simultaneous renderers. This includes having a DX9 renderer and D3D10 or D3D11 renderer in existence at the same time. Applications should ensure that any active renderer is completely shut down before creating another. The DX9 renderer does not support multiple simultaneous renderers, and behavior is undefined if more than one is used at a time. It is, however, possible to render to multiple heads of the same device using the same renderer.
