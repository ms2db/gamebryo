NiDX9TextureManager

: public NiRefObject

<NiDX9TextureManager.h>

Description

NiDX9TextureManager is the class that manages the DirectX textures.

Constructors

NiDX9TextureManager(NiDX9Renderer* pkRenderer)

This function creates a new instance of a NiDX9TextureManager. Since the renderer creates the default one automatically, no application should need to use this function.

Supported Member Functions

LPDIRECT3DTEXTURE9 GetTexture(unsigned int uiWidth, unsigned int uiHeight, unsigned int uiLevels, unsigned int uiUsage, D3DFORMAT eFormat, D3DPOOL ePool)

Returns a new DX texture object based on the parameters passed in. Refer to the DirectX documentation for details on the parameters.

LPDIRECT3DBASETEXTURE9 PrepareTextureForRendering(NiTexture* pNewTexIm, bool &bChanged, bool &bMipmap, bool &bNonPow2)

This function takes the NiTexture that is passed in and ensures that it is ready to be rendered. It returns a pointer to the DX texture that can be used with a NiD3DRenderState::SetTexture call, and three Boolean values indicating whether the texture needs to be updated, whether the texture contains mipmaps, and whether it has one or both dimensions that are not round powers of two.

bool PrecacheTexture(NiTexture* pkIm)

Loads the texture into a DX texture. This function should only be called from the NiDX9Renderer itself.

unsigned int GetFormatFromTexture(NiTexture& kIm)

Returns the format of a texture in the form of an NiDX9Renderer::TexFormat enumeration.
