NiDX9ShadowWriteShader

: public NiDX9FragmentShader

<NiDX9ShadowWrite.h>

Description

This is a specialized shader used by the shadowing system. The only difference from this shader and a normal NiDX9FragmentShader is that in the PreProcessPipeline() function this shader overrides the standard back face rendering state and sets it to back face rendering state the shader has been supplied.

Constructors

NiDX9ShadowWriteShader(NiMaterialDescriptor* pkDesc)

This constructor simply passes the NiMaterialDescriptor up to the NiDX9FragmentMaterialShader.

Supported Virtual Member Functions

virtual unsigned int PreProcessPipeline(NiRenderCallContext& kRCC)

The function passes all its parameters to its parent's PreProcessPipeline function and then manually sets the culling mode base on the render back face state the shader has been supplied.

Supported Static Member Functions

static void SetRenderBackfaces(bool bRenderBackfaces)

Sets the back face rendering state to be used when rendering geometry to a shadow map.
