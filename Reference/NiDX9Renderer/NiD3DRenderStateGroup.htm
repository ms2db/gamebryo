NiD3DRenderStateGroup

: public NiMemObject

<NiD3DRenderStateGroup.h>

Description

NiD3DRenderStateGroup is a collection of render states to be set on the device. There are two sets of states in the group, ones which are 'fire-and-forget' and ones which are saved and then restored later.

Constructors

NiD3DRenderStateGroup()

Creates an instance of the class, initializing its members to a valid state. In general, it is a good idea to not directly create NiD3DRenderStateGroup objects, but to call GetFreeRenderStateGroup to obtain one from the pool of available objects.

Supported Static Member Functions

static NiD3DRenderStateGroup* GetFreeRenderStateGroup()

Retrieves an instance of the class from the pool maintained by the renderer. This is provided to prevent fragmentation.

static void ReleaseRenderStateGroup(NiD3DRenderStateGroup* pkGroup)

Places the instance back into the pool.

Supported Member Functions

void SetRenderState(unsigned int uiRenderState, unsigned int uiValue, bool bSave = false)

Sets the given render state, uiRenderState, to the given value, uiValue. If bSave is true, the state will be saved for restoration with the RestoreRenderStates function.

void RemoveRenderState(unsigned int uiRenderState)

Removes the given render state, uiRenderState, from the group.

bool GetRenderState(unsigned int uiRenderState, unsigned int& uiValue, bool& bSave)

Retrieves the given render state, uiRenderState, from the group, setting the value and save parameters passed in. Returns true if the state was present, false if not.

NiD3DError SetRenderStates()

Sets the render states in the group on the device.

NiD3DError RestoreRenderStates()

Restores any render states marked for saving.
