NiDX9Direct3DTextureData

: public NiDX9TextureData

<NiDX9Direct3DTextureData.h>

Description

NiDX9Direct3DTextureData contains the DX representation of the data in a texture that has been provided by an application.

Constructors

This class contains no public constructors the static Create function below is used to create NiDX9Direct3DTextureData objects.

Static Member Functions

NiDX9Direct3DTextureData* Create(NiDirect3DTexture* pkTexture, NiDX9Renderer* pkRenderer, D3DBaseTexturePtr pkD3DTexture)

Creates the renderer data for the specified NiDirect3DTexture using the specified NiDX9Renderer and the provided pkD3DTexture.
