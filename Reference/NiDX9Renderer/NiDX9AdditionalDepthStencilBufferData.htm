NiDX9AdditionalDepthStencilBufferData

: public NiDX9DepthStencilBufferData

<NiDX92DBufferData.h>

Description

NiDX9AdditionalDepthStencilBufferData is a Gamebryo class that represents a depth/stencil buffer that is created at application request. Applications should not need to create or manipulate objects of this type directly.
