NiBinaryShader Class Reference: Introduction

The NiBinaryShader library demonstrates a fully developed NiD3DShader derivation. The library provides the basis for Gamebryo's binary shader (NSB) format. Developers should feel free to have their own custom written parser libraries generate NSB format files to remove the need to develop a custom binary format as well. The NSB format is provided to allow for platform-independent binary storage of shaders.

The NiBinaryShader library defines the following classes:

  • NiBinaryShader
  • NiBinaryShaderLibrary
