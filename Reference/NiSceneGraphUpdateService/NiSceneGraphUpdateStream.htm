Description

Custom stream class that is aware of the objects cached by NiSceneGraphUpdate singleton.

This class is used to serialize NiObjects that are to be sent from the editor to the viewer. In addition, this class can unserialize objects in-place. This means that the memory address of the updated object does not change in the replicated scene graph.

Objects that already exist in the NiSceneGraphUpdate cache (and are not dirty) are not serialized. In this case, the NiSceneGraphUpdateObjectId is serialized in place of the link id in the stream.

This class is used internally by NiSceneGraphUpdate and does not need to be instantiated by application code.
