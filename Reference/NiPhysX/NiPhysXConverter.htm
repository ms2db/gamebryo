Description

This class manages backwards compatibility by automatically converting NiGeometry-based assets to the NiMesh-based system.

The conversion function provided by this class is automatically registered when the NiPhysX library is included by an application. No explicit action is required by an application. Any NiPhysX-related, pre-NiMesh objects are converted to their NiMesh equivalents.
