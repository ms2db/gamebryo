Inheritance graph for NiPhysXFileStream

Description

This class implements the PhysX's NxStream streaming interface with a Gamebryo efd::File stream.

All the interface calls are passed directly to the file stream. Gamebryo-PhysX uses this class to stream in external cooked mesh files.

virtual ~NiPhysXFileStream()

virtual bool Ready() const

Returns true if the object succeeded in creating its file and is ready for reading or writing.

Applications should check this before using the stream.
