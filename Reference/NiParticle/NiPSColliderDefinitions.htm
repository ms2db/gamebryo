NiPSColliderDefinitions

<NiPSColliderDefinitions.h>

Description

Holds data structures used by particle colliders.

This class defines several structures that contain the data used by particle colliders. They must be defined in a separate class so that they can be easily included in the NiPSSimulatorCollidersKernel Floodgate kernel.

New colliders should add new struct definitions to this class.

The structures included in this class are the following:

  • ColliderData
  • PlanarColliderData
  • SphericalColliderData

Public Types

NiPSColliderDefinitions::ColliderType

The type of collider.

This is used by the Floodgate kernel to determine which NiPSColliderHelpers functions to call.

New colliders should add new values to this enumeration.

Name Description
COLLIDER_PLANAR NiPSPlanarCollider.
COLLIDER_SPHERICAL NiPSSphericalCollider.
NUM_COLLIDER_TYPES The maximum number of collider types. This should be increased if more collider types are added.
