NiPSCommonSemantics

<NiPSCommonSemantics.h>

Description

Defines common semantics strings for particle system data.

These strings are used by various particle system classes to identify the type of particle data being referenced.

Public Types

NiPSCommonSemantics::CommonSemantics

The enumerated types used to lookup strings.

Name Description
PS_INVALID An invalid semantic value.
PS_PARTICLEPOSITION The position of the particle.
PS_PARTICLEVELOCITY The velocity of the particle.
PS_PARTICLEAGE The age of the particle.
PS_PARTICLELIFESPAN The life span of the particle.
PS_PARTICLELASTUPDATE The last time at which the particle was updated.
PS_PARTICLEFLAGS A bit-field containing flags about the particle.
PS_PARTICLEINITSIZE The radius of the particle. Size times radius is the final size.
PS_PARTICLESIZE The size of the particle. Size times radius is the final size.
PS_PARTICLEROTAXIS The axis about which the particle will rotate.
PS_PARTICLEROTINITANGLE The initial rotation angle for the particle about the rotation axis.
PS_PARTICLEROTANGLE The rotation angle for the particle about the rotation axis.
PS_PARTICLEROTSPEED The speed at which the particle is rotating.
PS_PARTICLECOLOR The color of the particle.
PS_PARTICLEROTATION The rotation of a mesh particle.
PS_PARTICLESCALE The scale of a mesh particle.
PS_SEMANTICSCOUNT The maximum number of semantics. Not valid for use as a semantic.
