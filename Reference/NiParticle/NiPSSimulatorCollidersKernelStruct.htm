NiPSSimulatorCollidersKernelStruct

<NiPSSimulatorCollidersKernel.h>

Description

A structure containing data required by the NiPSSimulatorCollidersKernel Floodgate kernel.

Public Attributes

unsigned int m_uiColliderCount

The number of colliders being passed down to the kernel.

float m_fCurrentTime

The current simulation time.

size_t m_stDataBufferSize

The size of the collider data buffer in bytes.

size_t m_astDataRanges[NiPSColliderDefinitions::NUM_COLLIDER_TYPES]

An array containing range values in bytes for the collider data sections in the data buffer.

A data range is provided here for each type of collider. The data range includes from the first byte of the collider data section through the 16-byte-aligned end of that section. It does not include the 16 byte header preceding the data block that specifies the collider type.
