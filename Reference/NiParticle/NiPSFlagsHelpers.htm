NiPSFlagsHelpers

<NiPSFlagsHelpers.h>

Description

A helper class containing static functions that aid in dealing with particle flags.

This class contains functions that get and set data in the particle flags bitfield. This is similar to the way in which NiDeclareFlags flags are handled.

Public Static Methods

static NiUInt16 GetGeneration(NiUInt32 uiFlags)

Gets the particle generation from the particle flags bitfield.

static void SetGeneration(
NiUInt32& uiFlags,
NiUInt16 usGeneration)

Sets the particle generation in the particle flags bitfield.

static NiUInt16 GetSpawnerID(NiUInt32 uiFlags)

Gets the spawner ID from the particle flags bitfield.

static void SetSpawnerID(
NiUInt32& uiFlags,
NiUInt16 ucSpawnerID)

Sets the spawner ID in the particle flags bitfield.

static bool GetShouldDie(NiUInt32 uiFlags)

Gets whether or not the particle should die from the particle flags bitfield.

static void SetShouldDie(
NiUInt32& uiFlags,
bool bShouldDie)

Sets whether or not the particle should die in the particle flags bitfield.

static bool GetShouldSpawn(NiUInt32 uiFlags)

Gets whether or not the particle should spawn from the particle flags bitfield.

static void SetShouldSpawn(
NiUInt32& uiFlags,
bool bShouldSpawn)

Sets whether or not the particle should spawn in the particle flags bitfield.
