NiInputDevice::Axis

<NiInputDevice.h>

Description

The NiInputDevice::Axis sub-class provides a representation for an analog axis of a device. It is present in the NiInputDevice class as it is shared by both game pads and mice. The class handles the mapping of raw platform-specific data to the requested application range.

The button sub-class should be considered internal to the NiInputDevice class, but is described here to give some technical information about the make-up of a device in the system.

Notes

No application should ever create or access an instance of an NiInputDevice::Axis class. The entire class should be considered an 'internal use only'.
