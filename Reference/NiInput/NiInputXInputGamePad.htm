NiInputXInputGamePad

: public NiInputGamePad

<NiInputXInputGamePad.h>

Description

NiInputXInputGamePad is the Win32-specific implementation of an XInput-aware game pad device. NiInputDI8GamePad is used for legacy game pads.

Constructors

There are no public constructors for the NiInputXInputGamePad class. No application should ever create an instance - they should be retrieved via calls to NiInputSystem::OpenGamePad.

Supported Member Functions

There are no functions that expand on the definition supplied in the NiInputGamePad reference defined in this class. For more information on all supported functions, please refer to the NiInputGamePad reference.
