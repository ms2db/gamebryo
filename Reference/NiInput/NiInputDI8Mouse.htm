NiInputDI8Mouse

: public NiInputMouse

<NiInputDI8Mouse.h>

Description

NiInputDI8Mouse is the Win32-specific mouse implementation.

There are no public constructors for the NiInputDI8Mouse class. No application should ever create an instance - they should be retrieved via calls to NiInputSystem::OpenMouse.

Supported Member Functions

Only functions that expand on the definition supplied in the NiInputMouse reference are explained here. For more information on all supported functions, see the NiInputMouse reference.

virtual NiInputErr HandleRemoval()

Calls the DirectInputDevice::Unacquire function. For information on dealing with these situations, please refer to Handling Lost or Removed Devices.

virtual NiInputErr HandleInsertion()

Calls the DirectInputDevice::Acquire function. For information on dealing with these situations, please refer to Handling Lost or Removed Devices.
