NiInputErr

<NiInputErr.h>

Description

NiInputErr contains error code enumerations that are used by the system.

Supported Enumerations

The NiInputErr class contains a single enumeration that identifies errors that can occur in the system.

NiInputErr

The NiInputErr enumeration identifies errors that can occur in the system.

NiInputErr Description
NIIERR_OK No error has occurred; the call succeeded.
NIIERR_GENERIC A generic error has occurred.
NIIERR_UNSUPPORTED The attempted function is unsupported on the operating platform.
NIIERR_INVALIDPARAMS Invalid parameters were passed to a function.
NIIERR_INITFAILED Initialization failed.
NIIERR_INVALIDDEVICE The device is invalid.
NIIERR_DEVICENOTFOUND The device was not found.
NIIERR_DEVICELOST The device was lost.
NIIERR_DEVICECREATEFAILED The device creation failed.
NIIERR_DEVICEINITFAIL The device initialization failed.
NIIERR_DEVICEERROR There was an error with the device.
NIIERR_DEVICECHANGE The device was changed.
NIIERR_NOOPENPORTS There are no available ports.
NIIERR_APIFAILURE There was an underlying API failure. (For example, DirectInput had an error on Win32.)
NIIERR_ALLOCFAILED An allocation failed.
NIIERR_INVALIDACTION An invalid action was detected.
NIIERR_NOTMAPPED The device is not action mapped.
NIIERR_MAPPINGFAILED The mapping of the device failed.
NIIERR_MAPPINGFAILED_GAMEPAD The mapping of the game pad failed.
NIIERR_MAPPINGFAILED_KEYBOARD The mapping of the keyboard failed.
NIIERR_MAPPINGFAILED_MOUSE The mapping of the mouse failed.
NIIERR_MAPPING_TAKEN The requested mapping is already taken.
NIIERR_INVALID_AXIS The requested axis is invalid.
NIIERR_ENUM_NOTRUNNING The enumeration of devices is not currently running.
NIIERR_ENUM_NOTCOMPLETE The enumeration of devices is not completed.
NIIERR_ENUM_FAILED The enumeration of devices failed.
NIIERR_ENUM_COMPLETE The enumeration of device is completed.
