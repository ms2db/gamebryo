: public NiInputKeyboard

<NiInputDI8Keyboard.h>

Description

NiInputDI8Keyboard is the Win32-specific keyboard implementation.

There are no public constructors for the NiInputDI8Keyboard class. No application should ever create an instance - they should be retrieved via calls to NiInputSystem::OpenKeyboard.

Supported Member Functions

Only functions that expand on the definition supplied in the NiInputKeyboard reference are explained here. For more information on all supported functions, see the NiInputKeyboard reference.

virtual NiInputErr HandleRemoval()

Calls the DirectInputDevice::Unacquire function. For information on dealing with these situations, please refer to Handling Lost or Removed Devices.

virtual NiInputErr HandleInsertion()

Calls the DirectInputDevice::Acquire function. For information on dealing with these situations, please refer to Handling Lost or Removed Devices.
