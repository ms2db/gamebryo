NiActionMap

: public NiRefObject

<NiActionMap.h>

Description

NiActionMap represents a collection of actions that an application wishes to utilize in their application. Action maps can be hot-swapped during the run of an application. This allows for things such as having a specific action map for the game, and another for user interface code.

Constructors

NiActionMap()

Creates an 'empty' instance of an NiActionMap.

NiActionMap(const char* pcName, void* pvPlatformSpecificData = 0)

Creates an 'initialized' instance of an NiActionMap. The name of the action map is set to the given pcName. The pvPlatformSpecificData parameter is used only on the Win32 platform, in which case it should be set to a GUID associated with the action map.

Supported Member Functions

const char* GetName() const

Returns the name of the action map.

const void* GetPlatformSpecificData() const

Returns the platform specific data assigned to the action map.

const unsigned int GetActionCount() const

Returns the number of actions in the map.

const NiAction* GetAction(unsigned int uiIndex) const

Returns the action instance at the given uiIndex.

NiAction* AddAction(const char* pcName, unsigned int uiAppData, unsigned int uiSemantic, unsigned int uiFlags, unsigned int uiDeviceID = 0, int iRangeLow = 0, int iRangeHigh = 0, void* pvContext = 0, unsigned int uiKeyModifiers = 0, unsigned int uiMouseModifiers = 0, unsigned int uiPadModifiers = 0)

Adds an NiAction instance to the map. Refer to the NiAction section for details on the parameters.
