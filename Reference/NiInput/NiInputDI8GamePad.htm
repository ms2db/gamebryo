NiInputDI8GamePad

: public NiInputGamePad

<NiInputDI8GamePad.h>

Description

NiInputDI8GamePad is the Win32-specific implementation of a legacy game pad device. NiInputXInputGamePad is used for XInput-aware game pads.

Constructors

There are no public constructors for the NiInputDI8GamePad class. No application should ever create an instance - they should be retrieved via calls to NiInputSystem::OpenGamePad.

Supported Member Functions

There are no functions that expand on the definition supplied in the NiInputGamePad reference defined in this class. For more information on all supported functions, please refer to the NiInputGamePad reference.
