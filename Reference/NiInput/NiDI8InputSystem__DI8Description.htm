NiDI8InputSystem::DI8Description

: public NiInputDevice::Description

<NiDI8InputSystem.h>

Description

The NiDI8InputSystem::DI8Description extends the NiInputDevice::Description class to provide additional information required by DirectInput. Due to the fact that there is no NiDI8InputDevice class, the derived description class was actually placed in the NiDI8InputSystem class. This is not usual practice for Gamebryo code, but allowed for the description to be extended properly.

Constructors

No application should ever create an instance of an NiDI8InputSystem::DI8Description class. it should be considered an 'internal use only' function.

Supported Member Functions

const GUID* GetGUIDInstance() const

Returns the GUID associated with the device.
