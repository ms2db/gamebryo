NiProcessorAffinity

<NiProcessorAffinity.h>

Description

A platform-independent wrapper class for setting processor affinities. NiProcessorAffinity supports both an ideal processor and a processor affinity mask. Note that different platforms may only support one of these methods for specifying target processors.

Constructors

NiProcessorAffinity()

Creates an NiProcessorAffinity object with an ideal processor and an affinity of PROCESSOR_DONT_CARE.

NiProcessorAffinity(Processor eProcessor, unsigned int uiAffinityMask)

Creates an NiProcessorAffinity object with members fully qualified and initialized.

Supported Member Functions

unsigned int GetAffinityMask() const

void SetAffinityMask(unsigned int uiAffinityMask)

Get and set the processor affinity mask which should be an OR'd set of the Processor enumeration below.

Processor GetIdealProcessor() const

void SetIdealProcessor(Processor eProcessor)

Get and set the ideal processor.

NiProcessorAffinity::Processor

NiProcessorAffinity::Processor is an enumeration with the following values.

Open Mode Description
NiProcessorAffinity::PROCESSOR_0 - NiProcessorAffinity::PROCESSOR_31 Assigned a single bit from bit 0 to 31, these values map directly to the 0th through 31st processor on the system
NiProcessorAffinity::PROCESSOR_XENON_CORE_0_THREAD_0 An Xbox 360 specific processor enumeration that maps to the first hardware thread on the first core.
NiProcessorAffinity::PROCESSOR_XENON_CORE_0_THREAD_1 An Xbox 360 specific processor enumeration that maps to the second hardware thread on the first core.
NiProcessorAffinity::PROCESSOR_XENON_CORE_1_THREAD_0 An Xbox 360 specific processor enumeration that maps to the first hardware thread on the second core.
NiProcessorAffinity::PROCESSOR_XENON_CORE_1_THREAD_1 An Xbox 360 specific processor enumeration that maps to the second hardware thread on the second core.
NiProcessorAffinity::PROCESSOR_XENON_CORE_2_THREAD_0 An Xbox 360 specific processor enumeration that maps to the first hardware thread on the third core.
NiProcessorAffinity::PROCESSOR_XENON_CORE_2_THREAD_1 An Xbox 360 specific processor enumeration that maps to the second hardware thread on the third core.
NiProcessorAffinity::PROCESSOR_DONT_CARE Assigned the value of 0xFFFFFFFF this mask represents all processors.
