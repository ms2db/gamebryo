NiOS

<NiOS.h>

Description

This is a header file that includes system-specific headers as well as some compiler-specific macros.

Macros

NI_UNUSED

This macro can be used to declare a particular variable as being unused.

NI_FORCEINLINE

This function will try to hint to the compiler that a particular function should be inlined. When used, it replaces the inline keyword.

NI_NOINLINE

This function will try to hint to the compiler that a particular function should never be inlined. Place before the function name.
