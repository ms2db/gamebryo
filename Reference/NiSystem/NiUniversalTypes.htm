NiUniversalTypes

<NiUniversalTypes.h>

Description

This is a header file that defines a number of potentially platform-specific types so they may be used by the engine in a cross-platform way.

Basic Types

NiInt8

8-bit signed integer type.

NiUInt8

8-bit unsigned integer type.

NiInt16

16-bit signed integer type.

NiUInt16

16-bit unsigned integer type.

NiInt32

32-bit signed integer type.

NiUInt32

32-bit unsigned integer type.

NiInt64

64-bit signed integer type.

NiUInt64

64-bit unsigned integer type.

NiWChar

Wide 2-byte character.

NiFloat16

16-bit float type.

Windowing System Types

NiWindowRef

Generic window type (HWND).

NiStatusWindowRef

A subsection of the window to receive text updates, such as the status bar (HWND).

NiContextRef

A context for drawing to (HDC).

NiOglRenderContextRef

An OpenGL draw context (HGLRC).

NiInstanceRef

An instance of the running application (HINSTANCE).

NiAcceleratorRef

Accelerator table (HACCEL).

NiModuleRef;

An executable code module (HMODULE).

NiEventRecord

An structure that contains all the data the OS provides with a message in its message pump.

NiEventRef

A pointer to an NiEventRecord struct.

NiCreateStructRef

Application construction structure (LPCREATESTRUCT).

NiMenuRef

An menu bar (HMENU).
