System Query Routines

float NiGetCurrentTimeInSec()

Returns current time in seconds.

void NiResetBaseTime()

Reset the base time used by the system. NiGetCurrentTimeInSec will return the number of seconds elapsed since the last call to NiResetBaseTime. Applications that must run for long durations can avoid reductions in floating point precision by calling this function periodically to reset time to a range closer to zero. It is recommended that the function be called during level or mode transitions of an application to avoid discontinuities in update times for animated objects.

unsigned int GetFileSize(const char* pcFilename)

Returns the size of the given file.

unsigned int NiGetPerformanceCounter()

Returns a high performance counter. The unit may vary depending on the platform. It is not necessarily the CPU clock cycle counter; it may, for example, be the bus cycle counter, or some other unit. Used by the NiStopWatch class in the NiMain library.
