NiLogBehavior

<NiLogBehavior.h>

Description

NiLogBehavior is a mechanism for defining the log behavior before NiInit is called, such that any messages processed during NiInit will be handled as desired. Applications may derive their own NiLogBehavior implementations to redefine the default behavior.

Static Member Functions

static NiLogBehavior* Get()

static void Set(NiLogBehavior* pkBehavior)

Set or get the NiLogBehavior singleton.

Supported Member Functions

virtual void Initialize()

This function is called by NiLog::_SDMInit (which is called by NiInit). Override this virtual function in an application-derived NiLogBehavior implementation to set custom log behavior.

Notes

Builds that define NI_USE_MEMORY_MANAGEMENT use NiMemoryLogBehavior by default.
