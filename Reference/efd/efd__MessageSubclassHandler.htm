Inheritance graph for efd::MessageSubclassHandler

Description

Class used to declare a message handler object where the type of the handler function in the handler object does not match the actual type of the message being sent.

This template is used to subscribe to a specific message class, but to use a handler that expects a base class (e.g. StreamMessage, or IMessage).
