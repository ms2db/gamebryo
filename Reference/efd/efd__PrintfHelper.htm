efd::PrintfHelper

<Helpers.h>

Description

The PrintfHelper class is used to convert a macro parameter containing a printf-style format string and a list of arguments into a plain string.

The string is formatted into a fixed size buffer and the string will be truncated if required to fit into this buffer.
