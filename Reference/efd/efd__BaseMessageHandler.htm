efd::BaseMessageHandler

<MessageHandlerBase.h>

Inheritance graph for efd::BaseMessageHandler

Description

Class used to register message handler objects.

This class is not meant to be used directly. It is intended to be used by the EE_HANDLER, EE_HANDLER_SUBCLASS, EE_HANDLER_WRAP, and EE_HANDLER_SUBCLASS_WRAPPER macros. In cases where these macros are not sufficient the MessageHandlerImpl, MessageSubclassHandler or MessageWrapperSubclassHandler should be used instead.
