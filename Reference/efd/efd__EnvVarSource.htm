Description

Class that reads in all the environment variables and adds them to the configuration hierarchy.

All of the values that are read by the Environment Variables class are added to a special section defined by the constant EnvVarSource::kDefaultSection. The resulting section entries may be linked into other sections or used directly by the application and system services. The source generates a section such as:

        [Environment Variables]
        PATH = c:\windows\
        COMPUTERNAME = DEV_MACHINE
        PROCESSOR_ARCHITECTURE = x86

This class is a system service.
