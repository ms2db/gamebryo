Description

IAssetTagger is a virtual interface class used to define tagging extensions for use in the Asset Controller application.

Each tagging extension that is registered gets invoked for every new asset when it is first discovered. A child class can process the asset to determine whether certain tag or property values ought to be set for the asset. Tags that are created in this way become part of the generated metadata for the asset web.
