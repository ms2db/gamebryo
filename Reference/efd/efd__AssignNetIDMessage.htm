Description

Assign NetID Message.

This is sent when the NetID is set following connection to a channel manager. In addition, MessageWrapper subclasses of this class are sent with message type kMSGID_ChannelManagerClientJoin and kMSGID_ChannelManagerClientLeave when other processes join or leave the channel manager.
