efd::Semaphore

<Semaphore.h>

Description

A platform-independent wrapper class for a simple semaphore.

Some platforms support the concept of a maximum count for a semaphore. On platforms that do not, the maximum count is bounded only by the maximum of the storage type used by the OS. Regardless, the Semaphore class does not enforce this count internally, but relies on the OS for such management, due to the synchronization overhead that such enforcement would incur.

Public Methods

Semaphore()

Creates a Semaphore object with a count of 0.

Semaphores created with this default constructor are non-signaled from creation, since the initial count is zero. The maximum count is set to 1.

Semaphore(efd::SInt32 count)

Creates a Semaphore object with a count equal to iCount.

iCount should be a non-negative value. The maximum count is set to iCount + 1. Semaphores created with this constructor will be signaled if the count is positive.

Parameter Description
count Initial count. This value must be greater than or equal to 0.

Semaphore(
efd::SInt32 count,
efd::SInt32 maxCount)

Creates a Semaphore object with a count equal to iCount and the maximum count set to iMaxCount.

iCount should be a non-negative value less than iMaxCount. Semaphores created with this constructor will be signaled if the count is positive.

Parameter Description
count Initial count. This value must be greater than or equal to 0 and less than or equal to maxCount.
maxCount Maximum count. This value must be greater than zero.

virtual ~Semaphore()

Virtual destructor.

efd::SInt32 GetCount()

Get a count on the semaphore.

It is possible for this value to be stale since the semaphore only alters the count after system level operations complete. GetCount() is intended for debugging use only and should not be considered an accurate count for the purposes of thread synchronization.

Parameter Description
Return Value A signed 32-bit integer representing the current count.

efd::SInt32 GetMaxCount()

Get the maximum count of the semaphore, set during construction.

Parameter Description
Return Value A signed 32-bit integer representing the current maximum count.

efd::SInt32 Signal()

Signal the semaphore and increase the count by 1.

Parameter Description
Return Value The count of the semaphore after the signal operation.

efd::SInt32 Wait()

Wait on the semaphore.

If the count is 0, the calling thread will be suspended until the semaphore is signaled. If the semaphore is in a signaled state, then the count will be reduced by 1 and returned.

Parameter Description
Return Value The count of the semaphore after the wait operation.
