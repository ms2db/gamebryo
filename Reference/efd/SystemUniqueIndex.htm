SystemUniqueIndex

<SystemUniqueIndex.h>

Description

This class creates a cross-process unique index given a base name.

For example, if you pass in your process name as the base name then it will give you an index that is guaranteed not to be in use by any of the other processes which could be used to generate, say, a unique log file name like "MyProcessName%d.log". To accomplish this a named kernel object is used. As such the instance of the SystemUniqueIndex class must have the same lifespan as your usage of that index. Indices will be reused as soon as the SystemUniqueIndex class that used that index is destructed.

Indices start at 1, index 0 indicates an error.
