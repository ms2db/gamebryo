Inheritance graph for efd::AssetCacheResponse

Description

A message class for responses to requests to cache an asset.

Currently, individual services that cache assets use this message to inform a requestor that the cache action has been completed. There is no centralized service for caching all asset types.

ACR_UnknownError The asset load failed for an unspecified reason, check the log.
ACR_PartialSuccess The asset was loaded, but there were errors. Check the log.
ACR_Success The cache action was successful. Depending on the request, this may mean the asset was loaded into the cache, it was already cached or it was flushed from the cache.
