efd::SDMMacros

<Helpers.h>

Description

The SDMMacros are used to initialize and manage static data.

Public Methods

EE_DECLARE_SDM(libname, entrytag)

Declare the static data management class and members.

EE_DECLARE_SDM_WITH_TLS(libname, entrytag)

Declare the static data management class and members with threaded library support.

EE_IMPLEMENT_SDM_CONSTRUCTOR(libname, dependencies)

Implement the static data management class constructor.

EE_IMPLEMENT_SDM_CONSTRUCTOR_WITH_TLS(libname, dependencies)

Implement the static data management class constructor with threaded library support.

EE_IMPLEMENT_SDM_INIT_CHECK()

Implement the static data management method used to check if it has been initialized.

EE_IMPLEMENT_SDM_SHUTDOWN_CHECK()

Implement the static data management method used to check if it has been shutdown.

EE_IMPLEMENT_DLLMAIN(libname)

Implement the static data management method for a DLLMain.
