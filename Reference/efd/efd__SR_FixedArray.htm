efd::SR_FixedArray

<SerializeRoutines.h>

Description

Any chunk of serialization code that you find yourself repeating frequently can be made into a routine, even something as simple as iterating a fixed size array.

This combines well with the EE_ARRAYSIZEOF macro:

     MyClass data[10];
     SR_FixedArray<EE_ARRAYSIZEOF(data)>::Serialize(data, archive);
