efd::ConfigSource

: public efd::IConfigSource

<ConfigSource.h>

Inheritance graph for efd::ConfigSource

Description

Base class for all the configuration sources.

It provides the basic handlers for priority and source information. It is a system service.

Public Methods

virtual ~ConfigSource()

Destructor for the configuration source.

virtual int GetPriority() const

Returns the priority for this configuration source.

A higher number is a higher priority and overwrites lower values.

Parameter Description
Return Value Priority value for this source.

virtual unsigned int GetSourceType() const

Retrieves the source type of the configuration source as defined in tSourceType.

Other source types may be added and they should be assigned numbers starting with tSourceType::kSourceTypeMax.

Parameter Description
Return Value Source type.

virtual const efd::utf8string& GetSourceName() const

Returns the string name assigned to this configuration source.

In general, it will contain the file name or address of the data used by this configuration source.

Parameter Description
Return Value Name of the source.

virtual efd::utf8string SourceToString() const

Converts the information identifying this source into a string that may be used for printing.

Parameter Description
Return Value Identification string.

virtual void SetConfiguration(
ISection* pRoot,
int iPriority)

Sets the root so this source may add to the configuration.

Parameter Description
pRoot Pointer to the root of the configuration hierarchy.
iPriority Priority of the configuration source.
