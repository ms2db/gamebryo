efd::OS

<OS.h>

Description

OS is not a class but is instead a group of macros to handle Operating System specific tasks.

Public Methods

EE_ARCH_32

Detect the compiler.

Detect the target operating system. For each platform, standard tool-provided macros are checked first, then common user-supplied ones. Note that the order of platform evaluation is significant in some cases. Verify that a build configuration has been selected. Detect the current hardware architecture.

EE_UNUSED

A helper macro that tells the compiler that an attribute might not be used to stop the compiler from generating a warning.

EE_UNUSED_ARG(arg)

A helper macro that declares the argument as unused.

Useful when building at warning level-4 to indicate the unused argument is known and accepted.

EE_FORCEINLINE

A helper macro to attempt to force the compiler to inline the function.

EE_NOINLINE

A helper macro attempt to force the compiler to inline the function.

EE_RESTRICT

A macro to indicates that a symbol is not aliased in the current scope.

EE_EMPTY_THROW

A helper macro that declares an empty throw statement.

MAKEFOURCC

A macro for generating four-character codes.

PS3 Only.
