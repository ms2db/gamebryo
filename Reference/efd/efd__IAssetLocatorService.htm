ASSETS_NONE Initial state. No local metadata opened or remote server connection.

ASSETS_LOCAL_AND_REMOTE Both local and remote opened and connected.

efd::IAssetLocatorService::AssetLookupScope

Values for limiting which asset locations can be used.

Name Description
ALS_LOCAL_OR_REMOTE Lookup asset locations on a remote server first, if one has connected. This is the default scope for AssetLocate().
ALS_LOCAL_ONLY Limit asset location lookups to only local metadata.
ALS_REMOTE_ONLY Limit asset location lookups by requiring remote metadata accessed via a connected AssetServer service.
