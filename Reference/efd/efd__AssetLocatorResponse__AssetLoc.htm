efd::AssetLocatorResponse::AssetLoc

<AssetLocatorResponse.h>

Description

Response details are contained in a map of <AssetID, AssetLoc>.

The AssetID is the "urn:uuid:xxxx" format as used in the Asset Locator server, where 'xxxx' is the Asset logical ID. The request itself can take either the urn::tag format or the urn::uuid format. Most requests are made using the former.

See the How the Asset Locating Services Work page for further details.

Public Attributes
