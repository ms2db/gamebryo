NSBUserDefinedDataBlock

: public NSBConstantMap

<NSBUserDefinedDataBlock.h>

Description

NSBUserDefinedDataBlock represents a single user-defined data block from an NSF file. For more information on user-defined data blocks, please see User Defined Data.

Constructors

NSBUserDefinedDataBlock()

Creates an 'empty' NSBUserDefinedDataBlock instance.

Supported Member Functions

const char* GetName() const

Returns the name of the data block.

void SetName(const char* pcName)

Sets the name of the data block.

virtual bool AddEntry(char* pcKey, unsigned int uiFlags, unsigned int uiSize, unsigned int uiStride, void* pvSource, bool bCopyData = true)

Adds the given entry to the block. Returns true if it was successfully added.

Notes

Users should utilize the NSBConstantMap::GetFirstEntry, NSBConstantMap::GetNextEntry, and NSBConstantMap::GetEntryByKey functions to access the entries. For an example of accessing the data, please refer to the Picking sample.
