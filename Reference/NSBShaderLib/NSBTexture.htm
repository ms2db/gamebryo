NSBTexture

: public NiMemObject

<NSBTexture.h>

Description

NSBTexture represents a platform-independent texture for use in a shader in the Gamebryo engine.

Constructors

NSBTexture()

The constructor for NSBTexture creates an 'empty' instance.

Supported Member Functions

const char* GetName()

void SetName(const char* pcName)

Returns or sets the name of the texture.

unsigned int GetStage()

void SetStage(unsigned int uiStage)

Retrieves or sets the index of the stage.

NSBStateGroup* GetTextureStageGroup()

Retrieves the texture stage state group.

NSBStateGroup* GetSamplerStageGroup()

Retrieves the texture sampler state group.

unsigned int GetTextureFlags()

Retrieves the texture flags that indicate the source of the texture to use. Refer to the NiTextureStage class reference for more information on allowable values.

void SetSource(unsigned int uiNDLMap)

Set the given map flag. Note that decal and shader modes are first cleared.

void SetSourceDecal(unsigned int uiDecal)

Set decal mode and index.

void SetSourceShader(unsigned int uiShaderMap)

Set shader mode and index.

void SetSourceObject(NiShaderAttributeDesc::ObjectType eObjectType, unsigned int uiObjectIndex)

Set object texture flags to the object type and object index specified.
