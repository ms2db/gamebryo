NSBStateGroup::NSBSGEntry

: public NiMemObject

<NSBStateGroup.h>

Description

NSBSGEntry is a helper sub-class of the NSBStateGroup class. It is provided to allow for platform-independent states to be held in an array. It also allows for indicating whether the state should be saved/restored when being used in a run-time application.

See NiShaderConstantMapEntry for details on the various parameters.

Constructors

NSBStateGroup::NSBSGEntry()

The constructor for NSBSGEntry creates an 'empty' instance.

Supported Member Functions

const unsigned int GetFlags() const

Returns the flags set for the entry.

const bool IsSaved() const

Returns true if the entry is to be saved and restored on the hardware device.

const bool UsesAttribute() const

Returns true if the entry is set to be an attribute lookup. [NOTE: Attribute mapping of states is not supported in the current release of Gamebryo.]

const bool UsesMapValue() const

Returns true if the entry is to use the mapped value.

const unsigned int GetState() const

Gets the state value for the entry.

const unsigned int GetValue() const

Gets the value that should be set for the state.

const char* GetAttribute() const

Gets the name of the attribute to used for the states data value. [NOTE: Attribute mapping of states is not supported in Gamebryo.]

void SetFlags(unsigned int uiFlags)

Sets the flags for the entry. It is important to note that setting the flags directly via this function will override any previous settings made by calls to SetSaved or SetUsesAttribute.

void SetSaved(bool bSave)

If bSave is true, sets the appropriate flag to indicate the state should be saved/restored. Otherwise, clears the save flag.

void SetUseAttribute(bool bUse)

If bUse is true, sets the appropriate flag to indicate the states data value should be looked up in the attribute table. Otherwise, clears the attribute flag. [NOTE: Attribute mapping of states is not supported in the current release of Gamebryo.]

void SetUseMapValue(bool bUse)

If bUse is true, sets the appropriate flag to indicate the states data value should be looked up in the map.

void SetState(unsigned int uiState)

Sets the state of the entry.

void SetValue(unsigned int uiValue)

Sets the data value of the entry.

void SetAttribute(const char* pcAttribute)

Set the name of the attribute to use for the value. [NOTE: Attribute mapping of states is not supported in Gamebryo.]

Notes

Flag Enumeration

The following enumerations are provided to set the flags value:

Enumeration Meaning
NSBSG_SAVE Indicates the state should be saved.
NSBSG_ATTRIBUTE Indicates the states data value is obtained from an attribute.
NSBSG_USEMAPVALUE Indicates that the texture stage state should be read from the appropriate NiTexturingProperty::Map value.
