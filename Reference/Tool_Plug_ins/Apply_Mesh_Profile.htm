Apply Mesh Profile

Type: PROCESS

Version: 1.0

<ApplyMeshProfile.dll>

Description

This plug-in takes a provided mesh profile name and, if valid, applies that mesh profile to all meshes. This operation replaces any mesh profile that previously existed on the object. This plug-in should be the run prior to any other plug-ins.

This plug-in is primarily intended to be used for offline batch processing of older, existing assets to convert them to a newer version of Gamebryo, as the mesh profile can be specified in both Max and Maya.

Options

Apply Mesh Profile Options

  • Mesh Profile Name: The name of the mesh profile to apply to all meshes in the scene. This name must refer to a valid profile for the specified renderer.
