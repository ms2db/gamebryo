Normal Unifier

Type: PROCESS

Version: 2.0

<NormalUnifier.dll>

Description

This plug-in is designed to merge normals on adjacent objects in a scene graph. This is especially useful in Max where adjacent objects do not have coincident normals. This usually results in lighting artifacts. The way around this is to find the average normal from all vertices that share the same position in space and set all of the normals associated with those vertices to the average normal. When all of the normals are the same, the lighting artifacts will go away. Since this is a potentially expensive operation, it is limited to the objects in the scene with the text "NiUnifyNormals" somewhere in the string extra data on the object.

Options

This plug-in has no dialog options. However, to enable an object to have its normals unified, it must be tagged with "NiUnifyNormals" in its string extra data.

Notes

For normals to be properly unified, the two adjacent objects must have the same world transform.
