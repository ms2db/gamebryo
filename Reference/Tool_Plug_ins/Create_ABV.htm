Create ABV

Type: PROCESS

Version: 1.0

<CreateABV.dll>

Description

The Create ABV plug-in processes the scene graph and creates alternate bounding volumes (ABVs) for objects that are marked with a specific naming convention. It also allows for the creation of wire-frame object representations for ABV's, their proxies, and selections. See the following documentation for information on how to create collision objects in the specific art packages:

For Max, see the section on Collision Detection and 3ds Max. For Maya, see the section on Creating Collision Attributes.

Options

Create ABV Options

Create ABV

This option will create alternate bounding volumes for collision for objects named according to the convention.

Create WireFrames representing ABV Proxies

This option will create an NiMesh wire-frame geometry for assistance in visualizing the proxy boxes that are used to create the alternate bounding volumes within your application. This option is applied recursively to the tagged portion of the scene graph.

Create WireFrames representing ABVs

This option will create an NiMesh wire-frame geometry for assistance in visualizing the alternate bounding volumes within your application. This option is applied recursively to the tagged portion of the scene graph.

Create WireFrame for Selections

This option will create NiMesh wire-frame geometry representing bounding volumes. This action will be done for each child of the tagged portion of the scene graph.

Remove Proxy Geometry

This option will remove the NiMesh for the proxy geometry used to define the ABV in the CreateABV option.
