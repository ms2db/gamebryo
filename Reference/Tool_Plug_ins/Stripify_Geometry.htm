Stripify Geometry

Type: PROCESS

Version: 2.0

<StripifyGeometry.dll>

Description

This plug-in stripifies all the triangles in the scene to speed up rendering time. The stripification algorithms may optimize the triangle strips for cache coherency, and thus create degenerate triangles. In other words, the actual triangle count may go up after stripification, but the resulting dataset should render faster.

Note that stripification is mutually exclusive with the vertex cache optimization performed by the Scene Graph Optimization plug-in. Vertex cache optimization generally gives more of a performance improvement than stripification so it is recommended instead of stripification.

Options

Stripify Options dialog box

Stripification can be enabled on a per-renderer basis. The current target renderer for export determines which settings are used by the tool plug-in. If no target is set, the PC settings are used. Note that you can also make an object be ignored in the stripification process by adding the "NiNoStripify" tag to its extra data.
