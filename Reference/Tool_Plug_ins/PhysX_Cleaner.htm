PhysX Cleaner

Type: PROCESS

Version: 2.0

<PhysXCleaner.dll>

Description

If you have an PhysX plug-in running in Max and Maya, and PhysX content in your scene, the Gamebryo export process automatically finds this content and adds it as extra data in the scene graph. This plug-in removes this extra data. Use this plug-in if you are using PhysX but not using Gamebryo-PhysX or do not want the PhysX content in nodes.

Options

This plug-in has no options.

Notes

If you are using the PhysX Extractor plug-in then the extra data is removed by that plug-in, so this one is not needed.
