Material Modifier

Type: PROCESS

Version: 1.0

<MaterialModifier.dll>

Description

The material modifier is a process plug-in designed to automatically modify all of the materials in a scene to perform platform-specific optimization. The plug-in provides texture size reduction options for each supported platform.

The intention of this plug-in is to enable you to create a single source asset that can be used to export platform-specific assets that fit the memory and capability constraints of each platform.

Options

<Platform> Texture Size

There is one platform texture size dropdown for each supported platform. Use this to set a power-of-two size reduction on every texture found in the scenegraph depending on the selected export platform.
