    efd::AssetLoadResponse,
    DerivedClassName)
 The category found in the original request message SendLocal A message of this type, or a derived class, is sent by the service when an asset load request has been completed. The concrete type of the message is determined by the asset factory class that loaded the asset.

If the requested asset was not found using the efd::AssetLocatorService, an instance of efd::AssetLoadResponse is sent indicating the failure. If the asset is located, the response from the loading factory is sent. You must handle both the base efd::AssetLoadResponse and the derived message class to receive both success and failure response messages.
