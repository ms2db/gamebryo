Description

This class implements the PhysX's NxUserOutputStream interface with Emergent Foundation's various debug output and assertion macros.

If no alternative is explicitly passed to efdPhysX::PhysXSDKManager::Initialize then an instance of this class is created and passed to the PhysX SDK initialization function. Users wishing to override this behavior should create their own class that implements NxUserOutputStream and pass it to efdPhysX::PhysXSDKManager::Initialize.
