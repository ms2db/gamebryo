ParallelSplitShadowConfig Model

Found in project: StandardModelLibrary

Traits: Abstract, Active

Mixed Into: PSSMDirectionalLight

Description

Entity model that stores shadowing parameters for a shadow casting light using parallel split shadow mapping. The parallel split shadow mapping (PSSM) solution provides improved shadow quality for large scenes at the expense of an increase in CPU and GPU load.

Properties

Name Type Source Model Default Value Traits Description
PSSMEnabled Boolean ParallelSplitShadowConfig True Specifies if Parallel Split Shadow Mapping should be used.
SceneDependentFrustums Boolean ParallelSplitShadowConfig False Specifies if the scene should be analyzed per frame to allow the shadow camera frustum to be tightened to only the visible objects. Improves shadow quality at the expense of an increase in CPU load.
SliceCount UInt32 ParallelSplitShadowConfig 4 Specifies how many slices (sections) to break the scene camera's frustum into. Larger slice counts increase CPU & GPU load.
SliceDistanceExponentFactor Float32 ParallelSplitShadowConfig 0.5 Defines how much larger a given PSSM slice will be in comparison to the previous. A larger value will create a bigger difference in size; therefore biasing more shadowmap pixels to objects nearer to the visible camera.
SliceTransitionLength Float32 ParallelSplitShadowConfig 300 The distance, in world dimensions, of the transition from one slice to the next.
SliceTransitionNoiseGranularity Float32 ParallelSplitShadowConfig 0.05 The size of each noise pixel in a slice transition. A lower number will increase the size of each noise pixel.
SliceTransitions Boolean ParallelSplitShadowConfig False Enable smoother transitions between PSSM splits, using random noise. Increases GPU cost.
SuppressShimmer Boolean ParallelSplitShadowConfig True Quantize the shadow-map camera's movements so that they snap to the shadow maps pixels. This removes shadow shimmering when the view camera moves. It has no effect on any shimmering caused by movement of the casting light.
UseCustomFarPlane Boolean ParallelSplitShadowConfig False Enables use of a user specified far clipping plane for shadows. The scene camera's far clipping plane distance is used by default when one is not specified.
CustomFarPlane Float32 ParallelSplitShadowConfig 10000 The distance from the camera, in world space, that shadows will become clipped away. A shorter distance will improve shadow quality and performance.
CameraDistanceScaleFactor Float32 ParallelSplitShadowConfig 5 Defines how far the shadow map camera will be placed from the scene. A larger scale factor helps to eliminate shadow clipping, but reduces shadow map precision.
