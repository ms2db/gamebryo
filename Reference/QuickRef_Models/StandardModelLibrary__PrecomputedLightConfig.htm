PrecomputedLightConfig Model

Found in project: StandardModelLibrary

Traits: Active

Mixes In: PrecomputedLighting

Mixed Into: PCLDirectionalLight, PCLPointLight, PCLSpotLight

Description

Configures a light to behave appropriately in precomputed lighting mode with respect to both precomputed and non-precomputed objects.

Properties

Name Type Source Model Default Value Traits Description
LightPCLObjectsAtRuntime Boolean PrecomputedLightConfig False This light should dynamically illuminate objects which use precomputed lighting even when the LightService is in precomputed lighting mode.
LightNonPCLObjectsAtRuntime Boolean PrecomputedLightConfig True This light should dynamically illuminate objects which do not use precomputed lighting when the LightService is in precomputed lighting mode.
UseForPrecomputedLighting Boolean PrecomputedLighting True Indicates that this entity will be used in the generation of a precomputed lighting solution.
