ShadowGenerator Model

Found in project: StandardModelLibrary

Traits: Abstract, Active

Mixed Into: DirectionalLight, PointLight

Description

Entity model that determines which shadowing technique a light should use if it indeed casts shadows.

Properties

Name Type Source Model Default Value Traits Description
CastShadows Boolean ShadowGenerator False Determines whether or not this light casts shadows.
RenderBackfaces Boolean ShadowGenerator True Determines whether or not the backfaces are used when rendering an entity. For many shadowing algorithms, this reduces the visual artifact known as "shadow acne".
ShadowTechnique String ShadowGenerator NiStandardShadowTechnique Determines which shadowing algorithm is used to generate the shadow maps.
SizeHint UInt16 ShadowGenerator 512 Hint to the renderer used to describe the desired square resolution of the shadow map.
StrictlyObserveSizeHint Boolean ShadowGenerator False Forces the renderer to obey the SizeHint parameter.
StaticShadows Boolean ShadowGenerator False If a shadow generator is flagged as static its shadow maps will only be rendered when its shadow maps or render views are flagged as dirty.
UseDefaultDepthBias Boolean ShadowGenerator True Sets the shadow generator's depth bias to the default depth bias of the shadow technique assigned to the shadow generator.
DepthBias Float32 ShadowGenerator 0 The shadow depth bias used by the shadow generator. This value is ignored if UseDefaultDepthBias is set to true.
