PhysXCapsule Model

Found in project: PhysXModelLibrary

Traits: Active

Mixes In: PhysXShape

Mixed Into: PhysXCapsuleTrigger

Description

Defines a capsule collision shape for use with the PhysX system. The origin for a capsule is (0, 0, -z/2), or the center of the lower end-cap hemisphere.
