PhysXRagdoll Model

Found in project: PhysXModelLibrary

Traits: Abstract, Active

Mixes In: PhysXProp

Description

Defines the properties for a PhysX ragdoll character. This is used by egmPhysXRagdoll::RagdollService to mark animation sequences from a Core Runtime KF/KFM file as blended with ragdoll data from PhysX.
