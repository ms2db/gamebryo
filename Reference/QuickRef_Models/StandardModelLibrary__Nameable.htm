Nameable Model

Found in project: StandardModelLibrary

Traits: Abstract, Active

Mixed Into: Actor

Description

Entity model representing an entity having a property called "Name". The value of the "Name" property will be the same as the tool-time entity name. This name property is always considered overridden by the entity. Any "Name" value specified on a model or preset will seed the default name generated for placed entities.
