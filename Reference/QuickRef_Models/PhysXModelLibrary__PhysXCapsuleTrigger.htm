PhysXCapsuleTrigger Model

Found in project: PhysXModelLibrary

Traits: Active

Mixes In: PhysXCapsule, PhysXTrigger

Description

PhysXCapsule with an OnPhysXTrigger callback that is called when this PhysXCapsule fulfills the collision requirements (TriggerOnEnter, TriggerOnStay, TriggerOnLeave) with another shape.

Properties

ShapeRotation Rotator PhysXShape 0, 0, 0 The rotation to apply to the shape on top of the entity's rotation.

ShapeType String PhysXCapsule capsule Hidden One of: box, sphere or capsule
ShapeDimensions Point3 PhysXShape 1, 1, 1 The side length in X, Y and Z for a box. The radius of a sphere is X and Y and Z are unused. The radius of the capsule is X, Y is unused and Z is the height not including the end caps.
SceneName String PhysXShape PhysXDefaultSceneName Names the PhysXScene that this terrain object should be placed in.

TriggerOnEnter Boolean PhysXTrigger True If true, callbacks will be invoked when a shape enters the trigger shape.
TriggerOnStay Boolean PhysXTrigger False If true, callbacks will be invoked when a shape is penetrating the trigger shape.
TriggerMinInterval Float32 PhysXTrigger 0.1 The shortest time between trigger invocations when TriggerOnStay is active.
TriggerOnLeave Boolean PhysXTrigger True If true, callbacks will be invoked when a shape leaves the trigger shape.
AnchorAtBase Boolean PhysXTrigger True Hidden When True, the anchor is at the base of the 3D object. When False, the anchor is at the center of the 3D object.

Behaviors

Name Target Type Source Model Traits Description
OnPhysXTrigger Abstract PhysXTrigger Called when PhysX issues a trigger callback for this trigger. The behavior is passed the entity that caused the trigger and the enter/stay/leave status.
