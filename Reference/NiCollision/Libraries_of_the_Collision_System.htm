Libraries of the Collision System

The NiCollision library is a set of capabilities for detecting collisions among objects in a scene. It includes the fundamental data structures and algorithms necessary for the collision detection system, including 3D intersection processing for alternate bounding volumes (ABVs), oriented bounding boxes (OBBs), and triangles.

The Gamebryo collision detection system is implemented primarily in the NiCollision library, as a component of the Gamebryo game engine, whereas the NiMain library provides supporting capabilities via the Gamebryo infrastructure. Specifically, the NiMain library provides scene-graph infrastructure and the ability to assign and retrieve custom collision data from the scene-graph. It also supplements the NiCollision system with bounding sphere information that is efficiently calculated for use with scene graph culling. This supporting infrastructure in NiMain simplifies the substitution and integration of a collision detection system other than NiCollision, if desired.
