NiSphereTriIntersector

<NiSphereTriIntersector.h>

Description

NiSphereTriIntersector is a Gamebryo internal-only class used for collision detection between a sphere and a triangle, and should not be used directly in an application. It has no supported constructors or member functions, other than for internal use only.

However, it is worth noting that squared distance calculations are used to avoid the square root calculation (for performance), and relative measurements are used so that this calculation does not vary with the scale of the objects.
