Limit Accuracy by Having Multiple Triangles Per Leaf

The bin size is the number of triangles per leaf node in the OBB tree. It is a parameter that may be specified when adding an object to a NiCollisionGroup.

The default value for bin size is 1, which indicates that a full OBB tree should be built. A larger value will limit the OBB tree construction to providing only an OBB at a leaf node corresponding to more than 1 triangle. For situations where extreme accuracy of first time or point of contact is not required, the bin size should be set to a value larger than one.
