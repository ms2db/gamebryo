Description

The MmHUDService class is responsible for parsing the HUD definition file and rendering the splash screen and gameplay HUD elements.

The splash screens to display, the HUD elements to display, and the parameters of both are parsed from a file in the data directory named "MangledMetalHUD.xml". Using this XML, you can specify a texture to load as a HUD element, what screen alignment and offset it needs, optional re-scaling, and the option to automatically associate the width with a property on the player entity. This file does not support rapid iteration.

The supported elements and attributes and their default values are listed in the comments of the MangledMetalHUD.xml file.

static bool PreRenderCallback(
NiRenderClick* pkCurrentRenderClick,
void* pvCallbackData)

Updates any dynamic HUD elements immediately before render.

Since changing dynamic HUD elements might be a GPU-blocking operation, we want to give the longest time possible to the GPU to complete the previous frame's work before issuing an update.

Parameter Description
pkCurrentRenderClick the HUD service's click
pvCallbackData unused
