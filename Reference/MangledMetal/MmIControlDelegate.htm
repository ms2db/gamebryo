MmIControlDelegate

<MmIControlDelegate.h>

Inheritance graph for MmIControlDelegate

Description

The MmIControlDelegate class is an interface implemented by all control delegates managed by MmControlService.

Public Methods

virtual void SetLongitudinalMovement(float magnitude) = 0

Set the longitudinal movement magnitude of a control delegate.

This is movement forward or backward relative to the gameplay camera's orientation.

Parameter Description
magnitude Magnitude value

virtual void SetLateralMovement(float magnitude) = 0

Set the lateral movement magnitude of a control delegate.

This is movement sideways relative to the gameplay camera's orientation.

Parameter Description
magnitude Magnitude value

virtual void DoAttack() = 0

Perform an "attack" action.

Applies only to character control delegates.

virtual void DoHeavyAttack() = 0

Perform a "heavy attack" action.

Applies only to character control delegates.
