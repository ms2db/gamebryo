Introduction to NiD3DXEffectShaderLib Classes

The NiD3DXEffectShaderLib is largely an internal-use only library, and so its classes and functions are largely not documented. The purpose of the library is to load FX files and create NiShader-derived objects so they can be used to render Gamebryo objects.

The only interfaces that applications need to use are documented in the NiD3DXEffectShaderLib Files section.

Information on how applications can use the library can be found here, and information on using FX shaders with Gamebryo in general can be found here.
