Description

NiUISignal0, NiUISignal1, and NiUISignal2 implement the second half of the inter-UI Element communication for the NiUserInterface library. Note that NiUISignal1 and NiUISignal2 are templates allowing for different communication primitives. For more information on slots and signals, see the discussion.
