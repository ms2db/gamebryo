NiUIImage

: public NiUIBaseElement

<NiUIImage.h>

Description

NiUIImage is a simple class which allows easy rendering of any subtexture of the texture stored in the NiUIAtlasMap in NDC safe zone coordinates (see discussion for UI NDC).
