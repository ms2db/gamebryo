NiUIRenderGroup

: public NiUIBaseElement

<NiUIRenderGroup.h>

Description

NiUIRenderGroup serves to group UI elements together into render groups. By grouping screen elements, several results may be achieved. First, it decreases the number of separate groups of screen quads which must be drawn each frame, speeding render time. Also, it allows for UI elements to be layered with different depths: the NiUIRenderGroups are drawn in order of increasing depth.
