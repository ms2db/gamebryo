Ni2DString

: public NiFontString

<Ni2DString.h>

Description

Ni2DString creates 2D geometry that will display a text string using a font defined in an NiFont object.

Constructors

Ni2DString(NiFont* pkFont, unsigned int uiFlags, unsigned int uiMaxStringLen, const char* pcString, const NiColorA& unsigned short usPointSize = DEFAULT_POINT_SIZE, const NiRenderTargetGroup* pkTarget = NULL, bool bStatic = false)

pkFont points to the NiFont instance that defines the font being used.

uiFlags is a combination of the flags that are enumerated in the base class NiFontString.

uiMaxStringLen is the maximum length of the text that will be displayed with this Ni2DString object.

pcString is the initial text that will be displayed.

kColor defines the color of the text if uiFlags includes NiFontString::COLORED. If uiFlags does not include NiFontString::COLORED, the color will be white. This color will be multiplied by any color "baked" into the font by NiFontCreator.

sX and sY define the position of the text on screen. If uiFlags includes NiFontString::CENTERED, the text will be centered about the pixel (sX, sY). Otherwise, (sX, sY) will be the upper left corner of the text.

ucDirection defines how the string will be displayed. Options include left to right, right to left, top to bottom, and bottom to top.

usPointSize defines the font point size for drawing the string. If none is specified, then it will be drawn at the point size of the font.

pkTarget specifies target surface for drawing this string. If none is specified then the default back buffer is assumed.

bStatic indicates whether the string is static or will be changing dynamically. This includes not only the text itself, but also its position, size, leading, tracking, and color.
