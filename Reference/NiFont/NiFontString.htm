NiFontString

: public NiRefObject

<NiFontString.h>

Description

NiFontString is the base class for classes that take an ASCII or Unicode text string and create geometry to display the text string using a font defined in an NiFont object.

Constructors

The constructor for NiFontString is protected. The class cannot be created directly; only derived versions such as the Ni2DString class can be created.

Supported Member Functions

void SetText(const char* pcText, unsigned int uiFlags = 0xffffffff)

Set the text of the ASCII string to the string pointed to by pcText. If uiFlags is 0xffffffff, the current flags will be used.

void SetText(const NiWChar* pkText, unsigned int uiFlags = 0xffffffff)

Set the text of the Unicode string to the string pointed to by pkText. If uiFlags is 0xffffffff, the current flags will be used.

const char* GetText()

Returns the currently set ASCII text for the string. If the string contains Unicode characters then the ASCII interface will return NULL.

const NiWChar* GetUnicodeText()

Returns the currently set Unicode text for the string. If the string was set using the ASCII interfaces, GetUnicodeText() will return the Unicode version of the string.

void SetColor(const NiColorA& kColor)

const NiColorA& GetColor() const

Access the color of the string.

unsigned char GetDirection() const

void SetDirection(unsigned char ucDirection)

Get and set the direction the string will be drawn. Options include:

NIFONTSTRING_LEFT_TO_RIGHT
NIFONTSTRING_RIGHT_TO_LEFT
NIFONTSTRING_TOP_TO_BOTTOM
NIFONTSTRING_BOTTOM_TO TOP

float GetLeading() const

void SetLeading(float fLeading)

Gets and sets the inter-line spacing to be used in this string, measured in pixels.

float GetTracking() const

void SetTracking(float fTracking)

Gets and sets the inter-character spacing to be used in this string, measured in pixels.

Flags

The following values are enumerated in NiFontString and OR'd together to form the uiFlags arguments to various functions in NiFontString and derived classes such as Ni2DString.

Flag Meaning
NiFontString::COLORED Use the color member. If not specified, the string is white.
NiFontString::CENTERED Center the string about its assigned position. If not specified, the string's position specifies the upper left position of the string.
