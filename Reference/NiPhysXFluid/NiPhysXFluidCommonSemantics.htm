NiPhysXFluidCommonSemantics

<NiPhysXFluidCommonSemantics.h>

Description

Defines common semantics strings for fluid particle system data.

These strings are used by various classes in the NiPhysXFluid library to identify the type of particle data found in a stream.

Public Types

NiPhysXFluidCommonSemantics::CommonSemantics

The enumerated types used to lookup strings.

Name Description
PHYSXFLUID_INVALID An invalid semantic value.
PHYSXFLUID_POSITION The position of a fluid particle, in PhysX coordinates.
PHYSXFLUID_SEMANTICSCOUNT The maximum number of semantics. Not valid for use as a semantic.

Public Static Methods

static const NiFixedString& PHYSXFLUIDPOSITION()

Return a string representing the fluid position semantic.
