NiVisualTracker::GraphCallbackObject

: public NiMemObject

<NiVisualTracker.h>

Description

This is the abstract base class for all time series line graph callback objects. It defines the functional interface for taking the samples to be used by the NiVisualTracker. The TakeSample method will be called at the update frequency passed to the NiVisualTracker::AddGraph method and a line graph will be constructed showing current and past values.

Constructor

There is no base class constructor.Each derived class must define a constructor.

Destructor

virtual ~GraphCallbackObject()

The destructor does nothing.

Supported Methods

virtual float TakeSample(float fTime) = 0

This method takes an immediate sample and returns its value. fTime is the current time in seconds. The frequency with which this method is called is determined by the NiVisualTracker::AddGraph() call, and it is not guaranteed to be called each frame, so care must be taken for frame-specific statistics, such as frame rate.
