Description

A special NiMaterialSwapProcessor which inspects the ExtraData on a RenderObject before doing a material swap.

If there is a piece of ExtraData with the name "NiNoMaterialSwap," the material will not be swapped.

This is used for when certain objects in a render view should not have their material swapped but need to be in the same render click as objects that do require their material swapped.
