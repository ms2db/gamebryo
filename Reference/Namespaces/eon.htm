efd::Bool CreateOnlineServices(
efd::ServiceManager* pServiceManager,
efd::UInt32 flags = 0)

Create the common Online services needed by most applications.

If you use a fairly standard configuration you can simply call this method to create, configure and register the services.

This function always creates the following services: eon::OnlineEntityManager and eon::ReplicationService.

Parameter Description
pServiceManager The ServiceManager to which the services are registered
flags Flags from the OnlineServiceAllocatorFlags enum to change default behavior
