EE_ECRPHYSX_ENTRY void ApplyTorque(
egf::EntityID eid,
efd::Point3 torque)

This method applies a conventional torque (or impulse) defined in the global coordinate frame to the all the actors in the specified entity.

Parameter Description
eid The prop entity that is receiving the torque
torque The torque being applied.

EE_ECRPHYSX_ENTRY void ApplyForce(
egf::EntityID eid,
efd::Point3 force)

This method applies a conventional force (or impulse) defined in the global coordinate frame to the all the actors in the specified entity.

Note that if the force does not act along the center of mass of the actor, this will also add the corresponding torque.

Parameter Description
eid The prop entity that is receiving the force
force The force being applied.

EE_ECRPHYSX_ENTRY void ApplyForceAtPos(
egf::EntityID eid,
efd::Point3 force,
efd::Point3 pos)

This method applies a conventional force (or impulse) defined in the global coordinate frame, acting at a particular point in global coordinates, to the all the actors in the specified entity.

Note that if the force does not act along the center of mass of the actor, this will also add the corresponding torque.

Parameter Description
eid The prop entity that is receiving the force
force The force being applied.
pos The point in global coordinates where the force is applied.
