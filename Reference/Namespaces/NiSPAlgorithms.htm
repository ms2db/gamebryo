NiSPAlgorithms Namespace

Alignment Helper Methods

Helper methods to assist in the alignment of data.

NiUInt64 Align128(NiUInt64 uiSize)

Compute the lowest multiple of 128 bytes that is greater than or equal to the given size.

Parameter Description
uiSize Original size to align.
Return Value Aligned buffer size.

void* PreAlign128(void* uiSize)

Align the given pointer to a 128 byte boundary that is lower than or equal to the pointer.

Parameter Description
uiSize The original pointer to align.
Return Value Aligned address.

Sorting Methods.

Helper methods to assist in the sorting of data collections.

template <class C , class T > void QuickSortObjects(
C* pkCollection,
NiInt32 iLeft,
NiInt32 iRight)

Sorts a collection of objects using quick sort.

C is the type of container class pkCollection. T is the type of contained object in pkCollection (e.g. NiObject).

Parameter Description
pkCollection The collection to sort.
iLeft Leftmost index.
iRight Rightmost index.

template <class C , class T > void QuickSortPointers(
C* pkCollection,
NiInt32 iLeft,
NiInt32 iRight)

Sorts a collection of pointers-to-objects using quick sort.

C is the type of container class pkCollection. T is the type of contained object in pkCollection (e.g. NiObjectPtr or NiObject*).

Parameter Description
pkCollection The collection to sort.
iLeft Leftmost index.
iRight Rightmost index.
