Introduction to NiMeshTools

The NiMeshTools library contains utility methods for operating on an NiMesh or sub-components of an NiMesh object. These utility methods are implemented by the NiMeshTools class and are primarily used to Gamebryo tools and tool libraries. Additionally, the library includes the NiToolPipelineCloneHelper class which may be used by plug-ins and tools to manipulate shared data streams on a mesh.
