NiNBTGenerator

<NiNBTGenerator.h>

Description

This class contains functionality to generate binormal and tangent data streams for a mesh.

Constructor

There is no supported constructor for NiNBTGenerator. All of its functions are static.

Supported Member Functions

static void SetupBinormalTangentData(NiMesh* pkMesh, NiUInt16 usUVSet)

This function will populate both a binormal and tangent data stream. If binormal and tangent data already exists, the existing streams will be modified. In the case where the data didn't already exist, the binormal and tangent data will both be added as separate single element data streams with the same index as the specified UV set. In order for the algorithm to work, position and normal data must exist at index 0 and texture coordinate data must exist at the UV set specified.
