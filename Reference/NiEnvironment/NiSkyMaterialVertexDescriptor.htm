Inheritance graph for NiSkyMaterialVertexDescriptor

Description

This class encapuslates a bit field used to describe a vertex shader for NiSkyMaterial.

The vertex material descriptor provides a way of easily describing all the information needed to produce a vertex shader that matches the requirements of a provided NiSky object.

Public Methods

NiString ToString()

Returns a string representation of the state of the bitfield.

bool OutputNormals()

Returns true if the vertex shader needs to output the normals of the verts.

bool OutputWorldView()

Returns true if the vertex shader needs to output the world view vector.
