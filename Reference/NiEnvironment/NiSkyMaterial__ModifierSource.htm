NiSkyMaterial::ModifierSource

<NiSkyMaterial.h>

Description

A structure to encapsulate the selection of a blend stage modifier.

Public Types

NiSkyMaterial::ModifierSource::Value

Name Description
CUSTOM Specifies a custom blend modifier source.
DEFAULT Specifies to use the default values (fastest).
ALPHA Specifies to use the alpha of the image.
CONSTANT Specifies to use a shader constant for the modifier.
HORIZONBIAS Specifies to use shader constants and normal to blend.
DISABLED Specifies this entire blend stage is disabled.
