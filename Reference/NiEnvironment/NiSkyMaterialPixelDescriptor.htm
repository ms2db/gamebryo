Inheritance graph for NiSkyMaterialPixelDescriptor

Description

This class encapsulates a bit field used to describe a pixel shader for NiSkyMaterial.

The pixel material descriptor provides a way of easily describing all the information needed to produce a pixel shader that matches the requirements of a provided NiSky object.
