Description

The MeshModel class provides the built-in model functionality for the Mesh model in the StandardModelLibrary.

Entites that mix in Mesh use this built-in, and the Scene Graph Service makes use of its properties. This model is primarily a data container. However, it is responsible for notifying the ecr::SceneGraphService that an entity with the Mesh model has been created or deleted. This is done in the OnAdded and OnRemoved methods.
