Inheritance graph for ecr::AttachmentMessage

Description

The AttachmentMessage class is a generic message class that holds an entity and an attachment key name.

Messages indicating specific events are derived from this message.

The attachment key will match one of the keys in the "AttachedObjects" associative property on the entity.
