Inheritance graph for NiTerrainMaterialPixelDescriptor

Description

This class manages the bit fields that are constructed from the terrain material descriptor.

It is used when determining how to construct a shade tree in order to generate a pixel shader.

Public Methods

NiString ToString()

Convert the bit field to a human readable string.

This is useful when debugging the values set in the bit field.
