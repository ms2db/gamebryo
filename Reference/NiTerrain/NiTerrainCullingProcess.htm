static NiCullingProcess* CreateTerrainCullingProcess()

Simple factory method for creating a NiTerrainCullingProcess.

This method is registered as a NiShadowManager::CreateShadowCullingProcess call back. Note: Decal rendering will be disabled by default for the culling process returned by this method.

NiTerrainCullingProcess(
NiVisibleArray* pkVisibleSet,
NiSPWorkflowManager* pkWorkflowManager)

The default public constructor.

Parameter Description
pkVisibleSet Pointer to the visible set to use with this culling process. NULL may be passed if the application intends to provide a visible set to the call to Cull.
pkWorkflowManager The workflow manager for this culling process. Applications may pass NULL in for the workflow manager. If so, NiTerrainCullingProcess will create one from the heap.
