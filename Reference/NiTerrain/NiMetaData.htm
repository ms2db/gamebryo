Inheritance graph for NiMetaData

Description

The MetaData object is used to store extra data associated with an object.

Currently only supports INTEGER, STRING and FLOAT data-types.

Public Types

NiMetaData::KeyType

Meta-data data-types.

These are explicitly defined so the compiler does not change them. Remember, these are used during load/save operations.

Name Description
INTEGER Single integer value.
FLOAT Single floating point value.
STRING Arbitrary string value.
INTEGER_BLENDED Single integer value that will blend with other meta-data objects.
FLOAT_BLENDED Single floating point value that will blend with other meta-data objects.
ERROR_TYPE Error type used when a problem occurs when blending. Value is a string containing a possible error message

void Blend(
const NiMetaData* pkMetaData,
float fWeight)

Blend this meta-data object with another meta-data object.

Any keys that do not already exist will be added. Any keys that already exist will be blended according to their type. INTEGER_BLENDED and FLOAT_BLENDED will be blended together using the weighting provided. For all other types the value will remain the value of the meta-data with the highest weight.

Parameter Description
pkMetaData A pointer to the meta-data to blend with.
fWeight The weighting of this meta-data's values.
