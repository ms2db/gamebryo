NiTerrainSurfacePackageFile::DataField

<NiTerrainSurfacePackageFile.h>

Description

A structure with a list of enumeration values describing discrete data fields that may be loaded from the file.

Public Types

NiTerrainSurfacePackageFile::DataField::Value

The enumeration containing a list of data field type ID's.

Name Description
SURFACE_CONFIG Indicates all package configuration information and surface/texture information.
COMPILED_TEXTURES Indicates the set of pre-compiled textures that have been saved into the format.
