Description

This class encapsulates a bit field used to describe an NiWaterMaterial shader.

The NiWaterMaterial generates shaders based on the defined water properties and dynamic effects that affect the light. The NiWaterMaterialDescriptor provides an easy way of describing all the information that is needed to generate a shader.
