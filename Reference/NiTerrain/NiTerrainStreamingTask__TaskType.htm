NiTerrainStreamingTask::TaskType

<NiTerrainStreamingTask.h>

Description

An enumeration to indicate different types of tasks.

Public Types

NiTerrainStreamingTask::TaskType::VALUE

Name Description
UNINITIALIZED The task has been uninitialized and so does not have a type yet.
LOADING The task is for loading data into a sector.
UNLOADING The task is for removing data from a sector.
REBUILDING The task is for rebuilding data in a sector.
