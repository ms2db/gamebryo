Inheritance graph for NiTerrainStreamLocks

Description

This class simplifies the locking and accessing of terrain streams.

An instance of this class should only be used in the NiTerrain library when accessing data in the terrain streams.

Public Types

NiTerrainStreamLocks::StreamType

Enum defining the diferent stream types.

Name Description
POSITION Position stream.
TANGENT Tangent stream.
NORMAL Normal stream.
INDEX Index stream.
TEXCOORD Texture coordinate stream.

Public Methods

NiTerrainStreamLocks()

Default constructor.

NiTerrainStreamLocks(
NiTerrainSector* pkSector,
NiUInt8 ucLockMask,
NiUInt32 uiLevel = 0)

Creates a NiTerrainStreamLocks object and acquires locks for all available static streams for the given sector at the given level of detail and using the given lock mask.

Parameter Description
pkSector pointer to the sector to create locks for
ucLockMask the lock mask to use on the lock (read or write)
uiLevel the level of detail to get the lock for (0 by default)

NiTerrainStreamLocks(
NiTerrainCell* pkCell,
NiUInt8 ucLockMask)

Creates a NiTerrainStreamLocks object and acquires locks for all available streams for the given block.

If some streams are dynamic, it will lock the dynamic streams.

Parameter Description
pkCell pointer to the cell to get the lock for
ucLockMask the lock mask to use on the lock (read or write)

~NiTerrainStreamLocks()

Destructor.

Calls on ReleaseLocks.

bool GetPositionIterator(
NiTerrainCell* pkCell,
NiUInt8 ucLockMask,
NiTerrainPositionRandomAccessIterator& kPosIter)

Retrieves a position iterator for a block or for a sector at a specific level.

If a lock has been acquired and it does not point to the same stream or does not use the same mask, it will release the current lock and acquire the appropriate one. If the created iterator exists, this functions return true.
