Inheritance graph for NiTerrainSectorFileVersion5

Description

The class is used to iterate over and write terrain sector files.

Public Types

NiTerrainSectorFileVersion5::StreamType

Identifiers for each type of stream capable of being stored.

Currently, element size is assumed to be sizeof(float) for endian swapping purposes.

Name Description
STREAM_INVALID Invalid or unspecified stream.
STREAM_HEIGHT Height stream.
STREAM_NORMAL Normal stream.
STREAM_TANGENT Tangent stream.
STREAM_MORPH_HEIGHT Morph Height stream.
STREAM_MORPH_NORMAL Morph Normal stream.
STREAM_MORPH_TANGENT Morph Tangent stream.
STREAM_SURFACE_INDEX The surface index stream.
STREAM_BLEND_MASK A blend mask stream.
STREAM_LOW_DETAIL_DIFFUSE_TEXTURE A low detail diffuse texture stream.
STREAM_LOW_DETAIL_NORMAL_TEXTURE A low detail normal texture stream.
STREAM_MAX_NUMBER Maximum number of streams per block.

virtual void WriteFileHeader()

Write the file header.

This function must only be called once, and called BEFORE any calls to WriteBlock().

Can only be called on a writable file.

virtual void WriteBlock()

Write the current block's data to the file.

This function must be called for each block to be written to the file, all stream data, bounding data, etc must be set before this function is called. By calling this function all current block data is reset and must be set again before the next call to WriteBlock().

Can only be called on a writable file.

virtual bool PushBlock(int iChildID)

Jump to the [iChildID]th child of the current block.

Can only be called on a file opened for read access

Can be useful for jumping to a specific level of detail region of the file

Parameter Description

Return Value True if the block was successfully read, false otherwise.

virtual bool PopBlock()

Jump to the parent of the current block.

Can only be called on a file opened for read access

Using this function after reading a LOD region of the file (i.e. sequentially reading more than the 4 children of the parent block) will cause any further GetBlockID calls to return incorrect information

Parameter Description
Return Value True if the block was successfully read.
