Introduction to NiStripify

The NiStripify library contains code to convert unstripped, indexed meshes that use 3 indices to define each triangle into indexed meshes that define triangles using an array of strips, where an n-vertex strip defines n-2 triangles. The Gamebryo exporters create triangle list meshes which can be optionally converted to triangle strips using NiStripify.

The NiStripify code converts on an NiMesh with any number of sub-meshes and the PrimitiveType of PRIMITIVE_TRIANGLES into an NiMesh of the PrimitiveType PRIMITIVE_TRISTRIPS, preserving the existing sub-mesh layout. Additionally, it will remap the indices and vertices for cache coherency. All NiDataStreams attached to the NiMesh with a Usage type of USAGE_VERTEX_INDEX or USAGE_VERTEX will be affected by this operation.
