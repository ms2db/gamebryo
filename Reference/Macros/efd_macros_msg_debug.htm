Message Debugging Macros

#define EE_MESSAGE_BREAK(pMessage, flags, args)

Macro to debug break if the flags are set, or trace if the flags and TraceOnly are set.

Parameter Description
pMessage A standard pointer (non-smart) to the message.
cat The category the message is sent on.
flags Any currently set IMessage debug/trace flags.
args The message to log.

#define EE_MESSAGE_TRACE(pMessage, module, level1, level2, args)

Macro to log a trace entry for a message.

Parameter Description
pMessage A standard pointer (non-smart) to the message.
module The module the message is being sent from.
level1 The level1 log level is used if any debug flags are set on the IMessage.
level2 The level2 log level is used when there are no debug flags set on the IMessage.
args The message to log.
