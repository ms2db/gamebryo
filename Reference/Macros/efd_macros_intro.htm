Macros: Introduction

Gamebryo LightSpeed provides the following macros:

  • Assert Macros
  • Binary Stream Macros
  • Built-in Model Macros
  • EE_NEW and EE_DELETE Macros
  • Logging Macros
  • Message Debugging Macros
  • Message Handler Macros
  • Path Macros
  • Property Macros
  • Reference Counting Macros
  • RTTI Macros
  • Service Macros
  • Smart Pointer Macros
  • Static Data Macros
  • Thread Local Static Macros
