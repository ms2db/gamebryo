NiD3D10MeshMaterialBinding

: public NiRefObject

<NiD3D10MeshMaterialBinding.h>

Description

NiD3D10MeshMaterialBinding is a class that wraps functionality needed to build a D3D10 vertex layout for an NiRenderObject.

Constructors

There are no public constructors for this class. The Create function should be used instead.

Supported Member Functions

void UpdateInputLayout(void* pvInputSignature, NiUInt32 uiInputSignatureSize)

Updates the NiD3D10MeshMaterialBinding with the input signature from a specific vertex shader. The NiD3D10MeshMaterialBinding will remember which ID3D10InputLayout object is associated with which input signature, so it will not need to recreate the input layout each time the vertex shader is changed.

D3D10_INPUT_ELEMENT_DESC* GetInputElementDescArray() const

Returns the array of D3D10_INPUT_ELEMENT_DESC structures that was created from the provided NiSemanticAdapterTable to describe the vertex buffers.

NiUInt32 GetInputElementDescCount() const

Returns the number of elements in the array of D3D10_INPUT_ELEMENT_DESC structures that was created from the provided NiSemanticAdapterTable to describe the vertex buffers.

ID3D10InputLayout* GetInputLayout() const

Returns the D3D10 vertex layout represented by this instance. The layout will be based on the most recent input signature passed into UpdateInputLayout.

void ReleaseCachedInputLayouts()

Releases all the ID3D10InputLayout objects that have been created for the various input signatures that have been passed into UpdateInputLayout. This function is called in the NiD3D10MeshMaterialBinding destructor.

void ReleaseElementArray()

Releases the array of D3D10_INPUT_ELEMENT_DESC structures that was created from the provided NiSemanticAdapterTable to describe the vertex buffers. This function is called in the NiD3D10MeshMaterialBinding destructor.

const NiDataStreamRef* GetIndexStreamRef() const

Returns a pointer to the NiDataStreamRef representing the index buffer.

const NiUInt16* GetStreamsToSetArray() const

Returns a pointer to an internal array describing the indices of the data streams that will be set as vertex buffers.

NiUInt16 GetLastValidStream() const

Returns the index of the highest-numbered stream that will be set.s

Static Functions

static NiD3D10MeshMaterialBindingPtr Create(NiRenderObject* pkMesh, const NiSemanticAdapterTable& kAdapterTable)

This function should be used in place of the constructor. If no binding can be created or the D3D10 renderer fail to create a vertex declaration, then this function will return NULL.
