NiD3D10RenderStateGroup

: public NiRefObject

<NiD3D10RenderStateGroup.h>

Description

NiD3D10RenderStateGroup is a collection of render state values that will be used to create ID3D10BlendState, ID3D10DepthStencilState, ID3D10RasterizerState, and ID3D10SamplerState objects. The NiD3D10RenderStateGroup allows a shader or application to accumulate render states individually before passing them to the NiD3D10RenderStateManager to be turned into D3D10 state objects. Refer to Render State Management in D3D10 for further discussion of how to handle render states.

Constructors

NiD3D10RenderStateGroup()

Creates an instance of the class with no render states set.

const unsigned char* GetSamplerRemapping(NiGPUProgram::ProgramType eType)const

Set and get an array describing the sampler remapping. Because a shader program may require the samplers to be set in a different order than they appear in the code, a shader may need to reorder the samplers before setting them. This array will be used by the NiD3D10RenderStateManager to map the samplers that appear in the render state group to the order the shader program requires.

void RemoveSamplerRemapping(NiGPUProgram::ProgramType eType)

Removes the sampler remapping array from the render state group.
