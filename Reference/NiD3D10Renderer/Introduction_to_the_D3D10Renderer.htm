Introduction to the NiD3D10Renderer

The Gamebryo D3D10-based Direct3D renderer is designed to give developers access to the latest features available on D3D10-capable hardware. NiApplication links in the NiD3D10Renderer by default.

The NiD3D10Renderer requires the same version of the DX SDK to be installed on development machines that was used originally to build the library. It also requires the D3D10 runtime corresponding to this version of the DX SDK to be installed on end user machines. Refer to the documentation on DirectX Support for more details.

The NiD3D10Renderer, like the D3D10 libraries themselves, is only supported under Windows Vista. Refer to the documentation on Windows Support for more details.
