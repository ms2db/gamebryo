NiCubeMapDepthStencilBuffer

: public NiDepthStencilBuffer

<NiCubeMapDepthStencilBuffer.h>

Description

This class represents an depth/stencil buffer that is designed to be used with cube maps. It is primarily a utility class that can be used to work around some D3D10 limitations. Most applications will not need to interact with this class directly, but instances of the class will be created when using the NiRenderTargetGroup::CubeMapSetupHelper helper class with the NiD3D10Renderer.

Constructors

No public constructors are available because the creation functions must be able to fail in some cases. As a result, all creation is handled via the static Create functions below.

Static Member Functions

static NiCubeMapDepthStencilBuffer* Create(unsigned int uiWidth, unsigned int uiHeight, NiRenderer* pkRenderer, const NiPixelFormat& kFormat, Ni2DBuffer::MultiSamplePreference eMSAAPref = Ni2DBuffer::MULTISAMPLE_NONE)

This create method attempts to create an NiCubeMapDepthStencilBuffer with the preferred multisampling level matching the NiPixelFormat kFormat. If the renderer is unable to create a buffer that matches the specifications, this method will return NULL. Note the preferred multisampling level must match that of the render target, otherwise no multisampling will be used. See the release notes for each renderer to check multisample antialiasing support for rendered targets.
