NiD3D10TextureData

: public NiTexture::RendererData

<NiD3D10TextureData.h>

Description

NiD3D10TextureData contains the D3D10 representation of the data in a texture, as well as the ID3D10ShaderResourceView that describes how the data should be interpreted.

Constructors

NiD3D10TextureData(NiTexture* pkTexture)

Creates the renderer data for the specified NiTexture.

Supported Member Functions

unsigned short GetLevels() const

Returns the number of mipmap levels in the texture.

bool IsRenderedTexture() const

Returns whether this texture is an NiRenderedTexture.

bool IsSourceTexture() const

Returns whether this texture is an NiSourceTexture.

bool IsDynamicTexture() const

Returns whether this texture is an NiDynamicTexture.

bool IsCubeMap() const

Returns whether or not the texture data represents a cube map.

bool IsTexture1D() const

Returns whether or not the texture data represents a 1D texture or texture array.

bool IsTexture2D() const

Returns whether or not the texture data represents a 2D texture or texture array.

bool IsTexture3D() const

Returns whether or not the texture data represents a 3D texture or texture array.

ID3D10ShaderResourceView* GetResourceView() const

Returns a pointer to the shader resource view of the texture.

ID3D10Resource* GetResource() const

Returns a pointer to the D3D10 resource for the texture.

void SetResourceView(ID3D10ShaderResourceView* pkResourceView)

Sets the shader resource view of the texture. This function should generally not be called by applications.

Static Member Functions

static void ClearTextureData()

Release all texture data.
