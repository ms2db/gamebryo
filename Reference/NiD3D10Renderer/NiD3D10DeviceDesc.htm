NiD3D10DeviceDesc

: public NiMemObject

<NiD3D10SystemDesc.h>

Description

NiD3D10DeviceDesc is a descriptor of the capabilities of a piece of 3D rendering hardware using a specific driver type. It describes the multisample capabilities of that device..

Constructors

There are no public constructors, as only the Gamebryo D3D10 renderer should create instances of this class.

Supported Member Functions

D3D10_DRIVER_TYPE GetDeviceType() const

Returns the D3D10 device type for the calling device.

unsigned int GetHighestMultisampleCount() const

Returns the highest supported multisample count on the device.

bool GetMultisampleSupport(unsigned int uiSamples, unsigned int& uiQualityLevels) const

Returns the number of quality levels supported for the multisample count passed in through uiSamples. If the sample count is not supported, the function will return false.
