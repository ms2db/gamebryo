NiD3D10DataStream

: public NiDataStream

<NiD3D10DataStream.h>

Description

NiD3D10DataStream is a D3D10-specific implementation of an NiDataStream. Data streams will typically be created through the NiDataStream interface itself, which will create an instance of this class using the correct factory. NiD3D10DataStream objects encapsulate an ID3D10Buffer, whether as a vertex buffer, index buffer, or shader constant buffer.

Constructors

This class contains no public constructors. The static Create function below is used to create NiD3D10DataStream objects.

Static Member Functions

static NiD3D10DataStream* Create(ID3D10Buffer* pkBuffer)

Creates a new NiD3D10DataStream instance that encapsulates the provided ID3D10Buffer object.

static bool InterpretDataStreamFlags(NiUInt8 uiAccessMask, Usage eUsage, D3D10_USAGE& eD3D10Usage, NiUInt32& uiBindFlags, NiUInt32& uiCPUAccessFlags)

Takes the Gamebryo access mask that is provided in uiAccessMask and the usage enumeration passed into eUsage and returns the corresponding D3D10_USAGE enumeration, D3D10_BIND_FLAG flags, and D3D10_CPU_ACCESS_FLAG in the eD3D10Usage, uiBindFlags, and uiCPUAccessFlags parameters, respectively. If an exact match cannot be found, the function returns false.

static bool InterpretAccessMask(NiUInt8 uiAccessMask, D3D10_USAGE& eUsage)

Takes the Gamebryo access mask provided in uiAccessMask and returns D3D10_USAGE enumeration that matches the access mask in the eUsage parameter. If an exact match cannot be found, the function returns false.

static NiUInt32 InterpretUsage(Usage eUsage, NiUInt32 uiAccessMask)

Takes the Gamebryo usage enumeration passed into eUsage and the access mask provided in uiAccessMask and returns the combination of D3D10_BIND_FLAG flags that the input parameters correspond to.

static void InterpretD3D10Parameters(D3D10_USAGE eD3D10Usage, NiUInt32 uiD3D10BindFlags, NiUInt32 uiD3D10CPUAccessFlags, Usage& eUsage, NiUInt8& uiAccessFlags)

Takes the D3D10_USAGE enumeration, D3D10_BIND_FLAG flags, and D3D10_CPU_ACCESS_FLAG in the eD3D10Usage, uiBindFlags, and uiCPUAccessFlags parameters, respectively, and returns the corresponding Gamebryo usage enumeration in eUsage and the access mask in uiAccessMask.
