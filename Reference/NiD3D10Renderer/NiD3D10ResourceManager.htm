NiD3D10ResourceManager

: public NiMemObject

<NiD3D10ResourceManager.h>

Description

NiD3D10ResourceManager is a Gamebryo class that wraps the D3D10 resource-creation functions. It is not necessary for applications to use this interface for creating D3D10 resources at this time.

Constructors

NiD3D10ResourceManager(ID3D10Device* pkDevice)

The constructor initializes a new NiD3D10ResourceManager using the D3D10 device. An application should never construct a new resource manager, but should obtain a pointer to the NiD3D10Renderer object's instance instead.

Supported Member Functions
