NiD3D10PixelFormat

<NiD3D10PixelFormat.h>

Description

NiD3D10PixelFormat is little more than a namespace for some useful pixel format-related functions.

Constructors

There is no constructor as no instance of this class should ever exist.

Static Member Functions

static DXGI_FORMAT DetermineDXGIFormat(const NiPixelFormat& kDesc)

Returns the DXGI_FORMAT that matches the format described by kDesc. If a match cannot be found, DXGI_FORMAT_UNKNOWN will be returned.

static DXGI_FORMAT DetermineDXGIFormat(NiDataStreamElement::Format eFormat)

Returns the DXGI_FORMAT that matches the format described by eFormat. If a match cannot be found, DXGI_FORMAT_UNKNOWN will be returned.

static void InitFromDXGIFormat(DXGI_FORMAT eDXGIFormat, NiPixelFormat& kFormat)

Overwrites the contents of kFormat with settings that match the provided DXGI_FORMAT.

static const NiPixelFormat* ObtainFromDXGIFormat(DXGI_FORMAT eDXGIFormat)

Returns the NiPixelFormat instance from NiD3D10PixelFormat that describes the provided DXGI_FORMAT.

static unsigned int GetBitsPerPixel(DXGI_FORMAT eDXGIFormat)

Returns the number of bits per pixel for the provided DXGI_FORMAT.

static const char* const GetFormatName(DXGI_FORMAT eDXGIFormat, bool bWithPrefix = true)

Returns a text string containing the name of the provided DXGI_FORMAT. If bWithPrefix is false, then the prefix "DXGI_FORMAT_" text will not be included in the string.

Static Member Variables

Every DXGI_FORMAT_* enumeration in D3D10 is described by a corresponding NiPixelFormat object named NiD3D10PixelFormat::NI_FORMAT_*. For example, DXGI_FORMAT_R11G11B10_FLOAT is described in Gamebryo by NiD3D10PixelFormat::NI_FORMAT_R11G11B10_FLOAT.
