NiD3D10OutputDesc

: public NiMemObject

<NiD3D10SystemDesc.h>

Description

NiD3D10OutputDesc is a descriptor of the capabilities of a specific output device, such as a monitor. It describes the supported display modes that this output device can handle.

Constructors

There are no public constructors, as only the Gamebryo D3D10 renderer should create instances of this class.

Supported Member Functions

const DXGI_OUTPUT_DESC* GetDesc() const

Returns the DXGI_OUTPUT_DESC that describes this output.

unsigned int GetDisplayModeCount() const

Returns the number of display modes that are supported by this output device.

const DXGI_MODE_DESC* GetDisplayModeArray() const

Returns the array of DXGI_MODE_DESC structures that are supported by this output device.

IDXGIOutput* GetOutput() const

Returns the IDXGIOutput represented by this NiD3D10OutputDesc.

unsigned int GetIndex() const

Returns the index of this NiD3D10OutputDesc.
