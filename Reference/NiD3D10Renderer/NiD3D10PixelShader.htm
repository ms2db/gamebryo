NiD3D10PixelShader

: public NiD3D10ShaderProgram

<NiD3D10PixelShader.h>

Description

NiD3D10PixelShader is the shader program derivation for pixel shaders.

Constructors

NiD3D10PixelShader()

The default constructor creates an initialized NiD3D10PixelShader instance.

NiD3D10PixelShader(ID3D10PixelShader* pkPixelShader, ID3D10Blob* pkShaderByteCode)

This constructor creates an initialized NiD3D10PixelShader instance using the provided pixel shader and an ID3D10Blob containing the shader's byte code.

Supported Member Functions

ID3D10PixelShader* GetPixelShader() const

Returns the ID3D10PixelShader pointer.

void SetPixelShader(ID3D10PixelShader* pkPixelShader)

Sets the ID3D10PixelShader pointer. Note that the pixel shader and the shader byte code need to match up, so this function should not be called without a corresponding NiD3D10ShaderProgram::SetShaderByteCode.

virtual void DestroyRendererData()

Releases the ID3D10PixelShader reference.

virtual void RecreateRendererData()

Recreates the ID3D10PixelShader from the shader byte code, if it exists.
