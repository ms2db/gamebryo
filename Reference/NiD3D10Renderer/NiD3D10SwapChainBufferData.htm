NiD3D10SwapChainBufferData

: public NiD3D10RenderTargetBufferData

<NiD3D102DBufferData.h>

Description

NiD3D10SwapChainBufferData is a Gamebryo class that represents a swap chain. Gamebryo creates instances of this class internally to represent swap chains, but applications can also create instances if they have an IDXGISwapChain that they want to use as a render target.

Constructors

This class contains no public constructors. The static Create function below is used to create NiD3D10SwapChainBufferData objects.

Static Member Functions

static NiD3D10SwapChainBufferData* Create(IDXGISwapChain* pkSwapChain, Ni2DBuffer*& pkBuffer)

Creates a new instance of an NiD3D10RenderTargetBufferData object. The pkSwapChain parameter should be the swap chain that will be used as the render target buffer. The pkBuffer parameter will be the pointer that holds the Ni2DBuffer object that references the NiD3D10SwapChainBufferData . If pkBuffer is not empty, then it will be filled in with the newly created NiD3D10SwapChainBufferData object, but if it is empty, then when the function returns, pkBuffer will point to a new Ni2DBuffer object. This new object will need to be assigned to a smart pointer to prevent it from getting leaked accidentally. This function will add a reference to the passed-in IDXGISwapChain object.

Supported Member Functions

bool ResizeSwapChain(unsigned int uiWidth, unsigned int uiHeight)

Resizes the swap chain using IDXGISwapChain::ResizeBuffers.

IDXGISwapChain* GetSwapChain() const

Returns the swap chain for this render target.
