NiD3D10DataStreamFactory

: public NiDataStreamFactory

<NiD3D10DataStreamFactory.h>

Description

NiD3D10DataStreamFactory is a D3D10-specific implementation of an NiDataStreamFactory. Its methods will be invoked automatically through the NiDataStream::Create functions when using an NiD3D10Renderer.
