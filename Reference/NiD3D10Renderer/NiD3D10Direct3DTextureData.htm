NiD3D10Direct3DTextureData

: public NiD3D10TextureData

<NiD3D10Direct3DTextureData.h>

Description

NiD3D10Direct3DTextureData contains the D3D10 representation of the data in a texture that has been provided by an application.

Constructors

This class contains no public constructors the static Create function below is used to create NiD3D10Direct3DTextureData objects.

Static Member Functions

static NiD3D10Direct3DTextureData* Create(NiD3D10Direct3DTexture* pkTexture, ID3D10Resource* pkD3D10Texture, ID3D10ShaderResourceView* pkResourceView)

Creates the renderer data for the specified NiD3D10Direct3DTexture using the provided resource pkD3DTexture and resource view pkResourceView. This function should not be called by applications, since NiD3D10Renderer::CreateNiTextureFromD3D10Texture is the preferred API to use.
