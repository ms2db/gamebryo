NiD3D10GPUProgramCache

: public NiGPUProgramCache

<NiD3D10GPUProgramCache.h>

Description

NiD3D10GPUProgramCache is the implementation of NiGPUProgramCache specific to the D3D10 renderer.

Constructors

NiD3D10GPUProgramCache(unsigned int uiVersion, const char* pcWorkingDir, NiGPUProgram::ProgramType eType, const NiFixedString& kShaderProfile, const char* pcMaterialIdentifier, bool bAutoWriteToDisk, bool bWriteDebugHLSLFile, bool Locked, bool bLoad)

Create a cache with the specified version for shader programs of the type identified with eType. The working directory is set to pcWorkingDir, and the shaders will be compiled using the shader profile kShaderProfile. If the bAutoWriteToDisk flag is true, the cache will automatically write its files to disk after a compile. If bWriteDebugHLSLFile is true, then an additional .hlsl file will be written to disk each time a shader program is generated, and that file can be used to debug the shader program. If bLocked is true, the cache will not compile new programs, but will provide programs already in the cache. If bLoad is true, then the cache will attempt to load any cache file at the file location that it would ordinarily write its cache to, and add the loaded shaders to its own cache.

Supported Member Functions

virtual NiGPUProgram* GenerateProgram(const char* pcName, const char* pcProgramText, NiTObjectPtrSet<NiMaterialResourcePtr>& kUniforms)

Inherited from NiGPUProgramCache. This method compiles the input text into a GPU program. The uniform variables used by the program are returned by reference in the kUniforms variable. If the program is set to write, it will automatically write the compiled program and variables to the cache file.

virtual const char* GetPlatformSpecificCodeID() const

Inherited from NiGPUProgramCache. Returns "hlsl".

virtual const char* GetPlatform() const

Inherited from NiGPUProgramCache. Returns "D3D10".
