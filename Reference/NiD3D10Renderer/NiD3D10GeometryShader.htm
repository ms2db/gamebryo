NiD3D10GeometryShader

: public NiD3D10ShaderProgram

<NiD3D10GeometryShader.h>

Description

NiD3D10GeometryShader is the shader program derivation for geometry shaders.

Constructors

NiD3D10GeometryShader()

The default constructor creates an initialized NiD3D10GeometryShader instance.

NiD3D10GeometryShader(ID3D10GeometryShader* pkGeometryShader, ID3D10Blob* pkShaderByteCode)

This constructor creates an initialized NiD3D10GeometryShader instance using the provided geometry shader and an ID3D10Blob containing the shader's byte code.

Supported Member Functions

ID3D10GeometryShader* GetGeometryShader() const

Returns the ID3D10GeometryShader pointer.

void SetGeometryShader(ID3D10GeometryShader* pkGeometryShader)

Sets the ID3D10GeometryShader pointer. Note that the geometry shader and the shader byte code need to match up, so this function should not be called without a corresponding NiD3D10ShaderProgram::SetShaderByteCode.

virtual void RecreateRendererData()

Recreates the ID3D10GeometryShader from the shader byte code, if it exists.

virtual void DestroyRendererData()

Releases the ID3D10GeometryShader reference.

void SetStreamOutputDeclaration(const D3D10_SO_DECLARATION_ENTRY* pkSODeclaration, unsigned int uiNumEntries, unsigned int uiOutputStreamStride)

Set the stream output declaration, as well as the number of entries in the declaration and the total stride, for geometry shaders with stream output capability.

const D3D10_SO_DECLARATION_ENTRY* GetStreamOutputDeclaration() const;

Returns the stream output declaration for geometry shaders with stream output capability.

unsigned int GetNumStreamOutputEntries() const

Returns the number of entries in the stream output declaration for geometry shaders with stream output capability.

unsigned int GetOutputStreamStride() const

Returns the stream output stride for geometry shaders with stream output capability.
