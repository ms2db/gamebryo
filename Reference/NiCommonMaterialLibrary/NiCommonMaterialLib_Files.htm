NiCommonMaterialLib Files

The common material library project contains the following files:

  • NiCommonMaterialLib.h/cpp—Implements the DLL interfaces for the custom material library
  • NiCommonMaterialLibrary.h/cpp—Implements the common material library, containing the materials detailed below. Of particular note is the constructor, which creates the NiShaderDesc objects for the materials.
  • NiCommonMaterialSDM.h/cpp—Implements the static data manager for this project.
  • NiCommonMaterialLibType.h—Sets up the DLL import/export definitions for this project
  • NiCommonMaterialLibPCH.h/cpp—Sets up the pre-compiled header for this project
  • NiFlatWireframeMaterial.h/cpp—Implements a custom material that supports skinned and unskinned implementations. See NiFlatWireframeMaterial and NiMaterial Extension::Using Standard Shaders for more details
  • TextureBlendMaterial.h/cpp—Implements a texture blending material on top of the standard material. See TextureBlendMaterial and NiMaterial extension:: Extending the Standard Material for more details.
  • AdditiveLightMapsMaterial.h/cpp—Implements a material on top of the standard material that changes behavior of the texture pipeline so that projected lights lighten instead of darken the base map. See AdditiveLightMapsMaterial and NiMaterial extension:: Extending the Standard Material for more details.

If a developer wishes to use the materials specified in this file, they will need to perform one of the following actions:

  • For statically linked libraries, include the appropriate lib file for the build configuration and make sure to include NiCommonMaterialLib.h somewhere in the application source code.
  • For dynamically linked libraries, please use the NiMaterialLibrary::LoadMaterialLibraryDLL method to load the DLL into the application.
