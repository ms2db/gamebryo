NiMeshProfileComponent::NiClampData

<NiMeshProfileComponent.h>

Description

This structure is used to store the data needed to perform a clamp.

This class does not do any of the clamping functionality but serves solely as the data storage to be used to convert to a NiOpClamp operational node.

clamp="min=x max=y"
