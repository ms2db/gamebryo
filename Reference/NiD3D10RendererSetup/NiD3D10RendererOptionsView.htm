NiD3D10RendererOptionsView

: public NiBaseRendererOptionsView

<NiD3D10RendererOptionsView.h>

Description

This class handles the controls for the Direct3D 10-related settings in the renderer section of the renderer settings dialog.

These classes work with corresponding NiD3D10RendererDesc object, which manages the list of available options and store the user's choices to an NiRendererSettings object. This separates the data from the mechanism to view and manipulate that data. The NiD3D10RendererDesc object is instantiated when the settings dialog is created by this object, and destroyed when user presses OK or Cancel in the renderer settings dialog.

Virtual Member Functions

virtual NiWindowRef InitDialog(NiWindowRef pParentWnd)

Creates and returns a dialog with settings for a specific renderer. The dialog will be a child window of pParentWnd.

virtual char* GetName()

Returns the name of the renderer controlled by this object.

virtual unsigned int SetBasicHeight()

virtual unsigned int SetAdvancedHeight()

Resizes the child dialog to the default height for basic or advanced mode, and returns the new height in pixels. The width is constant and cannot be set. These functions are called when the user presses the "Advanced" button to switch between the basic and advanced modes of the renderer settings dialog.
