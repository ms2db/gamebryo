void SetEntityVisibility(
egf::EntityID entityID,
const efd::utf8string& source,
efd::UInt32 category,
bool isVisible)

Tells the service if a given entity is visible or not.

The service needs to know if an entity is visible so that the property visualizers bound to that entity match the entity's visibility. This uses the VisualizerVisibilityTracker to make final visibility determinations.

Parameter Description
entityID The entity.
source The source changing the visibility.
category A unique integer ID used by the source to differentiate between multiple methods of changing invisibility. Must be a number between 0-31.
isVisible Whether the entity is visible or not.
