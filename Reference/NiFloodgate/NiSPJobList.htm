Description

This class is a platform specific implementation of a set of jobs to be executed by the stream processor.

The documentation given here is PS3 specific, where this class manages the creation of a PS3 JOBLIST consisting of one or more jobs. For platforms other than PS3, this documentation is partial. Please refer to the platform specific header file for complete documentation.
