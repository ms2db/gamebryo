Inheritance graph for NiGeometryConverter

Description

Manages backwards compatibility by automatically converting NiGeometry-based assets to the NiMesh-based system.

Public Types

NiGeometryConverter::ConversionResult

Type to hold the conversion result.

Name Description
SUCCESS Successful conversion to an NiMesh based asset.
SUCCESS_CLONED Successful conversion to an NiMesh based asset when a clone of the asset has already been successfully converted; returns a clone of the previously converted asset.
FAIL Conversion has failed.

NiGeometryConverter(
bool bForce32BitIndices = true,
bool bIsRuntime = false,
bool bAttachModifiers = true)

Constructor.

Parameter Description
bForce32BitIndices If true, create 32-bit index buffers. If false, 16-bit index buffers are used.
bIsRuntime If true, NiDataStream objects will be renderer-specific if a renderer has been created and its NiDataStreamFactory initialized. If false, NiDataStream objects will be of type NiToolDataStream.
bAttachModifiers If true, mesh modifiers created in the conversion process are both added and attached. If false, they are added but not attached.
