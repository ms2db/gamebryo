NiScriptReader

: public NiRefObject

<NiScriptReader.h>

Description

This class is the interface for the parser of NiScriptInfo objects from character strings. An NiScriptReader is usually complemented with an NiScriptWriter.

Supported Member Functions

NiScriptInfo* Parse(const char* pcScript, NiString& strErrors)

Takes an input character script and parses it to create the NiScriptInfo object.  Any errors in the process are written to the strErrors string.

NiString GetFileExtensions()

This method enumerates the extensions that this reader can support.  The format for this string is "*.EXTENSION;*.EXTENSION2".
