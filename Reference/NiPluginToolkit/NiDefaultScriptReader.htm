NiDefaultScriptReader

: public NiScriptReader

<NiDefaultScriptReader.h>

Description

This class is the interface for the decoding of NiScriptInfo objects from the *.script file character strings. The complementary writer class is NiDefaultScriptWriter.

Format

Script("SCRIPT_NAME","SCRIPT_TYPE")
{
   Plugin("PLUGIN_CLASSNAME","PLUGIN_NAME","PLUGIN_VERSION", "PLUGIN_TYPE")
   {
      Parameter("PARAM_KEY", "PARAM_VALUE");
      ...
   }
   ...
}

Supported Member Functions

NiScriptInfo* Parse(const char* pcScript, NiString& strErrors)

This method takes a properly encoded character string and converts it into an NiScriptInfo object.

NiString GetFileExtensions()

This method returns "*.script".

NiPluginInfo* CreateByType(NiString strClassName, NiString strName, NiString strVersion, NiString strType)

Creates a NiPluginInfo object of the appropriate class based on the input parameters.
