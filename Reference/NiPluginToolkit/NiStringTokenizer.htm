NiStringTokenizer

<NiStringTokenizer.h>

Description

This class is designed to take an existing NiString and break it into tokens based on an NiString of delimiters.

Constructor

NiStringTokenizer(const char* pcSource)

Create a NiString tokenizer for the argument NiString.

Supported Member Functions

NiString GetNextToken(const char* pcDelimiters)

Get the next NiString token from the stream using the delimiters found in the string pcDelimiters.

bool IsEmpty()

Query if the end of the stream has been reached.

unsigned int Consume(const char* pcChars)

Consumes any of the following contiguous characters in pcChars from the stream. The number of consumed characters is returned.

unsigned int Consume(unsigned int uiNumChars)

Consumes uiNumChars from the stream. The number of consumed characters is returned.

const char* Peek()

This method allows a glimpse at what is remaining in the current stream.
