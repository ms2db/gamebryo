NiExecutionResult

: public NiRefObject

<NiExecutionResult.h>

Description

This class encapsulates the result from a plug-in execution.

Constructors

NiExecutionResult(ReturnCode eReturnCode)

The constructor takes the return code to store in this result. See the notes below for the definition of ReturnCode.

Supported Member Variables

ReturnCode m_eReturnCode

This variable stores the return code for this result. It is a public variable.

Notes

ReturnCode

ReturnCode Definition
EXECUTE_SUCCESS The plug-in executed successfully with no errors.
EXECUTE_FAILURE Some errors occurred during plug-in execution.
