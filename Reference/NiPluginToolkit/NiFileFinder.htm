NiFileFinder

: public NiMemObject

<NiFileFinder.h>

Description

This class is designed to assist in searching for files in a directory. A file finder is created for each directory or sub-tree you wish to search and you can poll the results for file information.

Constructor

NiFileFinder(const char* pcPath, bool bRecurseSubDirs = true, const char* pcExtension = "\0")

This constructor takes the path (for example, "c:\\test\\myfiles\\") and looks for files containing the extension, pcExtension. You can optionally choose to recurse all subdirectories of the path.

NiFoundFile

This class defines a found file during the search procedure.

NiString m_strPath

This variable defines the absolute path to the file.

unsigned int m_uiSize

This variable defines the size of the file on disk.

Supported Member Functions

NiFoundFile* GetNextFile()

This method returns the next file matching the extension requested by searching the path.

bool HasMoreFiles()

This method determines if there are more files matching the criteria in the list.

void ResetIterator()

This method resets the internal list iterator for the files found matching the criteria back to the beginning.

unsigned int GetFileCount()

This method return the number of files found.
