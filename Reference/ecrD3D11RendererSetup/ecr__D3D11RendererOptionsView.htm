Description

D3D11RendererOptionsView is the NiBaseRendererOptionsView that handles the D3D11 dialog.

This class handles the controls for the Direct3D 11-related settings in the renderer section of the renderer settings dialog.

These classes work with corresponding D3D11RendererDesc object, which manages the list of available options and store the user's choices to an NiRendererSettings object. This separates the data from the mechanism to view and manipulate that data. The NiD3D10RendererDesc object is instantiated when the settings dialog is created by this object, and destroyed when user presses OK or Cancel in the renderer settings dialog.
