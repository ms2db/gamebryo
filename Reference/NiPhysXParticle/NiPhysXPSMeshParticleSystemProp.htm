Description

NiPhysXPSMeshParticleSystemProp objects hold information about a PhysX mesh particle system prop.

Each prop stores an array of physics-based mesh particle systems that are part of the prop, and implement callback functions to attach the systems to a scene, or remove them from a scene.

Particle systems are added to the prop via the function NiPhysXPSMeshParticleSystem::AttachToProp, and functionality is provided here only to access the particle systems.
