NiPhysXPSCommonSemantics

<NiPhysXPSCommonSemantics.h>

Description

Define common semantics strings for physics-based particle system data.

These strings are used by various classes in the NiPhysXParticle library to identify the type of particle data in a stream.

Public Types

NiPhysXPSCommonSemantics::CommonSemantics

The enumerated types used to lookup strings.

Name Description
PHYSXPS_INVALID An invalid semantic value.
PHYSXPS_ACTORPOSE The pose of the actors after a simulation step.
PHYSXPS_ACTORVELOCITY The target velocity of the particle actors prior to a simulation step.
PHYSXPS_ACTORLASTVELOCITY The velocity of the particle actors at the end of a simulation step.
PHYSXPS_SEMANTICSCOUNT The maximum number of semantics. Not valid for use as a semantic.

Public Static Methods

static const NiFixedString& PARTICLEACTORPOSE()

Return a string representing the actor pose semantic.

static const NiFixedString& PARTICLEACTORVELOCITY()

Return a string representing the actor velocity semantic.

static const NiFixedString& PARTICLEACTORLASTVELOCITY()

Return a string representing the actor prior velocity semantic.
