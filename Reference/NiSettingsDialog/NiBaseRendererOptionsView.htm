NiBaseRendererOptionsView

: public NiMemObject

<NiBaseRendererOptionsView.h>

Description

This is an abstract base for classes that handle the controls in the renderer section of the renderer settings dialog. The NiRendererTabController object creates an instance of each, but the window for only one of them is visible at a time depending on the selection that the user has made. This allows each renderer to show special settings that apply only to that renderer in a child window.

These classes work with corresponding NiBaseRendererDesc-derived classes, which manage the list of available options and store the user's choices to an NiRendererSettings object. This separates the data from the mechanism to view and manipulate that data. The NiBaseRendererDesc-derived classes are instantiated when the settings dialog is created by an NiBaseRendererOptionsView-derived class, and destroyed when user presses OK or Cancel in the renderer settings dialog.

Virtual Member Functions

virtual NiWindowRef InitDialog(NiWindowRef pParentWnd)

Creates and returns a dialog with settings for a specific renderer. The dialog will be a child window of pParentWnd.

virtual char* GetName() const

Returns the name of the renderer controlled by this object.

virtual void Activate()

virtual void Deactivate()

These functions are called when the window for this object is selected or deselected.

virtual bool IsActive()

Returns true if the window for this object is active, false otherwise.

virtual unsigned int SetBasicHeight()

virtual unsigned int SetAdvancedHeight()

Resizes the child dialog to the default height for basic or advanced mode, and returns the new height in pixels. The width is constant and cannot be set. These functions are called when the user presses the "Advanced" button to switch between the basic and advanced modes of the renderer settings dialog.

virtual efd::SystemDesc::RendererID GetRendererID() const

Returns the renderer ID that the derived class was built to manage.
