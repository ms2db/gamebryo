NiBaseRendererDesc

: public NiMemObject

<NiBaseRendererDesc.h>

Description

This class is an abstract base class that defines an interface for communicating renderer settings to and from an object that derives from NiBaseRendererOptionsView.

A class that derives from NiBaseRendererDesc provides a set of options to a related class that derives from NiBaseRendererOptionsView. The class that derives from NiBaseRendererOptionsView displays those options, allows a user to change them, and then reports the changes to the class that derives from NiBaseRendererDesc. The class that derives from NiBaseRendererDesc saves the settings in an NiRendererSettings object.

Supported Member Functions

This class has no supported member functions.
