Inheritance graph for ecrInput::MouseMessage

Description

Abstract mouse message common across all mouse messages.

Messages of this type are only sent when the ecrInput::InputService object is constructed with the sendSingleMessages parameter true (not the default).

This class is not recommended for use outside of tools. Applications should only use it when working with Gamebryo tool services. Furthermore, the values returned by the GetX and GetY member functions are valid only inside a Gamebryo Toolbench plug-in or Toolbench core code, or in cases where application code explicitly sets the values. The (X, Y) values are not set by the Input Service.
