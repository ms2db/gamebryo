Introduction to NSFParserLib Classes

The NSFParserLib is largely an internal-use only library, and so its classes and functions are largely not documented. The purpose of the library is to load NSF files and create NSBShader representations of the shaders, which get streamed out to NSB files. Those NSB files can then be streamed in and converted to NiBinaryShader objects.

The only interfaces that applications need to use are documented in the NSFParserLib Files section.
