Gamebryo LightSpeed 3.0

  • The Win32 version of NiSample has been updated to always initialize the cursor to the center of the screen.

Gamebryo 2.6.0

  • A new method SetUISkinFilename has been added to NiSample, allowing the directory structure used for the sample UI files to be modified without modifying the source code of the NiSample library. For more information, see the NiSample reference documentation.

Gamebryo 2.3.0

  • NiSample has been modified to optionally support rendering using the new frame rendering system.
  • NiSample has been converted to using the new NiUserInterfacelibrary for UI widgets and consistent navigation controls.
  • A smoother integration for shaders has been developed.
  • The documentation has been rewritten.
