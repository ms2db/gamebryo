Usage of NiSample in Gamebryo

Most of the Gamebryo samples (found in %EMERGENT_PATH%\Samples\GraphicsTechDemos) are built on NiSample, however the SimpleApp shows how to create a Gamebryo application without using NiApplication or NiSample.

An application that uses NiSample must explicitly link the NiMain, NiSystem, NiApplication, NiVisualTracker, NiInput, NiFont, NiCursor, and NiUserInterface libraries, as well as any other Gamebryo libraries it uses. On Win32, the appropriate renderer library is automatically linked in via #pragma comment compiler directives.
