Introduction to NiPhysXTools

The NiPhysXTools library provided functionality for tools related to the Gamebryo-PhysX integration.

  • The NiPhysXSnapshotExtractor class is used to convert a NxScene object and all its PhysX contents into a set of Gamebryo descriptors. The result is a set of NiPhysXProp objects with descriptors filled with the PhysX data. The NiPhysXSnapshotExtractor library has a dependency on the PhysX NxCooking library, so Xbox applications must link against the latter in order to use the former.
  • The NiPhysXTools class provides general functionality useful for manipulating PhysX content in tools.
