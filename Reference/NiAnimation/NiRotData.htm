NiRotData

: public NiObject

<NiRotData.h>

Description

The NiRotData class provides a shareable wrapper for rotation animation keys.

Constructors

NiRotData()

The default constructor creates an NiRotData with no animation data.

Supported Member Functions

unsigned int GetNumKeys() const

This call simply returns the number of keys the NiRotData is currently storing.

NiRotKey* GetAnim(unsigned int &uiNumKeys, NiRotKey::KeyType &eType, unsigned char &ucSize) const

This call returns the key data that the NiRotData currently stores.

void ReplaceAnim(NiRotKey* pkRotKeys, unsigned int uiNumKeys, NiRotKey::KeyType eType)

This call gives the NiRotData animation data to store. Note that the NiRotData will not copy the NiRotKey objects - it will use them directly and delete any existing keys.

void SetAnim(NiRotKey* pkRotKeys, unsigned int uiNumKeys, NiRotKey::KeyType eType)

This call gives the NiRotData animation data to store. Note that the NiRotData will not copy the NiRotKey objects - it will use them directly but does not delete any existing keys. This method is used to transfer ownership of keys.
