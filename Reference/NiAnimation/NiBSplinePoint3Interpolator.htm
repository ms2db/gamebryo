NiBSplinePoint3Interpolator

: public NiBSplineInterpolator

<NiBSplinePoint3Interpolator.h>

Description

NiBSplinePoint3Interpolator uses B-Splines to animate an NiPoint3 value over time. This NiBSplineInterpolator only supports one control point channel.

Constructors

NiBSplinePoint3Interpolator()

This constructor creates an NiBSplinePoint3Interpolator containing invalid data.

NiBSplinePoint3Interpolator(NiBSplineData* pkData, NiBSplineData::Handle kPoint3CPHandle = NiBSplineData::INVALID_HANDLE, NiBSplineBasisData* pkBasisData = NULL)

This constructor creates an NiBSplinePoint3Interpolator with the specified NiPoint3Data.

NiBSplinePoint3Interpolator(const NiPoint3& kPoseValue)

This constructor creates an NiBSplinePoint3Interpolator with a pose value of kPoseValue.
