NiTextKeyExtraData

: public NiExtraData

<NiTextKeyExtraData.h>

Description

NiTextKeyExtraData holds an array of NiTextKey objects for use in animation sequences (NiSequenceData).

Constructors

NiTextKeyExtraData()

NiTextKeyExtraData(NiTextKey* pkKeys, unsigned int uiNumKeys)

The default constructor creates an empty NiTextKeyExtraData, while the second form supplies an array of NiTextKey objects. Note that NiTextKeyExtraData does not copy the NiTextKey objects—it uses them directly and deletes any existing keys.

Supported Member Functions

NiTextKey* GetKeys(unsigned int& uiNumKeys) const

void SetKeys(NiTextKey* pkKeys, unsigned int uiNumKeys)

These functions get and set the NiTextKey objects. Note that NiTextKeyExtraData does not copy the NiTextKey objects—it uses them directly and deletes any existing keys.
