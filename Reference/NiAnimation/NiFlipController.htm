NiFlipController

: public NiFloatInterpController

<NiFlipController.h>

Description

NiFlipController changes which image an NiTexturingProperty::Map will use. It will cycle through its internal list of images according to a set of user-specified parameters. This class uses an NiInterpolator supporting float values to animate the current texture index. NiFlipController is often used for performing flipbook animation, displaying a texture per frame to display a movie or video image, similar to an AVI file. IFL files used as textures in the art packages are converted to NiFlipController objects upon export.

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, float fFloat, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the associated texture index to the value specified by fFloat (rounding down to the closest integer). This function always returns true.

virtual bool UpdateValue(float fTime, NiControllerSequence* pkControllerSequence, NiEvaluator* pkEvaluator, NiEvaluatorSPData** ppkEvalSPData, unsigned short usIndex = 0)

Overridden from NiInterpController. This function calls Update on the specified evaluator to obtain its latest float value. The returned value (rounded down) is used to set the associated texture index. ppkEvalSPData may be NULL when the evaluator contains posed (i.e. constant) data. This function returns true when the specified evaluator contains valid data, and false otherwise. This function always returns true.
