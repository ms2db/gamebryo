Description

NiLinFloatKeys are used for linear animation between float keys.

Constructors

NiLinFloatKey()

NiLinFloatKey(float fTime, float fValue)

The default constructor creates an uninitialized NiLinFloatKey. The second form of the constructor supplies the time and floating-point value.
