NiSingleInterpController

: public NiInterpController

<NiSingleInterpController.h>

Description

NiSingleInterpController is the base class for all NiInterpControllers that use a single NiInterpolator to animate their target value.

Constructors

This is an abstract base class and therefore has no public constructors.

Virtual Function Overrides

virtual unsigned short GetInterpolatorCount() const

Overridden from NiInterpController. Always returns 1.

virtual const char* GetInterpolatorID(unsigned short usIndex = 0)

Overridden from NiInterpController. Returns NULL. The index passed in should always be 0.

virtual unsigned short GetInterpolatorIndex(const char* pcID) const

Overridden from NiInterpController. Always returns 0.

Overridden from NiInterpController. The index passed in should always be 0.

Overridden from NiInterpController. The index passed in should always be 0.
