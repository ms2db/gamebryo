NiAlphaController

: public NiFloatInterpController

<NiAlphaController.h>

Description

NiAlphaController animates the alpha value of an NiMaterialProperty using an NiInterpolator supporting float values. The float values should range from 0.0 (totally transparent) to 1.0 (opaque).

Constructors

NiAlphaController()

The constructor creates an empty NiAlphaController.

Virtual Function Overrides

virtual void Update(float fTime)

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, float fFloat, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the associated alpha attribute to the value specified by fFloat. This function always returns true.
