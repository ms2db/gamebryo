NiExtraDataController

: public NiSingleInterpController

<NiExtraDataController.h>

Description

NiExtraDataController serves as an abstract base class for all extra data controllers. It houses the name of and a pointer to the NiExtraData object that it modifies.

Constructors

NiExtraDataController is an abstract base class and as such has no public constructors.

Supported Member Functions

const NiFixedString& GetExtraDataName()

This function returns the name of the extra data object that the controller modifies.

NiExtraData* GetExtraDataPointer()

This function returns a pointer to the extra data object that the controller modifies. If the controller has no target, this function returns NULL.

virtual void SetTarget(NiObjectNET* pkTarget)

Overridden from NiTimeController. Adds logic to resolve the name of the extra data object to a pointer to it when the target is set.
