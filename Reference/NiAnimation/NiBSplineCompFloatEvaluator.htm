NiBSplineCompFloatEvaluator

: public NiBSplineFloatEvaluator

<NiBSplineCompFloatEvaluator.h>

Description

NiBSplineCompFloatEvaluator provides all of the functionality of NiBSplineFloatEvaluator with the additional information required for using compact control points. If the channel is not animated, the posed evaluator can be handled more efficiently as an NiConstFloatEvaluator.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.
