NiQuaternionInterpolator

: public NiKeyBasedInterpolator

<NiQuaternionInterpolator.h>

Description

NiQuaternionInterpolator uses NiRotKeys to animate an NiQuaternion value over time. This NiKeyBasedInterpolator only supports one key channel.

Constructors

NiQuaternionInterpolator()

This constructor creates an NiQuaternionInterpolator containing invalid data.

NiQuaternionInterpolator(NiRotData* pkRotData)

This constructor creates an NiQuaternionInterpolator with the specified NiRotData.

NiQuaternionInterpolator(const NiQuaternion& kPoseValue)

This constructor creates an NiQuaternionInterpolator with a pose value of kPoseValue and a NULL NiRotData.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget, NiQuaternion& kValue)

Overridden from NiInterpolator. If there are animation keys in this interpolator, the value will be the interpolated value from the Update call. Otherwise, this value will be the pose value if there are no keys and the pose has been set.

virtual bool IsQuaternionValueSupported() const

Overridden from NiInterpolator. Returns true to indicate that NiQuaternion values are supported.

virtual void Collapse()

Overridden from NiInterpolator.

virtual unsigned short GetKeyChannelCount() const

Overridden from NiKeyBasedInterpolator. This value is always one.

virtual unsigned int GetKeyCount(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiQuaternionInterpolator.

virtual NiAnimationKey::KeyType GetKeyType(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiQuaternionInterpolator.

virtual NiAnimationKey::KeyContent GetKeyContent(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiQuaternionInterpolator.

virtual NiAnimationKey* GetKeyArray(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiQuaternionInterpolator.

virtual unsigned char GetKeyStride(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiQuaternionInterpolator.

virtual bool GetChannelPosed(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiQuaternionInterpolator.

Supported Member Functions

NiRotData* GetQuaternionData() const

void SetQuaternionData(NiRotData* pkRotData)

These functions get and set the NiRotData for this NiQuaternionInterpolator.

void SetPoseValue(const NiQuaternion& kPoseValue)

This function sets the pose value for this interpolator. If the interpolator has an NiRotData object, it is set to NULL.

NiRotKey* GetKeys(unsigned int& uiNumKeys, NiRotKey::KeyType& eType, unsigned char& ucSize) const

This function returns the keys currently held by this interpolator. The number of keys, key type, and key stride are returned in uiNumKeys, eType, and ucSize.

void ReplaceKeys(NiRotKey* pkKeys, unsigned int uiNumKeys, NiRotKey::KeyType eType)

This function specifies the key data to store. Note that NiQuaternionInterpolator does not copy the NiRotKeys -- it assumes ownership of the keys and deletes any previous keys.

void SetKeys(NiRotKey* pkKeys, unsigned int uiNumKeys, NiRotKey::KeyType eType)

This function specifies the key data to store. Note that the NiQuaternionInterpolator does not copy the NiRotKeys -- it simply overwrites the pointers to the old keys. This action can easily cause memory leaks, so calls to SetKeys should generally be avoided in favor of ReplaceKeys.
