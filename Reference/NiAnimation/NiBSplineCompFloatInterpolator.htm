NiBSplineCompFloatInterpolator

: public NiBSplineFloatInterpolator

<NiBSplineCompFloatInterpolator.h>

Description

NiBSplineCompFloatInterpolator provides all of the functionality of NiBSplineFloatInterpolator with the additional information required for using compact control points.

Constructors

NiBSplineCompFloatInterpolator()

This constructor creates an NiBSplineCompFloatInterpolator containing invalid data.

NiBSplineCompFloatInterpolator(NiBSplineData* pkData, NiBSplineData::Handle kFloatACPHandle = NiBSplineData::INVALID_HANDLE, NiBSplineBasisData* pkBasisData = NULL)

This constructor creates an NiBSplineCompFloatInterpolator with the specified data objects and control point handles.

NiBSplineCompFloatInterpolator(const float fPoseValue)

This constructor creates an NiBSplineCompFloatInterpolator with the specified pose and no data object.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget, float& fValue)

Overridden from NiInterpolator. Returns true to indicate that the returned value is valid.

virtual bool UsesCompressedControlPoints() const

This function returns true since this class uses compressed control points.

Supported Member Functions

float GetOffset(unsigned short usChannel) const

void SetOffset(float fOffset, unsigned short usChannel)

Get and set the offset used by the specified channel of compressed control points. This value is used to decompress the control points.

float GetHalfRange(unsigned short usChannel) const

void SetHalfRange(float fHalfRange, unsigned short usChannel)

Get and set the range used by the specified channel of compressed control points. This value is used to decompress the control points.
