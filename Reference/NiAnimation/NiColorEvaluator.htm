NiColorEvaluator

: public NiKeyBasedEvaluator

<NiColorEvaluator.h>

Description

NiColorEvaluator uses NiColorKey objects to animate an NiColorA value over time. This NiKeyBasedEvaluator only supports one key channel. A posed (i.e. constant) channel is stored using the key array. As a result, such posed evaluators are handled more efficiently as an NiConstColorEvaluator.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Supported Member Functions

NiColorData* GetColorData() const

void SetColorData(NiColorData* pkColorData)

These functions get and set the NiColorData for this NiColorEvaluator.

NiColorKey* GetKeys(unsigned int& uiNumKeys, NiColorKey::KeyType& eType, unsigned char& ucSize) const

This function returns the keys currently held by this evaluator. The number of keys, key type, and key stride are returned in uiNumKeys, eType,and ucSize.

void ReplaceKeys(NiColorKey* pkKeys, unsigned int uiNumKeys, NiColorKey::KeyType eType)

This function specifies the key data to store. Note that NiColorEvaluator will not copy the NiColorKeys -- it assumes ownership of the keys and deletes any previous keys.

void SetKeys(NiColorKey* pkKeys, unsigned int uiNumKeys, NiColorKey::KeyType eType)

This function specifies the key data to store. Note that NiColorEvaluator will not copy the NiColorKeys -- it simply overwrites the pointers to the old keys. This action can easily cause memory leaks, so calls to SetKeys should generally be avoided in favor of ReplaceKeys.
