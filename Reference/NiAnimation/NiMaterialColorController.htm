NiMaterialColorController

: public NiPoint3InterpController

<NiMaterialColorController.h>

Description

NiMaterialColorController animates the ambient, diffuse, specular, or emittance color of an NiMaterialProperty using an NiInterpolator supporting NiPoint3 values.

Constructors

NiMaterialColorController()

The constructor creates an NiMaterialColorController.

Virtual Function Overrides

virtual const char* GetCtlrID()

Overridden from NiInterpController. The ID indicates the color field of the NiMaterialProperty that is being controlled.

virtual void Update(float fTime)

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, const NiPoint3& kPoint3, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the associated color attribute to the specified RGB value. This function always returns true.

Supported Member Functions

Field GetType()

void SetType(Field eType)

These functions get and set which color field of the NiMaterialProperty this animator will control. The field types are described below in the Notes section.

Notes

Field Enumerations

Constant Color Type Affected
