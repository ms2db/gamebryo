NiFloatsExtraDataController

: public NiExtraDataController

<NiFloatsExtraDataController.h>

Description

NiFloatsExtraDataController animates an NiFloatsExtraData object attached to an NiAVObject in the scene graph.

Constructors

NiFloatsExtraDataController(const NiFixedString& kFloatsExtraDataName, int iFloatsExtraDataIndex = -1)

The constructor takes the name of the NiFloatsExtraData object to control and the index of the desired float in the extra data to update.

Virtual Function Overrides

virtual void Update(float fTime)

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, float fFloat, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the float attribute associated with usIndex to the value specified by fFloat. This function always returns true.

Supported Member Functions

int GetFloatsExtraDataIndex() const

void SetFloatsExtraDataIndex(int iFloatsExtraDataIndex)

These functions get and set the index of the value in the NiFloatsExtraData object that the controller animates.
