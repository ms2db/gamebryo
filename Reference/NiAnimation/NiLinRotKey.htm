NiLinRotKey

: public NiRotKey

<NiLinRotKey.h>

Description

NiLinRotKeys are used for linearly interpolated rotation animations. Interpolation is actually performed by the spherical correlation to linear interpolation, namely Slerp.

Constructors

NiLinRotKey()

NiLinRotKey(float fTime, const NiQuaternion& kQuat)

NiLinRotKey(float fTime, float fAngle, const NiPoint3& kAxis)

The default constructor creates an uninitialized NiLinRotKey. The second form of the constructor supplies the time and quaternion while the third specifies the rotation using an angle axis representation.
