NiFloatData

: public NiObject

<NiFloatData.h>

Description

The NiFloatData class provides a shareable wrapper for one dimensional floating-point animation keys.

Constructors

NiFloatData()

The default constructor creates an empty NiFloatData.

Supported Member Functions

unsigned int GetNumKeys() const

This call simply returns the number of keys the NiFloatData is currently storing.

NiFloatKey* GetAnim(unsigned int &uiNumKeys, NiFloatKey::KeyType &eType, unsigned char &ucSize) const

This call returns the key data that the NiFloatData currently stores.

void ReplaceAnim(NiFloatKey* pkFloatKeys, unsigned int uiNumKeys, NiFloatKey::KeyType eType)

This call gives the NiFloatData animation data to store. Note that the NiFloatData will not copy the NiFloatKey objects—it will use them directly and delete any existing keys.

void SetAnim(NiFloatKey* pkFloatKeys, unsigned int uiNumKeys, NiFloatKey::KeyType eType)

This call gives the NiFloatData animation data to store. Note that the NiFloatData will not copy the NiFloatKey objects—it will use them directly but does not delete any existing keys. This method is used to transfer ownership of keys.
