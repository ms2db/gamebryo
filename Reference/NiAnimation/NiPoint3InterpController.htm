NiPoint3InterpController

: public NiSingleInterpController

<NiPoint3InterpController.h>

Description

The NiPoint3InterpController is the base class for all NiInterpController objects that use an NiInterpolator to animate their target NiPoint3 value. The interpolator set for this controller must support NiPoint3 values.

Constructors

This is an abstract base class and therefore has no public constructors.

Supported Member Functions

There are no public supported non-inherited member functions in this class.
