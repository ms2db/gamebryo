: public NiAnimationKey

Description

NiRotKey is the abstract base class from which all other rotation keys are derived. It provides the basic data access routines and interpolation code. This class should be considered pure virtual and never instanced directly.

Constructors

Because NiRotKey is an abstract base class, it has no public constructors.

static void GenInterp(float fTime, NiRotKey* pkKeys, KeyType eType, unsigned int uiNumKeys, unsigned int& uiLastIdx, unsigned char ucSize, NiQuaternion* pkResult)

GenInterp will interpolate the array of NiRotKeys pkKeys at time fTime and return the result. The number of keys in the array is given by uiNumKeys, and the interpolation method expected by the keys is given by eType. The index value of the last interpolation is given in uiLastIdx; this value acts as a hint to the function on where to restart the interpolation process, and can significantly increase performance. The index value of the current interpolation is returned in uiLastIdx for use during the next interpolation. The size of each individual key in the key array is specified by the ucSize variable.

static void GenCubicCoefs(float fTime, NiRotKey* pkKeys, KeyType eType, unsigned int uiNumKeys, unsigned int& uiLastIdx, unsigned char ucSize, float& fTime0, float& fTime1, NiQuaternion& kQuat0, NiQuaternion& kQuat1, NiQuaternion& kOutQuat0, NiQuaternion& kInQuat1)

GenCubicCoefs will return the cubic coefficients for the animation curve between the keys in the array of pkKeys that bound the specified fTime. The number of keys in the array is given by uiNumKeys, and the interpolation method expected by the keys is given by eType. The index value of the last curve segment that was used is given in uiLastIdx; this acts as a hint to the function on where to restart the bounding key search, and can significantly increase performance. The index value of the current curve segment is returned in uiLastIdx for use during the next search. The size of each individual key in the key array is specified by the ucSize variable. The values returned in fTime0 and fTime1 correspond to the times of the bounding keys. The cubic coefficients for the matching curve segment are returned via kQuat0, kQuat1, kOutQuat0, and kInQuat1. Data values may be computed using NiQuaternion::Squad(fT, kQuat0, kOutQuat0, kInQuat1, kQuat1) where fT is the normalized time along the curve segment. GenCubicCoefs is used internally for sequence animation and normally should not be called separately by an application. Rotation channels of type EULERKEY are not supported by this function.
