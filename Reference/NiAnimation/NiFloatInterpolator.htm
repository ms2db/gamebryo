NiFloatInterpolator

: public NiKeyBasedInterpolator

<NiFloatInterpolator.h>

Description

NiFloatInterpolator uses NiFloatKeys to animate a float value over time. This NiKeyBasedInterpolator only supports one key channel.

Constructors

NiFloatInterpolator()

This constructor creates an NiFloatInterpolator containing invalid data.

NiFloatInterpolator(NiFloatData* pkFloatData)

This constructor creates an NiFloatInterpolator with the specified NiFloatData.

NiFloatInterpolator(float fPoseValue)

This constructor creates an NiFloatInterpolator with a pose value of fPoseValue and a NULL NiFloatData.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget, float& kValue)

Overridden from NiInterpolator. Returns true to indicate that the returned float value is valid. If there are animation keys in this interpolator, the value will be the interpolated value. Otherwise, this value will be the pose value if there are no keys and the pose has been set.

virtual bool IsFloatValueSupported() const

Overridden from NiInterpolator. Returns true to indicate that float values are supported.

virtual void Collapse()

Overridden from NiInterpolator.

virtual unsigned short GetKeyChannelCount() const

Overridden from NiKeyBasedInterpolator. This value is always one.

virtual unsigned int GetKeyCount(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiFloatInterpolator.

virtual NiAnimationKey::KeyType GetKeyType(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiFloatInterpolator.

virtual NiAnimationKey::KeyContent GetKeyContent(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiFloatInterpolator.

virtual NiAnimationKey* GetKeyArray(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiFloatInterpolator.

virtual unsigned char GetKeyStride(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiFloatInterpolator.

virtual bool GetChannelPosed(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiFloatInterpolator.

Supported Member Functions

NiFloatData* GetFloatData() const

void SetFloatData(NiFloatData* pkFloatData)

These functions get and set the NiFloatData for this NiFloatInterpolator.

void SetPoseValue(float fPoseValue)

This function sets the pose value for this interpolator. If the interpolator has an NiFloatData object, it is set to NULL.

NiFloatKey* GetKeys(unsigned int& uiNumKeys, NiFloatKey::KeyType& eType, unsigned char& ucSize) const

This function returns the keys currently held by this interpolator. The number of keys, key type, and key stride are returned in uiNumKeys, eType,and ucSize.

void ReplaceKeys(NiFloatKey* pkKeys, unsigned int uiNumKeys, NiFloatKey::KeyType eType)

This function specifies the key data to store. Note that NiFloatInterpolator will not copy the NiFloatKeys -- it assumes ownership of the keys and deletes any previous keys.

void SetKeys(NiFloatKey* pkKeys, unsigned int uiNumKeys, NiFloatKey::KeyType eType)

This function specifies the key data to store. Note that NiFloatInterpolator will not copy the NiFloatKeys—it simply overwrites the pointers to the old keys. This action can easily cause memory leaks, so calls to SetKeys should generally be avoided in favor of ReplaceKeys.
