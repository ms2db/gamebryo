NiBlendFloatInterpolator

: public NiBlendInterpolator

<NiBlendFloatInterpolator.h>

Description

NiBlendFloatInterpolator is responsible for blending float values together.

Constructors

NiBlendFloatInterpolator(bool bManagerControlled, float fWeightThreshold, unsigned char ucArraySize)

The constructor creates an NiBlendFloatInterpolator using the specified weight threshold and array size. Stored times for the sub-interpolators are used if bManagerControlled is true. Also note that NiBlendInterpolators do not stream their sub-interpolators when the bManagerControlled flag is true, they are instead streamed by their parent NiControllerSequence.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget, float& fValue)

Overridden from NiInterpolator. Returns true to indicate that the returned float value is valid.

virtual bool IsFloatValueSupported() const

Overridden from NiInterpolator. Returns true to indicate that float values are supported.
