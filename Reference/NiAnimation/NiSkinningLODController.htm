NiSkinningLODController

: public NiTimeController

<NiSkinningLODController.h>

Description

NiSkinningLODController defines the levels of detail (LODs) for a given in-game character, and dictates the character's current LOD. Levels of detail are controlled on a character by character basis, and are frequently are used as a performance optimization.

Constructors

NiSkinningLODController()

The default constructor is the only constructor for this class.

Static Member Functions

static NiSkinningLODController* FindSkinningLODController(NiAVObject* pkObject)

This function returns the first NiSkinningLODController in the specified scene graph branch.

Virtual Function Overrides

virtual void Update(float fTime)

Overridden from NiTimeController. This function switches the current LOD level to the next LOD level. When the LOD level has changed, it will set selective update flags appropriately on all skins and bones associated with this controller.

Supported Member Functions

void SetBoneLOD(NiUInt32 uiLOD)

Sets the skinning LOD level to switch to during the next update. The valid range of uiLOD is 0 to GetNumberOfBoneLODs() - 1 where 0 is the highest level of detail.

NiUInt32 GetBoneLOD() const

Returns the current skinning LOD level.

NiUInt32 GetNextBoneLOD() const

Returns the skinning LOD level that will be switched to during the next update.

NiUInt32 GetNumberOfBoneLODs() const

Returns the total number of LOD levels.
