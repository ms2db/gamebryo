Note

This class has been deprecated in Gamebryo 2.6.0. This class became obsolete with the introduction of the NiMultiTargetPoseHandler class. NiMultiTargetPoseHandler is now responsible for propagating transform values from animated sequences to the appropriate scene graph nodes. Code continues to exist to stream NiMultiTargetTransformController objects which may reside in older NIF files.
