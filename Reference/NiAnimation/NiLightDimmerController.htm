NiLightDimmerController

: public NiFloatInterpController

<NiLightDimmerController.h>

Description

NiLightDimmerController animates the dimmer value of an NiLight-derived object. Since only one floating point value is used by this controller, it is derived from NiFloatInterpController.

Constructors

NiLightDimmerController()

The constructor creates an NiLightDimmerController.

Virtual Function Overrides

virtual void Update(float fTime)

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, float fFloat, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the associated dimmer attribute to the value specified by fFloat. This function always returns true.
