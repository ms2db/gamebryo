NiBSplineFloatInterpolator

: public NiBSplineInterpolator

<NiBSplineFloatInterpolator.h>

Description

NiBSplineFloatInterpolator uses B-Splines to animate a float value over time. This NiBSplineInterpolator only supports one control point channel.

Constructors

NiBSplineFloatInterpolator()

This constructor creates an NiBSplineFloatInterpolator containing invalid data.

NiBSplineFloatInterpolator(NiBSplineData* pkData, NiBSplineData::Handle kFloatACPHandle = NiBSplineData::INVALID_HANDLE, NiBSplineBasisData* pkBasisData = NULL)

This constructor creates an NiBSplineFloatInterpolator with the specified NiFloatData.

NiBSplineFloatInterpolator(float fPoseValue)

This constructor creates an NiBSplineFloatInterpolator with a pose value of fPoseValue.
