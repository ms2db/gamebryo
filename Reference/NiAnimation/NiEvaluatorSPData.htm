NiEvaluatorSPData

: public NiMemObject

<NiEvaluatorSPData.h>

Description

NiEvaluatorSPData acts as the main entry point to the scratch pad data for a given evaluator channel. Evaluators and their associated scratch pad data are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Supported Member Functions

There are no supported member functions as NiEvaluatorSPData is an Emergent internal class for accessing data within NiScratchPad.
