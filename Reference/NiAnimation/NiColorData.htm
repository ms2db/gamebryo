NiColorData

: public NiObject

<NiColorData.h>

Description

The NiColorData class provides a shareable wrapper for color animation keys.

Constructors

NiColorData()

The default constructor creates an NiColorData with no animation data.

Supported Member Functions

unsigned int GetNumKeys() const

This call simply returns the number of keys the NiColorData is currently storing.

NiColorKey* GetAnim (unsigned int &uiNumKeys, NiColorKey::KeyType &eType, unsigned char &ucSize) const

This call returns the key data that the NiColorData currently stores.

void ReplaceAnim(NiColorKey* pkColorKeys, unsigned int uiNumKeys, NiColorKey::KeyType eType)

This call gives the NiColorData animation data to store. Note that the NiColorData will not copy the NiColorKey objects—it will use them directly and delete any existing keys.

void SetAnim(NiColorKey* pkColorKeys, unsigned int uiNumKeys, NiColorKey::KeyType eType)

This call gives the NiColorData animation data to store. Note that the NiColorData will not copy the NiColorKey objects—it will use them directly but does not delete any existing keys. This method is used to transfer ownership of keys.
