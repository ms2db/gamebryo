Description

NiTCBFloatKeys are used to represent 1-D TCB splines. Each key contains a time value (the parameterization), a floating-point value and the tension, continuity and bias controls.

Constructors

NiTCBFloatKey()

NiTCBFloatKey(float fTime, float fValue, float fTension, float fContinuity, float fBias)

The default constructor creates an uninitialized NiTCBFloatKey. The second form of the constructor supplies the time, value and TCB controls.
