NiBoolInterpController

: public NiSingleInterpController

<NiBoolInterpController.h>

Description

NiBoolInterpController is the base class for all NiInterpControllers that use a NiInterpolator to animate their target boolean value. The interpolator set for this controller must support bool values.

Constructors

This is an abstract base class and therefore has no public constructors.

Supported Member Functions

There are no public supported non-inherited member functions in this class.
