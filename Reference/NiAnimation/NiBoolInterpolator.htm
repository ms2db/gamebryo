NiBoolInterpolator

: public NiKeyBasedInterpolator

<NiBoolInterpolator.h>

Description

NiBoolInterpolator uses NiBoolKeys to animate a bool value over time. This NiKeyBasedInterpolator only supports one key channel.

Constructors

NiBoolInterpolator()

This constructor creates an NiBoolInterpolator containing invalid data.

NiBoolInterpolator(NiBoolData* pkBoolData)

This constructor creates an NiBoolInterpolator with the specified NiBoolData.

NiBoolInterpolator(bool bPoseValue)

This constructor creates an NiBoolInterpolator with a pose value of bPoseValue and a NULL NiBoolData.

Overridden from NiInterpolator. Returns true to indicate that the returned boolean value is valid. If there are animation keys in this interpolator, the value will be the interpolated value. Otherwise, this value will be the pose value if there are no keys and the pose has been set.

Supported Member Functions

NiBoolData* GetBoolData() const

These functions get and set the NiBoolData for this NiBoolInterpolator.

void SetPoseValue(bool bPoseValue)

This function sets the pose value for this interpolator. If the interpolator has an NiBoolData object, it is set to NULL.

NiBoolKey* GetKeys(unsigned int& uiNumKeys, NiBoolKey::KeyType& eType, unsigned char& ucSize) const

This function returns the keys currently held by this interpolator. The number of keys, key type, and key stride are returned in uiNumKeys, eType,and ucSize.

void ReplaceKeys(NiBoolKey* pkKeys, unsigned int uiNumKeys, NiBoolKey::KeyType eType)

This function specifies the key data to store. Note that NiBoolInterpolator will not copy the NiBoolKeys -- it assumes ownership of the keys and deletes any previous keys.

void SetKeys(NiBoolKey* pkKeys, unsigned int uiNumKeys, NiBoolKey::KeyType eType)

This function specifies the key data to store. Note that NiBoolInterpolator will not copy the NiBoolKeys -- it simply overwrites the pointers to the old keys. This action can easily cause memory leaks, so calls to SetKeys should generally be avoided in favor of ReplaceKeys.
