NiConstColorEvaluator

: public NiEvaluator

<NiConstColorEvaluator.h>

Description

NiConstColorEvaluator supports a constant (i.e. posed) NiColorA value. This NiEvaluator only supports one channel.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Constructors

NiConstColorEvaluator()

This constructor creates an NiConstColorEvaluator containing invalid data.

NiConstColorEvaluator(const NiColorA& kPoseValue)

This constructor creates an NiConstColorEvaluator with the specified posed value.

Supported Member Functions

void SetPoseValue(const NiColorA& kPoseValue)

This function sets the posed value for this NiConstBoolEvaluator.

Notes

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying information about a specific animation channel in an NiConstColorEvaluator.

Value Meaning
COLORA This value specifies the color animation channel.
