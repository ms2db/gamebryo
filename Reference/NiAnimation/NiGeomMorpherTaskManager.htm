Description

NiGeomMorpherTaskManager was a singleton class used by previous versions of Gamebryo to allow morphing objects to be updated in the background of the normal update process. It was removed and replaced with the more general class NiParallelUpdateTaskManager and some flags in NiGeomMorpherController. Any legacy applications using NiGeomMorpherTaskManager should update code to use the new class.
