NiTransformData

: public NiObject

<NiTransformData.h>

Description

NiTransformData provides a shareable wrapper for the keys used to animate transformation data.

Constructors

NiTransformData()

The default constructor creates an NiTransformData with no animation data.

Supported Member Functions

unsigned int GetNumRotKeys() const

unsigned int GetNumPosKeys() const

unsigned int GetNumScaleKeys() const

These functions return the number of rotation, position, and scale keys.

void ReplaceScaleAnim(NiFloatKey* pkKeys, unsigned int uiNumKeys, NiFloatKey::KeyType eType)

These functions replace the rotation, position, and scale keys with the supplied array of keys. Note that these functions do not copy the keys; the keys are used directly, and any already existing keys are deleted.

void SetRotAnim(NiRotKey* pkKeys, unsigned int uiNumKeys, NiRotKey::KeyType eType)

void SetPosAnim(NiPosKey* pkKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

void SetScaleAnim(NiFloatKey* pkKeys, unsigned int uiNumKeys, NiFloatKey::KeyType eType)

These functions set the rotation, position, and scale keys using the supplied array of keys. Note that these functions do not copy the keys; the keys are used directly, but any already existing keys are not deleted. These functions are used for transferring ownership of keys.
