NiPosData

: public NiObject

<NiPosData.h>

Description

The NiPosData class provides a shareable wrapper for position animation keys.

Constructors

NiPosData()

The default constructor creates an NiPosData with no animation data.

Supported Member Functions

unsigned int GetNumKeys() const

This call simply returns the number of keys the NiPosData is currently storing.

NiPosKey* GetAnim(unsigned int &uiNumKeys, NiPosKey::KeyType &eType, unsigned char &ucSize) const

This call returns the key data that the NiPosData currently stores.

void ReplaceAnim(NiPosKey* pkPosKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

This call gives the NiPosData animation data to store. Note that the NiPosData will not copy the NiPosKey objects—it will use them directly and delete any existing keys.

void SetAnim(NiPosKey* pkPosKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

This call gives the NiPosData animation data to store. Note that the NiPosData will not copy the NiPosKey objects—it will use them directly but does not delete any existing keys. This method is used to transfer ownership of keys.
