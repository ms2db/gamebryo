efd::Bool Initialize(TerrainService* pTerrainService)

Initializes the material parser and registers asset locater responses for texture asset lookups.

Parameter Description
pTerrainService Pointer to the terrain service owning the parser. This is used to allow the parser to better communicate with the terrain system as material packages are loaded and discovered.
Return Value True if successful, false otherwise.

void Shutdown()

Unregisters any message handlers and cleans up the internal parser structures.

efd::Bool SavePackage(
NiSurfacePackage* pPackage,
const efd::utf8string& filePath)

Saves the specified package at the specified location on disk.

If NullString() is specified, this method assumes pPackage already has a valid filename since it was already saved once before.

Parameter Description
pPackage The package to save.
filePath Full path to the package (minus the file extension) or utf8string::NullString() if the package just needs to be re-saved.
Return Value True if saved successfully, false otherwise.

NiSurfacePackagePtr LoadPackage(const efd::utf8string& filePath)

Loads a package from the specified location.

Parameter Description
filePath Full path to the material package including the file extension.
Return Value The loaded package or NULL if the package could not be loaded.
