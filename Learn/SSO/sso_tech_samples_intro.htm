Online Tech Samples: Introduction

The following technical samples demonstrate functionality that is specific to networking for online games. For a more general introduction to Gamebryo LightSpeed's Online subsystem, including tutorial-style introductions, see Online Overview.

  • Matchmaking—demonstrates basic integration with the matchmaking services offered by supported platforms.
  • PC enet Reliable UDP—demonstrates adding a new network protocol using a custom NetLib. This sample adds reliable UDP for Windows using the ENet library.
  • Playstation 3 NetLib—demonstrates using UDPP2P over Playstation Network.
  • Xbox 360 NetLib—demonstrates using XRNM reliable UDP over Xbox Live.
  • Xbox 360 XSessionService—demonstrates how the Microsoft Xbox 360 XSession API can be integrated with Gamebryo LightSpeed.
