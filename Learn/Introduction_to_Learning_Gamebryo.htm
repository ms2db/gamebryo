Learning Gamebryo LightSpeed

This section provides an overview of Gamebryo LightSpeed architecture. It also includes a number of tutorials and explanations of the demos that ship with LightSpeed.

Architecture, tutorials, and demos are outlined in the following sections:

  • Architecture—Provides overviews of the product's principal subsystems and tools.
  • MangledMetal—Discusses our full-featured, robust application that demonstrates many of the features and best practices of the Emergent Game Framework.
  • Tutorials—Discusses introductory tutorials for programming with LightSpeed and using Gamebryo tools.
  • Graphics Tech Demos—Simple demo programs that demonstrate a particular Core Runtime feature.
  • Game Demos—Large demo applications combining several Gamebryo features.
