Gamebryo LightSpeed Tutorials: Introduction

The Gamebryo LightSpeed tutorials introduce you to the features and functionality of the product's major subsystems and tools.

  • Core Runtime—A series of tutorials that demonstrate key concepts of the Core Runtime subsystem, including renderers, shaders, and textures.
  • Game Framework—A set of tutorials that introduce the Hello World and Base Game applications, along with rapid iteration examples.
  • Online—A set of tutorials that introduce Peer Game and multiplayer rapid iteration.
  • Gamebryo-PhysX Integration—A tutorial that demonstrates the basics of creating a Gamebryo world with PhysX content.
  • Tools—A set of tutorials to familiarize you with the Animation Tool, World Builder, and the Digital Content Creation plug-ins.
  • MangledMetal—Information about adding trigger and enemy types and physical props to Emergent's full-featured demo application.
