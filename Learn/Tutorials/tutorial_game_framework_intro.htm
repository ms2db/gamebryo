Game Framework Tutorials: Introduction

The Emergent Game Framework is the Gamebryo LightSpeed subsystem that provides the basis for simulation and gameplay behavior. The tutorials in this section introduce you to Hello World, which a simple application that introduces you to the basic concepts and components of the Game Framework. You also learn about extending Hello World into Base Game, an application that adds some basic game play elements, physics support,and and camera control. We also introduce you to rapid iteration, including necessary setup for various target platforms.

A Note About Rapid Iteration on Consoles

Changes made during rapid iteration that result in metadata changes, for example adding a Model to the EMT project, are not automatically deployed to consoles. The application will use the AssetController to access this metadata. However, the next time you run the application stand alone, this metadata will not be found. This can result in unexpected behavior at runtime, including the inability to load your initial world file. You should ensure you re-deploy this metadata to your application working directory.
