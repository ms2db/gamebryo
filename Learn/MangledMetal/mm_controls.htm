MangledMetal Demo Controls

The demo renders a splash screen while loading. After loading is complete, the camera focuses on your Warjack. You are then free to run around the game world using the following controls.

Keyboard Game Pad Action
WASD/wasd Left Analog Controls the Warjack
Arrow Keys Right Analog Controls the Camera
Space A Button Light Attack
Left Ctrl X Button Heavy Attack
F11 L1 Button Display Visual Tracker
ESC Start+Select Quit Demo
R Right DPad Button Prints a performance report to the log file.
I Up DPad Button   Toggle Shadows
J Down DPad Button Toggle "Walk Anywhere"
K R1 Button Prints memory usage information to the log file.
