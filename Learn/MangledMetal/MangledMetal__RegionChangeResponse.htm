RegionChangeResponse Model

Found in project: MangledMetal

Traits: Active

Mixes In: AbstractTriggerResponse

Mixed Into: RegionChangeTrigger

Description

Trigger response that makes a given set of regions hidden and/or shown. Only affects entities that mix in the MMMesh model and requires the application to use the MMOcclusionService

ShowRegions SInt32 List RegionChangeResponse List Indicates what regions to show when trigger is hit
ActivationCount UInt32 AbstractTriggerResponse 0 Number of times this trigger has been activated.
MaxActivations UInt32 AbstractTriggerResponse 1 Maximum number of times this trigger is allowed to be activated. If the TriggerSource detects something that would have triggered, it should not call OnTrigger if ActivationCount >= MaxActivations.
