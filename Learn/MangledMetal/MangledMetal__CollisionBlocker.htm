CollisionBlocker Model

Found in project: MangledMetal

Traits: Active

Mixes In: PhysXTrigger, PhysXShape

Mixed Into: CollisionPushback

Description

Mixing in this model indicates that any other entity that mixes in the CollisionPushback model should move away when an interpenetration occurs with this entity
