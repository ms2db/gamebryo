MangledMetal Library

Overview

The MangledMetal EMT project (MangledMetal.emtproj) is an Entity Modeling Tool project which defines game logic and useful artifacts for the MangledMetal demo. Some constructs in the MangledMetal library may be applicable to other applications, but their design intent is to implement the MangledMetal game demo.
