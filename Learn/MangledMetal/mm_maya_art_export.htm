Exporting MangledMetal Maya art assets

All of the custom process scripts and MelScripts on this page can be found at: %EMERGENT_PATH%\Samples\GameDemos\MangledMetal\Art Files\ToolScripts\Maya\. If these are copied into the root folder of your Maya installation, you should be able to re-export the Maya assets.

Export Scripts

All MangledMetal assets use a set of custom export scripts. The three most common are:

  • MangledMetal PhysX Standard, which is similar to the Default Export script with the additions of the PhysX Extractor and Material Modifier plugins
  • MangledMetal PhysX Actor, which is similar to the MangledMetal PhysX Standard script with the addition of the Controller Extractor plugin
  • MangledMetal PhysX Character, which is similar to the MangledMetal PhysX Actor script with the addition of the PhysX Ragdoll plugin

Material Modifier

MangledMetal uses a custom export plugin called the Material Modifier for all assets. This modifier is designed to allow an artist to easily export for different platforms with customized settings from a single file. The settings used in this plugin for MangledMetal down-sizes all textures to half their original sizes for Xbox 360 and Playstation 3. These settings are automatically applied on a platform-specific export.

Multi-Export Utility

To expedite the process of exporting a single asset for multiple platforms, MangledMetal Maya assets are set up to take advantage of the Multi-Export Utility. Using this utility, an artist can export platform-specific and platform optimized versions of an asset for all supported platforms with a single click.

To use the Multi-Export utility inside of Maya, first make sure that the Gamebryo plug-in has been loaded. Next, open and run the GamebryoMultiExportUtility.mel inside the Script Editor. This will launch the Multi-Export Utility Interface.

To export a single file, simply check all desired platforms and click Export (the Source and Destination Directories do not apply to a single-file Export. The Multi-Export Utility will then prompt for a file name. The Multi-Export Utility will always export for Generic first, so the export filename entered into the prop should be under the Data/Generic directory. The Multi-Export Utility will then export platform-specific versions in the appropriate folders under the Data directory (Data/PS3, Data/DX9, and so on).

The Multi-Export Utility can also export every file under a given directory. To do this, set the source directory to where the Source Art Files are stored and the destination directory to the root of the desired destination (Data/Generic/Props, or Data/Generic/Characters, for example), and click Batch Export. The Multi-Export Utility will then open every file in the Source Directory and export platform-specific version for each of the selected platforms.

It is important to note that the Batch Export process will name the output files according to the name of the source file. For example, prop01.mb will export out as prop01.nif. Because of this, it is important to name the source art files according to the desired output file names if you plan on using the Batch Export option.
