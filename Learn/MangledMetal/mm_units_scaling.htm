MangledMetal units and scaling

Source Art Files

All assets in MangledMetal are modeled at a 1 unit = 1 centimeter scale. Before altering or adding assets to MangledMetal, make sure you are working at the correct scale inside of 3D Studio Max or Maya:

In 3D Studio Max:

  1. Go to Customize -> Units Setup
  2. Click System Unit Setup
  3. Set System Unit Scale to 1 unit = 1 centimeter

In Maya:

  1. Window -> Settings/Preferences -> Preferences
  2. Select Settings from the Category pane
  3. In Working Units set Linear to centimeter

Scaling inside of World Builder

Scaling MangledMetal assets inside of World Builder is not recommended for 2 reasons.

  • PhysX Rigid Bodies do not scale once exported from Max or Maya. Because of this, Assets with PhysX data will always simulate as though they are at Scale = 1, regardless of the size the geometry renders at.
  • Many assets have Nonwalkable meshes associated with them used to determine where Actors cannot walk. Because this calculation is performed based on the center of the character, if an asset with a walkable/nonwalkable mesh is scaled significantly, the distance between the edge of the rendered geometry and the edge of the Nonwalkable mesh may get very large, keeping Actors at a greater distance than desired.

Because of these limitations, it is recommended that any scale changes be made inside of the DCC tools and re-exported into World Builder.

PhysX Units

PhysX has no default scale, but works best within a certain range of values. Because of this, the PhysX scale inside the PhysX extractor plugin is set to .01. Roughly speaking, this means that PhysX mass values will be calculated as Kilograms.
