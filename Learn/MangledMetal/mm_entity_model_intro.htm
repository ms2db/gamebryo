MangledMetal Model Reference

MangledMetal makes heavy use of the Entity modeling Tool. Models are defined in two main libraries.

  • The TriggersLibrary (TriggersLibrary/TriggersLibrary.emtproj) contains a reusable definition of a set of trigger source and response models
  • The MangledMetal library (MangledMetal.emtproj) contains the bulk of the game logic.

All or some portions of the models in both libraries may be useful in your application.

You can find both EMT project files in the directory %EMERGENT_PATH%\Samples\GameDemos\MangledMetal\Data.
