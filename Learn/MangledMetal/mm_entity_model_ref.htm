MangledMetal Model Reference

MangledMetal makes heavy use of the Entity Modeling Tool. Models are defined in two main libraries:

  • TriggerLibrary (TriggerLibrary.emtproj)—contains a reusable definition of a set of trigger source and response models
  • MangledMetal library (MangledMetal.emtproj)—contains the bulk of the game logic

All or some portions of the models in both libraries may be useful in your application. By default, both EMT project files are installed in the following directory:

%EMERGENT_PATH%\Samples\GameDemos\MangledMetal\Data
