ObjectPickBinaryShader

This simple NiBinaryShader-derived class relies on the NSF files to do most of the work, setting up the vertex and pixel shaders. The only additional functionality it provides is in the SetupShaderPrograms function. Before calling the base class's implementation, the function will call GetColorForObject on the stored ColorPicker object. It then converts the color, currently stored as an unsigned int, into a set of four floats that it uses to update the appropriate global shader constant. That shader constant is used by the shader to flat-color the object.
