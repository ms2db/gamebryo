PhysX Timing Demo Overview

The PhysX Timing demo illustrates the effect of various update timing strategies when using Gamebryo and PhysX. An on-screen menu allows the user to control the relative PhysX and Gamebryo time-steps, as well as interpolation of PhysX state and whether or not the Gamebryo scene graph is updated before or after a PhysX simulation step is started.

PhysX Timing demo

Debug geometry is turned on by default. It shows you the state of the PhysX simulation at the end of the most recent FetchResults call. This is the latest PhysX state that Gamebryo has available to it for setting the pose of actors. Gamebryo also keeps the prior state, although this is not visualized.
