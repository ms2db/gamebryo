BadSushi Functional Overview

BadSushi loads a KFM file and organizes fish characters along a curve. Each model is independently animated via morphing, and the animation is chosen from two animations—one for non-startled swimming, and another for startled swimming—that are enumerated in the KFM file.

The menu has options to add and remove fish characters. There are also controls for animation speed, "startle fish" behavior, wireframe display, and navigation. With the "startle fish" behavior enabled, you can use the mouse, keyboard, and gamepad navigation controls to reposition the "gunsight" to startle arbitrary fish independently.

Console Display

The following statistics display in the console:

Statistic Description
Frame Rate Frames rendered per-second
Tris/model Total triangles for each character (fish) model
Models Total character models that are cloned in the scene
Tris/frame Total character triangles that are rendered per frame
Tris/sec Total character triangles that are rendered per second

Menu Options

The following menu items are available:

Menu item Description
Fish Count Total character models (fish) that are cloned in the scene
Animation Speed Speed of character and caustic animation
Startle Fish Toggle whether "gunsight" causes nearby fish to "puff out"
Wireframe Mode Toggle wireframe mode
Change Control Mode Change the navigation controller (default NiSample UI)
Display Control Help Display navigation help (default NiSample UI)
Worker Threads PCs: Total number of worker threads for Floodgate to use
PS3: Total number of SPUs for Floodgate to use
Xbox 360: Total number of worker threads for Floodgate to use
SPUs/Job (PS3 only) Total number of SPUs to use (1-5)
Quit Exit BadSushi (default NiSample UI)
