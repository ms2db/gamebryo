CharacterPerformance Demo Functional Overview

CharacterPerformanceDemo loads a set of KFM files and displays various statistics for each character. You can open the demo's menu by pressing Esc or Start. The menu contains options to add and remove characters and lights. You can also use the menun to control how the model is updated and rendered. Each model is independently animated, and the animation is chosen randomly from the set of animations that are enumerated in the KFM file.

Console Display

The following statistics display in the console:

Statistic Description
Model The name of the currently loaded KFM file.
Bones Total bones in the character.
Partitions Total number of sub-meshes.
Frame Rate Frames rendered per-second
Triangle drawn Total triangles rendered (per second)
Vertices drawn Total vertices rendered (per second)

Menu Options

The following menu items are available:

Menu item Description
Quit Exit the demo.
Objects Number of cloned characters.
Thread Count Number of threads to use to update the scene in parallel. By default, the number of threads will be the number of logical processors on the machine.
Switch Model Switch to the next set of characters
Toggle Bone LOD If there are multiple Bone LOD levels specified for this model, this switches to the next level
Add Spot Light Add a spot light to the scene.
Add Dir Light Add a directional light to the scene.
Add Point Light Add a point light to the scene.
Remove Light Remove the most recently added light.
UpdateSelected If On, uses NiAVObject::UpdateSelected for updating animations. If Off, uses NiAVObject::Update.
Wireframe Mode Toggle wireframe mode.
Draw Scene Toggle background geometry.
