Notes on PhysX Demos

PhysX-specific demo names are prefixed with "PhysX". These demos require an installed version of the supported NVIDIA PhysX™ system software as well as the SDK if recompilation is required. Contact NVIDIA to obtain an evaluation license for PhysX. If PhysX is not installed and you attempt to view a PhysX-specific demo, a dialog box will display, and the application will exit.

Additional issues:

  • A dialog will display if you have the wrong NVIDIA PhysX version. See the Gamebryo-PhysX Release Notes for information about supported PhysX versions.
  • PhysX debug rendering will not appear until a simulation step has been taken. This means that debug rendering will not immediately appear if the application is paused. Other simulation-related parameters also take effect only when the simulation is active.

For more information about some of our more full-featured demos, see Game Demos and MangledMetal.
