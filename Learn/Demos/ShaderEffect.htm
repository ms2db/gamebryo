ShaderEffect

The ShaderEffect class is the base class for all shader effects in the ShaderSample demo.

Init

Default initialization for the effect. This function calls RegisterShaderLibrary, which must be defined by each derived effect, to perform whatever actions are necessary to create and initialize the shaders. It also creates an NiDefaultClickRenderStep that derived classes can use to attach their own render clicks.

Shutdown

Default shutdown for the effect. This function releases all smart pointers that were stored during initialization.

UpdateScene

Default update function for the effect. This function simply calls Update on the scene and camera.

Enable

Default enabling function for the effect. This function sets the "active" flag of its stored render step to true and calculates a coordinate frame that will be used to manipulate the scene.

Disable

Default enabling function for the effect. This function simply sets the "active" flag of its stored render step to false.

GetScale

Returns the scale of the scene graph, used to control the granularity of the translation controls.

GetName

Returns the name of the effect.

GetDescription

Returns a description of the effect to display in the UI.

GetUpDir

Returns the up vector for the scene graph that should be used for the camera rotation.

MoreSubeffects

Returns a Boolean value indicating whether or not the effect has any additional sub-effects that can be displayed.

NextSubeffect

Advances the effect to display the next sub-effect, if it has any.

GetInitialized

Returns a Boolean value indicating whether or not the effect has been successfully initialized.

GetScene

Returns a pointer to the scene graph.

GetCamera

Returns a pointer to the camera.

GetRenderStep

This function returns a pointer to the render step that represents this rendering effect. The ShaderSample class queries each effect for its render step so it can be added to the render frame.
