CubeMap Demo Overview

The CubeMap demo shows how to use cube mapping to create realistic reflection maps in real time. The demo renders the mobile room containing a model of the solar system. A mirror-reflective sphere is wrapped around the selected planet and reflects a live, rendered reflection image. Because the reflection cube map is rendered each frame, the movements of the planets can be seen in the reflection.

To show how the effect works, a key may be used to replace the seamless rendered cube map with a static cube map that consists of marked squares, marking the edges of the six faces of the cube map, as well as the cube face identities (+/- X, Y, Z). Also, another keystroke switches the mapping mode for the reflection between specular (uses the reflection vector at each vertex) or diffuse (uses the normal vector at each vertex) modes.

On older hardware (graphics cards not Shader Model 2.0 compliant) the normal map will not appear on the sphere.
