Introduction to the UnicodeCharacterSets Demo

This sample demonstrates how to display text strings using fonts with 2-byte Unicode characters (UTC-2) through the Gamebryo NiFont library. The Gamebryo NiFont library has been extended to support the display of Unicode fonts as well as Ascii fonts. The Ni2DString class, which makes use of NiMeshScreenElements, is used to represent the text strings.

The sample displays a start-up screen for a fictional game, "Gamebryo World", including numerous start-up commands, initially in English. The "Language" line identifies the current language as English, using the English word for English, the Chinese word for English, the Japanese word for English, and the Korean word for English.

By pressing the space bar (or left analog stick press on the gamepad), successive frames indicate Chinese, Japanese, and Korean translations of the same start-up screen, in succession, as could be done in a game so the end user could configure a desired language. The Gamebryo implementation for Ni2DString supports both Unicode and Ascii strings, and makes use of NiMeshScreenElements objects as the underlying display technology.

The background "parchment" texture is also one of the NiMeshScreenElements objects, so it may be stretched, as needed, to accommodate different screen or window resolutions.

To create the NFF file representing the "Arial Unicode MS" font used for this demo, the Gamebryo NiFontCreator tool was used. NiFontCreator is a Windows application used to create NFF files, the files that contain font information for the Gamebryo NiFont library.
