BadSushi Overview

BadSushi demonstrates Gamebryo morphing performance and character animation, with an emphasis on NiFloodgate capabilities.

There is no instancing in this demo. Each fish has its own node in the scene graph, its own morphing, and a separate draw call, because different morphing animations may be activated arbitrarily and independently.

You can use the sliders to control the number of fish and the animation speed. When you turn on the Startle Fish option, fish will "puff out" when they are near the "gunsight". You can use the navigation controls to reposition the gunsight and startle arbitrary individual fish independently. You can turn on wireframe mode to see a graphical depiction of how complex the geometry is for each fish.

BadSushi demo (Win32)

On Playstation 3, you can also use a slider to control the number of SPUs. This provides a visual example of the effect that multiple SPUs via NiFloodgate have on morphing performance.
