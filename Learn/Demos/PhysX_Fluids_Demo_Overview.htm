PhysX Fluids Demo Overview

The PhysX Fluids sample demonstrates a basic PhysX fluid simulation integrated with Gamebryo. A stream of liquid falls and runs down a trough into a bowl. A fluid mesh-based particle system is being used to render the output of the PhysX simulation.

The fluid simulation will make use of PhysX hardware if available. Otherwise, it falls back to software simulation.

PhysX Fluids demo

The geometry, particle properties and PhysX properties of the scene were generated in Max and exported as a fluid system using the PhysX Extractor tool plug-in. The same scene could be generated in Maya, as the starting point is a standard Gamebryo particle system. The remainder of the application is standard Gamebryo-PhysX.
