Introduction to the ShadowMap Demo

The ShadowMap demo shows how to use Gamebryo's built in shadowing system. It shows the different shadowing techniques supported by Gamebryo and allows the user to adjust different shadowing parameters to see how the settings affects the resulting shadows. A shadow map is a shadow-casting method that creates high-quality shadows for lights that allows every object in the scene to serve as both a shadow caster and a shadow receiver, allowing for self-shadowing and complex shadow generation.

The ShadowMap sample allows the user to cycle through the different shadow techniques supported by Gamebryo, adjust the resolution of the shadow maps used in the scene, toggle use of back faces rendering, and provides a lighting only view. It should be noted that each shadow technique has its own set of hardware requirements and may not function on older hardware. The three shadow techniques that come with Gamebryo by default, NiStandardShadowTechnique, NiPCFShadowTechnique, and NiVSMShadowTechnique all require a SM2.0 card that supports floating point textures. If a shadow technique is used that is not supported by the hardware no shadows will be visible.
