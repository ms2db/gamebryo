BackgroundLoad Demo Functional Overview

The BackgroundLoad demo uses the NiStream::Load blocking function to load the ROOM.NIF file. It then launches a background load of the STUFF.NIF file. This file contains all of the contents of the room. While the background load of STUFF.NIF progresses, the demo renders the empty room that is extracted from ROOM.NIF and displays text that indicates the progress of the background load. As soon as STUFF.NIF finishes loading, the scene graph that is extracted from it is added to the scene graph that is being rendered.
