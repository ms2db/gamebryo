CharacterPerformance Demo Overview

CharacterPerformanceDemo is a demonstration of Gamebryo character animation and skinning performance. Several example characters are included, but you can also load your own character as an argument to the application. CharacterPerformanceDemo is designed to allow you to test and compare the performance of different models and parameters.

Load a character model into CharacterPerformanceDemo by supplying a KFM file argument to the application. You can export a KFM file from the Gamebryo 3ds Max Plug-in or the Gamebryo Maya Plug-in. Alternately, you can use the Gamebryo Animation Tool to create a KFM file.
