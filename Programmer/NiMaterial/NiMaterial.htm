NiMaterial

The base class for a Gamebryo material is in the NiMain library. This class is essentially a platform-independent interface class provided to enable a developer to easily control which NiShader is used to render a given piece of geometry.

An application using the material system will first ask the material if the last used shader is still valid. This is done through the IsShaderCurrent method. If that shader is not still valid or no previous shader exists, the GetCurrentShader method is called. This lightweight interface should be sufficient for a variety of material implementations.

Derived Classes

Two derived classes in particular are discussed in detail in this documentation:

  • NiFragmentMaterial - The base class for the additive high-level shading language system
  • NiSingleShaderMaterial - A class used to wrap the NiShader system in the material framework
