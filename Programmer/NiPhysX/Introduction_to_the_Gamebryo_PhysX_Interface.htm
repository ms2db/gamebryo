Introduction to the Gamebryo PhysX Interface

The core of Gamebryo-PhysX functionality is the run-time systems for interfacing between PhysX actors and Gamebryo's scene graph objects. This section describes the mechanisms by which data is transferred, along with issues in managing PhysX content at run-time.

The following topics are covered:

  • Destinations that extract PhysX actor state to drive Gamebryo scene graph elements.
  • Sources that use Gamebryo's animation system to drive PhysX actor motion.
  • Mesh Modifiers that push PhysX state directly into meshes or particle systems.
  • Gamebryo-PhysX Props and Scenes that interface between Gamebryo and a PhysX NxScene object.
  • Slaving and Coordinate Systems for placing PhysX content where it is needed, and controlling how components of a PhysX scene are put together.
  • Global Management of PhysX state in Gamebryo.
