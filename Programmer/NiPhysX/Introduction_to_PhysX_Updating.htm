Introduction to PhysX Updating

This section discusses the steps to updating physics within a game loop.

  • Basic Update Tasks discusses the sub-steps common to any PhysX update, but leaves open the question of how to order them within a game loop.
  • Single-threaded Updates looks briefly at the ways to order sub-steps in a single-threaded application, and some of the artifacts that might arise. For a more complete discussion, with example code, see the PhysX Timing Demo.
  • Multi-threaded Updates examines update issues specific to multi-threaded environments.
