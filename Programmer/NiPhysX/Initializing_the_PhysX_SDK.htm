Initializing the PhysX SDK

The PhysX SDK must be initialized before any other PhysX function calls are made. Initialization is done with the efdPhysX::PhysXSDKManager::Initialize function. Applications will typically call this function immediately after initializing Gamebryo.

The PhysX SDK and Gamebryo-PhysX system must be shut down when an application terminates. The efdPhysX::PhysXSDKManager::Shutdown function does this. Applications can also shut down and re-initialize the SDK to do a complete reset.

Some scene-graph elements contain pointers into PhysX objects, such as actors. All of these should be deleted before the PhysX SDK is shut down, particularly if an application re-initializes the SDK at some later time.
