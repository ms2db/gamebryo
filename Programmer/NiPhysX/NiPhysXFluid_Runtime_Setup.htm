Runtime Setup of NiPhysXFluid Content

The art pipeline for PhysX fluids is not sufficient to define all the fluid's properties. Hence, some degree of customization is typically required after loading and instantiating PhysX content in an application.

Creating Drains

Fluid drains should be created after the PhysX content has been instantiated.

Mixing Fluid and Rigid Bodies

The PhysX compartment system eases the burden of mixing rigid bodies and fluids in hardware and software. To make use of compartments, create the software rigid body scene without the fluid content, and save it out of the art package. This will create one prop which you can assign to a particular compartment. Then, create the fluid scene and save it separately. There should be no objects common to both scenes. When the PhysX scene is created in your application, set up compartments, inform the NiPhysXScene of them, and then inform the NiPhysXProp of the particular compartment it should be created in. Then add the prop to the scene.

See the PhysX documentation for more information on the interaction between fluids and rigid bodies and the structure of compartments.
