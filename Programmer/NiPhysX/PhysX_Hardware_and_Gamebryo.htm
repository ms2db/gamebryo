PhysX Hardware and Gamebryo

Gamebryo supports both hardware and software PhysX simulations. The most significant issues when working with hardware are in tailoring content to the feature set of hardware, although the PhysX SDK provides some help in managing a mixture of hardware and software simulation and can fall back to software if hardware is not suitable. The Gamebryo art tools provide some help in this area, but also see the General Gamebryo-PhysX Known Issues and PhysX Programming Known Issues.

Applications must manage hardware detection and the loading of specific content for hardware or software. See the PhysX demos for examples of hardware detection.
