NVIDIA PhysX™ Resources

The following external links provide helpful resources for Gamebryo users supporting physics capabilities via the NVIDIA PhysX integration:

  • http://developer.nvidia.com/object/physx.html - main developer site for NVIDIA PhysX.
  • http://origin-developer.nvidia.com/object/physx_downloads.html - for downloading NVIDIA PhysX.
  • physxlicensing@nvidia.com - to obtain appropriate licensing for NVIDIA PhysX.
  • The installers for the PhysX DCC plug-ins are shipped on the Gamebryo disc, in the PhysX directory. Use these rather than plug-ins found in other locations. Gamebryo LightSpeed was tested against the NVIDIA_PhysX_Plugin_for_3ds_Max_1.0.6.exe and NVIDIA_PhysX_Plug-in_3dsMax_x86_WithAPEX_2.40.0421.1156.msi versions of the Max PhysX plug-in, and the NVIDIA_PhysX_Plug-in_Maya_x86_2.01.0529.1836.msi version of the Maya PhysX plug-in. These are pre-release version of the plug-ins with a specific licensing agreement, found in the same directory.
