PhysX Global Management

In addition to initializing and shutting down the PhysX SDK, the singleton efdPhysX::PhysXSDKManager class provides services for global PhysX state:

  • The WaitSDKLock and ReleaseSDKLock functions can be used to ensure that only one thread accesses the PhysX SDK at any given time in a multi-threaded environment. See the multi-threaded update documentation for more details.
