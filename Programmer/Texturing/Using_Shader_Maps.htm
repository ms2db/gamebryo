Using Shader Maps

Shader maps (using NiTexturingProperty::ShaderMap) can be used to provide texture maps to an NiShader object that will make use of them. As a result, any map placed into a shader map slot will be ignored by the default texturing pipeline in all renderers. If an NiShader requires more information than is provided in a standard NiTexturingProperty::ShaderMap, applications can derive a class from NiTexturingProperty::ShaderMap and add any required data.
