Adding or Updating Texture Coordinates On-the-fly

Texture coordinates are specified as part of the model data of NiRenderObject subclasses specifically, NiMesh. Users who wish to add texture coordinates within an application can add a data stream using NiMesh::AddStream or other interfaces on NiMesh. Existing data streams on an NiMesh object can be modified by getting the appropriate NiDataStream object and locking the stream with write access.
