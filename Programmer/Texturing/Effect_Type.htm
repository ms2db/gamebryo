Effect Type

The texture effect type controls the placement of the projected multitexture effect in the texture pipeline (similar to the different Map slots in an NiTexturingProperty). The possible values are: PROJECTED_LIGHT, PROJECTED_SHADOW, ENVIRONMENT_MAP, and FOG_MAP. Each of these corresponds to a type of effect described in the projected multitexturing overview. This setting and the texture coordinate generation settings are the two most pivotal settings in terms of forming a projected texture effect.
