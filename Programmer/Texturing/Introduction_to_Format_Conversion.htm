Introduction to Format Conversion

Gamebryo uses a system of "image converters" that allows export tools (the Gamebryo 3ds max Plug-in, the Gamebryo Maya Plug-in, etc.) to generate platform specific content at export time, rather than relying on just-in-time processes to convert image formats for specific renderer support. These image converters are found in the NiMain library of Gamebryo. The base class for these converters is NiImageConverter. Alternatively, the image converters can be used by applications to pre-process image data so that the renderers do not need to support image format conversion code.

In previous versions of the engine, each renderer had its own unique code base for handling the tasks of image conversion and mipmap generation. The image converters allow this code to be centralized and isolated and make it available to external applications. Additionally, the converters are extendable, making it a much simpler task for applications to support custom image formats.
