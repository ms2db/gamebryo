Shader Map

The Shader Map is a way of providing texture maps that do not fall into any other category to an NiShader object. Note that the default texture pipeline in all renderers will ignore any Shader Map on an object.

If an NiShader requires more information than is provided in a standard NiTexturingProperty::ShaderMap, applications can derived a class from NiTexturingProperty::ShaderMap and add any required data.
