The NiTexturingProperty and its Components

An NiTexturingProperty contains two major pieces of information. The first is a set of one or more "Maps", which define the settings for each texture that is to be applied to the object. Different Maps represent different kinds of texturing effects, but for the moment, the only type of map that we will discuss is the Base Map, a texture map that defines the color of the object at any point on the object. Base maps are sometimes called diffuse maps in other packages.

The other piece of information in an NiTexturingProperty is a texture-apply mode setting, which defines the way that all of the textures (as a group) are affected by the object's final vertex colors. No matter how many Maps are contained in a NiTexturingProperty, there is only one apply mode setting.

An NiTexturingProperty::Map object contains all of the information that is needed to render a given texture as applied to the object. The values stored in a Map include (each of these will be described in detail in the following sections):

  • A pointer to a texture image object, which is a subclass of the NiTexture object. This value defines the image that is to be applied as a texture.
  • The clamping mode, which defines how a texture is to be "wrapped" when the texture coordinates at any point on the object fall outside of the range [0, 1].
  • The filtering mode, which defines how to interpolate between pixels in the texture.
  • The texture coordinate index, which selects the set of texture coordinates that are to be used to map the texture onto the geometry object (a geometry object may have several independent sets of texture coordinates for each vertex). For an object with a single texture, this index is generally irrelevant, as there only tends to be one set of texture coordinates on an object with a single texture. This setting will be described in more detail in the section on multitexturing.

The Base Map Map object for a given NiTexturingProperty is accessed by the calls NiTexturingProperty::GetBaseMap and SetBaseMap. Since the Base Map is the most commonly accessed Map in an NiTexturingProperty, the values within the Base Map object itself may be accessed through the convenience functions NiTexturingProperty::GetBaseTexture, SetBaseTexture, GetBaseClampMode, SetBaseClampMode, GetBaseFilterMode, SetBaseFilterMode, GetBaseTextureIndex, and SetBaseTextureIndex.
