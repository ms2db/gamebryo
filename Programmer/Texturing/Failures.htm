Image Conversion Failures

Generally, the image conversion process completes and renders the textures in the pipeline. There are conditions that can break the image conversion process. Commonly, this occurs because of external images that cannot be read or by loading data that was exported for a different platform than the one on which the application is currently executing. In this case, a default image will be substituted in the pipeline to indicate to the developer that a failure has occurred. The table below shows the possible default failure textures and describes the situations that will cause them to be used. Note that not all renderers use all of the options below.

Failure Condition NiTexture::RendererData Enumeration Description Image
No Matching 2D Format NO2DMATCH No valid format was found to which the renderer could convert the supplied pixel data for a standard NiSourceTexture.
No Matching Cube Map Format NOCUBEMAPMATCH No valid format was found to which the renderer could convert the supplied pixel data for an NiSourceCubeMap.
No Converter for Selected Format NOCONVERTER A valid format was found, but the supplied image converter could not convert to that format.
Failed Conversion FAILEDCONVERT The image converter failed to convert to this format. Note that in some renderers this may indicate a failure to find a matching format depending on the logic flow of that renderer.
Failed Cube Map Conversion CUBEMAPFAILEDCONVERT The image converter failed to convert a cube map to this format. Note that in some renderers this may indicate a failure to find a matching format depending on the logic flow of that renderer.
File Not Found FILENOTFOUND The specified file was not found.
Cube Map File Not Found CUBEMAPFILENOTFOUND The specified cube map file was not found.
Bad Dimensions BADDIMENSIONS The image has dimensions that are unsupported by the renderer for the given device. The dimensions may be too large for the device or may not be a power of 2 for a device that requires power-of-two dimensions for 2D textures.
Bad Cube Map Dimensions CUBEMAPBADDIMENSIONS The cube map image has dimensions that are unsupported by the renderer for the given device. The dimensions may be too large for the device or may not be a power of 2 for a device that requires power-of-two dimensions for 2D textures.
File Format Not Supported FILEFORMATNOTSUPPORTED The texture file was found, but is in a file format not supported by the current image converter.
Cube Map File Format Not Supported CUBEMAPFILEFORMATNOTSUPPORTED The cube map file was found, but is in a file format not supported by the current image converter.
File Load Failed FILELOADFAILED The texture file was found, and the image converter attempted to load the file but failed for some reason.
Cube Map File Load Failed CUBEMAPFILELOADFAILED The cube map file was found, and the image converter attempted to load the file but failed for some reason.
Generic GENERIC An undefined error has occurred.
