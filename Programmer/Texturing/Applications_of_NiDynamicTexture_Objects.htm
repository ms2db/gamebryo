Applications of NiDynamicTexture Objects

Applications use NiDynamicTextures to represent image data that is generated by an application via mechanisms other than rendering. For example, an application might use an NiDynamicTexture to represent a procedural texture. As another example, an NiDynamicTexture might represent the image coming from a video clip (which is then applied as a texture to an object in the main scene).

NiDynamicTexture effects are fairly straightforward. They require a "write-only" edit of the texture by the application, followed by rendering of the main scene, including objects to which the texture is applied.

See the DynamicTexture and VideoTexture demos, for specific examples of dynamic textures.
