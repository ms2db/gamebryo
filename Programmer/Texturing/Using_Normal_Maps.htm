Using Normal Maps

  • Image: The image used is an encoding of the normals for a geometric object into a texture. This image should encode the normals in tangent space, as Gamebryo normal mapping assumes that NBT vectors are required to convert the texture back into world space for lighting.
  • Filter: Any mode is reasonable, dependent upon the desired effect.
  • Clamping: Completely dependent upon the effect.
  • Texture Coordinates: Completely dependent upon the desired effect.
