Entity Properties: Introduction

Entities are composed of a set of properties and a set of behaviors. Entity properties are variables that are associated with a runtime entity. They're defined in the entity's model using the Entity Modeling Tool (EMT). Properties have associated traits that define how the property is used in the runtime system. They can have default values defined in the entity model, and these default values can be overridden in mix-in models and in World Builder.

After reading this section you will:

  • understand what entity properties represent
  • be familiar with property types that are supported by the Game Framework
  • know how to add properties to entity models
  • understand how property overrides work
  • be able to retrieve and update properties at runtime in C++ and script
  • have the information you need to add your own custom property types
