Asset Loading Services Overview

Gamebryo provides a message-based system for loading assets from media. System services or application level code send request messages for assets. Later, the services or application-level code receive a response containing the asset.

You can control how assets are loaded, you can re-use assets without reloading them, and you can create your own asset loaders. Gamebryo's asset loading services support rapid iteration.
