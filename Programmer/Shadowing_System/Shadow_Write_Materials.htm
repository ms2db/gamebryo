Shadow Write Materials

Shadow write materials define the data written to a shadow map and the mechanism for writing that data. The shadow write materials derive from NiStandardMaterial but they are simpler and used only to write depth information to shadow maps. An NiShadowTechnique object holds a reference to a shader fragment for each light type capable of casting shadows: point lights, directional lights, and spot lights. When geometry is being drawn to a shadow map, the appropriate shader fragment will be used to write the depth information. This allows the different NiShadowTechnique objects to write data to a shadow map in a way that is specific to that shadow technique.

The following three shadow write materials are used by default in the Gamebryo shadowing system when writing to a shadow map. Each material corresponds to one light type:

  • NiPointShadowWriteMaterial
  • NiDirectionalShadowWriteMaterial
  • NiSpotShadowWriteMaterial
