Shadow Map

Shadow map objects provide the rendering pipeline with a way of retrieving the needed NiRenderedTexture objects to read and write the shadow mapping data for a piece of geometry. The shadow map object contains a reference to the NiShadowRenderClick objects needed to render the data to the associated NiRenderedTexture object.

The shadow manager contains a pool of NiShadowMap objects that can be assigned to NiShadowGenerator objects. It is the responsibility of the shadow click generator to choose how to assign these shadow maps and when to create new ones.
