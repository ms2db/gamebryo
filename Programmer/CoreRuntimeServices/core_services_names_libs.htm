The Core Runtime Services Libraries

The Core Runtime Services are found in several libraries, each with its own C++ namespace. Libraries with the prefix "ecr" provide the most basic functionality and we expect them to be used in any application that renders a scene, uses input or uses PhysX. Libraries with the prefix "egm" are Game Modules that provide a packaged block of functionality that some applications, but not all, are expected to require. Game Modules also simplify the provision of debug only features, such as visualization of non-rendered entities.

The following table provides basic information and links for each library.
