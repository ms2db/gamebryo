The egmPhysXWhitebox Library

The egmPhysXWhitebox::PhysXWhiteboxService class can optionally replace the egmWhitebox::WhiteboxService class. The PhysXWhitebox service does everything the Whitbox service does, but adds the capability to create static or dynamic rigid bodies associated with an entity. When an entity is created that mixes in the Whitebox model, the service creates a scenegraph containing mesh geometry and associates that scenegraph with the entity via the SceneGraphService. When the service discovers and entity that mixes in PhysXWhitebox, it also creates a PhysX prop and registers it was well. The service listens for property changes related to the box geometry or physical properties for those entities and modifies their scene graphs and/or props to reflect the new property values.

The WhiteboxService doesn't have any relevant public interfaces or built-in functions. The only method of interaction with Whiteboxes is through the entity properties. For more information on authoring whiteboxes, see Working with Whiteboxes.
