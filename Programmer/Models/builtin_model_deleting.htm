Deleting built-in models

Built-in models can be deleted as normal using the Entity Modeling Tool. This should work even in rapid iteration scenarios with the normal limitations on changing mix-in composition at run-time. You should remove the data for the built-in model before you remove the code. Once the data has been removed the built-in model C++ implementation can be removed at your leisure.
