NiEnvironment Introduction

The environment is an extension to Gamebryo's Scene Graph system. An environment is created and rendered by adding a NiEnvironment object to the scene graph of an application. The creation of an NiEnvironment object can be assisted using the Game Framework block document system by adding an entity with a Environment model mixin to the scene. Furthermore, the NiEnvironment's property animation can be controlled using a NiTimeOfDay object.

NiTimeOfDay is an extension of Gamebryo's NiAVObject. It can be attached to any parent node. When updated, it will animate any properties registered with it. The creation of a time of day animation can be assisted using the Game Framework block document system by adding an entity with a TimeOfDay model mixin to the scene.
