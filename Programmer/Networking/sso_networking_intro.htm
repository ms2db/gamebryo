Networking: Introduction

Gamebryo LightSpeed's networking system supports Windows and console-specific network transports. The system provides seamless remote messaging and connection management. It provides remote communication capabilities, and it transparently abstracts both remote and local messaging. The networking system is designed for flexibility, providing an extensible network transport layer abstraction.
