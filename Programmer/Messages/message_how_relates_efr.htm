How messages relate to the Emergent Game Framework

Many services in the Emergent Game Framework send local and remote notifications using messages. Some services subscribe to specific Categories and handle messages as a way of interfacing with that service or notifying the service of a specific event. For more details see the documentation for the specific service you wish to use.
