Requirements for working with messages

The table below lists the libraries, system services and header files required to work with messages in a C++ module:

Library System Services Header Files Description
efd
  • Message Service
  • efd/MessageService.h
 Provides access to the core message service functionality. Includes headers for IMessage, MessageHandler and associated macros.
efdNetwork
  • Net Service
  • efdNetwork/NetService.h
 Provides access to the remote message service functionality.
