Using Built-in API Functions

You use built-in API functions to expose API calls to script. Whether you need to call these global functions in your C++ behavior, Lua behavior, or any supported scripting language, you will need to load the built-in API before you have access to them in script. The Emergent Game Framework supports dynamic and static loading of built-in APIs.
