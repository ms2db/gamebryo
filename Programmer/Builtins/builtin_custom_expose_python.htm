Exposing Custom Built-in API Functions to Python

To expose a custom built-in API function to Python

Python has been added as an example of adding an additional scripting language but is not supported by all modules, and is not supported by Emergent at this time. Currently only BehaviorAPI in Game Framework has built-in API functions for Python, however adding support to other modules can be done by following the examples in egfPython. The process is very similar to Exposing custom built-in API Functions to Lua.
