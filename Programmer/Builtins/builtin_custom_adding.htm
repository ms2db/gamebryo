Adding Custom Built-in API Functions to the Scripting System

After you have written your C++ built-in API functions, then next step is to expose them to one or more scripting languages.

To add a custom built-in API function to the scripting system

  1. Create a SWIG .i file.
  2. Use SWIG to generate a _wrap.cpp for your built-in API.
  3. Add the generated _wrap.cpp file to your project.
  4. Register the _wrap file initialization function with the appropriate scripting runtime.
