ColorA Module

Description

The ColorA module contains functions to create, access, and manipulate instances of the efd::ColorA class in Lua. Colors are represented by four float values (r,g b,a), with black represented as (0,0,0,1), and white as (1,1,1,1).

All functions are in the ColorA module namespace in Lua.

ColorA values passed from C++ to Lua are always converted into a Lua ColorA userdata object. The functions described below represent the functions available to these objects.

ColorA values passed from Lua to C++ can either be a ColorA value constructed using Color.Color(r,g,b,a) or a Lua table containing 4numbers representing r, g, b, a.

Library Constants

ColorA_BLACK

A Color constant containing the values (0,0,0,1).

ColorA_WHITE

A Color constant containing the values (1,1,1,1).

Module Attributes

ColorA.r

The red value.

ColorA.g

The green value.

ColorA.b

The blue value.

ColorA.a

The alpha value.

    c1 = ColorA.ColorA()

boolean ==

Determine if two ColorsA are equal.

Return ValueDescription
boolean True if the two colors are equal, false otherwise.

Example: 

    c1 = ColorA.ColorA(0.8, 0.6, 0.4, 0.9)
    c2 = ColorA.ColorA(0.8, 0.6, 0.4, 0.9)
    c3 = ColorA.ColorA_BLACK
    
    b = (c1 == c2)  -- true
    b = (c1 == c3)  -- false
    b = (c2 ~= c3)  -- true

userdata +

Add two colors.

Return ValueDescription
ColorA Add two colors.

Example: 

    c1 = ColorA.ColorA(0.1, 0.1, 0.1, 0.5)
    c2 = ColorA.ColorA(0.5, 0.4, 0.3)   
    c1 = c1 + c2 -- ColorA(0.6, 0.5, 0.6, 0.5)

userdata -

Subtract two colors.

Return ValueDescription
ColorA Subtract two colors.

Example: 

    c1 = ColorA.ColorA(0.1, 0.1, 0.1, 0.5)
    c2 = ColorA.ColorA(0.5, 0.4, 0.3)    
    c1 = c1 - c2 -- ColorA(-0.4, -0.3, -0.2, 0.5)

userdata *

Multiply two colors.

Return ValueDescription
ColorA Multiply two colors.

Example: 

    c1 = ColorA.ColorA(0.5, 0.4, 0.3, 0.5)
    c2 = ColorA.ColorA(0.1, 0.2, 0.3, 0.5)    
    c1 = c1 * c2 -- ColorA(0.05, 0.08, 0.09, 0.25)

userdata /

Divide two colors.

Return ValueDescription
ColorA Divide two colors.

Example: 

    c1 = ColorA.ColorA(3.0, 4.0, 5.0, 100.0)
    c2 = ColorA.ColorA(9.0, 8.0, 15.0, 10.0)    
    c2 = c2 / c1 -- ColorA(3.0, 2.0, 3.0, 10.0)

userdata Unary-

Divide two colors.

Return ValueDescription
Color Negate a color.

Example: 

    c1 = ColorA.ColorA(0.5, -0.4, 0.3, -0.25)
    c2 = -c1  ColorA(-0.5, 0.4, -0.3, 0.25)

nil Scale

Ensure RGBA components are within the range [0.0, 1.0] by scaling down all components by the max component. This function sets the components in the correct range and preserves the hue of the final color. Note: all components are assumed to be >= 0.0, so no such check is performed.

Return ValueDescription
nil Scale the color to the range [0.0, 1.0]

Example: 

    c1 = ColorA.ColorA(1.65, 0.8, 1.11, 1.65)
    c1:Scale()  -- ColorA(1.0, 0.4848, 0.6727, 1.0)

nil Clamp

Ensure RGBA components are within the range [0.0, 1.0] by clamping any components above 1.0 to 1.0. Note: all components are assumed to be >=0, so no such check is performed.

Return ValueDescription
nil Clamp the color to the range [0.0, 1.0]

Example: 

    c1 = ColorA.ColorA(10.0, 0.001, 1.52, 1.23)
    c1:Clamp()  -- ColorA(1.0, 0.001, 1.0, 1.0)

string tostring(ColorA)

Create a pretty string for this color, of the form "Color(r.000000, g.000000, b.000000, a.000000)".

Return ValueDescription
string String containing the pretty version of the given color.

Example: 

    c1 = Color.Color(1.65, 0.8, 1.11, 0.9)
    local s = tostring(c1)   -- "Color(1.650000, 0.800000, 1.110000, 0.900000)"
