egmAnimation

Description

This built-in API library contains methods for interacting with entities that implement the Actor model. It interacts with the AnimationService.

Lua Constants

NoTransition

Not transitioning

Blending

In a blending transition.

Morphing

In a morphing transition.

CrossFading

In a cross-fading transition.

Lua Variables

kInvalidSequence

Sequence ID constant that represents an invalid sequence.

kSequenceDataFreq

Frequency constant used to specify that the sequence's frequency itself should be used.

kSequenceNone

Sequence ID constant used for specifying that a sequence should not be synchronized.

number entityID,
number sequenceID,

number priority,
number weight,
number easeInTime,

number timeSyncID,
number freq,
number startFrame,
boolean additiveBlend,
float additiveRefFrame)

Activates a layered sequence on an entity. See NiActorManager documentation for more information about these arguments.

Parameter Description
entityID Actor that owns the sequence.
sequenceID Sequence ID to activate.
autoDeactivate Whether to deactivate the sequence when it is complete.
priority (Optional) Sequence priority.
weight (Optional) Sequence weight.
easeInTime (Optional) Duration to ease this sequence in.
easeOutTime (Optional) Duration to ease this sequence out (if auto-deactivating.)
timeSyncID (Optional) Sequence to synchronize with. kInvalidSequence means no synchronization.
freq (Optional) Frequency to play this animation at.
startFrame (Optional) Frame to begin playing this animation at.
additiveBlend (Optional) Whether or not to blend this sequence additively with other active sequences.
additiveRefFrame (Optional) The frame to use as the base when calculating deltas for additive sequences.
Return Value Description
boolean True if the sequence activated successfully.

boolean ListenForTextKeyEvents(
number entityID,
string behaviorName,
string textKey,
string seqName)

Registers a text key callback behavior function. The data stream elements passed to the behavior are, in order:

  • sequence ID (UInt32)
  • text key (String)
  • current time (Float32)
  • event time (Float32)
Parameter Description
entityID The actor.
behaviorName The behavior to call when this text key is encountered.
textKey (Optional) The text key to match against (Default is all text keys.)
seqName (Optional) The sequence name to match text keys in (Default is all sequences.)
Return Value Description
boolean True if the registration succeeded.
