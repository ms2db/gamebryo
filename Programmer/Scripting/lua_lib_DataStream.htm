ParameterList

Description

The ParameterList library contains only a single function which is used to create a new ParameterList object. ParameterList are used to send and return data between behaviors.

Library Functions

userdata ParameterList()

Create a new userdata object to wrapper an efd::ParameterList instance. The returned object has a ParameterList metatable to provide access methods. The newly created efd::ParameterList will initially be empty.

Return ValueDescription
ParameterListA userdata object wrapping an empty ParameterList.
