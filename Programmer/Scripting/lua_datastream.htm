ParameterList.ParameterList

Description

A ParameterList knows how to read and write Emergent data types to marshal and demarshal this data to a binary buffer. ParameterList is implemented in Lua as a custom userdata type with a metatable to provide an object-like syntax. To create a ParameterList from script, call the ParameterList.new() function. The ParameterList Lua module is automatically loaded and made available to all behaviors. The following functions are available in the ParameterList metatable:

ParameterList value)
