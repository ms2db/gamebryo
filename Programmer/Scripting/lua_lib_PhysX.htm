PhysX

Description

The PhysX Lua bindings provide access to the ecrPhysX library and, through it, to the Gamebryo Core Runtime's NiPhysXScene and NiPhysXProp classes.

Library Functions

ApplyForce()

Apply a force to (every actor in) a particular entity. Since this is applied to the actor's center of mass, it will not create a torque.

Parameter Description
eid ID of the entity to have a force applied.
force Force to be applied.

ApplyForceAtPos()

Apply a force to (every actor in) a particular entity at a particular position, possibly creating torque.

Parameter Description
eid ID of the entity to have a force applied.
force Force to be applied.
pos Point at which to apply the force.

bool ResetState()

Sets all Props in the scene to the state stored in a particular snapshot.

Parameter Description
sceneName Name of the PhysX scene to reset.
stateID Index into array of state snapshots stored by the art tool in the scene's NiPhysXProp objects.
Return Value Description
ResetState True if the named scene was found; false otherwise.
