Programming Basics

This section describes Gamebryo's programming standards, object systems, and basic programming concepts such as run-time type information and memory management. Topics include:

  • Class Inheritance Structure: describes Gamebryo's predominant class hierarchy structures.
  • Object System: describes components of Gamebryo's object systems.
  • Static Gamebryo Objects and Object Lifetime: describes management of static data.
  • Naming Conventions: describes naming conventions for files, namespaces, classes, functions and variables used throughout Gamebryo.
  • Template Classes: provides an overview of templated container classes and algorithms provided by and used in Gamebryo.
  • Gamebryo and Unicode: describes the extent of support for Unicode characters provided by Gamebryo.
