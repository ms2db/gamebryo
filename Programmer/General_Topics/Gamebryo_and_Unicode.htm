Gamebryo and Unicode

Gamebryo provides the capability to display 2-byte Unicode characters (UTC-2), as well as Ascii characters, via its NiFont library.

The UnicodeCharacterSets sample demonstrates this capability. It makes use of an NFF file, from which font information is streamed into the application to create an NiFont instance. NFF files are created by the NiFontCreator tool.

Nevertheless, other strings in Gamebryo, such as file names, file paths, object names, etc. remain in Ascii.

Furthermore, builds of Gamebryo with UNICODE or _UNICODE defined are not supported.
