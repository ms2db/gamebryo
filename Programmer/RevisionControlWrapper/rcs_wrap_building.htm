Building Revision Control

The SDKs for Perforce or Subversion are not included with Gamebryo LightSpeed. If you plan to integrate either of these revision control systems with Toolbench, you will need to build them. If you plan to make changes to the DLLs, or if you simply need to recompile them, you can use the Visual Studio solutions that ship with the product.

Build Order

  • Build\Tools\Win32\VC90\RevisionControl.sln
  • Build\Tools\Win32\VC90\Toolbench.sln
  • Build\Tools\Win32\VC90\ToolbenchRevisionControl.sln

Building Perforce

Toolbench was built against the Perforce r09.1 API. You can download the headers and libs from this Perforce version from the following location:

ftp://ftp.perforce.com/perforce/r09.1/bin.ntx86/

You need to download all of the following compressed zip files to build Toolbench. There are also Visual Studio 2005 zips if you plan to re-purpose our revision control layer for something else that can be built under Visual Studio 2005.

  • p4api_vs2008_dyn.zip
  • p4api_vs2008_dyn_vsdebug.zip
  • p4api_vs2008_static.zip
  • p4api_vs2008_static_vsdebug.zip

The Perforce project in RevisionControl.sln is set up to look for an environment variable called P4SDKPATH. You need to define this environment variable, and you also need to set up the folder hierarchy in which you unzip the p4api_vs2008*.zip files so that it looks like the following (using p4api_vs2008_dyn_vsdebug as an example):

$(P4SDKPATH)/bin.ntx86/p4api_vs2008_dyn_vsdebug/include

Building Subversion

Toolbench was built against the Subversion 1.6.2 API. You can download the headers and libs from this Subversion version from the following location:

http://subversion.tigris.org/files/documents/15/45947/svn-win32-1.6.2_dev.zip

The Subversion project in RevisionControl.sln is set up to look for a an environment variable called SVNSDKPATH. You need to define this environment variable, and you also need to set up the folder hierarchy in which you unzip svn-win32-1.6.2_dev.zip so that it looks like this:

$(SVNSDKPATH)/include
