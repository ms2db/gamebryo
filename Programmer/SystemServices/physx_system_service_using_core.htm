Using PhysX services with Gamebryo Core Runtime

Using services from the ecrPhysX, egmPhysXTerrain, and egmPhysXRagdoll libraries gives you access to the full set of PhysX features supported in the Core Runtime.

PhysX services:

  • ecrPhysX::PropService — This service handles the Gamebryo Core Runtime version of the PhysX scene and all PhysX props that are created. This service takes over the responsibility of scene simulation from the Scene Service, although the Scene Service still controls when the simulation takes place.
  • egmPhysXRagdoll::RagdollService — This optional service allows you to use ragdolls in your application by controlling which animation sequences of a prop are overridden by ragdoll data from PhysX.
  • egmPhysXTerrain::PhysXTerrainService — This optional service extends egmTerrain::TerrainService to support physical interactions with terrain sectors.
