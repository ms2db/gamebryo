Using PhysX service without Gamebryo Core Runtime

PhysX services:

  • egfPhysX::SceneService — This service handles the life cycle of PhysX scenes and is responsible for simulation of those scenes. It can delegate simulating some scenes to other services. When it "owns" a scene, Scene Service provides some of the same time management features as NiPhysXScene.
  • egfPhysX::TriggerService — This service handles collisions between PhysX shapes. It does not respond to collisions that involve PhysX props.
