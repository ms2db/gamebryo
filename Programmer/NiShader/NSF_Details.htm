NSF Details

The NDL Shader File (NSF) text-based format is a utility format supplied with the Gamebryo Shader System. This format allows for shaders to be setup in a simple text format, allowing for rapid prototyping during the development of your application. These text-based files can be processed by the supplied parser library, NSFParserLib, to generate corresponding NSB files which can, in turn, be loaded by the NiBinaryShaderLib library.

The parser library utilizes flex and bison to generate NSB files from the text-base format. For more information on writing shaders in the NSF format, please refer to the NSF Reference documentation.
