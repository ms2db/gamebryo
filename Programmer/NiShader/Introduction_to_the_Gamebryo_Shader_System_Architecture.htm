Introduction to the Gamebryo Shader System Architecture

The Gamebryo Shader System consists primarily of several classes which are contained in the NiMain library, as well several which are shared amongst D3D-based renderers (NiDX9Renderer and NiXenonRenderer). These classes constitute the core of the shader system.

This section contains a general overview of how the classes relate to each other and are used together to implement the functionality of the shader system. Refer to the class reference for specific information about each class.
