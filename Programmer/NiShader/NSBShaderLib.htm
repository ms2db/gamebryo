NSBShaderLib

The NSBShaderLib implements a platform-independent library that will load any valid NSB files found in a directory.

Several classes also exist in this library, primarily to provide a platform-independent representation of the data that is stored in the binary file. By providing a platform-independent representation of the data, a single data file may be used on any supported renderer. For a complete description of these classes, refer to the NSBShaderLib reference.
