NiShaderParser

This class defines the base interface for implementing shader parser. A shader parser is a class that transforms a shader source text file into one or more binary files.

At its base level, the parser contains functions for parsing shader source files and retrieving mime type information relevant to the types of shaders that the parser can take as input or writes as outputs. These functions are empty in the base class, and rely on derived classes to perform the work.

Shader parsers are registered with the static methods in the NiShaderParser class. For a complete description of the functions contained in this class, please refer to the NiShaderParser class reference.
