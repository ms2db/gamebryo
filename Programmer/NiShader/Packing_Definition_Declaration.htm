Packing Definition Declaration

This section simply declares the packing definition that the implementation requires. The packing definition must have been defined previously in the shader file. This declaration is required for any implementation that uses a vertex shader in any of its passes. If an implementation uses only the fixed-function vertex pipeline, then it may safely leave out the packing definition declaration.

Note: The Packing Definition is now deprecated, and does not directly affect packing of geometry.  If present, it will be converted into a semantic adapter table, which specifies how data from a mesh is bound to varying parameters in a shader.
