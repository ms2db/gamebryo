NiShaderConstantMap

This base class provides a way to setup the shader constant registers for use with shader programs. It contains an array of entries, which are represented by the class NiShaderConstantMapEntry, each one of which maps its data value into one or more shader constant registers.

The shader constant map provides the ability to write shaders that are completely data-driven. The data required by the shader can be hung off of each object via the NiExtraData interface. These items can be exposed to the artist for per-object tweaking, or can just provide an easy interface for setting data on a per-object manner. Constant mapping and the extra data system combine to be one of the more powerful features of the Gamebryo Shader System.

For a complete description of the functions and members contained in this class, please refer to the NiShaderConstantMap reference.
