NiShaderLibrary

This class defines the base interface for implementing shader libraries. A shader library is a collection of Gamebryo shaders that can be exposed to both your application and the tool chain, allowing for artists and designers to preview artwork with exactly the same shaders that will be utilized in the game itself.

At its base level, the library contains only a name and functions for retrieving shaders, GetShader, and releasing them, ReleaseShader. These functions are empty in the base class, and rely on derived classes to perform the work.

Shader libraries are registered with the shader factory, which is used to attach shaders to geometry during streaming. The shader factory class is described in the next section.

For a complete description of the functions contained in this class, please refer to the NiShaderLibrary class reference.
