Use Descriptions for the Artist

The NSF file format allows descriptions to be embedded that will show up in the shader rollouts for 3ds Max and Maya. Below is a fragment from an NSF illustrating this.

NSFShader BaseBumpWithSpatialGlow
{
    "Apply a base map, normal map, and a glow map. The normal map is used "
    "to modify the lighting from two directional lights, and the glow map "
    "intensity is scaled by the distance to the glow source. The quadratic "
    "and linear attenuation components of the glow map are controlled by "
    "the x and y components of the 'GlowAttenuation' attribute, while the "
    "w component of that attribute represents the size of the hot spot in "
    "the middle of the glow. This shader is used in the Eturnum sample for "
    "the walls of the temple. "

Use this section to outline any special instructions that the artist may need in setting up the shader. Explain what the shader does, and any special conditions that need to be met for the shader to function properly. For example, this is a good place to explain how the artist should set constant values on the object or to explain which slots should contain specific textures.
