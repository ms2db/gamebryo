Error Handling

Error handling occurs on several levels in the Gamebryo Shader System. Individual functions return NiShaderError values, the base values of which are defined in NiShaderError.h in NISHADERERR_OK, and all other values should be considered failures. As Gamebryo develops, the error code definitions will be expanded to describe a wider variety of error conditions.

The NiShaderFactory and NiD3DShaderProgramFactory classes utilize a callback method to report errors back to the user. This technique is useful during development to easily report errors to the user including missing shader and/or shader program files, errors in shader and/or shader program files, and failure to create shaders.

The parser library, NSFParserLib, and the binary shader library, NiBinaryShaderLib, also take advantage of the error callback system to inform of errors that occur during the parsing of NSF files and loading of NSB files.
