NSB Details

One of the shader-related libraries provided with the Gamebryo engine is the NSBShaderLib library. This library defines a set of classes for treating shaders in a purely binary format called NSB. These classes allow for a completely data-driven approach to shader development and usage in your application. The NSBShaderLib library is responsible for loading/processing of any NSB shader files found in a given directory. (Optionally, a recursive search of a directory and its subdirectories may be performed.) This library, coupled with the parsing of NSF text-based files, provides a powerful rapid-prototyping system for shader development, which is also full-featured enough for use in a released application.

Another shader-related library, NiBinaryShaderLib, implements an NiD3DShaderLibrary which automatically handles the conversion of the NSB structures provided by NSBShaderLib into NiD3DShader objects. The same functionality can be found for the other renderers in the NiD3D10BinaryShaderLib, ecr::D3D11BinaryShaderLib, and NiPS3BinaryShaderLib libraries.
