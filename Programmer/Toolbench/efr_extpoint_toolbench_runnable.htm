Runnable (Extension-Point)

Emergent.Toolbench.Runnable

When Toolbench starts it will search all of the extensions for any that are of the runnable type. These runnable extensions make up the possible applications that Toolbench runs. As an example the standard UI base of Toolbench is Workbench. But by adding a new console based runnable extension, you could effectively create a console based version of Toolbench for batch processing.

<extension point="Emergent.Toolbench.Runnable">
  <runnable id="Workbench"
            class="Emergent.Toolbench.UI.Workbench.UIApplication" />
  ...
</extension>
Attribute Type Description
class String The full class name of the runnable class to use when loading this runnable plug-in. The runnable class must implement the Emergent.Toolbench.IRunnable interface.
