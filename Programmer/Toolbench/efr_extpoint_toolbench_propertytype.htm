PropertyType (Extension-Point)

Emergent.Toolbench.PropertyType

The PropertyType extension point allows the specification of data types which entity properties will use. This extension point also allows plug-ins to specify which editor(s) will be used to modify properties of the given type.

<extension point="Emergent.Toolbench.PropertyType">
  <type name="Float" managed-type="System.Single"/>
  ...
</extension>

type

Identifier Type Description
name String The name of the property type which will show up when selecting the type of a property.
managed-type String The underlying managed type that will store the value of properties of this type.
typeconverter String [optional] A converter which will convert data entered by the user to this property type.
ui-single-editor String [optional] The editor (which must extend UITypeEditor) alllowing the user to edit properties of this type as though it was a single value.
ui-collection-editor String [optional] The editor (which must extend UITypeEditor) allowing the user to edit properties of this type as though they were a collection of values.
