SimEvent (Extension-Point)

Emergent.Toolbench.SimDebugger.SimEvent

The SimEvent extension point allows you to register new event handlers in the Simulation Debugger. They are factoried in response to notifications received from a connected game. SimEvents make up the history of the game recorded by the Simulation Debugger and can be "processed" to update the debugger world state, which is in turn visualized in the Simulation Debugger World View. 

<extension point="Emergent.Toolbench.SimDebugger.SimEvent">
  <sim-event class="Emergent.Toolbench.SimDebugger.SessionStartedSimEvent"/>
  <sim-event class="Emergent.Toolbench.SimDebugger.PropertyChangedSimEvent"/>
</extension>

sim-event

Attribute Type Description
class String The class factoried to manage the event in the debugger.
