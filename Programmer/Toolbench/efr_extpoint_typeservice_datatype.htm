DataType extension-point

The data type extension point allows you to specify new data types for use with entity properties. These new data types can be given custom restrictions and can also have their own customized editors for setting their values.

<extension point="Emergent.Toolbench.TypeService.DataType">
  <datatype name="Boolean" 
            type="Emergent.Toolbench.TypeService.Types.Boolean"
            editor="Emergent.Toolbench.TypeService.Editors.BooleanEditor"
            use-text-editor="false">
    <restriction></restriction>
  </datatype>
</extension>

datatype

Identifier Type Description
name String The unique name of the data type.
type String The full name of the class that gets created to store data for the data type.
editor String The editor used to edit data of this type.
export-name String [optional] The name of the data type when being exported.
use-text-editor Boolean [optional] [default: true] A flag indicating whether to allow using the standard text edit box to edit a property of this type.

restriction

Identifier Type Description
id String [optional] A uniquely identifying string for the restriction.
