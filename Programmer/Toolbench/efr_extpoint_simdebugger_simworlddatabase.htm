SimWorldDatabase (Extension-Point)

Emergent.Toolbench.SimDebugger.SimWorldDatabase

The SimWorldDatabase extension point is used to specify new databases of game information in the Simulation Debugger. Declaring a SimWorldDatabase extension will create one instance of the database per .simdebug session document, managed via the SimWorldAspect. The class specified for the extension must implement the ISimWorldDatabase interface, which has only a simple Reset method.

Typically, SimEvents will add and modify the data stored in SimWorldDatabases, and SimVisuals will render a visualization of this data. 

<extension point="Emergent.Toolbench.SimDebugger.SimWorldDatabase">
  <sim-event class="Emergent.Toolbench.SimDebugger.ModelDatabase"/>
  <sim-event class="Emergent.Toolbench.SimDebugger.EntityDatabase"/>
</extension>

sim-world-database

Attribute Type Description
class String The database class factoried in the debugger.
