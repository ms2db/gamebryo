BrushElementDropDownUI (Extension-Point)

Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushElementDropDownUI

The Terrain BrushElementDropDownUI allows the developer to attach a custom UI to a preiously defined custom BrushElement to allow the user to configure the various settings available on the element.

The class indicated by the class attribute of the extension point must refer to a class implementing the UserControl class. 

<!-- New brush element UI. -->
<extension point="Emergent.Toolbench.Worldbuilder.Terrain.UI.BrushElementDropDownUI">
    <brushelementdropdownui class="default"
                    brush-element="New IBrushElementName"/>
</extension>

Attribute Type Description
class String ...

Adding New Brush Elements

BrushElementDropDownUI is required to add an existing BrushElement to the existing brush element user interface. For example, the complete elevate brush element extension point looks like: 

<!-- Elevate brush element. -->
<extension point="Emergent.Toolbench.WorldBuilder.TerrainCore.BrushElement">
    <brushelement class="Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushElements.ElevateBrushElement"
                   name="Elevate" 
                   icon="Icons/terrain_brushelement_elevate.png" 
               category="Terrain Sculpt">
        Increases or decreases the height of the terrain. Hold down the Ctrl key to decrease 
        height.
    </brushelement>
</extension>
  
<extension point="Emergent.Toolbench.Worldbuilder.Terrain.UI.BrushElementDropDownUI">
    <brushelementdropdownui class="default"
                    brush-element="Elevate"/>
    </extension>
<!-- End Elevate brush element. -->
