OutputChannel (Extension-Point)

Emergent.Toolbench.OutputView.OutputChannel

The OutputChannel extension point allows users to create new channels where output messages will be written. These channels show up in Toolbench's Output View.

<extension point="Emergent.Toolbench.OutputView.OutputChannel">
  <channel name="Game"
           visible="true"
           steal-focus="false"
           tooltip="Displays messages received from the connected game instance"
           class="Emergent.Toolbench.CodeEditor.GotoCodeOutputChannel"/>
</extension>

channel

Identifier Type Description
name String A user-friendly name for the output channel.
visible Boolean [optional] Determines whether the channel is initially visible in the output view.
steal-focus Boolean [optional] Determines whether the output channel steals focus when something is written to it.
icon String [optional] Icon associated with the channel.
class String [optional] The class is the full class name of the optional output channel class to support custom logic. The output channel class must implement the Emergent.Toolbench.OutputView.IOutputChannel interface.
button-priority Integer [optional] The priority of the output channel button.
tooltip String [optional] A tooltip which will be displayed for the output channel.
