EditableAdapter (Extension-Point)

Emergent.Toolbench.UI.EditableService.EditableAdapter

This extension point allows developers to create an adapter to existing controls that want to expose their editability to the edit system. That is, do they support cut/copy/paste/delete when the control has focus. In a situation where you're using 3rd party libraries or just the .Net framework in general you may want to create an editable adapter for the control type you want to work with the edit menu, and edit toolbars in Toolbench. If however you're creating a control from scratch, you can implement the Emergent.Toolbench.UI.EditableService.IEditable interface on the control itself and disregard creating an adapter for it.

<extension point="Emergent.Toolbench.UI.EditableService.EditableAdapter">
  <adapter class="Emergent.Toolbench.UI.EditableService.TextBoxBaseEditableAdapter"
           control-class="System.Windows.Forms.TextBoxBase"/>
</extension>

adapter

Attribute Type Description
class String The full name of the class to instantiate to handle the editability of a control type. The class must implement the Emergent.Toolbench.UI.EditableService.IEditableAdapter interface.
control-class String The full class name of the control that this adapter will work for. The class name can be the base class and will work for any subclasses of that control.
