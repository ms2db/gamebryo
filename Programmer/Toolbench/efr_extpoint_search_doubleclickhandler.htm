FindResultsDoubleClickHandler (Extension-Point)

Emergent.Toolbench.Search.UI.FindResults.DoubleClickHandler

This extension point is used to specify a handler class for responding to double-clicks on Find in Files search results. The specified class must implement the IDoubleClickHandler interface, which consists of a single HandleDoubleClick method. 

<extension point="Emergent.Toolbench.Search.UI.FindResults.DoubleClickHandler">
  <handler name="Search Results Double-Click Handler"
               class="Emergent.Toolbench.CodeEditor.FindResultsDoubleClickHandler">
  </handler>
</extension>

handler

Attribute Type Description
name String A name used only for context.
class String The class to instantiate for processing double-clicks on results.
