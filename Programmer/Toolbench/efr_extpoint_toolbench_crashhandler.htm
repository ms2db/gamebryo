CrashHandler (Extension-Point)

Emergent.Toolbench.CrashHandler

The CrashHandler extension point allows plugins to be called when a crash occurs inside Toolbench. These crash handlers can both capture crash information and will also be called to report their information when Toolbench shuts down.

<extension point="Emergent.Toolbench.CrashHandler">
  <class id="Emergent.Toolbench.CrashHandlerClass"/>
</extension>

handler

Identifier Type Description
class String The class is the full class name of the crash handler class. The crash handler class must implement the Emergent.Toolbench.ICrashHandler interface.
