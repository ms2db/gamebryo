ImportLocation (Extension-Point)

Emergent.Toolbench.GameSolution.ImportLocation

The ImportLocation extension point allows users to provide Toolbench with locations where the game solution can look for new import libraries for the game solution.

<extension point="Emergent.Toolbench.GameSolution.ImportLocation">
  <location path="%EMERGENT_PATH%\Media"/>
</extension>

location

Identifier Type Description
path String The directory to scan for import libraries.
