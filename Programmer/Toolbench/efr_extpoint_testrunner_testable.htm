Testable (Extension-Point)

Emergent.Toolbench.TestRunner.Testable

The testable extension point is needed if your plug-in wants to claim that it is testable and contains unit tests. If you include this extension in your plug-in it will be scanned for classes that inherit from the Emergent.Toolbench.TestRunner.TestFixture class. These test fixtures will be run whenever tests are executed.

<extension point="Emergent.Toolbench.TestRunner.Testable"/>
