TerrainCreationWizardUI (Extension-Point)

Emergent.Toolbench.WorldBuilder.Terrain.UI.TerrainCreationWizardUI

TerrainCreationWizardUI extension point allows a developer to register a new method of creating terrain and attach it to the Terrain Creation Wizard. Once attached, the creation method will be presented to the user on the first page of the Terrain Creation Wizard, and the user may then select the method. An internal set of WizardPages allow the method to specify a list of properties and settings to configure in order to generate an appropriate action to create the terrain.

The developer must define a class deriving from TerrainCreationWizard and register it as an extension using the following syntax in the plugin.xml file. 

<!-- New Terrain creation method. -->
<extension point="Emergent.Toolbench.Worldbuilder.Terrain.UI.TerrainCreationWizardUI">
    <TerrainCreationWizardUI class="Emergent.Toolbench.WorldBuilder.Terrain.UI.Wizards.NewCreateBlankTerrainMethod"/>
</extension>
