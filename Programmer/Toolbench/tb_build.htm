Building Toolbench

Toolbench is at the center of many Gamebryo LightSpeed content development workflows. This page outlines requirements and steps to build this application, including all of its default plug-ins that implement "sub-tools," like World Builder and the Entity Modeling Tool.

Prerequisites

See Installing Gamebryo LightSpeed for prerequisites.

Build Dependencies

Toolbench draws on base functionality that is provided by many different parts of the Gamebryo LightSpeed codebase. Solutions must be built in the following order to prepare dependencies:

  1. Build\Foundation\Win32\VC90\Foundation.sln
  2. Build\CoreRuntime\Win32\VC90\Core00_ThirdPartyCode.sln
  3. Build\CoreRuntime\Win32\VC90\Core01_EngineLibs.sln
  4. Build\CoreRuntime\Win32\VC90\Core03_ToolLibs.sln
  5. Build\CoreRuntime\Win32\VC90\Core05_ShaderLibs.sln
  6. Build\CoreRuntime\Win32\VC90\Core06_UtilityLibs.sln
  7. Build\GameFramework\Win32\VC90\GameFramework.sln
  8. Build\GameFramework\Win32\VC90\CoreServices.sln

These solutions should be built in a DLL configuration to satisfy Toolbench dependencies. For example, if you are building Debug Toolbench, the solutions above should be built in DebugDLL configuration. The exception is Core00_ThirdPartyCode.sln, which lacks a DLL configuration and can be built in Debug.

The Toolbench Solution

Toolbench itself is built via the following solution:

Build\Tools\Win32\VC90\Toolbench.sln

Use the "Mixed Platforms" build configuration and verify that ToolbenchWindowed is specified as the startup project when you launch the application from Visual Studio.

The source and project files for Toolbench are located under Source\Tools\Toolbench. Build output is routed to SDK\Win32\Bin\VC90.

Additional Toolbench Solutions

Due to dependencies on external libraries, the following solutions are kept seperate from the main solution.

  • Support Library for Revision Control - Build\Tools\Win32\VC90\RevisionControl.sln
  • Toolbench Revision Control - Build\Tools\Win32\VC90\ToolbenchRevisionControl.sln
  • Toolbench FBX - Build\Tools\Win32\VC90\ToolbenchFBX.sln
