Provider (Extension-Point)

Emergent.Toolbench.RevisionControl.Provider

Revision control providers wrap calls to revision control providers in Toolbench. This extension point facilitates exposing a new revision control provider.

<extension point="Emergent.Toolbench.RevisionControl.Provider">
  <provider name="Perforce"
            id="Perforce"
            class="Emergent.Toolbench.RevisionControl.Perforce.PerforceProvider"/>
</extension>

provider

Attribute Type Description
name String The name of the revision control provider. This name is used for display purposes.
id String This id is what will be used to factory the corresponding native RCSProvider.
class String This is the corresponding class that will be created for your provider. This class must implement the Emergent.Toolbench.RevisionControl.IRevisionProvider interface.
