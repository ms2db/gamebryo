Particle System Controllers

Time controllers can be attached to the NiPSParticleSystem object to control aspects of the system or any of its emitters or forces. Time controllers that affect emitters are derived from NiPSEmitterCtlr, and each must be provided with the name of the emitter that it controls. Similarly, time controllers that affect forces are derived from NiPSForceCtlr, and each must be provided with the name of the force that it controls. All NiPSEmitter and NiPSForce objects must have a unique name assigned; this is ensured by the art package exporters. The name is used when the controller is attached to the NiPSParticleSystem object to resolve a pointer to that modifier.

System Controllers

Controllers that affect the particle system itself are listed below:

Time Controller Description
NiPSResetOnLoopCtlr Calls ResetParticleSystem on an NiPSParticleSystem target upon looping.

Emitter Controllers

Controllers that affect particle emitters are listed below:

Time Controller Description
NiPSEmitParticlesCtlr Animates particle emission and birth rate.
NiPSEmitterDeclinationCtlr Animates the declination value on an NiPSEmitter object.
NiPSEmitterDeclinationVarCtlr Animates the declination variation value on an NiPSEmitter object.
NiPSEmitterLifeSpanCtlr Animates the life span value on an NiPSEmitter object.
NiPSEmitterLifeSpanVarCtlr Animates the life span variance value on an NiPSEmitter object.
NiPSEmitterPlanarAngleCtlr Animates the planar angle value on an NiPSEmitter object.
NiPSEmitterPlanarAngleVarCtlr Animates the planar angle variation value on an NiPSEmitter object.
NiPSEmitterRadiusCtlr Animates the radius value on an NiPSEmitter object.
NiPSEmitterRadiusVarCtlr Animates the radius variance value on an NiPSEmitter object.
NiPSEmitterRotAngleCtlr Animates the rotation angle value on an NiPSEmitter object.
NiPSEmitterRotAngleVarCtlr Animates the rotation angle variation value on an NiPSEmitter object.
NiPSEmitterRotSpeedCtlr Animates the rotation speed value on an NiPSEmitter object.
NiPSEmitterRotSpeedVarCtlr Animates the rotation speed variation value on an NiPSEmitter object.
NiPSEmitterSpeedCtlr Animates the speed value on an NiPSEmitter object.
NiPSEmitterSpeedVarCtlr Animates the speed variance value on an NiPSEmitter object.
NiPSEmitterSpeedFlipRatioCtlr Animates the speed flip ratio on an NiPSEmitter object.

Force Controllers

Controllers that affect particle forces are listed below:

Time Controller Description
NiPSAirFieldAirFrictionCtlr Animates the air friction value of an NiPSAirFieldForce object.
NiPSAirFieldInheritedVelocityCtlr Animates the inherited velocity value of an NiPSAirFieldForce object.
NiPSAirFieldSpreadCtlr Animates the spread value of an NiPSAirFieldForce object.
NiPSFieldAttenuationCtlr Animates the attenuation value of an NiPSFieldForce object.
NiPSFieldMagnitudeCtlr Animates the magnitude value of an NiPSFieldForce object.
NiPSFieldMaxDistanceCtlr Animates the max distance value of an NiPSFieldForce object.
NiPSForceActiveCtlr Animates whether or not an NiPSForce object is active.
NiPSGravityStrengthCtlr Animates the strength value of an NiPSGravityForce object.
