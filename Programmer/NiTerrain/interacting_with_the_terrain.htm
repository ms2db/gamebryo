Interacting with the Terrain

The terrain system provides several methods of interaction with the terrain.

  • Querying with Rays
    Using rays to intersect with the terrain.
  • Querying Terrain Metadata
    How to query the terrain for metadata. Meta-data can be assigned to different positions or materials of the terrain. It allows the terrain to hold arbitrary information and can facilitate scripted events.
  • Integrating with PhysX
    How to integrate the terrain into a PhysX scene.
  • Applying Decals to the Terrain
    Decals are small meshes that can be applied to the terrain. An example of this is scorch marks or footprints.
