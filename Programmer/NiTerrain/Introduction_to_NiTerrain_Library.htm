Introduction to NiTerrain

This section of the documentation introduces the architecture of the Terrain Engine This introduction is divided into the following sections, each dealing with a specific aspect of the terrain engine.

  • Overview of the Terrain Hierarchy
    Describes the organization of terrain data.
  • NiTerrain Geometry
    Details how the terrain stores and uses its geometry data.
  • NiTerrain Multi-sector System
    Describes the terrain multisector system used to support very large terrains.
  • NiTerrain Paging System
    Describes the terrain paging system used to load and unload parts of the terrain.
  • NiTerrain Materials and Masks
    Details how the terrain displays different materials, allowing blending of different shader effects and texturing.
  • NiTerrain Meta-Data System
    Describes the meta-data system used to associate arbitrary user data with terrain materials.
  • NiTerrain Resource Management
    Describes the resource management system for handling all the resources required by the terrain.
  • NiTerrain Storage
    Describes the storage system of how the data is stored for the terrain.
  • NiTerrain Streaming
    Describes the streaming system of how the terrain data is streamed from storage.
  • NiTerrain Shader
    Describes the shaders used by the terrain.
  • NiTerrain Painting and Deformation
    Describes how the terrain paints and deforms.
  • NiTerrain Collision System
    Describes the collision detection interface which is essential for objects interacting with the terrain.
