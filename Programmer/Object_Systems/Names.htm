Names

The class NiObjectNET supports character string names to facilitate identifying objects and searching for objects in a hierarchy. The following functions are used to access the name of an object:

void SetName(const NiFixedString& kName);

const NiFixedString& GetName() const;

SetName increments the reference to kName.

NiAVObject::GetObjectByName can be used to recursively traverse scene graphs in search of an object with a specific name. Its prototype is as follows:

NiAVObject* GetObjectByName(const NiFixedString& kName);

Note that when cloning an NiObjectNET, the clone's objects will have NULL names by default. This behavior can be changed. See Cloning for more details.
